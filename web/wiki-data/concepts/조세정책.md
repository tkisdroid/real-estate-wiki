# 조세정책

## 1. 부동산조세 분류

| 단계 | 지방세 | 국세 |
|------|--------|------|
| **취득** | 취득세, 등록면허세, 지방교육세, 지방소비세 | 상속세, 증여세, 농어촌특별세, 부가가치세, 인지세 |
| **보유** | **재산세**, 지방교육세, 지방소비세 | **종합부동산세**, 종합소득세(임대업), 농어촌특별세, 부가가치세(임대업) |
| **처분** | 지방소득세, 지방소비세 | **양도소득세**, 종합소득세(매매업), 농어촌특별세, 인지세 |

> **주의**: 상속세 = **취득**단계 **국세**, 재산세 = **보유**단계 **지방세**, 증여세 = **취득**단계 **국세**

## 2. 조세의 전가와 귀착

- **조세의 전가(shifting)**: 법적 납세의무자로부터 다른 경제주체에게 부담이 이전
- **조세의 귀착(incidence)**: 조세를 최종적으로 부담하게 되는 것

### 핵심 원리

- **탄력성이 큰 쪽**이 세금을 **덜** 부담, **탄력성이 작은 쪽**이 **더** 부담
- 수요·공급 모두 완전비탄력적/완전탄력적이 아닌 한, 수요자와 공급자가 **같이 부담**

### 수요자 부담액 공식

$$수요자\ 부담\ 비율 = \frac{공급의\ 가격탄력성}{수요의\ 가격탄력성 + 공급의\ 가격탄력성}$$

### 극단적 경우

| 조건 | 세금 부담자 |
|------|-----------|
| 수요 **완전탄력적** 또는 공급 **완전비탄력적** | **공급자** 전부 부담 |
| 수요 **완전비탄력적** 또는 공급 **완전탄력적** | **수요자** 전부 부담 |

### 주택시장 적용

- **임대주택 재산세**: 수요 비탄력적 → 임차인 더 부담 / 공급 비탄력적 → 임대인 더 부담
- 공급 **완전비탄력적** → 재산세 **전부 임대인** 부담
- 수요 **완전비탄력적** → 재산세 **전부 임차인** 부담

### 양도소득세 중과 효과

1. 매도자 부담 → 주택공급 감소 (**동결효과**, lock-in effect)
2. 대체주택 감소 → 수요 가격탄력성 **감소**
3. 매수자 부담 **증가** → 주택가격 **상승** 가능

> **시험 주의**: 양도소득세 중과는 매각을 **늦추게(미루게)** 하는 동결효과 발생 (앞당기는 것이 아님)

## 3. 조세 부과의 효과

- 공급자는 소비자에게 전가하기 위해 **공급 감소**
- 수요자 실질 지불금액 **상승**, 공급자 실질 수령금액 **하락**
- **소비자잉여 감소**, **생산자잉여 감소**
- **사회적 비용(경제적 순손실, deadweight loss)** 발생
- 공급이 **탄력적**일수록 사회적 비용 **더 큼**

## 4. 재산세·종합부동산세

- 과세기준일: 매년 **6월 1일** (재산세, 종합부동산세 동일)

## 핵심 암기 포인트

- 재산세: **보유**단계 **지방세** / 종합부동산세: **보유**단계 **국세**
- 양도소득세: **처분**단계 **국세** / 취득세: **취득**단계 **지방세**
- 상속세·증여세: **취득**단계 **국세**
- 과세기준일: 매년 **6월 1일** (재산세, 종합부동산세 동일)
- 수요자 부담 비율 = **공급탄력성 / (수요탄력성 + 공급탄력성)**
- 탄력성 **큰 쪽**이 세금 **덜** 부담, **작은 쪽**이 **더** 부담
- 수요 완전탄력적 또는 공급 완전비탄력적 → **공급자** 전부 부담
- 수요 완전비탄력적 또는 공급 완전탄력적 → **수요자** 전부 부담
- 양도소득세 중과 → **동결효과**(lock-in effect) → 공급 감소 → 가격 상승 가능
- 공급이 **탄력적**일수록 사회적 비용(deadweight loss) **더 큼**

## OX 퀴즈

**Q1.** 재산세는 보유단계의 국세이다. → **X** (재산세는 보유단계의 **지방세**, 국세는 종합부동산세)

**Q2.** 상속세는 처분단계의 세금이다. → **X** (상속세는 **취득**단계의 국세)

**Q3.** 조세의 귀착에서 탄력성이 큰 쪽이 세금을 더 많이 부담한다. → **X** (탄력성이 **작은 쪽**이 더 많이 부담)

**Q4.** 공급이 완전비탄력적일 때 재산세는 전부 임대인(공급자)이 부담한다. → **O**

**Q5.** 양도소득세 중과는 매각을 앞당기는 효과가 있다. → **X** (매각을 **미루게(늦추게)** 하는 **동결효과** 발생)

**Q6.** 재산세와 종합부동산세의 과세기준일은 매년 1월 1일이다. → **X** (매년 **6월 1일**)

**Q7.** 증여세는 취득단계의 지방세이다. → **X** (증여세는 취득단계의 **국세**)

**Q8.** 수요가 완전비탄력적이면 재산세는 전부 수요자(임차인)가 부담한다. → **O**

## 함정 노트

- ⚠️ "재산세 = 국세" → 재산세는 **지방세**, 종합부동산세가 국세
- ⚠️ "상속세 = 처분단계" → 상속세·증여세는 **취득**단계 국세
- ⚠️ "탄력성 큰 쪽이 더 부담" → 정반대. 탄력성 **작은 쪽**이 세금 **더** 부담
- ⚠️ "양도소득세 중과 → 매각 촉진" → 매각을 **미루는** 동결효과 발생, 공급 감소로 가격 상승 가능
- ⚠️ "과세기준일 = 1월 1일" → 재산세·종합부동산세 과세기준일은 **6월 1일**
