# 취득세 - 과세표준과 세율

## 1. 과세표준의 기준

취득세의 과세표준은 **취득 당시의 가액**(취득당시가액)으로 한다.

| 취득 유형 | 과세표준 |
|-----------|---------|
| **유상승계취득** | **사실상취득가격** |
| **무상취득 (상속)** | **시가표준액** |
| **무상취득 (시가표준액 1억 이하)** | 시가인정액과 시가표준액 중 **납세자 선택** |
| **무상취득 (기타)** | **시가인정액** (산정 곤란시 시가표준액) |
| **원시취득** | **사실상취득가격** (확인 불가 시 시가표준액) |

## 2. 유상승계취득의 과세표준

### 사실상취득가격 = 직접비용 + 간접비용

| 구분 | 법인 | 개인 |
|------|:----:|:----:|
| **부가가치세** | 제외 | 제외 |
| **건설자금이자** | 포함 | **제외** |
| **할부·연부이자, 연체료** | 포함 | **제외** |
| **중개보수** | 포함 | 포함 |
| **농지보전부담금** | 포함 | 포함 |
| **취득 용역비·수수료** | 포함 | 포함 |

> **개인 제외 3가지**: 건설자금이자, 할부이자, 연체료 (법인은 모두 포함)

### 특수관계인 부당행위계산

시가인정액과 사실상취득가격의 차액이 **3억원 이상** 또는 시가인정액의 **5% 이상** → 시가인정액을 과세표준으로 결정

### 부담부증여

| 부분 | 과세표준 |
|------|---------|
| **채무부담액** | 유상승계취득 과세표준 (사실상취득가격) |
| **시가인정액 - 채무부담액** | 무상취득 과세표준 (시가인정액) |

## 3. 무상취득의 과세표준

### 시가인정액

취득시기 현재 불특정 다수인 사이에 자유롭게 거래가 이루어지는 경우 통상적으로 성립된다고 인정되는 가액

| 시기 | 범위 |
|------|------|
| 취득일 **전 6개월** ~ **후 3개월** | 매매사례가액, 감정가액, 공매가액 등 |

### 시가표준액

| 대상 | 시가표준액 |
|------|----------|
| **토지** | 개별공시지가 |
| **단독주택** | 개별주택가격 |
| **공동주택** | 공동주택가격 |
| **건축물** | 건축물 시가표준액 |

## 4. 연부취득

- 연부로 취득하는 경우: 과세표준 = **연부금액** (매회 사실상 지급되는 금액, 계약보증금 포함)
- 면세점 판단: **연부금액 총액** 기준

## 5. 표준세율 (부동산)

### 유상승계취득

| 구분 | 세율 |
|------|:----:|
| **농지** | **3%** (1,000분의 30) |
| **농지 이외 부동산** (주택 제외) | **4%** (1,000분의 40) |

### 유상거래 주택 세율 (구간별)

| 취득가액 | 세율 |
|----------|:----:|
| **6억원 이하** | **1%** |
| **6억원 초과 ~ 9억원 이하** | **1.01% ~ 2.99%** (산식 적용) |
| **9억원 초과** | **3%** |

> 주택 산식: (취득당시가액 x 2/3 - 3) x 1/100

### 상속

| 구분 | 세율 |
|------|:----:|
| **농지** | **2.3%** |
| **농지 이외** | **2.8%** |

### 상속 외 무상취득 (증여 등)

| 구분 | 세율 |
|------|:----:|
| **비영리사업자** | **2.8%** |
| **일반** | **3.5%** |

### 원시취득·공유물분할

| 구분 | 세율 |
|------|:----:|
| **원시취득** (신축, 매립·간척) | **2.8%** |
| **공유물 분할** | **2.3%** |

## 6. 세율의 특례

### 표준세율 - 중과기준세율(2%) = 특례세율

형식적 소유권 취득에 적용:
- 환매, 상속 1세대1주택, 공유물 분할, 재산분할, 건축물 이전 등

### 중과기준세율(2%) 적용

간주취득 등:
- 개수(대수선), 종류변경, **지목변경**, **과점주주** 취득 등

### 중과기준세율

**1,000분의 20 (2%)** — 모든 세율의 특례 적용의 기준

## 7. 세율 적용 가감 조례

지방자치단체의 장은 조례로 표준세율의 **50% 범위** 내에서 가감 가능

## 핵심 암기 포인트

- 유상승계: **사실상취득가격**, 무상(상속): **시가표준액**, 무상(기타): **시가인정액**
- 개인 제외: 건설자금이자, 할부이자, 연체료 (법인은 모두 포함)
- 주택: 6억 이하 **1%**, 9억 초과 **3%**
- 농지 유상: **3%**, 농지 외 유상: **4%**
- 상속: 농지 **2.3%**, 비농지 **2.8%**
- 증여(일반): **3.5%**, 원시취득: **2.8%**
- 중과기준세율: **2%** (간주취득·특례세율 기준)
- 시가인정액: 취득일 **전 6개월 ~ 후 3개월**

## OX 퀴즈

**Q1.** 상속에 의한 취득의 과세표준은 시가인정액이다. → **X** (시가표준액)

**Q2.** 개인이 부동산 유상취득 시 건설자금이자는 사실상취득가격에 포함된다. → **X** (개인은 제외, 법인만 포함)

**Q3.** 농지의 유상승계취득 세율은 4%이다. → **X** (3%, 4%는 농지 이외 부동산)

**Q4.** 주택 취득가액 5억원인 경우 취득세율은 1%이다. → **O** (6억 이하)

**Q5.** 원시취득의 표준세율은 3.5%이다. → **X** (2.8%, 3.5%는 일반 무상취득)

**Q6.** 부가가치세는 법인·개인 모두 사실상취득가격에서 제외된다. → **O**

**Q7.** 중과기준세율은 3%이다. → **X** (2%)

**Q8.** 시가표준액이 1억원 이하인 부동산의 무상취득 시 납세자가 과세표준을 선택할 수 있다. → **O** (시가인정액·시가표준액 중 선택)

## 함정 노트

- **상속 = 시가표준액**: 무상취득이지만 상속만 **시가표준액** 적용 (기타 무상은 시가인정액)
- **개인 vs 법인 간접비용**: 건설자금이자·할부이자·연체료 → 법인 포함, 개인 제외. **중개보수는 둘 다 포함**
- **주택 구간별 세율**: 6억 초과~9억 이하 구간은 **산식**으로 계산 (단순 세율 아님)
- **공유물 분할 2.3%**: 상속(농지)과 동일한 세율이지만 성격이 다름
- **부담부증여 이중과세표준**: 채무부담액=유상(사실상취득가격), 나머지=무상(시가인정액). 한 건에 두 가지 과세표준 적용
