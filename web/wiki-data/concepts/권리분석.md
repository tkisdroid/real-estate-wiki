# 권리분석

## 1. 권리분석의 의의
- 개업공인중개사가 중개대상물의 **권리관계를 조사·확인·분석**하여 거래 안전을 도모하는 것
- [[확인설명의무]]의 핵심 내용 중 하나

## 2. 확인해야 할 권리관계

### 등기부 확인 사항
| 구분 | 확인 내용 |
|---|---|
| **갑구** (소유권) | 소유자, 가등기, 가처분, 가압류, 환매특약, 신탁등기 등 |
| **을구** (소유권 외) | 전세권, 저당권(근저당), 지상권, 지역권, 임차권 등 |
| 표제부 | 부동산 표시 (소재, 지번, 면적, 구조 등) |

### 공부 확인
| 공부 | 확인 내용 |
|---|---|
| 등기사항증명서 | 소유권 및 제한물권 |
| 토지대장 | 지목, 면적, 소유자 |
| 건축물대장 | 용도, 구조, 면적, 위반건축물 여부 |
| 토지이용계획확인서 | 용도지역·지구·구역, 도시계획시설 |
| 지적도·임야도 | 토지 형상, 경계 |

### 비등기 권리 확인
- **법정지상권**: 등기 없이 성립
- **유치권**: 등기 없이 성립 (점유 필요)
- **주택임대차보호법상 대항력**: 주민등록 + 점유
- **확정일자부 임차인**: 우선변제권
- 전입세대확인서 열람

## 3. 주택임대차 확인사항

### 임대인 정보제시의무 (주택임대차보호법 제3조의7)
- 확정일자 부여현황 정보 (임대인이 제출 또는 열람 동의)
- 국세·지방세 체납정보

### 확인·설명서 기재사항 (주택 임대차)
| 항목 | 내용 |
|---|---|
| 확정일자 부여현황 | 임대인 자료제출 / 열람동의 여부 |
| 임차인 권리 설명 | 대항력, 우선변제권 등 |
| 국세·지방세 체납정보 | |
| 소액임차인 최우선변제금 | 선순위 담보물권 설정 당시 기준 |
| 전입세대확인서 | 확인/미확인 |
| 민간임대 등록여부 | 등록/미등록/해당없음 |
| 계약갱신요구권 행사여부 | 확인/미확인/해당없음 |
| 임대보증금 보증 | 민간임대주택에 한함 |
| 관리비 | 직전 1년간 월평균 관리비 등 |

## 4. 토지 이용계획 확인

### 확인 사항
- **용도지역**: 건폐율·용적률 상한
- **용도지구·용도구역**
- **도시·군계획시설**
- **허가·신고 구역**: 토지거래허가구역
- **투기지역**: 토지투기지역, 주택투기지역, 투기과열지구
- 지구단위계획구역
- 기타 이용제한·거래규제사항

## 5. 실무상 유의사항

### 등기부 확인 시점
- 중개의뢰 접수 시
- **계약 체결 직전**
- **잔금 지급 전** (판례: 재확인 의무, 수원지법 1997.8.26)

### 신탁등기 부동산
- 수탁자·신탁원부 확인 필수
- 임대차계약 체결 시 수탁자·수익자 동의·승낙 등 요건 확인

### 공동담보
- 공동담보 목록 확인 → 해당 물건의 **실제 채권최고액** 파악

## 6. 비등기 권리 세부 - 법정지상권

### 법정지상권의 성립 유형
| 유형 | 근거 | 요건 |
|---|---|---|
| 전세권설정에 의한 | 민법 제305조 | 동일인 소유 대지+건물에서 건물에 전세권 설정 → 대지 특별승계인에게 지상권 인정 |
| **저당물 경매**로 인한 | 민법 제366조 | 저당물 경매로 토지+건물 소유자가 달라진 경우 |
| 가등기담보에 의한 | 가등기담보법 제10조 | 담보가등기에 기한 본등기 시 |
| 입목 경매에 의한 | 입목법 제6조 | 경매로 토지+입목 소유자가 달라진 경우 |
| **관습법상** | 판례 | 매매 등으로 소유자 달라진 경우 (철거 약정 없을 때) |

### 법정지상권 주요 판례
- 저당권자가 건축에 동의하더라도, 저당권 설정 당시 건물이 없으면 → 법정지상권 **불성립** (대판 2003.9.5)
- 무허가건물·미등기건물도 건물 요건 갖추면 → 법정지상권 **성립** (대판 1988.4.12)
- 원인무효 등기로 소유자 달라진 경우 → 관습법상 법정지상권 **불허** (대판 1999.3.26)
- 법정지상권은 독립된 물권 → 건물과 **별도 처분** 가능 (대판 2001.12.27)

## 7. 비등기 권리 세부 - 유치권

### 유치권 성립 요건
- 타인의 물건 점유 + 그 물건에 관하여 생긴 채권

### 유치권이 성립하는 경우
- 건물공사 **미지급 공사대금** (대판 1995.9.15)
- 임차인의 **필요비·유익비** 반환청구권

### 유치권이 성립하지 않는 경우
- 임차보증금 반환청구권 → **유치권 불성립** (대판 1976.5.11)
- 권리금 → **유치권 불성립**
- **원상회복의무** 약정 시 → 필요비·유익비 유치권 **불성립** (대판 1993.2.26)

### 유치권의 효력
- 모든 사람에게 주장 가능 (채무자, 양수인, **경매 매수인** 포함)
- 점유 상실 시 소멸

## 8. 비등기 권리 세부 - 분묘기지권

### 의의
- 타인의 토지에 분묘를 설치한 자의 **지상권 유사 물권**

### 성립 요건
- 타인 소유 토지에 **소유자의 승낙** 받아 분묘 설치
- 또는 타인의 토지에 **20년간** 점유 (시효취득)
- 자기 토지에 분묘 설치 후 토지 양도 시 → **관습상 법정지상권** 유사 인정

### 효력
- 분묘 존속 기간 동안 토지사용권 인정
- 등기 없이 **제3자에게 대항** 가능
