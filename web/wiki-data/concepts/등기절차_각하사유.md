# 등기절차 - 각하사유 (법 제29조)

## 1. 의의

등기신청이 법정 요건을 갖추지 못한 경우, 등기관이 해당 신청을 **각하**(거부)하는 사유. 총 **11가지**가 법정되어 있다.

## 2. 각하사유 11가지

| 호수 | 각하사유 | 키워드 |
|:----:|----------|--------|
| **1** | 사건이 그 등기소의 **관할에 속하지 아니**한 경우 | 관할 위반 |
| **2** | 사건이 **등기할 것이 아닌** 경우 | 등기 부적격 |
| **3** | **신청할 권한이 없는 자**가 신청한 경우 | 신청권한 없음 |
| **4** | 당사자나 대리인이 **출석하지 아니**한 경우 (방문신청) | 미출석 |
| **5** | 신청정보의 제공이 **방식에 맞지 아니**한 경우 | 방식 위반 |
| **6** | 신청정보의 부동산 또는 등기목적인 권리의 표시가 **등기기록과 일치하지 아니**한 경우 | 부동산표시 불일치 |
| **7** | 신청정보의 등기의무자 표시가 **등기기록과 일치하지 아니**한 경우 | 등기의무자 불일치 |
| **8** | 신청정보와 등기원인을 증명하는 정보가 **일치하지 아니**한 경우 | 신청정보·증명정보 불일치 |
| **9** | 등기에 필요한 **첨부정보를 제공하지 아니**한 경우 | 첨부정보 미제출 |
| **10** | **취득세·등록면허세·수수료**를 내지 아니하거나, 등기신청과 관련하여 다른 법률에 따라 부과된 **의무를 이행하지 아니**한 경우 | 조세·수수료 미납, 의무 불이행 |
| **11** | 신청정보 또는 등기기록의 부동산의 표시가 **토지대장·임야대장 또는 건축물대장과 일치하지 아니**한 경우 | 공적장부 불일치 |

### 각하사유의 분류

| 분류 | 해당 호수 |
|------|----------|
| **관할·자격** | 1, 2, 3호 |
| **출석·방식** | 4, 5호 |
| **내용 불일치** | 6, 7, 8호 |
| **첨부·납부** | 9, 10, 11호 |

### 제7호 예외

가. 포괄승계인(상속인, 합병법인 등)이 신청하는 경우에는 등기의무자 표시가 등기기록과 일치하지 않더라도 각하사유에 해당하지 **않는다**.

나. 신청정보와 등기기록의 등기의무자가 **동일인임을 대법원규칙으로 정하는 바에 따라 확인**할 수 있는 경우 (2024.9.20. 신설)

## 3. 보정 (법 제29조 단서)

| 항목 | 내용 |
|------|------|
| **보정 기한** | 각하사유 해당 시 보정을 명한 날의 **다음 날**까지 |
| **보정 대상** | 보정이 가능한 각하사유에 한함 |
| **보정 불이행** | 보정 기한 경과 시 **각하** |

> **보정기한 주의**: "보정명령일 **다음 날**"까지이지, 보정명령을 받은 날이 아님

## 4. 각하에 대한 이의신청 (법 제100조~제103조)

### 이의신청

| 항목 | 내용 |
|------|------|
| **이의신청 대상** | 등기관의 결정 또는 처분이 부당하다고 인정하는 자 |
| **이의신청 기관** | **관할 지방법원** |
| **이의가 이유 있을 때** | 법원이 등기관에 **상당한 처분을 명령** |

### 이의신청 처리

- 이의가 **이유 있다**: 법원이 등기관에 기록명령·말소명령 등 상당한 처분 명령
- 이의가 **이유 없다**: 이의 기각
- 이의 결정 전: **가등기명령** 또는 **부기등기명령** 가능

## 5. 등기관의 직권 각하 후 말소

등기관이 법 제29조 **제1호(관할위반)** 또는 **제2호(등기할 것이 아닌 경우)**에 해당하는 사실을 간과하고 등기를 마친 경우 → **직권으로 말소** 가능

## 6. 각하사유 vs 직권등기 비교

| 구분 | 각하사유 | 직권등기 |
|------|----------|----------|
| **주체** | 등기관의 거부 | 등기관의 직권 등기 |
| **근거** | 법 제29조 | 법률의 규정 |
| **내용** | 신청 거부 | 보존·변경·경정·말소 등 |
| **간과 시** | 1호·2호 간과 → 직권 말소 | - |

## 핵심 암기 포인트

- **11가지 각하사유**: 관할, 부적격, 무권한, 미출석, 방식위반, 부동산표시불일치, 의무자불일치, 신청정보·증명정보불일치, 첨부미제출, 조세·수수료미납·의무불이행, 공적장부불일치
- **보정기한**: 보정명령일의 **다음 날**까지
- **제7호 예외**: 포괄승계인 신청 시, 또는 대법원규칙에 따라 동일인 확인 가능 시 → 각하 아님
- **조세·수수료 미납(10호)**: 취득세·등록면허세·수수료 미납 또는 다른 법률상 의무 불이행도 각하사유
- **공적장부 불일치(11호)**: 부동산 표시가 토지대장·임야대장·건축물대장과 불일치 시 각하
- **1호·2호 간과**: 등기관이 간과하고 등기한 경우 **직권 말소**
- 이의신청: **관할 지방법원**에 제기

## OX 퀴즈

**Q1.** 각하사유에 해당하면 보정 없이 즉시 각하된다. → **X** (보정 명한 날의 다음 날까지 보정 가능)

**Q2.** 취득세를 부족하게 납부한 경우 등기신청의 각하사유에 해당한다. → **O** (제10호)

**Q3.** 포괄승계인이 신청한 경우 등기의무자 표시 불일치는 각하사유이다. → **X** (포괄승계인 신청은 제외)

**Q4.** 보정기한은 보정을 명한 날까지이다. → **X** (보정을 명한 날의 **다음 날**까지)

**Q5.** 등기관의 각하에 대한 이의신청은 등기관에게 한다. → **X** (관할 지방법원)

**Q6.** 등기관이 관할 위반을 간과하고 등기한 경우 직권으로 말소할 수 있다. → **O** (제1호 간과)

**Q7.** 수수료를 전액 납부했으나 등록면허세를 미납한 경우 각하사유에 해당한다. → **O** (제10호)

**Q8.** 각하사유는 총 10가지이다. → **X** (11가지)

**Q9.** 전자신청의 경우에도 당사자 미출석은 각하사유이다. → **X** (제4호는 방문신청에 한함)

**Q10.** 등기원인일자가 등기의무자의 권리등기 후의 일자이면 각하사유이다. → **X** (등기 **전**의 일자로 된 경우가 각하사유)

## 함정 노트

- **보정기한 "다음 날"**: "보정명령을 받은 날"이 아니라 "보정을 **명한 날**의 다음 날"까지. 일자 기산 주의
- **10호 vs 11호**: 10호=조세(취득세·등록면허세), 11호=수수료. 구분하여 출제
- **제4호 "미출석"**: 전자신청에는 해당 없음. 방문신청에서만 적용
- **제8호 신청정보·증명정보 불일치**: 신청정보와 등기원인을 증명하는 정보가 일치하지 않으면 각하
- **제10호 통합**: 취득세·등록면허세·수수료가 모두 10호에 통합, 다른 법률상 의무 불이행도 포함
- **제11호 공적장부 불일치**: 토지대장·임야대장·건축물대장과의 일치 여부 확인
- **제1호·제2호 간과**: 이 두 가지만 간과 시 **직권 말소** 대상. 나머지 호수는 간과해도 직권 말소 불가
