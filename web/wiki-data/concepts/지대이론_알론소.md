# 알론소(W. Alonso)의 입찰지대이론

> 📖 상위 문서: [[지대이론]]

## 1. 의의

- 튀넨의 **고립국이론을 도시공간에 적용**하여 확장·발전시킨 이론
- 튀넨의 **농촌 토지이용**에 대한 사고를 **도시공간**으로 확장
- 고립국 이론의 성과인 **위치지대 및 입찰지대곡선**을 도시 토지이용분화를 설명하는 데 도입

## 2. 입찰지대(Bid Rent)

### 개념

- 입찰지대란 단위 면적의 토지에 대해 토지이용자가 지불하고자 하는 **최대금액**
- **초과이윤이 '0'이 되는 수준**의 지대
- 완전경쟁 하에서 잉여부분이 토지소유자에게 지대로 귀착

### 지대 공식

$$지대(초과이윤) = 매상고 - 생산비 - 운송비$$

- 도심에 가까운 입지일수록 수송비 감소 → **지대 증가**
- 가장 높은 지대를 지불할 수 있는 산업에 토지 할당

## 3. 입찰지대곡선

### 형태

- 생산요소 간의 **대체가 일어날 경우**: **우하향**하며 **원점을 향하여 볼록**한 형태
- 도심으로부터 한계지까지 각 지점의 토지를 경매에 부칠 경우, 토지이용자가 부를 수 있는 **최고가격을 반영**
- 도심에서 교외로 이동하면서 **가장 높은 지대를 지불할 수 있는 여러 산업들의 지대곡선을 연결한 선**

### 기울기

$$입찰지대곡선의\ 기울기 = \frac{기업의\ 한계교통비(단위당\ 한계운송비)}{기업의\ 토지사용량}$$

- 단위당 **한계수송비가 증가**할수록 → 기울기 **가파름**
- 기울기가 **가파른 업종**일수록 → **중심지에 가까이** 입지
- 기울기가 **완만한 업종**일수록 → **외곽**에 입지

### 도시 토지이용분화

- 기울기 가파름 → 도심 입지: **상업**(소매업)
- 기울기 중간 → 중간지대: **공업**
- 기울기 완만 → 외곽: **주거**

## 4. 튀넨과의 비교

| 구분 | 튀넨 | 알론소 |
|------|------|--------|
| 적용 공간 | 농촌 | **도시** |
| 곡선 명칭 | 한계지대곡선 | **입찰지대곡선** |
| 곡선 형태 | 우하향, 원점 볼록 | 우하향, 원점 볼록 (동일) |
| 지대 결정 | 수송비 | 수송비 (교통비) |
| 토지이용 | 농작물 배치 | 상업·공업·주거 배치 |

## 핵심 암기 포인트

- 알론소 = **입찰지대이론**, 튀넨의 고립국이론을 **도시공간에 적용**
- 입찰지대 = 토지이용자의 **최대지불용의액**, 초과이윤 = **0**
- 입찰지대곡선: **우하향, 원점 볼록** (생산요소 간 대체 발생 시)
- 기울기 = **한계교통비 / 토지사용량**
- 기울기 가파를수록 **도심 가까이** 입지 (상업 > 공업 > 주거)
- 한계수송비↑ → 기울기↑ → 도심 입지

## OX 퀴즈

**Q1.** 알론소의 입찰지대이론은 튀넨의 고립국이론을 도시공간에 적용·확장한 것이다. → **O**

**Q2.** 입찰지대란 초과이윤이 극대화되는 수준의 지대를 말한다. → **X** (초과이윤이 **'0'이 되는 수준**의 지대)

**Q3.** 입찰지대곡선은 우상향하며 원점에 오목한 형태이다. → **X** (**우하향**하며 **원점에 볼록**)

**Q4.** 입찰지대곡선의 기울기는 토지이용자의 토지사용량을 한계교통비로 나눈 값이다. → **X** (**한계교통비를 토지사용량으로** 나눈 값)

**Q5.** 입찰지대곡선의 기울기가 가파른 업종일수록 도심에 가까이 입지한다. → **O**

**Q6.** 알론소의 입찰지대이론에서 입찰지대란 토지이용자가 지불하고자 하는 최소금액이다. → **X** (**최대금액(최대지불용의액)**)

**Q7.** 단위당 한계수송비가 증가할수록 입찰지대곡선의 기울기는 완만해진다. → **X** (한계수송비 증가 → 기울기 **가파름**)

## 함정 노트

- ⚠️ "입찰지대 = 최소지불용의액" → **최대지불용의액** (초과이윤 0인 수준)
- ⚠️ "입찰지대곡선: 우상향" → **우하향**, 원점 **볼록**
- ⚠️ "기울기 = 토지사용량/한계교통비" → 기울기 = **한계교통비/토지사용량** (분자·분모 혼동 주의)
- ⚠️ "기울기 가파를수록 외곽 입지" → 기울기 가파를수록 **도심 가까이** 입지
- ⚠️ "알론소 = 리카도의 이론 도시 적용" → **튀넨**의 고립국이론을 도시에 적용
