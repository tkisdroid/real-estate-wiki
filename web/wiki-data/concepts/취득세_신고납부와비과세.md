# 취득세 - 신고납부와 비과세

## 1. 신고납부 기한

| 취득유형 | 신고기한 |
|----------|----------|
| **일반 취득 (매매 등)** | 취득일부터 **60일** 이내 |
| **상속** | 상속개시일이 속하는 달의 말일부터 **6개월** (외국 거주: **9개월**) |
| **증여** (부담부증여 포함) | 취득일이 속하는 달의 말일부터 **3개월** |
| **등기·등록하는 경우** | 등기·등록 **접수일**까지 (60일·6개월·3개월보다 빠르면 우선) |
| **중과세대상이 된 경우** | 중과세대상이 된 날부터 **60일** 이내 |

### 핵심 원리

```
일반취득: 취득일 + 60일 이내
상속: 상속개시월 말일 + 6개월 (외국 9개월)
증여: 취득월 말일 + 3개월
공통: 등기·등록 접수일 전이면 접수일까지
```

> **등기 선행 시**: 60일 이내와 등기접수일 중 **빠른 날**까지 신고·납부

### 대위자 신고납부
- 채권자대위권에 의한 등기신청 시 채권자가 납세의무자를 **대위**하여 신고·납부 가능
- 대위 납부 시 지방자치단체장은 납세의무자에게 **즉시 통보**

## 2. 가산세

### 무신고가산세

| 유형 | 가산세율 |
|------|:--------:|
| **일반** 무신고 | 무신고납부세액의 **20%** |
| **부정** 무신고 (사기 등) | 무신고납부세액의 **40%** |

### 과소신고가산세

| 유형 | 가산세율 |
|------|:--------:|
| **일반** 과소신고 | 과소신고분 세액의 **10%** |
| **부정** 과소신고 | 부정분 세액의 **40%** + 나머지 과소분의 10% |

### 납부지연가산세

- **산식**: 미납(과소납부) 세액 x 경과일수 x **1일 0.022%** (10만분의 22)
- **한도**: 미납세액의 **75%**

### 중가산세

| 항목 | 내용 |
|------|------|
| **대상** | 취득 후 **신고하지 않고 매각**하는 경우 |
| **가산세율** | 산출세액에 **80%** 가산 |
| **제외 대상** | 등기·등록 불요 과세물건(골프회원권 등 제외), 지목변경, 차량 종류변경, 주식취득 등 간주취득 |

## 3. 비과세

| 번호 | 비과세 사유 |
|------|-----------|
| 1 | **국가·지방자치단체·외국정부** 등의 취득 |
| 2 | **기부채납** 조건 취득 (반대급부 없는 경우) |
| 3 | **신탁재산**의 이전 (위탁자↔수탁자, 수탁자 변경) |
| 4 | **환매권 행사**로 매수하는 부동산 |
| 5 | **임시건축물** 취득 (존속기간 **1년 이하**, 사치성재산 제외) |
| 6 | 공동주택 **개수** (시가표준액 **9억원 이하** 주택의 노후시설물 교체·수선) |
| 7 | 상속개시 이전 **사용불가** 차량의 상속취득 |

### 비과세 핵심 포인트

- **신탁재산 이전**: 위탁자→수탁자, 수탁자→위탁자, 수탁자 변경 모두 **비과세**
- **임시건축물**: 존속기간 **1년 이하** + 사치성재산 **아닌** 경우
- **공동주택 개수**: 시가표준액 **9억원 이하** 조건 충족 필요

## 4. 면세점

| 항목 | 내용 |
|------|------|
| **면세점** | 취득가액 **50만원 이하** → 취득세 미부과 |
| **인접 토지·건축물** | 취득일로부터 **1년 이내** 인접 토지·건축물 취득 시 **합산** 판단 |
| **연부취득** | **연부금액 총액** 기준 (연부금 1회분이 아님) |

## 5. 부가세

### 농어촌특별세

| 산식 | 내용 |
|------|------|
| 과세표준 x **2%**(표준세율) x **10%** | 취득세 표준세율을 2%로 적용한 세액의 10% |
| 감면세액의 **20%** | 취득세 감면 시 감면세액의 20% |

### 지방교육세

| 산식 | 내용 |
|------|------|
| 과세표준 x **(표준세율 - 2%)** x **20%** | 표준세율에서 중과기준세율 2%를 뺀 세율의 20% |
| 유상거래 주택: 해당세율 x **50%** x **20%** | 주택 세율의 50%에 20% |

## 6. 취득 후 중과세대상이 된 경우

- 취득 후 **5년 이내** 중과세 대상이 된 경우: 중과세율 **추징**
- 추가 납부세액 = (과세표준 x 중과세율) - 기납부세액(가산세 제외)
- 신고기한: **60일** 이내

## 핵심 암기 포인트

- 일반 **60일**, 상속 **6개월**(외국 9개월), 증여 **3개월**
- 등기 선행 시: 등기접수일까지 (빠른 날 기준)
- 무신고가산세: 일반 **20%**, 부정 **40%**
- 과소신고가산세: 일반 **10%**, 부정 **40%**
- 중가산세: 무신고 매각 시 **80%**
- 납부지연: 1일 **0.022%**, 한도 **75%**
- 면세점: **50만원** 이하
- 비과세: 국가 등, 기부채납, 신탁이전, 환매, 임시건축물(1년), 공동주택 개수(9억)

## OX 퀴즈

**Q1.** 취득세 신고기한은 취득일부터 30일 이내이다. → **X** (60일 이내)

**Q2.** 상속에 의한 취득세 신고기한은 상속개시일부터 6개월이다. → **X** (상속개시일이 속하는 **달의 말일**부터 6개월)

**Q3.** 증여에 의한 취득세 신고기한은 3개월이다. → **O** (취득일이 속하는 달의 말일부터 3개월)

**Q4.** 취득 후 신고하지 않고 매각하면 산출세액에 50%를 가산한다. → **X** (80%)

**Q5.** 신탁재산의 수탁자 변경은 취득세 과세대상이다. → **X** (비과세)

**Q6.** 임시건축물은 존속기간 2년 이하일 때 비과세된다. → **X** (1년 이하)

**Q7.** 면세점은 취득가액 100만원 이하이다. → **X** (50만원 이하)

**Q8.** 연부취득의 면세점은 연부금 1회분 금액으로 판단한다. → **X** (연부금액 총액 기준)

**Q9.** 취득 후 3년 이내에 중과세대상이 되면 추징된다. → **X** (5년 이내)

**Q10.** 외국에 주소를 둔 상속인의 취득세 신고기한은 9개월이다. → **O**

## 함정 노트

- **상속 기산점**: "상속개시일"이 아니라 "상속개시일이 속하는 **달의 말일**"부터 6개월. 기산점 혼동 주의
- **등기접수일 vs 60일**: 60일 이내라도 등기를 먼저 하면 **등기접수일까지** 신고·납부. "빠른 날" 기준
- **중가산세 80%**: 가산세 중 가장 높은 세율. "무신고 + 매각"이라는 특수한 경우에만 적용
- **임시건축물 1년**: "1년" 이하와 "사치성재산 아닌" 두 요건 모두 충족 필요
- **납부지연가산세 한도 75%**: 가산세가 원래 세액의 75%를 넘지 못함
