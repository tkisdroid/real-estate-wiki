# 마샬(A. Marshall)의 준지대설과 경제적 지대

> 📖 상위 문서: [[지대이론]]

## 1. 준지대(Quasi-Rent)

### 의의

- 마샬의 준지대론은 리카도의 **차액지대론을 재편성**한 이론
- 준지대란 생산을 위하여 **사람이 만든 기계·기구** 등 **고정생산요소**로부터 얻는 소득

### 핵심 내용

- 준지대의 크기는 해당 고정생산요소를 이용하여 만든 **생산물의 가격**에 의해 결정
- 단기적으로 고정생산요소의 공급량은 **고정** → 준지대는 **수요에 의해 결정**
- 준지대는 고정요소가 존재하는 **단기에만 발생**
- **장기에는 사라지는 지대** (장기에는 고정요소도 가변요소가 됨)

### 토지지대와의 차이

| 구분 | 준지대 | 토지지대(지대) |
|------|--------|---------------|
| 대상 | 기계·기구 등 **인공 고정생산요소** | **토지** |
| 기간 | **단기**에만 발생 | **영구적** |
| 장기 | **소멸** | 지속 |

## 2. 경제적 지대(Economic Rent)와 전용수입

### 전용수입(Transfer Earning)

- 어떤 생산요소가 다른 용도로 **전용(사용)되지 않도록** 하기 위하여 현재의 용도에서 지급되어야 하는 **최소한의 지급액(보수)**
- 생산요소의 **기회비용**과 같은 개념

### 경제적 지대(Economic Rent)

- 어떤 생산요소를 현재의 용도에 머물도록 하기 위하여 필요한 지급액을 **초과해서 지불된 소득**
- 즉, **전용수입을 초과하는 소득**
- 생산물시장에서는 **생산자잉여**라고도 함

### 기본 공식

$$생산요소\ 공급자의\ 총수입 = 경제적\ 지대 + 전용수입(기회비용)$$

### 공급의 가격탄력성과의 관계

| 공급탄력성 | 경제적 지대 | 전용수입 |
|-----------|------------|---------|
| 탄력성 **클수록** | **작아짐** | **커짐** |
| 탄력성 **작을수록** | **커짐** | **작아짐** |
| **완전비탄력적** (수직) | 총수입 **전액** = 경제적 지대 | **0** |
| **완전탄력적** (수평) | **0** | 총수입 **전액** = 전용수입 |

### 직관적 이해

- 공급이 **비탄력적**이라는 것 = 대체 용도가 별로 없다 → 기회비용(전용수입) 작음 → 경제적 지대 큼
- 공급이 **탄력적**이라는 것 = 대체 용도가 많다 → 기회비용(전용수입) 큼 → 경제적 지대 작음

## 3. 헤이그(Haig)의 마찰비용이론

- 교통수단은 공간의 마찰을 극복하기 위해 고안된 것
- **지대 = 마찰비용 - 교통비** (교통비의 절약분)
- 교통수단이 **양호할수록** 마찰이 **적어짐**

## 핵심 암기 포인트

- 마샬 = **준지대설**, 리카도의 차액지대론 **재편성**
- 준지대 = **기계·기구 등 인공 고정생산요소**의 소득 (토지 아님!)
- 준지대는 **단기**에만 발생, **장기**에는 사라짐
- 총수입 = **경제적 지대 + 전용수입(기회비용)**
- 공급 **완전비탄력적** → 전액 **경제적 지대**
- 공급 **완전탄력적** → 전액 **전용수입**
- 헤이그: 지대 = **마찰비용 - 교통비**

## OX 퀴즈

**Q1.** 마샬의 준지대는 장기에도 계속 발생한다. → **X** (**단기**에만 발생, 장기에는 사라짐)

**Q2.** 준지대란 토지로부터 얻는 소득을 말한다. → **X** (사람이 만든 **기계·기구 등 고정생산요소**의 소득)

**Q3.** 공급이 완전비탄력적이면 총수입 전액이 전용수입이다. → **X** (완전비탄력적이면 전액 **경제적 지대**, 완전탄력적이면 전액 전용수입)

**Q4.** 경제적 지대는 전용수입을 초과하는 소득이다. → **O**

**Q5.** 전용수입은 생산요소의 기회비용과 같은 개념이다. → **O**

**Q6.** 공급의 가격탄력성이 커질수록 경제적 지대는 커진다. → **X** (탄력성 커질수록 경제적 지대 **작아짐**, 전용수입 커짐)

**Q7.** 공급이 완전탄력적일 경우 전용수입이 발생하지 않는다. → **X** (완전탄력적이면 전액이 **전용수입**. 발생하지 않는 것은 경제적 지대) - 감정평가사 2025 출제

**Q8.** 헤이그에 의하면 지대는 마찰비용에서 교통비를 차감한 것이다. → **O**

## 함정 노트

- ⚠️ "준지대 = 토지지대" → 준지대는 **기계·기구 등 인공 고정생산요소**의 소득 (토지 아님)
- ⚠️ "준지대 = 장기에도 발생" → **단기**에만 발생, 장기에는 사라짐
- ⚠️ "공급 완전비탄력적 → 전용수입" → 완전**비**탄력적 = 전액 **경제적 지대**, 완전**탄력적** = 전액 전용수입
- ⚠️ "경제적 지대 = 기회비용" → 경제적 지대 = **전용수입(기회비용)을 초과하는 소득**. 기회비용은 전용수입
- ⚠️ "탄력성↑ → 경제적 지대↑" → 탄력성↑ → 경제적 지대 **↓**, 전용수입 **↑**
- ⚠️ "완전탄력적 → 전용수입 미발생" → 완전탄력적이면 전액 **전용수입** (감정평가사 2025 출제 함정)
