# 기타 금지행위 (법 제33조)

## 1. 중개대상물의 매매를 업으로 하는 행위 (제1항 제1호)

### 의의
- 개업공인중개사등이 중개가 아닌 **직접 매매업**을 하는 행위 금지
- 매각물건을 싸게 사서 비싸게 파는 등의 행위로 **공정한 거래질서 파괴 방지**

### 주의사항
- 매매를 **업으로** 하는 것만 금지 → 일회성 매매는 해당하지 않을 수 있음
- 업의 판단: **반복·계속성, 영업성, 수익 목적, 규모·횟수·태양** 등 종합 판단
- **임대를 업으로 하는 것**은 해당되지 않음
- 벌칙: **1년/1천만원**

### 판례
| 판례 | 내용 |
|---|---|
| 대판 1988.8.9 | 매매를 업으로 한다 = 계속적이고 **반복적**인 것 |
| 대판 1995.11.7 | 거래태양·규모·횟수·보유기간 등 사회통념상 사업활동으로 볼 수 있는 계속성·반복성 + **수익 목적** |

## 2. 무등록업자 명의 이용 중개의뢰 (제1항 제2호)

- 무등록업자임을 **알면서** 그를 통해 중개의뢰받거나 명의 이용하게 하는 행위
- 무등록업자와의 협력 차단이 목적
- 무등록업자인 줄 **몰랐다면** 해당 안 됨 (단, 스스로 소명 필요)
- 벌칙: **1년/1천만원**

## 3. 보수 초과수수 (제1항 제3호)

- 사례·증여 등 **어떠한 명목으로도** 법정 보수·실비 초과 금품 수수 금지
- 의뢰인이 **스스로** 초과 금액을 준 경우에도 위반
- 초과 수수 후 **되돌려준 경우**에도 위반 (수수 시점에 죄 성립)
- 초과 부분은 **무효** → 반환 의무
- 부가가치세법상 부가가치세를 별도 징수하는 것은 위반 **아님** (일반과세자에 한함)
- 벌칙: **1년/1천만원**

### 판례
| 판례 | 내용 |
|---|---|
| 대판 1990.11.13 | 사례금 명목 초과수수도 유죄 |
| 대판 1999.2.9 | 수고비 명목 초과수수도 위반 |
| 대판 2004.11.12 | 당좌수표 교부 시점에 죄 성립 (부도·반환 무관) |

## 4. 거짓 언행으로 의뢰인 판단을 그르치게 하는 행위 (제1항 제4호)

- 거래의 **중요사항**(물리적 상태, 환경, 가격, 거래조건 등)에 대한 거짓된 언행
- 가격 조작, 과장광고, 사술·기만 등 포함
- **침묵**(어떠한 사실을 알면서도 알리지 않는 것)도 포함
- 벌칙: **1년/1천만원**

### 판례
| 판례 | 내용 |
|---|---|
| 대판 1991.12.24 | 매도의뢰가격을 숨기고 높은 가격으로 매도 후 차액 취득 → 금지행위 + 불법행위 |
| 대판 1980.2.26 | 개발제한구역 결정 사실 속이고 매매 중개 → 불법행위 |
| 대판 2008.2.1 | "거래상의 중요사항"에는 대상물 자체 + **가격 등**도 포함 |

## 5. 증서의 매매·교환 중개 또는 매매업 (제1항 제5호)

- 관계법령에서 양도·알선 등이 금지된 부동산 분양·임대 관련 **증서**의 매매·교환 중개 또는 매매업
- 증서: 주택청약예금(저축)통장, 주택상환사채, 주택조합원 지위, 무허가건물확인서 등
- **분양권(입주자로 선정된 지위)**은 증서에 해당하지 않음 → 건물 중개에 해당 (대판 1990.2.13)
- 벌칙: **3년/3천만원**

## 6. 부동산투기 조장 (제1항 제7호)

- 탈세 목적 **미등기 전매** 중개
- 관계법령에 의한 **전매제한 부동산** 매매 중개
- 전매차익이 **실제로 나지 않아도** 위반 (대판 1990.11.23)
- 포괄적으로 투기를 조장하는 모든 행위 포함
- 벌칙: **3년/3천만원**

## 7. 시세 부당영향 (제1항 제8호)

- 부당이익 목적으로 **거짓 거래 완료** 꾸미는 행위
- 제3자에게 부당한 이익을 얻게 할 목적도 포함
- 시세에 부당한 영향을 주거나 **줄 우려가 있는 행위**까지 포함
- 벌칙: **3년/3천만원**

## 8. 중개제한 단체 구성 (제1항 제9호)

- 특정 대상물에 대해 중개를 **제한하는 단체** 구성
- 단체 외 자와 **공동중개 제한** 행위
- 벌칙: **3년/3천만원**

## 9. 개업공인중개사등에 대한 업무방해 금지 (제2항)

### 적용 대상: **누구든지** (비 중개업자 포함)
- 시세에 부당한 영향을 줄 목적으로 다음 행위 금지

| 금지행위 |
|---|
| 안내문·온라인 커뮤니티로 특정 개업공인중개사에 중개의뢰 **제한·유도** |
| 시세보다 현저히 높게 표시·광고하는 특정 개업공인중개사에게만 **의뢰 유도** |
| 특정 가격 이하로 **중개의뢰 금지 유도** |
| 정당한 표시·광고 행위 **방해** |
| 시세보다 현저히 높게 표시·광고하도록 **강요·유도** |

- 벌칙: **3년/3천만원**

## 10. 중개대상물 표시·광고 관련

### 부당 표시·광고 (개업공인중개사)
- 위반 시: **500만원 이하 과태료**
- 행정처분: 업무정지

### 아닌 자의 표시·광고
- 개업공인중개사가 아닌 자가 중개업을 하기 위해 표시·광고
- 벌칙: **1년/1천만원**

## 11. 위반 시 제재 종합

| 금지행위 | 형벌 | 행정처분(개업) | 행정처분(소속) |
|---|---|---|---|
| 직접거래·쌍방대리 | 3년/3천만원 | 임의 등록취소/업무정지 | 자격정지 |
| 투기조장 | 3년/3천만원 | 임의 등록취소/업무정지 | 자격정지 |
| 증서 중개·매매업 | 3년/3천만원 | 임의 등록취소/업무정지 | 자격정지 |
| 시세 부당영향 | 3년/3천만원 | 임의 등록취소/업무정지 | 자격정지 |
| 단체 구성 | 3년/3천만원 | 임의 등록취소/업무정지 | 자격정지 |
| 업무방해(누구든지) | 3년/3천만원 | - | - |
| 매매업 | 1년/1천만원 | 임의 등록취소/업무정지 | 자격정지 |
| 무등록 명의이용 | 1년/1천만원 | 임의 등록취소/업무정지 | 자격정지 |
| 초과보수 | 1년/1천만원 | 임의 등록취소/업무정지 | 자격정지 |
| 거짓언행 | 1년/1천만원 | 임의 등록취소/업무정지 | 자격정지 |

- 고용인 위반 시 → **양벌규정** 적용 (개업공인중개사에게 벌금형)
- 개업공인중개사가 **상당한 주의와 감독**을 한 경우 → 양벌규정 미적용

## 핵심 암기 포인트

1. 매매업: **업**으로 하는 것만 금지 (일회성 매매 허용), 임대업은 해당 안 됨
2. 초과보수: **어떤 명목**이든, 자발적이든, 반환해도 → 위반
3. 거짓언행: **침묵**도 포함, "중요사항"에 **가격**도 포함
4. 투기조장: 전매차익 없어도 **위반**
5. 업무방해: **누구든지** 적용 → 3년/3천만원
6. 분양권 ≠ 증서: 분양권은 **건물 중개**에 해당
7. 부당 표시·광고 → **500만원 과태료** (형벌 아님)

## OX 퀴즈

**Q1.** 개업공인중개사가 임대를 업으로 하는 것은 금지행위에 해당한다.
> **X** — 매매를 업으로 하는 것만 금지이며, **임대를 업으로 하는 것은 해당하지 않는다**.

**Q2.** 전매차익이 실제로 발생하지 않았더라도 미등기 전매 중개를 하면 투기조장에 해당한다.
> **O** — 전매차익이 실제로 나지 않아도 **투기조장에 해당**한다 (대판 1990.11.23).

**Q3.** 무등록업자인 줄 모르고 중개를 의뢰받은 경우에도 금지행위에 해당한다.
> **X** — 무등록업자인 줄 **몰랐다면** 해당하지 않으나, 스스로 **소명** 필요.

**Q4.** 개업공인중개사가 알면서도 침묵한 것은 거짓 언행에 해당하지 않는다.
> **X** — **침묵**도 거짓 언행에 포함된다.

**Q5.** 업무방해 금지행위는 개업공인중개사에게만 적용된다.
> **X** — **누구든지** 적용된다 (비 중개업자 포함).

**Q6.** 분양권(입주자로 선정된 지위)의 매매를 중개하는 것은 증서 매매 중개에 해당한다.
> **X** — 분양권은 증서가 아니라 **건물 중개**에 해당한다 (대판 1990.2.13).

**Q7.** 부당한 표시·광고를 한 개업공인중개사에게는 1년 이하 징역이 부과된다.
> **X** — 부당 표시·광고는 형벌이 아닌 **500만원 이하 과태료**이다.

**Q8.** 소속공인중개사가 금지행위를 위반하면 자격이 취소된다.
> **X** — 소속공인중개사는 자격**정지** 처분을 받으며, 징역형을 받아야 자격**취소**된다.

**Q9.** 중개보수를 초과하여 받았다가 되돌려주면 금지행위 위반이 아니다.
> **X** — **수수 시점**에 죄가 성립하므로 되돌려줘도 위반이다.

**Q10.** 시세에 부당한 영향을 줄 우려가 있는 행위는 실제로 영향을 주지 않았으면 위반이 아니다.
> **X** — 시세에 영향을 주거나 **줄 우려가 있는 행위**까지 금지된다.

## 함정 노트

| 함정 포인트 | 정확한 내용 |
|---|---|
| 임대업도 금지 | **매매업만** 금지, 임대업은 해당 안 됨 |
| 전매차익 없으면 합법 | 전매차익 없어도 **투기조장 위반** |
| 침묵 ≠ 거짓언행 | **침묵도** 거짓언행에 포함 |
| 분양권 = 증서 | 분양권은 **건물 중개** (증서 아님) |
| 업무방해 → 중개업자만 | **누구든지** 적용 |
| 부당광고 → 형벌 | 부당광고 → **500만원 과태료** |
| 초과보수 반환 → 면책 | 반환해도 **면책 안 됨** |
| 소속 금지행위 → 자격취소 | 소속 금지행위 → **자격정지** (취소 아님) |
| 시세 영향 실제 발생 필요 | **우려**만으로도 위반 |
