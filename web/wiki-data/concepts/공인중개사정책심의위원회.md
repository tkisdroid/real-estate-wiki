# 공인중개사 정책심의위원회 (법 제3조의2)

## 1. 설치 목적
공인중개사의 시험·교육·중개보수 등에 관한 사항을 심의·의결하기 위해 **국토교통부**에 설치

## 2. 심의사항 (법 제3조의2)

1. 공인중개사의 **시험 등 자격취득**에 관한 사항
2. 부동산 **중개업의 육성**에 관한 사항
3. **중개보수 변경**에 관한 사항
4. **손해배상책임의 보장** 등에 관한 사항

### 시·도지사의 준수의무
- 심의사항 중 **자격취득에 관한 사항** → 시·도지사는 이에 **따라야** 한다

## 3. 위원 구성 (영 제1조의2)

| 항목 | 내용 |
|---|---|
| 위원장 | 국토교통부 **제1차관** |
| 위원 수 | 위원장 포함 **7인 이상 11인 이내** |
| 위원 임기 | **2년** (보궐위원: 전임위원 잔여기간) |

### 위원 자격 (국토교통부장관 임명·위촉)
1. 국토교통부 **4급 이상** 또는 고위공무원단 일반직 공무원
2. 대학 **부교수 이상**
3. **변호사** 또는 **공인회계사** 자격자
4. **공인중개사협회** 추천 자
5. 시험 위탁기관장 추천 자
6. **비영리민간단체** 추천 자
7. **소비자단체** 또는 한국소비자원 임직원
8. 부동산·금융 분야 학식·경험 풍부한 자

## 4. 위원의 제척·기피·회피·해촉

| 구분 | 내용 |
|---|---|
| 제척 | 위원 본인이 안건 당사자이거나 친족관계 등인 경우 자동 배제 |
| 기피 | 당사자가 공정성 문제로 신청 → 위원회 의결 (대상 위원은 의결 참여 불가) |
| 회피 | 위원 본인이 제척 사유에 해당 시 스스로 회피 |
| 해촉 | 제척 사유에도 회피하지 않은 경우 → 국토교통부장관이 해촉 가능 |

## 5. 운영

| 항목 | 내용 |
|---|---|
| 개의 | 재적위원 **과반수 출석** |
| 의결 | 출석위원 **과반수 찬성** |
| 회의 통보 | 개최 **7일 전**까지 (긴급 시 전날까지) |
| 간사 | 국토교통부 소속 공무원 중 위원장이 지명, **1명** |
| 수당 | 출석 위원·전문가에게 예산 범위 내 수당·여비 지급 (소관업무 공무원 제외) |

## 6. 운영세칙
- 시행령에서 규정한 사항 외 운영 등에 필요한 사항은 **위원회 의결**을 거쳐 **위원장이 정함**
