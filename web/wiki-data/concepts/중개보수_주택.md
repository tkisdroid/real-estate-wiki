# 주택 중개보수 (규칙 제20조 별표1)

## 1. 주택 보수의 법적 근거

- 주택(부속토지 포함)의 중개보수는 **국토교통부령이 정하는 범위 안에서** 특별시·광역시·도의 **조례**로 정함 (법 제32조 제2항)
- 조례로 정하는 요율 한도 이내에서 의뢰인과 개업공인중개사가 **서로 협의**하여 결정 (규칙 제20조 제1항)

## 2. 매매·교환 요율표

| 거래금액 | 상한요율 | 한도액 |
|---|---|---|
| 5천만원 미만 | 천분의 6 | **25만원** |
| 5천만원 이상 ~ 2억원 미만 | 천분의 5 | **80만원** |
| 2억원 이상 ~ 9억원 미만 | 천분의 4 | - |
| 9억원 이상 ~ 12억원 미만 | 천분의 5 | - |
| 12억원 이상 ~ 15억원 미만 | 천분의 6 | - |
| **15억원 이상** | **천분의 7** | - |

## 3. 임대차 등 요율표

| 거래금액 | 상한요율 | 한도액 |
|---|---|---|
| 5천만원 미만 | 천분의 5 | **20만원** |
| 5천만원 이상 ~ 1억원 미만 | 천분의 4 | **30만원** |
| 1억원 이상 ~ 6억원 미만 | 천분의 3 | - |
| 6억원 이상 ~ 12억원 미만 | 천분의 4 | - |
| 12억원 이상 ~ 15억원 미만 | 천분의 5 | - |
| **15억원 이상** | **천분의 6** | - |

### 임대차 등의 범위
- 임대차뿐 아니라 **저당권·전세권·지상권·지역권·임차권** 등의 설정행위 중개 시에도 적용

## 4. 주택 판정 기준 (규칙 제20조 제6항)

- 건축물 중 **주택 면적이 1/2 이상** → **주택 보수율** 적용
- 주택 면적이 1/2 **미만** → **주택 외 보수율** 적용 (천분의 9 이내)

## 5. 한도액 계산

### 한도액이 있는 구간 (5천만원 미만, 5천만원~2억원)

한도액이 있는 경우: **거래금액 x 요율** 과 **한도액** 중 **적은 금액**

#### 계산 예시 (매매)

**예시 1**: 매매가격 4,000만원
- 4,000만원 x 천분의 6 = 24만원
- 한도액 25만원
- 보수: **24만원** (계산액이 한도액보다 작으므로)

**예시 2**: 매매가격 1억원
- 1억원 x 천분의 5 = 50만원
- 한도액 80만원
- 보수: **50만원** (계산액이 한도액보다 작으므로)

## 6. 소재지 차이 시

- 중개대상물 소재지와 사무소 소재지가 다를 때 → **사무소 소재지** 시·도 조례 적용 (규칙 제20조 제3항)
- 분사무소의 경우 → **분사무소 소재지** 시·도 조례 적용

## 7. 분양권 중개보수

- 아파트 특정 동·호수에 대한 피분양자 선정 또는 분양계약 체결 후 → 특정 아파트 매매 중개는 **건물 중개**에 해당
- **주택 보수율** 적용 (대판 2005.5.27)

## 핵심 암기 포인트

1. 주택 보수: 시·도 **조례** 위임 (국토교통부령 범위 내)
2. 매매 한도액: 5천만원 미만 → **25만원**, 5천~2억원 → **80만원**
3. 임대차 한도액: 5천만원 미만 → **20만원**, 5천~1억원 → **30만원**
4. 최고 요율: 매매 **천분의 7** (15억 이상), 임대차 **천분의 6** (15억 이상)
5. 주택 면적 기준: **1/2 이상** → 주택 보수율
6. 소재지 차이: **사무소 소재지** 조례 적용
7. 분양권: **주택** 보수율 적용

## OX 퀴즈

**Q1.** 주택의 중개보수는 국토교통부령으로 직접 정한다.
> **X** — 국토교통부령이 정하는 범위 안에서 **시·도 조례**로 정한다.

**Q2.** 매매가격 5천만원 미만 주택의 보수 한도액은 20만원이다.
> **X** — 매매가격 5천만원 미만의 한도액은 **25만원**이다 (임대차가 20만원).

**Q3.** 주택 임대차 거래금액 15억원 이상의 상한요율은 천분의 7이다.
> **X** — 임대차 15억원 이상의 상한요율은 **천분의 6**이다 (매매가 천분의 7).

**Q4.** 건축물 중 주택 면적이 1/2 미만이면 주택 보수율을 적용한다.
> **X** — 주택 면적 1/2 **미만**이면 **주택 외 보수율**(천분의 9 이내)을 적용한다.

**Q5.** 중개대상물 소재지와 사무소 소재지가 다르면 대상물 소재지 조례를 적용한다.
> **X** — **사무소 소재지** 시·도 조례를 적용한다.

**Q6.** 임대차의 경우 전세권 설정행위 중개에도 주택 임대차 보수율이 적용된다.
> **O** — 임대차 등에는 전세권·저당권·지상권 등의 설정행위 중개도 포함된다.

## 함정 노트

| 함정 포인트 | 정확한 내용 |
|---|---|
| 보수 = 국토교통부령 직접 | 주택은 국토교통부령 범위 내 **시·도 조례** |
| 매매 한도액 20만원 | 매매 5천만원 미만 **25만원** (20만원은 임대차) |
| 임대차 최고 천분의 7 | 임대차 최고 **천분의 6** (천분의 7은 매매) |
| 주택 면적 1/2 미만 → 주택 | 1/2 미만 → **주택 외** 보수율 |
| 대상물 소재지 조례 | **사무소** 소재지 조례 적용 |
| 2억~9억 한도액 80만원 | 2억~9억은 **한도액 없음** (80만원은 5천~2억) |
