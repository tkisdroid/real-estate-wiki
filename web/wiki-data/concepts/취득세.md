# 취득세

취득세는 지방세법에 열거된 과세대상의 소유권을 취득한 경우 그 취득자에게 부과되는 **도세**(보통세)이다. 매년 시험에 **3~5문항** 출제되는 핵심 과목이다.

## 1. 취득세의 특징

- **행위세**: 취득이라는 법률행위에 대하여 과세
- **물세**: 취득자의 인적사항 불문, 물건별 개별과세
- **종가세**: 취득물건의 가액을 과세표준으로 함
- **차등비례세율**: 물건 종류에 따라 다른 세율 적용
- **신고납부**: 납세의무자가 자진 신고·납부 (신고로 확정)
- **거래세·유통세**: 소유권 이전 거래단계마다 과세
- **면세점 제도**: 취득가액 **50만원 이하**인 경우 비과세

## 2. 취득의 개념과 범위

### 취득의 구분

```
취득
├── 사실상 취득
│   ├── 승계취득
│   │   ├── 유상 승계취득 (매매, 교환, 현물출자)
│   │   └── 무상 승계취득 (상속, 증여, 기부)
│   └── 원시취득
│       ├── 토지: 공유수면 매립·간척
│       ├── 건축물: 신축, 재축
│       ├── 민법상 시효취득
│       └── 광업권·어업권 출원취득 (면제)
└── 간주(의제)취득
    ├── 토지 지목변경 (가액 증가시)
    ├── 건축물 개수 (대수선 등)
    ├── 차량·기계장비·선박 종류변경
    └── 과점주주의 주식취득 (비상장법인 50% 초과)
```

### 과점주주 취득세

- **과점주주**: 비상장법인의 주식 합계가 발행주식 총수의 **100분의 50을 초과**하면서 법인의 지배권을 실질적으로 행사하는 자
- 법인 **설립 시** 과점주주가 된 경우: 취득으로 보지 않음
- **최초** 과점주주가 된 경우: 소유 주식 **전체** 취득으로 봄
- **기존** 과점주주가 주식비율 증가: **증가분**만 취득으로 봄
- **재차** 과점주주가 된 경우: 이전 최고비율 초과 **증가분**만 과세

## 3. 과세대상

| 분류 | 과세대상 |
|------|----------|
| 부동산 | 토지 (28개 지목), 건축물 (건축법상 + 시설물) |
| 부동산에 준하는 것 | 차량, 기계장비, 선박, 항공기, 입목 |
| 권리 | 광업권, 어업권, 양식업권 |
| 회원권 | 골프회원권, 콘도미니엄회원권, 종합체육시설이용회원권, 승마회원권, 요트회원권 |

## 핵심 비교 정리

### 과세표준 기준 비교

| 취득유형 | 과세표준 |
|----------|---------|
| 유상승계취득 | **사실상취득가격** |
| 상속 무상취득 | **시가표준액** |
| 기타 무상취득 | **시가인정액** (곤란시 시가표준액) |
| 원시취득 | **사실상취득가격** (확인 불가 시 시가표준액) |

### 표준세율 요약

| 취득원인 | 세율 |
|----------|------|
| 유상 농지 | **3%** |
| 유상 농지외 (주택제외) | **4%** |
| 유상 주택 6억 이하 | **1%** |
| 유상 주택 9억 초과 | **3%** |
| 상속 농지/비농지 | **2.3% / 2.8%** |
| 증여 일반/비영리 | **3.5% / 2.8%** |
| 원시취득 | **2.8%** |

### 중과세율 요약

| 대상 | 중과분 |
|------|--------|
| 사치성 재산 | 표준세율 **+ 8%** (2%x4배) |
| 과밀억제권역 신·증축/공장 | 표준세율 **+ 4%** (2%x2배) |
| 대도시 법인설립 | 표준세율x**3배** - 2%x2배 |

### 신고기한 비교

| 취득유형 | 기한 | 기산점 |
|----------|------|--------|
| 일반 | **60일** | 취득일 |
| 상속 | **6개월** (외국 9개월) | 속하는 달 말일 |
| 증여 | **3개월** | 속하는 달 말일 |

## 핵심 암기 포인트

- **표준세율**: 유상(농지 3%, 농지외 4%), 상속(농지 2.3%, 농지외 2.8%), 증여(일반 3.5%, 비영리 2.8%), 원시취득 2.8%
- **중과기준세율**: 2%
- **과점주주**: 비상장법인 50% 초과 → 최초 전체, 기존 증가분만
- **과세표준**: 유상 = 사실상취득가격, 상속 = 시가표준액, 기타 무상 = 시가인정액
- **면세점**: 50만원 이하

## OX 퀴즈

**Q1.** 취득세의 유상승계취득 표준세율은 농지도 4%이다. → **X** (농지는 3%, 농지 외가 4%)

**Q2.** 과점주주는 상장법인의 주식 50% 초과 시 해당한다. → **X** (비상장법인)

**Q3.** 법인이 설립 시 과점주주가 된 경우 취득세가 부과된다. → **X** (취득으로 보지 않는다)

**Q4.** 기존 과점주주가 주식비율이 증가한 경우 소유 주식 전체가 취득으로 간주된다. → **X** (증가분만 취득)

**Q5.** 상속에 의한 무상취득의 과세표준은 시가인정액이다. → **X** (상속은 시가표준액. 기타 무상이 시가인정액)

## 함정 노트

- 세율·중과·신고납부 관련 상세 함정은 각 세부 페이지 참조
- **과세표준**: 유상 = 사실상취득가격이지만, 상속 무상 = 시가표준액 (시가인정액 아님). 기타 무상만 시가인정액
- **개인 vs 법인 차이**: 건설자금이자·할부이자 포함 여부가 다름 (법인 포함, 개인 제외)
- **과점주주**: 설립 시 = 과세 안됨, 최초 = 전체, 기존 = 증가분만
