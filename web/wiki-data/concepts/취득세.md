# 취득세

취득세는 지방세법에 열거된 과세대상의 소유권을 취득한 경우 그 취득자에게 부과되는 **도세**(보통세)이다. 매년 시험에 **3~5문항** 출제되는 핵심 과목이다.

## 1. 취득세의 특징

- **행위세**: 취득이라는 법률행위에 대하여 과세
- **물세**: 취득자의 인적사항 불문, 물건별 개별과세
- **종가세**: 취득물건의 가액을 과세표준으로 함
- **차등비례세율**: 물건 종류에 따라 다른 세율 적용
- **신고납부**: 납세의무자가 자진 신고·납부 (신고로 확정)
- **거래세·유통세**: 소유권 이전 거래단계마다 과세
- **면세점 제도**: 취득가액 **50만원 이하**인 경우 비과세

## 2. 취득의 개념과 범위

### 취득의 구분

```
취득
├── 사실상 취득
│   ├── 승계취득
│   │   ├── 유상 승계취득 (매매, 교환, 현물출자)
│   │   └── 무상 승계취득 (상속, 증여, 기부)
│   └── 원시취득
│       ├── 토지: 공유수면 매립·간척
│       ├── 건축물: 신축, 재축
│       ├── 민법상 시효취득
│       └── 광업권·어업권 출원취득 (면제)
└── 간주(의제)취득
    ├── 토지 지목변경 (가액 증가시)
    ├── 건축물 개수 (대수선 등)
    ├── 차량·기계장비·선박 종류변경
    └── 과점주주의 주식취득 (비상장법인 50% 초과)
```

### 과점주주 취득세

- **과점주주**: 비상장법인의 주식 합계가 발행주식 총수의 **100분의 50을 초과**하면서 법인의 지배권을 실질적으로 행사하는 자
- 법인 **설립 시** 과점주주가 된 경우: 취득으로 보지 않음
- **최초** 과점주주가 된 경우: 소유 주식 **전체** 취득으로 봄
- **기존** 과점주주가 주식비율 증가: **증가분**만 취득으로 봄
- **재차** 과점주주가 된 경우: 이전 최고비율 초과 **증가분**만 과세

## 3. 과세대상

| 분류 | 과세대상 |
|------|----------|
| 부동산 | 토지 (28개 지목), 건축물 (건축법상 + 시설물) |
| 부동산에 준하는 것 | 차량, 기계장비, 선박, 항공기, 입목 |
| 권리 | 광업권, 어업권, 양식업권 |
| 회원권 | 골프회원권, 콘도미니엄회원권, 종합체육시설이용회원권, 승마회원권, 요트회원권 |

## 4. 과세표준

### 유상승계취득

**사실상취득가격** = 직접비용 + 간접비용

| 구분 | 법인 | 개인 |
|------|------|------|
| 부가가치세 | 제외 | 제외 |
| 건설자금이자 | 포함 | **제외** |
| 할부·연부이자, 연체료 | 포함 | **제외** |
| 중개보수 | 포함 | 포함 |

### 무상취득

| 구분 | 과세표준 |
|------|----------|
| **상속** 무상취득 | **시가표준액** |
| 시가표준액 **1억원 이하** 무상취득 | 시가인정액과 시가표준액 중 **납세자 선택** |
| 기타 무상취득 | **시가인정액** (산정 곤란시 시가표준액) |

- **시가인정액**: 취득일 전 6개월~후 3개월 이내 매매사례가액, 감정가액, 공매가액 등
- **시가표준액**: 개별공시지가, 개별(공동)주택가격, 건축물 시가표준액

### 원시취득

**사실상취득가격** (개인이 건축물을 건축하여 사실상취득가격 확인 불가 시 시가표준액)

### 부담부증여

- 채무부담액 → **유상승계취득** 과세표준 적용
- 시가인정액 - 채무부담액 → **무상취득** 과세표준 적용

## 5. 세율

### 표준세율 (부동산)

| 취득원인 | 구분 | 세율 |
|----------|------|------|
| **유상승계취득** | 농지 | **1,000분의 30 (3%)** |
| | 농지 이외 부동산 (주택 제외) | **1,000분의 40 (4%)** |
| **상속** | 농지 | **1,000분의 23 (2.3%)** |
| | 농지 이외 | **1,000분의 28 (2.8%)** |
| **상속 외 무상취득** | 비영리사업자 | **1,000분의 28 (2.8%)** |
| | 일반 | **1,000분의 35 (3.5%)** |
| **원시취득** | 신축, 매립·간척 | **1,000분의 28 (2.8%)** |
| **공유물 분할** | | **1,000분의 23 (2.3%)** |

### 유상거래 주택 세율

| 취득가액 | 세율 |
|----------|------|
| **6억원 이하** | **1,000분의 10 (1%)** |
| **6억원 초과 ~ 9억원 이하** | **1.01% ~ 2.99%** (산식 적용) |
| **9억원 초과** | **1,000분의 30 (3%)** |

> 주택 세율 산식: (취득당시가액 × 2/3 - 3) × 1/100

### 특례세율

형식적 소유권 취득: **표준세율 - 중과기준세율(2%)**
- 환매, 상속 1세대1주택, 공유물 분할, 재산분할, 건축물 이전 등

간주취득 등: **중과기준세율 2%**
- 개수, 종류변경, 지목변경, 과점주주 취득 등

### 중과세율

| 중과세 대상 | 세율 |
|-------------|------|
| **사치성 재산** (골프장·고급오락장·고급선박·고급주택) | 표준세율 + **중과기준세율(2%)의 4배 = +8%** |
| **과밀억제권역 내** 본점·주사무소 신·증축 | 표준세율 + **중과기준세율의 2배 = +4%** |
| **과밀억제권역 내** 공장 신·증설 | 표준세율 + **중과기준세율의 2배 = +4%** |
| **대도시 내** 법인설립 등 | **표준세율의 3배 - 중과기준세율의 2배** |

> **고급주택 기준**: 단독주택 연면적 331㎡ 초과 또는 대지면적 662㎡ 초과 + 시가표준액 **9억원 초과** / 공동주택 전용면적 245㎡(복층 274㎡) 초과 + 시가표준액 **9억원 초과**

### 중과기준세율

**1,000분의 20 (2%)** — 세율의 특례 적용 기준이 되는 세율

## 6. 신고납부

| 취득유형 | 신고기한 |
|----------|----------|
| 일반 취득 | 취득일부터 **60일 이내** |
| 상속 | 상속개시일이 속하는 달의 말일부터 **6개월** (외국 주소 **9개월**) |
| 증여 (부담부증여 포함) | 취득일이 속하는 달의 말일부터 **3개월** |
| 등기·등록하는 경우 | 등기·등록 **접수일까지** |
| 중과세대상이 된 경우 | **60일 이내** |

### 중가산세

취득세 과세물건을 사실상 취득한 후 **신고하지 않고 매각**하는 경우: 산출세액에 **80%** 가산

## 7. 부가세

### 농어촌특별세

- 과세표준 × **2%** (표준세율 적용) × **10%**
- 감면세액의 **20%**

### 지방교육세

- 과세표준 × **(표준세율 - 2%)** × **20%**
- 유상거래 주택: 해당세율 × **50%** × **20%**

## 8. 비과세

1. **국가·지방자치단체·외국정부** 등의 취득
2. 기부채납 조건 취득 (반대급부 없는 경우)
3. **신탁재산**의 이전 (위탁자↔수탁자, 수탁자 변경)
4. 환매권 행사로 매수하는 부동산
5. **임시건축물** 취득 (존속기간 **1년 이하**, 사치성재산 제외)
6. 공동주택 **개수** (시가표준액 9억원 이하 주택의 노후시설물 교체·수선)
7. 상속개시 이전 사용불가 차량의 상속취득

## 9. 면세점

- 취득가액 **50만원 이하**: 취득세 미부과
- 1년 이내 인접 토지·건축물 취득 시 합산하여 판단
- 연부취득: **연부금액 총액** 기준

## 핵심 암기 포인트

- **표준세율**: 유상(농지 3%, 농지외 4%), 상속(농지 2.3%, 농지외 2.8%), 증여(일반 3.5%, 비영리 2.8%), 원시취득 2.8%, 공유물분할 2.3%
- **유상거래 주택 세율**: 6억 이하 1%, 6~9억 1.01~2.99%, 9억 초과 3%
- **중과기준세율**: 2%
- **사치성재산 중과**: 표준세율 + 8% (중과기준세율 2% × 4배)
- **과밀억제권역 신·증축**: 표준세율 + 4% (2% × 2배)
- **대도시 법인설립**: 표준세율의 3배 - 중과기준세율의 2배
- **신고기한**: 일반 60일, 상속 6개월(외국 9개월), 증여 3개월, 등기 시 접수일까지
- **면세점**: 50만원 이하
- **과점주주**: 비상장법인 50% 초과 → 최초 전체, 기존 증가분만
- **과세표준**: 유상 = 사실상취득가격, 상속 무상 = 시가표준액, 기타 무상 = 시가인정액
- **시가인정액**: 취득일 전 6개월~후 3개월 이내
- **부가세**: 농어촌특별세(2%분의 10%), 지방교육세((세율-2%)분의 20%)
- **중가산세**: 미신고 매각 시 산출세액의 80% 가산

## OX 퀴즈

**Q1.** 취득세의 유상승계취득 표준세율은 농지도 4%이다. → **X** (농지는 3%, 농지 외가 4%)

**Q2.** 유상거래 주택 취득가액이 5억원이면 취득세율은 3%이다. → **X** (6억원 이하는 1%)

**Q3.** 사치성 재산의 취득세 중과세율은 표준세율 + 4%이다. → **X** (표준세율 + 8%)

**Q4.** 상속에 의한 취득세 신고기한은 상속개시일부터 60일 이내이다. → **X** (상속개시일이 속하는 달의 말일부터 6개월)

**Q5.** 취득가액이 100만원이면 취득세가 면제된다. → **X** (면세점은 50만원 이하)

**Q6.** 과점주주는 상장법인의 주식 50% 초과 시 해당한다. → **X** (비상장법인)

**Q7.** 취득세의 과세표준에서 개인의 건설자금이자는 포함한다. → **X** (개인은 제외, 법인은 포함)

**Q8.** 취득세를 신고하지 않고 매각하면 산출세액의 50%가 가산된다. → **X** (80%가 가산)

**Q9.** 임시건축물(존속기간 1년 이하)의 취득은 취득세 비과세이다. → **O** (단, 사치성재산 제외)

**Q10.** 증여에 의한 취득세 신고기한은 취득일이 속하는 달의 말일부터 6개월이다. → **X** (3개월)

**Q11.** 상속에 의한 무상취득의 과세표준은 시가인정액이다. → **X** (상속 무상취득은 시가표준액이다. 기타 무상취득이 시가인정액이다)

**Q12.** 유상거래 주택의 취득가액이 7억원이면 취득세율은 1%이다. → **X** (6억원 초과~9억원 이하 구간은 1.01%~2.99% 산식 적용)

**Q13.** 과밀억제권역 내 공장 신·증설의 중과세율은 표준세율 + 8%이다. → **X** (표준세율 + 4%이다. +8%는 사치성 재산)

**Q14.** 고급주택의 시가표준액 기준은 6억원 초과이다. → **X** (9억원 초과이다)

**Q15.** 법인이 설립 시 과점주주가 된 경우 취득세가 부과된다. → **X** (법인 설립 시 과점주주가 된 경우는 취득으로 보지 않는다)

**Q16.** 기존 과점주주가 주식비율이 증가한 경우 소유 주식 전체가 취득으로 간주된다. → **X** (증가분만 취득으로 본다. 최초 과점주주가 된 경우에만 전체 취득)

**Q17.** 부담부증여의 채무부담액 부분에는 무상취득 과세표준이 적용된다. → **X** (채무부담액 부분은 유상승계취득 과세표준이 적용된다)

**Q18.** 취득세의 중과기준세율은 4%이다. → **X** (중과기준세율은 2%이다)

## 함정 노트

- ⚠️ **세율 숫자 혼동**: 유상 농지 3% vs 농지외 4% vs 주택 1~3% vs 상속 2.3%/2.8% — 모두 다른 세율
- ⚠️ **신고기한 혼동**: 일반 60일, 상속 6개월, 증여 3개월 — 각각 기산점도 다름 (일반 = 취득일, 상속 = 속하는 달 말일, 증여 = 속하는 달 말일)
- ⚠️ **사치성재산 중과분 +8% vs 과밀억제 +4%**: 4배(8%) vs 2배(4%) 혼동 주의
