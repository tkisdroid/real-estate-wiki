# 벌칙 (행정형벌)

## 1. 3년 이하 징역 또는 3천만원 이하 벌금 (법 제48조)

| 번호 | 사유 | 핵심 키워드 |
|---|---|---|
| ① | **무등록 중개업** | 미등록 |
| ② | **거짓·부정 등록** | 위조·거짓 서류 |
| ③ | 관계법령 양도·알선 금지 **증서의 매매·교환 중개** 또는 **매매업** | 증서 중개 |
| ④ | **직접거래·쌍방대리** | 직접·쌍방 |
| ⑤ | 미등기 전매 중개 등 **부동산투기 조장** | 투기 |
| ⑥ | 거짓 거래 완료로 **시세 부당 영향** | 시세 조작 |
| ⑦ | **중개제한 단체 구성** | 단체 |
| ⑧ | 개업공인중개사등에 대한 **업무방해 금지행위** (누구든지) | 업무방해 |

### 업무방해 금지행위 (법 제33조 제2항) - "누구든지"

| 세부 사항 |
|---|
| 안내문·온라인 커뮤니티로 특정 개업공인중개사에 중개의뢰 **제한·유도** |
| 시세보다 현저히 높게 표시·광고하는 특정 개업공인중개사에게만 **의뢰 유도** |
| 특정 가격 이하로 **중개의뢰 금지 유도** |
| 정당한 표시·광고 행위 **방해** |
| 시세보다 현저히 높게 표시·광고하도록 **강요·유도** |

### 3년/3천만원 암기법
> **"무거(거짓)증직투시단업"** = 무등록, 거짓등록, 증서, 직접거래·쌍방대리, 투기조장, 시세부당영향, 단체구성, 업무방해

## 2. 1년 이하 징역 또는 1천만원 이하 벌금 (법 제49조)

| 번호 | 사유 | 핵심 키워드 |
|---|---|---|
| ① | 자격증 **양도·대여** / 양수·대여받은 자 / 알선한 자 | 자격증 대여 |
| ② | **이중등록·이중소속** | 이중 |
| ③ | **2이상 사무소** / 임시 중개시설물 설치 | 이중사무소 |
| ④ | 아닌 자의 **유사명칭** 사용 (공인중개사, 공인중개사사무소, 부동산중개) | 유사명칭 |
| ⑤ | 아닌 자의 **중개대상물 표시·광고** | 아닌 자 광고 |
| ⑥ | **등록증 양도·대여** / 양수·대여받은 자 / 알선한 자 | 등록증 대여 |
| ⑦ | 거래정보사업자의 **정보 차별공개** | 차별공개 |
| ⑧ | **비밀누설** (**반의사불벌죄**) | 비밀 |
| ⑨ | 중개대상물 **매매업** | 매매업 |
| ⑩ | 무등록자 명의 이용 **중개의뢰** | 무등록 명의 |
| ⑪ | **보수 초과수수** | 초과보수 |
| ⑫ | **거짓 언행**으로 의뢰인 판단 그르치게 한 행위 | 거짓언행 |
| ⑬ | **중개보조원 채용제한** 위반 | 채용제한 |

### 주의사항
- **비밀누설**은 **반의사불벌죄**: 피해자의 명시한 의사에 반하여 벌하지 않음
- 자격증 양도·대여와 등록증 양도·대여는 **별개** 항목

## 3. 양벌규정 (법 제50조)

### 적용 요건
- **소속공인중개사·중개보조원** 또는 **법인 사원·임원**이 중개 업무에 관하여 형벌사항 위반

### 효과
- **행위자** 처벌 + **개업공인중개사**에게 **벌금형** 부과
- 징역형은 개업공인중개사에게 부과되지 않음 (벌금형만)

### 면책 사유
- 개업공인중개사가 **상당한 주의와 감독을 게을리하지 않은 경우** → 양벌규정 미적용

### 양벌규정과 결격사유
- 양벌규정으로 받은 벌금 → **결격사유 해당 안 됨**

### 양벌규정과 구상권
- 양벌규정으로 받은 벌금 → 고용인에게 **구상 불가**
- 손해배상금은 고용인에게 **구상 가능** (손해액 한정)

## 4. 3년 vs 1년 벌칙 분류 핵심

| 3년/3천만원 (8가지) | 1년/1천만원 |
|---|---|
| 무등록, 거짓등록 | 자격증·등록증 양도대여 |
| 증서 중개·매매업 | 이중등록·이중소속·이중사무소 |
| 직접거래·쌍방대리 | 유사명칭, 아닌 자 광고 |
| 투기조장 | 비밀누설(반의사불벌) |
| 시세 부당영향 | 매매업, 무등록 명의이용 |
| 단체 구성 | 보수 초과수수 |
| 업무방해(누구든지) | 거짓언행, 보조원 채용제한 |

## 핵심 암기 포인트

1. **3년/3천만원**: 무등록, 거짓등록, 증서, 직접거래·쌍방대리, 투기, 시세영향, 단체, 업무방해 (8가지)
2. **1년/1천만원**: 양도대여(자격증·등록증), 이중(등록·소속·사무소), 유사명칭, 비밀누설, 매매업, 무등록명의, 초과보수, 거짓언행, 채용제한
3. 비밀누설 = **반의사불벌죄**
4. 양벌규정: 행위자 + 개업공인중개사 **벌금형**
5. 양벌규정 벌금 → **결격사유 아님**
6. 상당한 주의·감독 → 양벌규정 **미적용**
7. 업무방해 금지행위: **"누구든지"** 적용 → 비 중개업자도 처벌

## OX 퀴즈

**Q1.** 직접거래·쌍방대리 위반 시 1년 이하 징역 또는 1천만원 이하 벌금에 처한다.
> **X** — 직접거래·쌍방대리는 **3년 이하 징역 또는 3천만원 이하** 벌금이다.

**Q2.** 비밀누설은 피해자의 고소 없이도 처벌할 수 있다.
> **X** — 비밀누설은 **반의사불벌죄**로, 피해자의 명시한 의사에 반하여 벌하지 않는다.

**Q3.** 양벌규정에 의해 개업공인중개사에게 징역형이 부과될 수 있다.
> **X** — 양벌규정으로는 **벌금형만** 부과된다.

**Q4.** 개업공인중개사가 상당한 주의와 감독을 게을리하지 않았으면 양벌규정이 적용되지 않는다.
> **O** — 상당한 주의와 감독 시 양벌규정 **미적용**.

**Q5.** 양벌규정으로 받은 벌금은 결격사유에 해당한다.
> **X** — 양벌규정으로 받은 벌금은 결격사유에 **해당하지 않는다**.

**Q6.** 업무방해 금지행위는 개업공인중개사에게만 적용된다.
> **X** — 업무방해 금지행위는 **"누구든지"** 적용된다.

**Q7.** 보수 초과수수 위반 시 3년 이하 징역에 처한다.
> **X** — 보수 초과수수는 **1년 이하** 징역 또는 1천만원 이하 벌금이다.

**Q8.** 중개대상물의 매매를 업으로 하는 행위는 3년 이하 징역에 처한다.
> **X** — 매매업은 **1년 이하** 징역 또는 1천만원 이하 벌금이다.

## 함정 노트

| 함정 포인트 | 정확한 내용 |
|---|---|
| 직접거래 → 1년 | 직접거래·쌍방대리 → **3년/3천만원** |
| 매매업 → 3년 | 매매업 → **1년/1천만원** |
| 초과보수 → 3년 | 초과보수 → **1년/1천만원** |
| 비밀누설 → 친고죄 | 비밀누설 → **반의사불벌죄** (친고죄 아님) |
| 양벌규정 → 징역형 가능 | 양벌규정 → **벌금형만** |
| 양벌규정 벌금 → 결격 | 양벌규정 벌금 → **결격 아님** |
| 업무방해 → 개업공인중개사만 | 업무방해 → **누구든지** |
| 거짓언행 → 3년 | 거짓언행 → **1년/1천만원** |
