# 등기절차

등기절차의 흐름: **신청 → 접수 → 조사(심사) → 기입 → 교합 → 등기필정보 작성·통지**

## 비교 정리

### 공동신청 vs 단독신청

| 비교 항목 | 공동신청 | 단독신청 |
|-----------|---------|---------|
| **원칙/예외** | 원칙 | 예외 |
| **신청인** | 등기권리자 + 등기의무자 | 일방만 |
| **대표 유형** | 매매 이전등기, 저당권 설정 | 보존, 상속, 판결, 신탁 등 |

### 등기신청의 강제 비교

| 구분 | 근거법 | 기한 |
|------|--------|------|
| 건물표제부변경·멸실등기 | 부동산등기법 | **1개월** 이내 |
| 소유권보존·이전등기 | 부동산등기특별조치법 | **60일** 이내 |
| 소유권이전등기 | 실명법 | **3년** 이내 |

### 신청주의의 원칙 (법 제22조)

등기는 **당사자의 신청** 또는 **관공서의 촉탁**에 따라 한다. 예외적으로 등기관의 **직권** 또는 법원의 **명령**에 의한다.

### 등기관의 직권등기 유형

| 유형 | 내용 |
|------|------|
| 보존 | 미등기부동산에 대한 처분제한등기 촉탁 시 소유권보존등기 |
| 변경 | 행정구역 변경에 의한 부동산표시변경·등기명의인 주소변경 |
| 경정 | 등기관의 과오로 인한 착오·유루 발생 시 |
| 말소 | 법 제29조 제1호·제2호 간과 등기, 가등기에 기한 본등기 시 중간처분등기 등 |
