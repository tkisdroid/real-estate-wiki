# 부동산 실권리자명의 등기에 관한 법률 (부동산실명법)

## 목적

부동산에 관한 소유권과 그 밖의 물권을 실체적 권리관계와 일치하도록 **실권리자 명의로 등기**하게 함으로써 부동산등기제도를 악용한 투기·탈세·탈법행위 등 반사회적 행위를 방지하고, 부동산거래의 정상화와 부동산가격의 안정을 도모

## 적용대상

### 명의신탁약정의 효력 (제4조)

- **명의신탁약정은 무효** (제4조 제1항)
- 명의신탁약정에 따른 등기로 이루어진 부동산 물권변동도 **무효** (제4조 제2항)
- 무효는 **제3자에게 대항하지 못함** (제4조 제3항)
  - '제3자': 명의신탁약정의 당사자 및 포괄승계인 이외의 자로서, 명의수탁자가 물권자임을 기초로 직접 새로운 이해관계를 맺은 사람
  - 소유권 취득자뿐 아니라 **압류·가압류채권자**도 포함
  - **선의·악의를 불문**

### 불법원인급여 해당 여부
- 명의신탁약정 자체가 사회질서에 위반하는 것은 아니므로 → 타인 명의 등기 자체만으로 **불법원인급여에 해당한다고 볼 수 없다** (대판 2003다41722)

## 유효인 명의신탁 (제8조 특례)

### 적용 제외 대상
다음의 경우로서 **조세 포탈·강제집행 면탈·법령상 제한 회피를 목적으로 하지 않는** 경우에는 명의신탁이 **유효**:
1. **종중**이 보유한 부동산을 종중 외의 자 명의로 등기한 경우
2. **배우자** 명의로 부동산 물권을 등기한 경우
3. **종교단체** 명의로 산하 조직이 보유한 부동산 물권을 등기한 경우

### 배우자 명의신탁
- **법률상 배우자**만 해당 (사실혼 배우자 제외)
- 사실혼 배우자가 혼인한 경우: 혼인 시점부터 특례 적용 → 명의신탁 **유효로** 됨 (대판 2001마1235)
- 배우자 일방 사망: 유효로 인정된 명의신탁은 사망 후에도 **여전히 유효** (대판 2011다99498)

### 유효인 명의신탁의 법률관계
- **대내적**: 신탁자가 소유자 → 언제든지 신탁해지하고 소유권이전등기 청구 가능 (물권적 청구권, 소멸시효 불적용)
- **대외적**: 수탁자가 소유자 → 제3자 침해 시 수탁자만 물권적 청구권 행사 가능, 신탁자는 대위행사만 가능
- 수탁자의 점유: **타주점유** (대판 77다1079)
- 수탁자는 **유효하게 제3자에게 처분** 가능 (선의·악의 불문, 다만 배임행위에 적극 가담하면 무효)

## 부동산실명법이 적용되지 않는 경우

1. **양도담보·가등기담보** (제2조 제1호 가목)
2. **신탁법·자본시장법에 따른 신탁재산** (제2조 제1호 다목)
3. **상호명의신탁 (구분소유적 공유)** (제2조 제1호 나목)

### 상호명의신탁
- 부동산의 위치·면적을 특정하여 2인 이상이 구분소유하기로 약정하고 공유로 등기하는 경우
- **내부관계**: 각자가 특정부분을 단독소유, 배타적 사용·수익 가능 (대판 93다49871)
- **대외관계**: 필지 전체에 관해 공유자로서의 권리만 주장 가능
- **특정부분의 처분**: 각자 단독으로 처분하고 공유지분등기를 이전할 수 있음
- **해소**: 명의신탁해지를 원인으로 한 **지분이전등기**의 이행을 구하면 됨 (공유물분할청구 불가)

## 무효인 명의신탁의 3유형

### 1. 2자간 등기명의신탁 (이전형)

#### 구조
**명의신탁자(甲)** → 자기 소유 부동산을 **명의수탁자(乙)** 명의로 이전등기

#### 법률관계

| 관계 | 내용 |
|------|------|
| 甲 → 乙 | 명의신탁약정 **무효**, 물권변동 **무효** → 소유권은 甲에게 남아 있음 |
| 甲의 청구 | 소유권에 기한 **등기말소** 또는 **진정명의회복을 원인으로 한 이전등기** 청구 가능 |
| 甲의 불가 | 명의신탁 **해지**를 원인으로 한 이전등기 청구 **불가** (대판 2005다5140) |
| 乙의 처분 | 제3자는 선의·악의 **불문** 유효하게 취득 (제4조 제3항) |
| 甲의 손해배상 | 乙에게 손해배상 청구 **가능** (대판 2008다1132) |

#### 벌칙
- 명의신탁자: **과징금 + 형벌** (제5조 제1항, 제7조 제1항)
- 명의수탁자: **형벌** (제7조 제2항)

### 2. 3자간 등기명의신탁 (중간생략등기형)

#### 구조
**명의신탁자(甲)**이 **매도인(丙)**과 매매계약 체결 후, **명의수탁자(乙)** 명의로 직접 이전등기

#### 법률관계

| 관계 | 내용 |
|------|------|
| 甲↔乙 | 명의신탁약정 **무효**, 乙 앞으로의 물권변동 **무효** → 소유권은 **丙에게** 있음 |
| 甲↔丙 | 매매계약 **유효** → 甲은 丙에게 이전등기 청구 가능 (대판 99다21738) |
| 甲이 점유 중이면 | 甲의 丙에 대한 이전등기청구권은 **소멸시효 불진행** (대판 2013다26647) |
| 丙→乙 | 丙은 소유권에 기한 **등기말소** 또는 **진정명의회복** 이전등기 청구 가능 |
| 甲→乙 | 甲은 소유자가 아니므로 乙에게 직접 이전등기·말소 청구 **불가** |
| 甲의 대위 | 甲은 **丙을 대위**하여 乙에게 무효인 등기의 말소 청구 가능 (대판 2001다61654) |
| 乙→甲 | 乙가 甲 앞으로 이전등기해 주었다면 실체관계에 부합하여 **유효** (대판 2004다6764) |
| 乙의 처분 | 제3자(丁)는 선의·악의 **불문** 유효하게 취득 (대판 2001다61654) |

### 3. 계약명의신탁 (위임형)

#### 구조
**명의신탁자(甲)**이 매수자금을 제공하고, **명의수탁자(乙)**이 매수인이 되어 **매도인(丙)**과 매매계약 체결 후 乙 명의로 등기

#### 매도인(丙)이 **선의**인 경우

| 관계 | 내용 |
|------|------|
| 甲↔乙 | 명의신탁약정 **무효** |
| 丙↔甲 | 아무런 법률관계 없음 |
| 丙↔乙 | 매매계약 **유효** → **乙이 유효하게 소유권 취득** (대판 98다4347) |
| 甲→乙 | 甲은 물권적 청구권·불법행위 손해배상 **불가** |
| 甲의 부당이득 | 乙에게 **매수자금** 상당의 부당이득반환 청구 가능 (대판 2008다34828) |
| 취득비용 | 甲이 乙에게 지급한 취득세·등록세 등도 **부당이득**으로 반환의무 (대판 2007다90432) |
| 유치권 | 甲의 부당이득반환청구권으로 **유치권 행사 불가** (대판 2008다34828) |

#### 매도인(丙)이 **악의**인 경우

| 관계 | 내용 |
|------|------|
| 丙↔乙 | 매매계약 **무효** → 물권변동 **무효** → 소유권은 **丙에게** 귀속 |
| 丙→乙 | 이전등기 말소 청구 가능 |
| 乙→丙 | 매매대금 반환 청구 가능 |
| 甲→乙 | 이전등기 청구 **불가** |
| 甲→丙 | 아무런 법률관계 없음 → 이전등기 청구 **불가** |

- 매도인 악의 + 법률효과를 甲에게 귀속시킬 의도 인정 시 → **3자간 등기명의신탁**으로 봄 (대판 2010다52799)

#### 경매에서의 계약명의신탁
- 타인 명의로 매각허가결정을 받은 경우 → **매도인 선의인 계약명의신탁**과 동일
- 매수인은 명의인(乙)이므로 乙이 소유권 취득 (대판 2012다69197)
- 경매 소유자(丙)가 명의신탁 사실을 알더라도 乙이 소유권 취득

## 명의신탁 3유형 비교표

| 구분 | 2자간 (이전형) | 3자간 (중간생략) | 계약명의신탁 (선의) |
|------|--------------|-----------------|-------------------|
| **매매계약** | 없음 (자기 부동산) | 甲↔丙 유효 | **乙↔丙 유효** |
| **소유권 귀속** | **甲** (물권변동 무효) | **丙** (물권변동 무효) | **乙** (유효 취득) |
| **甲의 물권적 청구** | **가능** (말소·진정명의회복) | **불가** (대위만 가능) | **불가** |
| **甲의 부당이득** | 해당 없음 (소유권 보유) | 해당 없음 | **매수자금** 반환 청구 |
| **乙의 처분 → 제3자** | 유효 (선의·악의 불문) | 유효 (선의·악의 불문) | 유효 (완전한 소유자) |

## 벌칙

### 과징금 (제5조)
- **명의신탁자**에게 부과
- 부동산가액의 **30%** 범위 내
- 의무등기기간 경과 후 부과

### 이행강제금 (제6조)
- 과징금 부과 후 **1년** 이내에 실명등기 등의 조치를 하지 않은 경우
- 부동산가액의 **10%** → **20%** 범위에서 부과
- **매년 1회** 부과 가능

### 형사처벌 (제7조)
- **명의신탁자**: 5년 이하의 징역 또는 2억원 이하의 벌금 (제7조 제1항)
- **명의수탁자**: 3년 이하의 징역 또는 1억원 이하의 벌금 (제7조 제2항)

## 핵심 암기 포인트

- **명의신탁약정**: 원칙적 **무효**, 물권변동도 **무효**
- **무효의 제3자 대항 불가**: 선의·악의 **불문**, 압류·가압류채권자도 포함
- **유효인 명의신탁**: 종중·배우자·종교단체 (조세포탈 등 목적이 아닌 경우)
- **2자간**: 소유권 **甲**에게 → 甲이 말소·진정명의회복 청구
- **3자간**: 소유권 **丙**에게 → 甲은 丙에게 매매 이전등기 청구, 乙에게는 대위 말소만
- **계약(선의)**: 소유권 **乙**이 취득 → 甲은 **매수자금** 부당이득반환만 청구
- **계약(악의)**: 매매 무효 → 소유권 **丙**에게 → 甲은 乙·丙 모두에게 청구 불가
- **과징금**: 부동산가액의 **30%**
- **이행강제금**: 10%→20%, 매년 1회
- **형벌**: 신탁자 5년/2억, 수탁자 3년/1억

## OX 퀴즈

**Q1.** 명의신탁약정의 무효는 선의의 제3자에게만 대항하지 못한다. → **X** (제3자의 선의·악의를 불문한다, 제4조 제3항)

**Q2.** 배우자 명의신탁은 항상 유효하다. → **X** (조세 포탈·강제집행 면탈·법령상 제한의 회피를 목적으로 하지 않는 경우에만 유효, 제8조)

**Q3.** 2자간 등기명의신탁에서 명의신탁자는 명의신탁 해지를 원인으로 한 이전등기를 청구할 수 있다. → **X** (명의신탁약정이 무효이므로 해지를 원인으로 한 이전등기 청구는 불가, 소유권에 기한 말소 또는 진정명의회복을 원인으로 한 이전등기를 청구해야 한다, 대판 2005다5140)

**Q4.** 3자간 등기명의신탁에서 명의신탁자와 매도인 사이의 매매계약은 무효이다. → **X** (매매계약은 유효하다)

**Q5.** 계약명의신탁에서 매도인이 선의인 경우 명의수탁자가 유효하게 소유권을 취득한다. → **O** (대판 98다4347)

**Q6.** 계약명의신탁에서 명의신탁자는 명의수탁자에게 부동산 자체의 반환을 부당이득으로 청구할 수 있다. → **X** (부동산실명법 시행 이후에는 매수자금의 반환만 청구할 수 있다, 대판 2008다34828)

**Q7.** 명의신탁자에게는 과징금과 형벌이 모두 부과될 수 있다. → **O** (제5조, 제7조)

**Q8.** 명의수탁자에게는 과징금이 부과된다. → **X** (과징금은 명의신탁자에게 부과되고, 명의수탁자에게는 형벌만 부과된다)

**Q9.** 양도담보로 인한 등기명의 이전은 명의신탁약정에 해당한다. → **X** (양도담보는 명의신탁약정에 해당하지 않는다, 제2조 제1호 가목)

**Q10.** 상호명의신탁(구분소유적 공유)은 부동산실명법 위반으로 무효이다. → **X** (상호명의신탁은 부동산실명법이 적용되지 않아 유효하다, 제2조 제1호 나목)

## 함정 노트

⚠️ "명의신탁 무효 = 제3자에게도 무효" → **제3자에게 대항하지 못함** (선의·악의 불문)

⚠️ "2자간에서 해지 원인 이전등기 가능" → **불가** (말소 또는 진정명의회복만 가능)

⚠️ "3자간에서 甲이 乙에게 직접 청구" → **불가** (丙을 대위해서만 말소 청구 가능)

⚠️ "계약명의신탁(선의)에서 甲이 소유권 주장" → **불가** (乙이 완전한 소유자, 甲은 매수자금 부당이득만)

⚠️ "계약명의신탁 = 매매 무효" → 매도인 **선의**면 매매 유효, **악의**면 매매 무효

⚠️ "과징금 = 수탁자에게" → 과징금은 **신탁자**에게, 수탁자에게는 **형벌만**

⚠️ "배우자 = 사실혼 포함" → **법률상 배우자**만 해당 (사실혼 제외)
