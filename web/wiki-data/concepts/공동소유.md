# 공동소유

하나의 물건을 수인이 공동으로 소유하는 형태. **공유, 합유, 총유**의 3가지 유형이 있다.

## 공유·합유·총유 비교표

| 구분 | [[공동소유_공유\|공유]] | [[공동소유_합유\|합유]] | [[공동소유_총유\|총유]] |
|------|------|------|------|
| **주체** | 일반 수인 | 조합체 | 법인 아닌 사단 |
| **지분** | 독립적 지분 | 지분 있으나 처분 제한 | **지분 없음** |
| **지분처분** | 자유 | 전원 동의 필요 | 불가 |
| **보존행위** | 각자 가능 | 각자 가능 | 각자 가능 |
| **관리행위** | 지분 **과반수** | **전원** 동의 | **사원총회** 결의 |
| **처분·변경** | **전원** 동의 | **전원** 동의 | **사원총회** 결의 |
| **분할** | 청구 가능 | 존속 중 불가 | 청구 불가 |
| **등기** | 각 공유자 명의 + 지분 | 전원 명의 + 합유 취지 | 사단 명의 |

## 하위 페이지

- [[공동소유_공유]] — 지분처분 자유, 보존행위(단독), 관리(과반수), 변경(전원), 분할청구
- [[공동소유_합유]] — 조합체 재산, 지분처분 불가, 분할청구 불가(존속 중), 탈퇴 시 반환
- [[공동소유_총유]] — 비법인사단, 처분·변경(총회 결의), 보존(각자), 구성원 지분 없음

## 핵심 비교 포인트

| 비교 항목 | 공유 | 합유 | 총유 |
|-----------|------|------|------|
| 지분 포기 | 다른 공유자에게 귀속 | 가능(등기 필요) | 불가(지분 없음) |
| 탈퇴 시 | - | 지분 반환 | 반환 불가 |
| 대표 예시 | 부동산 공동매수 | 조합 재산 | 종중·교회 재산 |
