# 재산세

재산세는 시·군·구 내에 소재하는 **토지, 건축물, 주택, 선박, 항공기**에 대하여 과세하는 지방세(보유세)이다. 매년 **2~3문항** 출제된다.

## 1. 특징

- **보유세**: 과세기준일(매년 **6월 1일**) 현재 납세의무 성립
- **시·군·구세** (보통세)
- **물세 + 인세 성격** (토지의 소유자별 합산과세 부분은 인세)
- **보통징수**: 과세관청이 부과·징수 → **가산세 없음** (납부지연가산세만 적용)
- **물납 및 분납** 허용
- **현황부과 원칙**: 공부상 등재 현황과 사실상 현황이 다르면 **사실상 현황**에 따라 과세
- **소액징수 면제** 제도 및 **과세표준 상한** 제도 운용

## 2. 과세대상

| 과세대상 | 내용 |
|----------|------|
| **토지** | 지적공부 등록대상 토지 + 사실상 토지 (주택 부속토지 제외) |
| **건축물** | 건축법상 건축물 + 시설물 (주택용 건축물 제외) |
| **주택** | 주거용 건축물 + 부속토지 (**통합과세**) |
| **선박** | 기선, 범선, 부선 등 모든 배 |
| **항공기** | 비행기, 비행선, 활공기, 회전익 항공기 등 |

### 겸용주택

- 1동의 건물: 주거용 부분만 주택으로 보고, 부속토지는 면적비율로 안분
- 1구의 건축물: 주거용 면적이 전체의 **50% 이상**이면 **전체를 주택**으로 봄
- 다가구주택: 1가구가 독립 거주 가능한 부분을 각각 **1구의 주택**으로 봄

## 3. 토지분 과세대상 구분

### 분리과세대상 토지 (차등비례세율)

| 분리과세 대상 | 세율 |
|--------------|------|
| **농지** (전·답·과수원), **목장용지**, **임야** | **0.07%** (저율) |
| **공장용지**, 주택용·산업용 공급목적 토지 | **0.2%** |
| **골프장**, **고급오락장**용 토지 | **4%** (고율) |

**농지 분리과세 조건** (개인 소유):
- 과세기준일 현재 **실제 영농에 사용**
- 도시지역 내: **개발제한구역과 녹지지역**에 한정
- 주거·상업·공업지역 내 농지 → **종합합산과세**

**목장용지**: 기준면적 이내 분리과세, 초과분 종합합산과세

### 종합합산과세대상 토지 (초과누진세율)

분리과세·별도합산과세 **이외**의 모든 토지:
- 나대지, 잡종지
- 영농에 사용되지 않는 농지 (도시지역 주거·상업·공업지역 내 농지)
- 기준면적 초과 목장용지
- [[별장]]의 부속토지 등

### 별도합산과세대상 토지 (초과누진세율)

**일반건축물의 부속토지** 등 사업용으로 사용되는 토지:
- 사무실·점포·공장 등의 부속토지 (기준면적 이내)
- 기준면적 초과분 → 종합합산과세

## 4. 납세의무자

**원칙**: 과세기준일(**6월 1일**) 현재 **사실상 소유자**
- 신탁재산의 경우 **위탁자**가 소유한 것으로 봄

**예외** (의제납세의무자):

| 경우 | 납세의무자 |
|------|-----------|
| 사실상 소유자 알 수 없을 때 | **공부상 소유자** |
| 상속등기 미이행 | **주된 상속자** (상속지분 최고, 동순위 시 연장자) |
| 국가 등으로부터 연부매매 + 무상사용 | **매수계약자** |
| 환지방식 도시개발사업 체비지·보류지 | **사업시행자** |
| 소유권 귀속 불명 | **사용자** |

## 5. 과세표준

**시가표준액 × 공정시장가액비율**

| 과세대상 | 공정시장가액비율 |
|----------|-----------------|
| **토지·건축물** | **70%** |
| **주택** (일반) | **60%** |
| **주택** (1세대1주택, 시가표준액별) | 3억 이하: **43%** / 3~6억: **44%** / 6억 초과: **45%** |
| **선박·항공기** | 시가표준액 그대로 |

- 주택의 경우 **과세표준 상한제** 적용 (2024년~)

## 6. 세율

### 토지

**종합합산과세대상** (3단계 초과누진):

| 과세표준 | 세율 |
|----------|------|
| 5,000만원 이하 | **0.2%** |
| 5,000만원 초과 ~ 1억원 이하 | 10만원 + 초과액의 **0.3%** |
| 1억원 초과 | 25만원 + 초과액의 **0.5%** |

**별도합산과세대상** (3단계 초과누진):

| 과세표준 | 세율 |
|----------|------|
| 2억원 이하 | **0.2%** |
| 2억원 초과 ~ 10억원 이하 | 40만원 + 초과액의 **0.3%** |
| 10억원 초과 | 280만원 + 초과액의 **0.4%** |

**분리과세대상** (차등비례):
- 농지·목장용지·임야: **0.07%**
- 공장용지 등: **0.2%**
- 골프장·고급오락장용: **4%**

### 건축물

| 건축물 구분 | 세율 |
|-------------|------|
| 일반 건축물 | **0.25%** |
| 골프장·고급오락장용 | **4%** |
| 주거지역·상업지역 내 공장용 건축물 | **0.5%** |

### 주택 (4단계 초과누진)

| 과세표준 | 일반 세율 | 1세대1주택 특례세율 |
|----------|-----------|---------------------|
| 6,000만원 이하 | **0.1%** | **0.05%** |
| 6,000만원 ~ 1.5억원 | 6만원 + 초과액의 **0.15%** | 3만원 + 초과액의 **0.1%** |
| 1.5억원 ~ 3억원 | 19.5만원 + 초과액의 **0.25%** | 12만원 + 초과액의 **0.2%** |
| 3억원 초과 | 57만원 + 초과액의 **0.4%** | 42만원 + 초과액의 **0.35%** |

> **1세대1주택 특례**: 시가표준액 **9억원 이하** 주택에 한정, 유효기간 6년

### 선박·항공기

| 구분 | 세율 |
|------|------|
| **일반선박** | **0.3%** (1,000분의 3) |
| **고급선박** | **5%** (1,000분의 50) |
| **항공기** | **0.3%** (1,000분의 3) |

## 7. 재산세 도시지역분

### 과세대상
국토계획법에 따른 **도시지역** 중 해당 지방의회의 의결을 거쳐 고시한 지역 안에 있는 토지·건축물·주택

| 과세대상 | 내용 |
|----------|------|
| **토지** | 재산세 과세대상 토지 중 **전·답·과수원·목장용지·임야를 제외**한 토지 |
| **건축물** | 재산세 과세대상 건축물 |
| **주택** | 재산세 과세대상 주택. 단, 개발제한구역에서는 **고급주택만** 해당 |

> 공공시설용지 또는 개발제한구역으로 지정된 토지 중 지상건축물·골프장·유원지 등이 **없는** 토지는 **과세 제외**

### 세율
- **기본 재산세 세율** + 과세표준 × **1,000분의 1.4** (0.14%)
- 조례에 의해 **1,000분의 2.3** (0.23%) 이내에서 다르게 정할 수 있음
- **지방교육세** 부과 대상에서 도시지역분 재산세는 **제외**

## 8. 재산세 특례

### 1세대1주택 특례세율
시가표준액 **9억원 이하** 주택에 한정, **유효기간 6년** (2026년까지)

### 과세표준 상한제 (2024년~)
해당 연도 과세표준이 직전 연도 과세표준의 일정 비율을 초과하면 그 초과분은 과세표준에서 제외

### 세 부담 상한 (150%)
해당 연도 재산세 산출세액이 직전 연도 재산세액의 **150%**를 초과 → 150%를 해당 연도 세액으로 함 (법인 소유 주택 제외, 주택은 과세표준 상한제로 대체)

### 과밀억제권역 내 공장 중과세
과밀억제권역(산업단지·공업지역 제외) 내 공장 신설·증설 시 건축물에 대한 재산세를 최초 과세기준일부터 **5년간** 표준세율의 **5배** (1,000분의 12.5 = **1.25%**) 적용. 도시형 업종은 제외

## 9. 부과·징수

### 납기

| 과세대상 | 납기 |
|----------|------|
| **건축물** | 매년 **7월 16일 ~ 7월 31일** |
| **토지** | 매년 **9월 16일 ~ 9월 30일** |
| **주택** (1/2) | **7월 16일 ~ 7월 31일** |
| **주택** (나머지 1/2) | **9월 16일 ~ 9월 30일** |

> 주택 부과세액 **20만원 이하**인 경우 7월에 한꺼번에 부과·징수 가능

### 세 부담의 상한

해당 연도 재산세 산출세액이 직전 연도 재산세액의 **150%**를 초과하는 경우 → 150%를 해당 연도 세액으로 함 (단, **법인 소유 주택 제외**, 주택은 과세표준 상한제로 대체)

### 분할납부

- 납부세액 **250만원 초과** 시 분납 가능
- 분납 한도: 500만원 이하 → 250만원 초과분 / 500만원 초과 → 세액의 **50% 이하**
- 분납기한: 납부기한 경과 후 **3개월 이내**
- 납부기한 **10일 전**까지 신청

### 물납

- 납부세액 **1,000만원 초과** 시 물납 가능
- 물납대상: 해당 지방자치단체 관할구역 내 **부동산**
- 물납 신청: 납부기한 **10일 전**까지
- 물납부동산 평가: 과세기준일 현재의 **시가**
- 지방교육세·지역자원시설세는 물납 **불가**

### 1세대1주택자 납부유예

**요건**:
- 1세대1주택 소유자 (시가표준액 9억원 초과 포함)
- 만 **60세 이상** 또는 **5년 이상** 보유
- 총급여 **7,000만원 이하** 또는 종합소득금액 **6,000만원 이하**
- 납부세액 **100만원 초과**
- 체납 없을 것

납부유예 신청: 납부기한 만료 **3일 전**까지

## 8. 부가세

| 부가세 | 세율 | 비고 |
|--------|------|------|
| **지방교육세** | 재산세액의 **20%** | 도시지역분 재산세 제외 |
| **지역자원시설세** (소방분) | 병기 고지 가능 | |

## 9. 비과세

1. **국가·지방자치단체** 등 소유 재산 (유료 사용·유상이전 약정 제외)
2. 국가 등이 **1년 이상** 공용·공공용으로 사용하는 재산
3. **도로, 하천, 제방, 구거, 유지, 묘지**
4. 산림보호구역, 군사시설 보호구역 내 토지 (전·답·과수원·대지 제외)
5. **임시건축물** (과세기준일 현재 **1년 미만**)
6. 철거명령을 받은 건축물 (해당 연도 철거계획 확정)

## 핵심 암기 포인트

- **과세기준일**: 매년 **6월 1일**
- **납기**: 건축물 7/16~7/31, 토지 9/16~9/30, 주택 7월(1/2)+9월(1/2). 주택 20만원 이하 시 7월 일괄
- **공정시장가액비율**: 토지·건축물 70%, 주택 일반 60%, 1세대1주택(3억 이하 43%, 3~6억 44%, 6억 초과 45%)
- **토지 세율**: 종합합산 0.2~0.5%, 별도합산 0.2~0.4%, 분리(농지 등 0.07%, 공장 0.2%, 골프장 4%)
- **건축물 세율**: 일반 0.25%, 골프장·고급오락장 4%, 주거·상업지역 공장 0.5%
- **주택 세율**: 일반 0.1~0.4%, 1세대1주택 특례 0.05~0.35% (시가표준액 9억 이하 한정)
- **분할납부**: 250만원 초과 시. 500만원 이하 → 250만원 초과분, 500만원 초과 → 50% 이하. 납부기한 후 3개월 이내
- **물납**: 1,000만원 초과 시. 관할구역 내 부동산. 시가 평가. 납부기한 10일 전까지 신청
- **세 부담 상한**: 직전 연도의 150%
- **신탁재산**: 위탁자가 소유한 것으로 봄
- **납부유예**: 1세대1주택, 만 60세 이상 또는 5년 이상 보유, 총급여 7,000만원(종합소득 6,000만원) 이하, 100만원 초과
- **비과세 토지 6종**: 도로, 하천, 제방, 구거, 유지, 묘지
- **도시지역분**: 기본세율 + 과세표준 x **0.14%** (1,000분의 1.4). 도시지역 중 의회 의결·고시 지역. 전·답·과수원·목장용지·임야 **제외**. 지방교육세 부과대상에서 **제외**
- **과밀억제권역 공장 중과**: 5년간 표준세율의 5배 (**1.25%**). 도시형 업종 제외
- **선박·항공기 세율**: 일반선박·항공기 0.3%, 고급선박 5%

## 함정 노트

- ⚠️ **납기 혼동**: 건축물+주택1/2 = 7월, 토지+주택1/2 = 9월 — 건축물과 토지의 납기를 바꿔 출제하는 함정
- ⚠️ **분할납부 vs 물납 금액기준**: 분납 250만원 초과, 물납 1,000만원 초과 — 금액 혼동 주의
- ⚠️ **공정시장가액비율**: 토지·건축물 70% vs 주택 60% vs 1세대1주택 43~45% — 과세대상별 다른 비율
- ⚠️ **분리과세 세율**: 농지 0.07%, 공장 0.2%, 골프장 4% — 같은 분리과세인데 세율이 크게 다름
- ⚠️ **재산세는 가산세 없음**: 보통징수이므로 신고불성실가산세 없음. 납부지연가산세만 적용
