# 도시계획체계

[[국토계획법]]에 따른 국토계획의 위계 구조. 시험에서 각 계획의 성격·수립권자·절차를 비교 출제.

## 계획 체계도

```
① 국토종합계획 (국토 전체, 최상위)
    ↓
② 광역도시계획 (광역계획권, 장기발전방향)
    ↓
③ 도시·군기본계획 (시·군, 종합계획)
    ↓
④ 도시·군관리계획 (시·군, 구속적 행정계획)
```

## 비교표

| 구분 | 광역도시계획 | 도시·군기본계획 | 도시·군관리계획 |
|------|-----------|-------------|-------------|
| **성격** | 비구속적 | 비구속적 | **구속적** |
| **행정쟁송** | 불가 | 불가 | **가능** |
| **수립기준** | 국토교통부장관 정함 | 국토교통부장관 정함 | 광역도시계획·도시군기본계획에 **부합** |
| **정비주기** | - | **5년마다** | **5년마다** |
| **효력발생** | - | - | **지형도면 고시일** |

## 광역도시계획

### 수립권자
| 상황 | 수립권자 |
|------|---------|
| 같은 도 | 관할 **시장·군수** 공동 |
| 둘 이상 시·도 | 관할 **시·도지사** 공동 |
| 3년 내 미신청 (시·군) | **도지사** |
| 3년 내 미신청 (시·도) | **국토교통부장관** |

### 승인권자
- 시·도지사 수립 → **국토교통부장관** 승인
- 시장·군수 수립 → **도지사** 승인

### 절차
기초조사 → 공청회 → 의견청취(30일 이내 제시) → 협의(30일 이내 제시) → 심의 → 승인 → 공고·열람(30일 이상)

## 도시·군기본계획

### 수립권자
- 시장·군수 등 (예외규정 **없음** - 국토교통부장관·도지사 수립 불가)

### 수립 생략
- 수도권 밖 + 광역시 비접경 + 인구 **10만명 이하** 시·군
- 광역도시계획에 기본계획 내용 **모두** 포함된 시·군

### 승인
- 시장·군수 → **도지사** 승인

### 내용
- 공간구조·인구배분, 토지이용·개발, 환경보전, 기반시설, 공원녹지, 경관, 기후변화·에너지, 방재·방범 등

## 도시·군관리계획

### 입안권자
- **원칙**: 시장·군수 등
- 국토교통부장관: 국가계획 관련, 둘 이상 시·도 걸치는 경우
- 도지사: 둘 이상 시·군 걸치는 경우

### 입안 제안 (주민)
| 제안대상 | 동의요건 |
|---------|---------|
| 기반시설·입체복합구역 | **4/5** 이상 |
| 지구단위계획구역·산업유통개발진흥지구 | **2/3** 이상 |

- 처리결과 통보: **45일** 이내 (1회 30일 연장 가능)

### 결정권자
- **원칙**: 시·도지사 (시장·군수 신청에 따라)
- 시장·군수 직접 결정: 지구단위계획구역 지정·변경, 지구단위계획 수립·변경
- 국토교통부장관: 국토교통부장관 입안, 개발제한구역, 시가화조정구역(국가계획 연계)
- 해양수산부장관: 수산자원보호구역

## 시험 핵심 포인트
- 도시·군기본계획 수립권자에는 **예외가 없음** (국토교통부장관·도지사 수립 불가)
- 도시·군관리계획의 효력: **지형도면 고시한 날**부터
- 도시·군관리계획 입안 시 주민의견 청취 **생략**: 국방상·국가안전보장상 기밀, 경미한 사항

## 핵심 암기 포인트

- **광역도시계획**: 비구속적, 행정쟁송 불가
- **도시·군기본계획**: 비구속적, 행정쟁송 불가, **5년마다** 정비
- **도시·군관리계획**: **구속적**, 행정쟁송 가능, **5년마다** 정비, 효력 **지형도면 고시일**
- **도시·군기본계획 수립권자**: 시장·군수 등 (**예외 없음**)
- **도시·군기본계획 수립 생략**: 수도권 밖 + 광역시 비접경 + 인구 **10만명 이하**
- **광역도시계획 미수립 시**: 시·군 3년 내 미신청 → 도지사, 시·도 3년 내 미신청 → 국토교통부장관
- **입안 제안 동의**: 기반시설 **4/5**, 지구단위계획 **2/3**
- **결정권자**: 원칙 시·도지사, 지구단위계획은 시장·군수 직접 결정
- **주민의견 청취 생략**: 국방·안전보장 기밀, 경미한 사항

## OX 퀴즈

**Q1.** 도시·군기본계획은 구속적 행정계획으로 행정쟁송의 대상이 된다. → **X** (비구속적, 행정쟁송 불가)

**Q2.** 도시·군기본계획의 수립권자에는 국토교통부장관도 포함된다. → **X** (시장·군수 등만, 예외 없음)

**Q3.** 도시·군관리계획은 지형도면 고시한 날부터 효력이 발생한다. → **O**

**Q4.** 광역도시계획 수립 시 시장·군수가 3년 내 미신청하면 국토교통부장관이 수립한다. → **X** (도지사가 수립, 국토교통부장관은 시·도지사 3년 내 미신청 시)

**Q5.** 도시·군관리계획의 입안 제안 시 기반시설은 토지소유자 2/3 이상의 동의가 필요하다. → **X** (기반시설은 4/5, 지구단위계획이 2/3)

**Q6.** 도시·군관리계획의 결정권자는 원칙적으로 시장·군수이다. → **X** (원칙 시·도지사)

**Q7.** 인구 10만명 이하의 시·군은 모두 도시·군기본계획 수립을 생략할 수 있다. → **X** (수도권 밖 + 광역시 비접경 + 10만명 이하, 3가지 요건 모두 충족 시)

**Q8.** 도시·군관리계획 입안 시 주민의견 청취는 항상 필요하다. → **X** (국방·안전보장 기밀, 경미한 사항은 생략)

## 함정 노트

- ⚠️ **비구속적 vs 구속적**: 광역도시계획·도시군기본계획은 **비구속적**, 도시군관리계획만 **구속적**. 계획 성격 혼동 주의
- ⚠️ **수립권자 예외**: 도시·군기본계획만 예외가 **없음**. 광역도시계획·도시군관리계획은 국토교통부장관 등 예외 있음
- ⚠️ **4/5 vs 2/3**: 기반시설 입안 제안 **4/5**, 지구단위계획 입안 제안 **2/3**. 비율 교차 출제 주의
- ⚠️ **효력발생**: "결정 고시일"이 아니라 **"지형도면 고시한 날"**. 시점 혼동 빈출
- ⚠️ **3년 미신청**: 시·군이면 **도지사**, 시·도이면 **국토교통부장관**이 수립. 주체 바꿔 출제
- ⚠️ **수립 생략 3가지 요건**: 수도권 밖 + 광역시 비접경 + 10만명 이하. 하나라도 빠지면 생략 불가
