# 저당대출 상환방식

> 📖 상위 문서: [[부동산금융론]]

## 1. 원리금균등상환 (고정상환저당)

- 매기 **동일한 금액**(원금+이자)을 상환
- **초기**: 이자 비중 **크고**, 원금 비중 **작음**
- **후기**: 이자 비중 **작고**, 원금 비중 **큼**
- **저당상수** 활용: 원리금상환액 = 대출잔금 x 저당상수

### 저당상수(연부상환율)

$$저당상수 = \frac{1}{연금의\ 현재가치계수}$$

- 저당상수와 연금의 현재가치계수는 **역수관계**
- 원리금균등상환방식에서 매기 상환액 계산 시 사용
- 원리금상환액 = 대출원금(현재가치) x 저당상수

> **주의**: 원금균등상환방식에서는 저당상수를 사용하지 않는다!

## 2. 원금균등상환 (체감식상환)

- 매기 **동일한 원금** + 잔금 기준 이자 상환
- 상환액이 점차 **체감** (시간이 지날수록 잔금 감소 → 이자 감소)
- **초기 상환부담 가장 큼** (원금 고정 + 이자 최대)
- **총이자부담**: 원리금균등상환보다 **적음**

## 3. 체증식상환 (점증식상환)

- 상환액이 점차 **증가**
- **초기 상환부담 가장 적음** → 소득증가 기대 **젊은 층**에 적합
- **총이자부담**: 세 가지 중 **가장 큼** (초기에 원금 상환이 적어 이자가 오래 쌓임)

## 4. 3대 상환방식 비교

| 구분 | 원리금균등 | 원금균등 | 체증식 |
|------|-----------|---------|--------|
| 매기 상환액 | **일정** | **체감** | **체증** |
| 초기 부담 | 중간 | **가장 큼** | **가장 작음** |
| 총이자부담 | 중간 | **가장 작음** | **가장 큼** |
| 적합 대상 | 일반적 | 상환능력 양호 | 젊은 층 (소득증가 기대) |
| 저당상수 사용 | **O** | **X** | **X** |

## 5. 원리금균등 vs 원금균등 심층 비교

| 비교 항목 | 원리금균등 | 원금균등 |
|----------|-----------|---------|
| 초기 원리금상환액 | **적음** | **큼** |
| 초기 소득 낮은 차입자 | **유리** | 불리 |
| 대출자 원금회수위험 | **높음** | **낮음** |
| 조기상환 시 저당잔금 | **높음** | 낮음 |
| 누적원리금상환액 | **큼** | 작음 |
| 대출자 총 이자수입 | 중간 | **가장 작음** |

**대출채권 평균회수기간(duration) 순서** (짧은 순):
원금균등 < 원리금균등 < 체증식 < **만기일시상환**

- 거치기간 설정 시 → **이자지급총액 증가** → 원리금지급총액도 **증가**
- 체증식 상환의 **부(-)의 상환**: 초기 원리금 납입액이 이자지급분에도 미치지 못해 미상환이자가 **원금에 가산**되는 현상
- 체증식은 예상보유기간이 **단기**인 차입자에게도 적합

## 6. 기타 상환 유형

| 유형 | 내용 |
|------|------|
| **만기일시상환** | 보유기간 중 **이자만**, 만기에 원금 **일시 상환**. 대출기관 이자수입 **가장 큼** |
| **부분분할상환(풍선형)** | 기간 중 일부 원금상환 + 만기에 **잔액 일시상환** |
| **무이자저당** | 대출 시 저당액수를 **할인**해서 적게 대출. 이자 미상환, **원금만 상환** |

## 7. 잔금비율과 상환비율

$$잔금비율 + 상환비율 = 1$$

- 잔금비율 = 미상환된 원금 / 대출원금 = **연금의 현가계수(잔여기간) / 연금의 현가계수(저당기간)**
- 상환비율 = 상환된 원금 / 대출원금

## 8. 대출비율 규제 (LTV·DTI·DSR)

| 규제 | 산식 | 의미 | 엄격도 |
|------|------|------|--------|
| **LTV** | 대출금 / 담보가치 | 담보가치 대비 대출 한도 | - |
| **DTI** | 해당 대출 연간 원리금상환액 / 연소득 | 소득 대비 해당 대출 상환능력 | 중간 |
| **DSR** | **모든 대출** 연간 원리금상환액 / 연소득 | 소득 대비 **전체** 상환능력 | **가장 엄격** |

- DTI는 해당 주택담보대출만 고려, DSR은 **모든 대출** 합산
- DSR이 DTI보다 **엄격**한 규제

## 핵심 암기 포인트

- 원리금균등: 매기 **일정**, 초기 이자↑원금↓, 후기 이자↓원금↑, **저당상수** 활용
- 원금균등: **체감**, 초기부담 **가장 큼**, 총이자 **가장 적음**
- 체증식: **체증**, 초기부담 **가장 작음**, 총이자 **가장 큼**, **젊은 층** 적합
- 저당상수 = **1/연금의 현재가치계수** (역수관계)
- 잔금비율 + 상환비율 = **1**
- LTV·DTI·DSR 중 **DSR이 가장 엄격** (모든 대출 합산)

## OX 퀴즈

**Q1.** 원리금균등상환에서 초기에는 원금 비중이 크고 이자 비중이 작다. → **X** (초기에는 **이자** 비중 크고, **원금** 비중 작음)

**Q2.** 원금균등상환방식의 저당대출 매기 상환액은 저당상수로 계산한다. → **X** (저당상수는 **원리금균등상환**에서만 사용)

**Q3.** 체증식상환은 초기 상환부담이 가장 크다. → **X** (초기 부담 **가장 작음**, 가장 큰 것은 **원금균등**)

**Q4.** 세 가지 상환방식 중 총이자부담이 가장 적은 것은 원금균등상환이다. → **O**

**Q5.** 저당상수와 연금의 미래가치계수는 역수관계이다. → **X** (저당상수와 **연금의 현재가치계수**가 역수관계)

**Q6.** 잔금비율과 상환비율의 합은 0이다. → **X** (**1**이다)

**Q7.** DSR은 해당 주택담보대출의 원리금상환액만 고려한다. → **X** (**모든 대출** 합산. 해당 대출만은 **DTI**)

## 함정 노트

- ⚠️ "원리금균등: 초기 원금 비중 큼" → 초기에는 **이자 비중**이 큼 (후기에 원금 비중 증가)
- ⚠️ "원금균등에 저당상수 사용" → 저당상수는 **원리금균등**에서만 사용
- ⚠️ "체증식: 총이자 가장 적음" → 총이자 **가장 큼** (가장 적은 것은 원금균등)
- ⚠️ "저당상수 역수 = 연금의 미래가치계수" → 역수 = **연금의 현재가치계수**
- ⚠️ "잔금비율 + 상환비율 = 0" → **1** (제29회 출제 함정)
- ⚠️ "DTI가 DSR보다 엄격" → **DSR**이 DTI보다 엄격 (모든 대출 합산)
