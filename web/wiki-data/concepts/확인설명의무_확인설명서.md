# 확인·설명서 작성·교부·보관 (법 제25조 제3항)

## 1. 확인·설명서 작성

### 작성 시기
- 중개가 완성되어 **거래계약서를 작성하는 때**에 작성 (법 제25조 제3항)

### 작성 주체

| 주체 | 작성 가능 여부 | 비고 |
|---|---|---|
| **개업공인중개사** | **의무** | 법인: 대표자, 분사무소: 책임자 |
| **소속공인중개사** | 작성 가능 (의무 아님) | 거래계약서 작성과 동일 |
| **중개보조원** | **불가** | 작성 시 **대여행위로 처벌** |

### 기재사항
- 확인·설명 자료 항목: 개업공인중개사가 설명 과정에서 제시한 자료 기재
- 대상물건의 상태에 관한 자료요구 사항: 매도·임대 의뢰인에게 요구한 사항 및 자료 제출 여부
- 내·외부 시설물 상태(건축물·벽면·도배), 환경조건(비선호시설 제외)의 확인 자료 요구 및 불응 여부

## 2. 서명·날인

### 서명·날인 의무자

| 주체 | 의무 | 비고 |
|---|---|---|
| **개업공인중개사** | **서명 및 날인** | 법인: **대표자**, 분사무소: **책임자** |
| **소속공인중개사** | 중개행위를 한 경우 **함께 서명 및 날인** | 개업공인중개사와 연명 |
| **중개보조원** | 서명·날인 의무 **없음** | 하여도 되고 하지 않아도 됨 |

### 위반 시 제재
- 개업공인중개사: **업무정지** 처분
- 소속공인중개사: **자격정지** 처분 (3개월)

## 3. 교부 및 보관

### 교부
- 거래당사자 **쌍방**에게 교부 (법 제25조 제3항)
- 확인·설명 상대방은 권리취득 의뢰인이지만, 확인설명서 교부는 **쌍방**

### 보관
- **3년간 보관** (영 제21조 제3항)
- **공인전자문서센터 보관 시 면제** (전자문서 및 전자거래 기본법에 따른 센터)

### 위반 시 제재
- 교부·보관 위반: **업무정지** 처분

## 4. 확인·설명서 서식 종류

| 서식 | 대상 | 분량 |
|---|---|---|
| 별지 제20호 서식 | **주거용 건축물** | 6쪽 |
| 별지 제21호 서식 | **비주거용 건축물** | 4쪽 |
| 별지 제22호 서식 | **토지** | 4쪽 |
| 별지 제23호 서식 | **입목·광업재단·공장재단** | - |

## 5. 확인·설명서 vs 거래계약서 비교

| 구분 | 확인·설명서 | 거래계약서 |
|---|---|---|
| 작성 시기 | 거래계약서 작성 시 | 중개 완성 시 |
| 교부 대상 | 쌍방 | 쌍방 |
| **보관 기간** | **3년** | **5년** |
| 서명·날인 | 개업공인중개사 + 소속공인중개사 | 개업공인중개사 + 거래당사자 |
| 전자문서 보관 면제 | 있음 | 있음 |

## 6. 주요 판례

| 판례 | 핵심 내용 |
|---|---|
| 대판 2002.2.5 | 무상위임의 경우에도 선량한 관리자의 주의의무 면제 안 됨 → 무료 중개도 설명의무 동일 |
| 서울남부지원 2000.2.11 | 선순위 임차인 존재 미설명 후 전세권만 설정 설명 → 손해배상 |
| 수원지법 1997.8.26 | 잔금 지급 전 등기사항증명서 재확인 의무 → 제한물권 설정 여부 고지 의무 |
| 대구지법 1987.10.30 | 확인설명 게을리하여 계약 해제 시 → 중개보수 반환 + 손해배상 책임 |

## 핵심 암기 포인트

1. 확인설명서 작성 시기: **거래계약서 작성 시**
2. 작성 주체: 개업공인중개사(의무), 소속공인중개사(가능), 중개보조원(**불가 - 대여행위**)
3. 서명날인: 개업공인중개사 + 중개행위 한 소속공인중개사 (**연명**)
4. 중개보조원: 서명날인 의무 **없음**
5. 교부: 거래당사자 **쌍방**
6. 보관: **3년** (거래계약서는 5년)
7. 공인전자문서센터 보관 시 **보관의무 면제**
8. 주거용 건축물: **6쪽**, 비주거용/토지: **4쪽**

## OX 퀴즈

**Q1.** 확인·설명서는 권리를 취득하고자 하는 의뢰인에게만 교부하면 된다.
> **X** — 확인·설명서는 거래당사자 **쌍방**에게 교부해야 한다.

**Q2.** 확인·설명서의 보관 기간은 거래계약서와 동일하게 5년이다.
> **X** — 확인·설명서는 **3년**, 거래계약서는 5년이다.

**Q3.** 중개보조원이 확인·설명서를 작성하면 대여행위로 처벌받을 수 있다.
> **O** — 중개보조원의 확인설명서 작성은 **대여행위**로 처벌 대상이다.

**Q4.** 중개보조원은 확인·설명서에 서명·날인할 의무가 있다.
> **X** — 중개보조원에게는 서명·날인 **의무가 없다** (하여도 되고 하지 않아도 됨).

**Q5.** 확인·설명서를 공인전자문서센터에 보관한 경우에도 별도 보관의무가 있다.
> **X** — 공인전자문서센터 보관 시 **보관의무 면제**된다.

**Q6.** 비주거용 건축물의 확인·설명서 서식은 6쪽이다.
> **X** — 비주거용 건축물은 **4쪽**, 주거용 건축물이 **6쪽**이다.

**Q7.** 소속공인중개사가 중개행위를 한 경우 개업공인중개사와 함께 서명·날인해야 한다.
> **O** — 중개행위를 한 소속공인중개사는 개업공인중개사와 **함께 서명·날인**해야 한다.

**Q8.** 확인·설명서 서명·날인을 위반한 개업공인중개사에게는 과태료가 부과된다.
> **X** — 서명·날인 위반 시 개업공인중개사에게는 **업무정지** 처분이 내려진다 (과태료 아님).

## 함정 노트

| 함정 포인트 | 정확한 내용 |
|---|---|
| 설명 상대방 = 교부 대상 | 설명은 **권리취득 의뢰인**, 교부는 **쌍방** |
| 보관 3년 = 계약서 3년 | 확인설명서 **3년**, 거래계약서 **5년** |
| 중개보조원도 작성 가능 | 중개보조원 작성은 **대여행위 처벌** |
| 중개보조원 서명날인 의무 | 중개보조원은 서명날인 **의무 없음** |
| 서명날인 위반 → 과태료 | 서명날인 위반 → **업무정지** (과태료 아님) |
| 주거용 = 4쪽 | 주거용 **6쪽**, 비주거용/토지 **4쪽** |
| 법인 → 아무나 서명 | 법인은 **대표자**, 분사무소는 **책임자**가 서명날인 |
