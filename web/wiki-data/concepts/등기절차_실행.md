# 등기절차 - 실행

## 1. 신청주의의 원칙

등기는 **당사자의 신청** 또는 **관공서의 촉탁**에 따라 하는 것을 원칙으로 한다 (부동산등기법 제22조).

### 등기절차 개시의 유형

| 구분 | 유형 | 예시 |
|------|------|------|
| **신청에 의한 개시** | 당사자의 신청 | 매매에 의한 소유권이전등기 |
| | 관공서의 촉탁 | 법원의 촉탁등기 |
| **법률규정에 의한 개시** | 등기관의 직권 | 행정구역 변경에 따른 표시변경등기 |
| | 법원의 명령 | 이의신청 인용 시 상당한 처분 명령 |

### 등기관의 직권에 의한 등기

| 등기 유형 | 내용 |
|----------|------|
| **보존** | 미등기부동산에 대한 법원의 처분제한등기 촉탁 시 소유권보존등기 |
| **변경** | 행정구역·명칭 변경에 따른 부동산표시변경등기·등기명의인 주소변경등기 |
| **경정** | 등기관의 과오로 인한 착오·유루의 경정 |
| **말소** | 법 제29조 제1호·제2호에 해당하는 것을 간과한 등기의 말소 등 |

## 2. 공동신청의 원칙

### 원칙: 공동신청

등기는 법률에 다른 규정이 없는 경우 **등기권리자와 등기의무자가 공동으로 신청**한다 (법 제23조 제1항).

| 구분 | 정의 |
|------|------|
| **등기의무자** (절차법상) | 등기부상으로 **권리를 잃거나** 기록형식상 불이익을 받는 자 |
| **등기권리자** (절차법상) | 등기부상으로 **권리를 얻거나** 기록형식상 이익을 받는 자 |

> 실체법(민법)상의 등기의무자·등기권리자와 절차법(부동산등기법)상의 등기의무자·등기권리자는 **대부분 일치하지만 반드시 일치하는 것은 아니다**.

### 공동신청의 예시

| 등기 유형 | 등기의무자 | 등기권리자 |
|----------|----------|----------|
| 매매에 의한 소유권이전등기 | 매도인 | 매수인 |
| 지상권 설정등기 | 토지소유자 | 지상권자 |
| 전세권 말소등기 | 전세권자 | 전세권설정자 |
| 전세금 증액 변경등기 | 전세권설정자 | 전세권자 |
| 전세금 감액 변경등기 | 전세권자 | 전세권설정자 |
| 채권최고액 증액 근저당권변경등기 | 근저당권설정자 | 근저당권자 |

## 3. 단독신청 (공동신청의 예외)

### 단독신청이 허용되는 경우

| 유형 | 신청인 | 근거 |
|------|--------|------|
| **소유권보존등기** | 등기명의인으로 될 자 | 법 제23조 제2항 |
| **소유권보존등기의 말소등기** | 등기명의인 | 법 제23조 제2항 |
| **상속·법인합병 등 포괄승계** | 등기권리자 단독 | 법 제23조 제3항 |
| **판결에 의한 등기** | 승소한 등기권리자 또는 등기의무자 | 법 제23조 제4항 |
| **부동산 표시의 변경·경정** | 소유권의 등기명의인 | 법 제23조 제5항 |
| **등기명의인 표시의 변경·경정** | 해당 권리의 등기명의인 | 법 제23조 제6항 |
| **신탁등기** | 수탁자 | 법 제23조 제7항 |

### 제3자에 의한 등기신청 (당사자신청의 예외)

| 유형 | 내용 |
|------|------|
| **포괄승계인에 의한 등기신청** | 등기원인 발생 후 상속 등 포괄승계 시, 포괄승계인이 등기 신청 가능 |
| **대위자에 의한 등기신청** | 채권자가 채무자의 등기신청권을 자기 이름으로 행사 (채무자의 무자력 불요) |
| **대리인에 의한 등기신청** | 임의대리인·법정대리인 모두 가능. 자기계약·쌍방대리 **허용** |

### 판결에 의한 등기의 요건

1. **이행판결**이어야 한다 (확인판결·형성판결 원칙적 불가, 공유물분할판결은 예외)
2. **확정판결**이어야 한다 (가집행선고 붙은 미확정판결로는 불가 → 각하)
3. 판결주문에 **등기의 종류·등기원인과 그 연월일** 등이 명시되어야 한다
4. 확정시기에 관계없이 (10년 경과 후에도) 등기신청 가능

## 4. 등기관의 심사 (형식적 심사주의)

우리나라 부동산등기법은 등기신청 시 등기원인의 **실질적 유효성**을 심사하지 않고, 법 제29조에 기록된 **각하사유에 해당하는지 여부만 확인**하는 **형식적 심사주의**에 의한다.

> 형식적 심사주의의 의미: 등기관은 서면상의 심사만 하며, 실체적 권리관계의 진위는 심사하지 않는다.

## 5. 등기절차 흐름: 접수 → 심사 → 실행 → 완료

### (1) 접수

- 등기신청서를 등기소에 접수하면 접수번호를 부여
- 접수된 부동산에 관하여는 등기관이 등기를 마칠 때까지 등기사항증명서 발급 제한 (다만 처리 중 표시 발급 가능)
- 등기는 **접수번호의 순서**에 따라 처리

### (2) 심사 (각하사유 확인)

- 등기관은 각하사유에 해당하는지 심사
- 각하사유 제1호~제4호: **보정명령 없이 즉시 각하**
- 각하사유 제5호~제11호: **보정명령 가능** (보정을 명한 날의 다음 날까지 보정하면 각하하지 않음)

### (3) 기입

- 신청내용을 등기부에 기록
- 동일 부동산에 관하여 여러 건의 신청 시 **반드시 접수번호 순서대로** 기입

### (4) 교합 (등기관의 전자서명)

- 교합 = 등기관이 전자서명을 함으로써 등기사무 처리한 등기관을 나타내는 조치
- 교합으로 **등기 실행이 완료**
- 교합 후 등기사항증명서 발급 가능
- 교합 후 등기신청 **취하 불가**, 자구정정 **불가**

### (5) 등기필정보 작성·통지 및 등기완료 통지

- **등기필정보**: 새로운 권리자가 기록되는 경우 등기관이 작성한 정보 (일련번호 + 비밀번호)
- 부동산 및 등기명의인별로 작성
- 방문신청: 등기필정보통지서 교부 / 전자신청: 전산정보처리조직으로 송신

## 6. 각하사유 (법 제29조) — 전 11호

| 호수 | 각하사유 | 보정 |
|------|---------|------|
| **1호** | 사건이 그 등기소의 **관할이 아닌** 경우 | 불가 |
| **2호** | 사건이 **등기할 것이 아닌** 경우 | 불가 |
| **3호** | **신청할 권한이 없는 자**가 신청한 경우 | 불가 |
| **4호** | 당사자나 그 대리인이 **출석하지 아니**한 경우 | 불가 |
| **5호** | 신청정보의 제공이 **방식에 맞지 아니**한 경우 | 가능 |
| **6호** | 부동산 또는 권리의 표시가 **등기기록과 불일치** | 가능 |
| **7호** | 등기의무자의 표시가 **등기기록과 불일치** (포괄승계인 제외) | 가능 |
| **8호** | 신청정보와 **등기원인을 증명하는 정보가 불일치** | 가능 |
| **9호** | 등기에 필요한 **첨부정보를 미제공** | 가능 |
| **10호** | **취득세·등록면허세 또는 수수료** 미납 등 부과된 의무 불이행 | 가능 |
| **11호** | 부동산의 표시가 **대장과 불일치** | 가능 |

### "사건이 등기할 것이 아닌 경우" (규칙 제52조)

1. 등기능력 없는 물건·권리에 대한 등기 신청
2. 법령에 근거 없는 특약사항의 등기 신청
3. 전유부분과 대지사용권의 분리처분 금지 위반
4. 농지를 전세권설정 목적으로 하는 등기 신청
5. 저당권을 피담보채권과 분리하여 양도하는 등기 신청
6. 일부지분에 대한 소유권보존등기 신청
7. 공동상속인 중 일부가 자기 상속지분만에 대한 상속등기 신청
8. 관공서·법원의 촉탁으로 실행되어야 할 등기를 신청한 경우
9. 이미 보존등기된 부동산에 대하여 다시 보존등기 신청
10. 기타 법률상 허용될 수 없음이 명백한 등기 신청

## 7. 등기필정보

| 항목 | 내용 |
|------|------|
| **정의** | 새로운 권리자가 기록되는 경우 권리자 확인을 위해 등기관이 작성한 정보 |
| **구성** | 아라비아 숫자 + 부호의 **일련번호**와 **비밀번호** |
| **작성 단위** | 부동산 및 등기명의인별 (접수일·접수번호 동일 시 부동산 달라도 명의인별 작성 가능) |
| **통지 방법** | 방문신청: 통지서 교부 / 전자신청: 전산정보처리조직 송신 |

### 등기필정보를 작성·통지하지 않는 경우

1. 등기권리자가 통지를 **원하지 않는** 경우
2. **국가 또는 지방자치단체**가 등기권리자인 경우
3. 전자신청 수신가능 시부터 **3개월** 이내 미수신
4. 등기필정보통지서를 등기완료 후 **3개월** 이내 미수령
5. 승소한 **등기의무자**가 판결에 의한 등기신청을 한 경우
6. **채권자가 등기권리자를 대위**하여 등기신청을 한 경우
7. 미등기부동산에 대하여 등기관이 **직권으로** 소유권보존등기를 한 경우

## 핵심 암기 포인트

- 등기는 **당사자 신청** 또는 **관공서 촉탁**이 원칙 (신청주의)
- 공동신청이 원칙, 예외적으로 **단독신청** 허용
- 단독신청: 보존등기, 포괄승계(상속·합병), 판결에 의한 등기, 표시변경·경정, 신탁등기
- 등기관의 심사: **형식적 심사주의** (실질적 유효성 심사 안 함)
- 각하사유 1~4호: 보정 **불가**, 5~11호: 보정 **가능**
- 등기절차: **접수 → 심사 → 기입 → 교합** → 등기필정보 작성·통지
- 교합 후: 등기사항증명서 발급 가능, 취하·자구정정 **불가**
- 등기필정보: 일련번호 + 비밀번호, 부동산·등기명의인별 작성

## OX 퀴즈

**Q1.** 등기는 반드시 당사자의 신청에 의해서만 개시된다. → **X** (관공서의 촉탁, 등기관의 직권, 법원의 명령에 의해서도 개시)

**Q2.** 상속에 의한 등기는 등기권리자가 단독으로 신청할 수 있다. → **O** (포괄승계에 따른 단독신청)

**Q3.** 판결에 의한 등기는 확인판결로도 신청할 수 있다. → **X** (이행판결이어야 하며, 확인판결로는 불가)

**Q4.** 등기관은 등기신청에 대하여 실질적 심사를 한다. → **X** (형식적 심사주의)

**Q5.** 각하사유 제1호(관할 위반)에 해당하는 경우 보정명령이 가능하다. → **X** (1~4호는 보정 불가, 즉시 각하)

**Q6.** 교합 이후에는 등기신청을 취하할 수 없다. → **O**

**Q7.** 등기필정보는 모든 등기에 대하여 반드시 작성·통지된다. → **X** (국가·지방자치단체가 등기권리자인 경우 등 예외 있음)

**Q8.** 포괄승계인에 의한 등기신청은 등기의무자의 표시가 등기기록과 일치하지 않아도 각하되지 않는다. → **O** (법 제29조 제7호 단서)

**Q9.** 농지를 전세권설정의 목적으로 하는 등기를 신청하면 각하된다. → **O** ("사건이 등기할 것이 아닌 경우")

**Q10.** 등기신청에 있어서 자기계약과 쌍방대리는 금지된다. → **X** (등기신청행위는 채무의 이행에 준하므로 허용)

## 함정 노트

- ⚠️ **각하사유 보정 가부**: 1~4호 보정 불가 / 5~11호 보정 가능. 보정기한은 보정명령일의 **다음 날**까지
- ⚠️ **판결에 의한 등기의 요건**: 이행판결 + 확정판결이어야 함. 공유물분할판결은 형성판결이지만 예외적으로 허용
- ⚠️ **포괄승계인 등기신청**: 등기의무자의 표시가 등기기록과 불일치해도 **각하되지 않음** (제7호 단서)
- ⚠️ **등기의무자 vs 등기권리자 혼동**: 전세금 증액 변경등기에서 등기의무자는 전세권설정자(불리), 등기권리자는 전세권자(유리)
- ⚠️ **등기필정보 미작성 사유**: 승소한 "등기권리자"가 아니라 승소한 **"등기의무자"**가 단독신청 시 미작성
