# 등기절차 - 공동신청 (법 제23조 제1항)

## 1. 공동신청주의의 원칙

등기는 법률에 다른 규정이 없는 경우에는 **등기권리자**와 **등기의무자**가 **공동으로 신청**한다.

### 취지
- 등기의무자의 **불측의 손해** 방지
- 등기의 **진정성 보장**

## 2. 등기권리자와 등기의무자

### 개념

| 구분 | 정의 |
|------|------|
| **등기의무자** | 등기부상 권리를 **잃거나** 기록형식상 **불이익**을 받는 자 |
| **등기권리자** | 등기부상 권리를 **얻거나** 기록형식상 **이익**을 받는 자 |

### 등기 유형별 구분

| 등기 유형 | 등기의무자 | 등기권리자 |
|-----------|-----------|-----------|
| 매매에 의한 **소유권이전** | 매도인 | 매수인 |
| **저당권 설정** | 저당권설정자(소유자) | 저당권자 |
| **저당권 말소** | 저당권자 | 저당권설정자 |
| **환매특약등기** | 매수인 | 매도인 |
| **전세금 증액** 변경등기 | 전세권설정자 | 전세권자 |
| **전세금 감액** 변경등기 | 전세권자 | 전세권설정자 |
| **채권최고액 증액** 변경등기 | 근저당권설정자 | 근저당권자 |
| **채권최고액 감액** 변경등기 | 근저당권자 | 근저당권설정자 |

> **핵심 원리**: "권리가 늘어나면 등기권리자, 줄어들면 등기의무자"

### 설정·말소에서의 전환

| 단계 | 설정 시 | 말소 시 |
|------|---------|---------|
| **저당권** | 설정자=의무자, 저당권자=권리자 | 저당권자=**의무자**, 설정자=**권리자** |
| **전세권** | 설정자=의무자, 전세권자=권리자 | 전세권자=**의무자**, 설정자=**권리자** |

## 3. 공동신청의 예외 (단독신청이 허용되는 경우)

→ 상세 내용은 [[등기절차_단독신청]] 참조

### 단독신청 사유 요약 (7가지)

1. 소유권보존등기 / 소유권보존등기의 말소
2. 상속·합병·포괄승계에 따른 등기
3. 판결에 의한 등기
4. 부동산표시변경·경정 등기
5. 등기명의인표시변경·경정 등기
6. 신탁등기 (수탁자 단독)
7. 멸실등기

## 4. 제3자에 의한 등기신청 (당사자 신청의 예외)

### 포괄승계인에 의한 등기신청
- 등기의무자·등기권리자의 **상속인·합병법인** 등 포괄승계인이 신청 가능

### 대위자에 의한 등기신청
- **채권자대위권**(민법 제404조)에 기한 등기신청
- 채권자가 채무자를 대위하여 등기 신청 가능

### 대리인에 의한 등기신청
- **자격자대리인**: 변호사, 법무사
- 자격자대리인의 **사무원**이 출석하여 신청 가능 (방문신청)

## 5. 등기신청 방법

| 방문신청 | 전자신청 |
|----------|----------|
| 신청인 또는 대리인이 등기소에 **출석**하여 서면 제출 | 전산정보처리조직을 이용하여 신청정보·첨부정보 **전송** |
| 자격자대리인(변호사·법무사)인 경우 **사무원 출석** 가능 | 방문신청과 **동등한** 지위 |

> 전부개정 부동산등기법에서는 방문신청과 전자신청을 **동등**하게 병렬적으로 규정

## 핵심 암기 포인트

- 등기는 원칙적으로 **등기권리자 + 등기의무자** 공동신청
- **증액 변경등기**: 이익 받는 쪽이 등기권리자 (전세금 증액 → 전세권자가 권리자)
- **감액 변경등기**: 반대로 전환 (전세금 감액 → 전세권설정자가 권리자)
- **설정 vs 말소**: 등기의무자와 등기권리자가 **역전**됨
- 단독신청은 공동신청의 **예외** → 7가지 암기 필수

## OX 퀴즈

**Q1.** 저당권 설정등기에서 저당권자는 등기의무자이다. → **X** (등기권리자)

**Q2.** 저당권 말소등기에서 저당권자는 등기권리자이다. → **X** (등기의무자, 설정 시와 역전)

**Q3.** 전세금 증액 변경등기에서 전세권설정자는 등기의무자이다. → **O**

**Q4.** 매매에 의한 소유권이전등기는 단독신청이 가능하다. → **X** (공동신청)

**Q5.** 전자신청은 방문신청의 보충적 수단이다. → **X** (동등하게 병렬 규정)

**Q6.** 자격자대리인의 사무원은 등기소에 출석할 수 없다. → **X** (출석 가능)

**Q7.** 채권자대위권에 의한 등기신청은 불가능하다. → **X** (대위 신청 가능)

## 함정 노트

- **증액·감액에 따른 권리자·의무자 전환**: 전세금 증액 시 전세권설정자=의무자이나, 감액 시 전세권자=의무자. 반드시 "누가 불이익을 받는가"로 판단
- **환매특약등기**: 매수인이 의무자, 매도인이 권리자. 일반적인 매매 이전등기와 의무자·권리자가 **역전**
- **저당권 이전 후 말소**: 저당권이 양도된 후 말소 시, 등기의무자는 **저당권 양수인** (원래 저당권자가 아님)
- **소유권보존등기의 말소**: 등기의무자 없이 **단독신청** (보존등기 자체가 단독이므로 말소도 단독)
