# 감정평가 3방식 (허브 페이지)

감정평가 3방식은 부동산의 가치를 평가하는 3가지 접근법으로, 비용성(원가방식), 시장성(비교방식), 수익성(수익방식)에 각각 착안한다.

## 하위 문서

| 방식 | 평가법 | 핵심 키워드 | 링크 |
|------|--------|-------------|------|
| 원가방식 | 원가법 | 재조달원가(복성식/대치식), 감가수정, 적산가액 | [[감정평가_원가법]] |
| 비교방식 | 거래사례비교법 | 사정보정→시점수정→지역요인→개별요인, 비준가액 | [[감정평가_거래사례비교법]] |
| 수익방식 | 수익환원법 | 직접자본환원법(NOI/R), DCF, 환원율·할인율, 수익가액 | [[감정평가_수익환원법]] |

## 3방식 종합 비교표

| 구분 | 원가방식 | 비교방식 | 수익방식 |
|------|---------|---------|---------|
| **원리** | 비용성 | 시장성 | 수익성 |
| **가격원칙** | 대체의 원칙 | 대체의 원칙 | 예측·수익배분 |
| **평가법** | 원가법 | 거래사례비교법 | 수익환원법 |
| **시산가액** | **적산가액** | **비준가액** | **수익가액** |
| **적합 대상** | 건물, 특수부동산 | 일반 거래 활발 부동산 | 수익용 부동산 |
| **핵심 공식** | 재조달원가 - 감가수정 | 사례가격 x 보정·비교 | NOI / R |
| **임대료 평가법** | 적산법 (적산임대료) | 임대사례비교법 (비준임대료) | 수익분석법 (수익임대료) |
| **임대료 공식** | 기초가액 x 기대이율 + 필요경비 | 사례 비교·보정 | 기초가액 x 환원율 + 필요경비 |

## 핵심 비교 요약

- **시산가액 명칭**: 원가법=적산가액, 비교법=비준가액, 수익법=수익가액
- **가격원칙**: 원가·비교=대체의 원칙, 수익=예측·수익배분의 원칙
- **환원율과 할인율**: 순수익 일정 R=Y, 증가 R<Y, 감소 R>Y
- **감가 3유형**: 물리적·기능적=치유가능/불가능, 경제적=**치유불가능만**
