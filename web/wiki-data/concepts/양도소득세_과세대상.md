# 양도소득세 - 과세대상

## 1. 양도의 개념 (소득세법 제88조)

**양도**란 자산에 대한 등기·등록에 관계없이 매도, 교환, 법인에 대한 현물출자 등으로 그 자산을 **유상으로 사실상 이전**하는 것을 말한다.

### 양도에 해당하는 경우

| 유형 | 예시 |
|------|------|
| **유상이전** | 매매, 교환, 현물출자 |
| **강제이전** | 수용, 공매, 경매 |
| **대물변제** | 채무 대신 자산 이전 |
| **부담부증여** | 채무인수액에 상당하는 부분 |

### 양도에 해당하지 않는 경우

| 유형 | 이유 |
|------|------|
| **신탁재산**의 위탁자→수탁자 이전 | 소유권의 실질적 이전이 아님 |
| **환지처분**으로 지목·면적 변경 없이 교체 | 교환이 아닌 대체 |
| 도시개발법에 따른 **환지** | 동일 가치의 대체 |
| 농지의 **교환·분합** (비과세 요건 충족 시) | 비과세 대상 |

> **핵심**: "유상 + 사실상 이전"이 양도의 두 가지 핵심 요소

## 2. 과세대상 자산

| 분류 | 과세대상 |
|------|----------|
| **토지·건물** | 모든 토지와 건물 |
| **부동산에 관한 권리** | 지상권, 전세권, **등기된** 부동산임차권 |
| **부동산을 취득할 수 있는 권리** | 조합원입주권, **분양권** 등 |
| **기타 자산** | 사업용 고정자산과 함께 양도하는 영업권, 특정시설물 이용권(골프·콘도 등) |
| **주식 등** | 상장·비상장 주식 |

### 부동산에 관한 권리 vs 부동산을 취득할 수 있는 권리

| 구분 | 부동산에 관한 권리 | 부동산을 취득할 수 있는 권리 |
|------|------------------|--------------------------|
| **성격** | 이미 존재하는 부동산에 대한 권리 | 장래 부동산을 취득할 수 있는 권리 |
| **예시** | 지상권, 전세권, 등기된 임차권 | 조합원입주권, 분양권 |
| **개산공제** | **7%** | 적용 없음 |

> **미등기 부동산임차권**: 과세대상 아님 (등기된 임차권만 해당)

## 3. 양도시기 및 취득시기

### 원칙

| 구분 | 양도(취득)시기 |
|------|---------------|
| **대금청산일** 분명 | **잔금지급일** |
| 대금청산일 **불분명** | **등기접수일** |
| 대금청산 **전** 소유권이전등기 | **등기접수일** |

> **원칙**: 잔금일 > 등기접수일 (잔금이 먼저이면 잔금일, 등기가 먼저이면 등기접수일)

### 특례

| 구분 | 양도(취득)시기 |
|------|---------------|
| **장기할부** | 소유권이전등기접수일, 인도일, 사용수익일 중 **빠른 날** |
| **자기건설** | **사용검사필증 교부일** (임시사용승인일) |
| **상속** | **상속개시일** (피상속인 사망일) |
| **증여** | **증여등기 접수일** |
| **수용** | **대금을 청산한 날** (수용개시일) |
| **경매·공매** | **대금청산일** |

### 장기할부 조건

| 요건 | 내용 |
|------|------|
| **잔금 기간** | 소유권이전등기일·인도일·사용수익일 중 빠른 날 ~ 최종 할부금 지급일까지 **1년 이상** |
| **할부 횟수** | **2회 이상** 분할 수령 |

## 4. 과세대상 비해당 (양도소득세 과세 안 됨)

| 유형 | 이유 |
|------|------|
| **법인의 양도** | 법인세로 과세 (양도소득세는 **개인**에게만 과세) |
| **무상이전** (순수증여) | 증여세로 과세 |
| **미등기 임차권** | 과세대상 아님 |

## 핵심 암기 포인트

- **양도 = 유상 + 사실상 이전** (등기 여부 불문)
- 부담부증여: 채무인수액 부분만 **양도**로 봄
- 과세대상: 토지·건물, 부동산 권리, 취득할 수 있는 권리, 기타자산, 주식
- **등기된** 임차권만 과세대상 (미등기 임차권 X)
- 양도시기 원칙: **잔금일** (불분명·등기 선행 시 등기접수일)
- 상속: **상속개시일**, 증여: **증여등기접수일**, 수용: **대금청산일**
- 장기할부: 등기접수일·인도일·사용수익일 중 **빠른 날**

## OX 퀴즈

**Q1.** 양도소득세는 법인에게도 과세된다. → **X** (개인에게만 과세, 법인은 법인세)

**Q2.** 신탁재산의 위탁자에서 수탁자로의 이전은 양도에 해당한다. → **X** (양도 아님)

**Q3.** 대금청산일이 분명하지 않으면 등기접수일이 양도시기이다. → **O**

**Q4.** 미등기 부동산임차권은 양도소득세 과세대상이다. → **X** (등기된 임차권만 해당)

**Q5.** 부담부증여 시 채무인수액 부분은 양도에 해당한다. → **O**

**Q6.** 상속에 의한 취득시기는 상속등기 접수일이다. → **X** (상속개시일 = 피상속인 사망일)

**Q7.** 장기할부의 양도시기는 최종 할부금 수령일이다. → **X** (등기접수일·인도일·사용수익일 중 빠른 날)

**Q8.** 분양권은 양도소득세 과세대상이다. → **O** (부동산을 취득할 수 있는 권리)

## 함정 노트

- **등기 여부 불문**: 양도는 등기·등록에 **관계없이** 사실상 이전. "등기 안 했으니 양도 아님"은 오답
- **부담부증여**: 채무인수액 = **양도** (양도소득세), 나머지 = **증여** (증여세). 이중 과세 구조
- **상속개시일 vs 증여등기접수일**: 상속은 사망 시점(개시일), 증여는 등기접수일. 혼동 주의
- **장기할부 3가지 중 빠른 날**: 등기접수일·인도일·사용수익일. 잔금일이 아님 (장기할부는 잔금이 나중에 오므로)
- **분양권 포함**: 분양권도 주택 수에 포함되며 양도소득세 과세대상. 보유기간별 세율 적용
