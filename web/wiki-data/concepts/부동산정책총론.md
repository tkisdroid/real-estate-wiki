# 부동산 정책론

## 1. 정부의 시장개입 방법

### 직접적 개입방법

부동산시장에 직접 참여하여 수요자·공급자 역할 수행.

- 공영개발, 토지은행제도, 공공임대보유제, 공공투자사업
- 토지수용, 공공임대주택공급, 가격규제(분양가·임대료 상한제), 선매제도

### 간접적 개입방법

시장기능을 통해 효과를 유도. 효과 불확실하나 시장기능을 해치지 않음.

- 보조금제도, 조세제도, 부담금제도
- 대출규제(LTV, DTI, DSR), 부동산가격공시제도

### 시장개입의 기능

| 기능 | 내용 |
|------|------|
| **정치적** | 효율성·형평성의 조화 (사회적 목표) |
| **경제적** | 시장실패 수정 |

## 2. 시장실패 (Market Failure)

시장기구에 맡겼을 때 **효율적 자원배분을 실현하지 못하는 상황**. 근본 원인은 **불완전경쟁시장**.

### 원인

| 원인 | 내용 |
|------|------|
| **외부효과** | 시장기구 거치지 않고 제3자에게 의도하지 않은 이익/손해 |
| **공공재** | 비경합성·비배제성 → 무임승차자 문제 → 과소생산 |
| **독과점기업** | 가격설정자 → 과소생산, 사회적 후생손실 |
| **정보의 비대칭성** | 역선택, 도덕적 해이 → 자원배분 비효율 |

## 3. 외부효과

### 정(+)의 외부효과 (외부경제)

| 항목 | 내용 |
|------|------|
| 효과 | 부동산가치 **상승** |
| 생산량 | **과소생산** (사적비용 > 사회적비용) |
| 현상 | **PIMFY** (Please In My Front Yard) |
| 대책 | **보조금 지급**, 세금감면 |

### 부(-)의 외부효과 (외부불경제)

| 항목 | 내용 |
|------|------|
| 효과 | 부동산가치 **하락** |
| 생산량 | **과다생산** (사적비용 < 사회적비용) |
| 현상 | **NIMBY** (Not In My Back Yard) |
| 대책 | **규제**, 용도지역지구제, 세금 부과 |

### 비교표

| 구분 | 정(+)의 외부효과 | 부(-)의 외부효과 |
|------|-----------------|-----------------|
| 사적편익 vs 사회적편익 | 사적 < 사회적 | 사적 > 사회적 |
| 사적비용 vs 사회적비용 | 사적 > 사회적 | 사적 < 사회적 |
| 시장에 맡길 경우 | **과소생산** | **과다생산** |

### 외부효과 대책

1. **직접규제**: 완전금지(용도지역지구제) 또는 허용기준 설정
2. **외부성의 내부화**: 세금 부과 또는 보조금 지급
3. **재산권 부여** (코오즈 정리): 민간 자발적 협상으로 해결
4. **합병**: 피해기업과 가해기업 합병
5. **오염배출부과금**: 배출량에 따른 비용 부담
6. **오염정화보조금**: 영세업체에 방지시설 비용 보조

## 4. 공공재의 특징

- **소비의 비경합성**: 추가 소비자가 다른 소비자 혜택 줄이지 않음
- **소비의 비배제성**: 대가 미지불자도 소비 가능 → **무임승차자** 문제
- 시장에 맡기면 **과소생산** → 정부가 세금·공공기금으로 직접 공급

## 5. 부동산문제

**부동산문제**란 부동산과 인간과의 관계가 악화되는 현상을 말한다. 크게 토지문제와 주택문제로 대별된다.

### 부동산문제의 특징

| 특징 | 내용 |
|------|------|
| **악화성향** | 주택공급 중단 시 양적·질적 문제 악화, 규제 소홀 시 스프롤 확대 |
| **비가역성** | 지가 폭등 후 원상회복 불가능, 도시계획 오류의 환원 불가 |
| **지속성** | 양적 문제 해결 후에도 질적 수준 향상 노력 필요, 수급불균형 지속 |
| **해결수단의 다양성** | 여러 수단 이용 가능 → 종합정책 성격, 해결이 어려움을 의미 |

### 지가고(地價高)의 문제

지가고란 사회·경제적으로 요구되는 합리적 지가수준보다 현실의 지가수준이 훨씬 높게 형성된 상태를 말한다.

- 공공용지 확보 난 → 국고·조세부담 증가
- 물가·생산비 상승 → 기업경영 악영향
- 택지취득 곤란 → 양적 주택문제 발생
- 도심 토지의 집약적 이용 심화, 직주분리·스프롤 현상 초래
- 토지투기 조장 → 근로의욕 저해

### 토지분배 측정방법

| 방법 | 내용 |
|------|------|
| **로렌츠곡선** | 소득분포의 불평등도 측정, 굴곡이 클수록 불평등 |
| **지니계수** | 0(완전평등) ~ 1(완전불평등), 로렌츠곡선의 불평등도를 수치화 |

## 핵심 암기 포인트

- 직접적 개입: 공영개발, 토지수용, 공공임대주택, 가격규제(분양가·임대료 상한제), 선매제도
- 간접적 개입: 보조금, 조세, 부담금, 대출규제(LTV·DTI·DSR), 가격공시제도
- 시장실패 원인: **외부효과, 공공재, 독과점, 정보비대칭**
- 정(+)의 외부효과 → **과소생산** → PIMFY → 보조금 지급
- 부(-)의 외부효과 → **과다생산** → NIMBY → 규제·세금 부과
- 공공재: 비경합성 + **비배제성** → 무임승차자 문제 → **과소생산**
- 코오즈 정리: **재산권 부여** → 민간 자발적 협상으로 외부효과 해결

## OX 퀴즈

**Q1.** 보조금제도는 정부의 직접적 시장개입 방법이다. → **X** (보조금은 **간접적** 개입방법)

**Q2.** 정(+)의 외부효과가 있는 재화는 시장에 맡기면 과다생산된다. → **X** (정의 외부효과는 **과소생산**, 부의 외부효과가 과다생산)

**Q3.** 공공재는 무임승차자 문제로 인해 과소생산되는 경향이 있다. → **O**

**Q4.** NIMBY 현상은 정(+)의 외부효과와 관련된다. → **X** (NIMBY는 **부(-)의 외부효과**, 정(+)은 PIMFY)

**Q5.** 부(-)의 외부효과가 있을 때 사적비용은 사회적비용보다 크다. → **X** (부의 외부효과 시 사적비용 < 사회적비용)

**Q6.** 코오즈 정리는 재산권을 부여하면 민간 자발적 협상으로 외부효과를 해결할 수 있다는 이론이다. → **O**

**Q7.** LTV, DTI, DSR 규제는 정부의 직접적 개입방법이다. → **X** (대출규제는 **간접적** 개입방법)

## 함정 노트

- ⚠️ "보조금 = 직접적 개입" → 보조금·조세·부담금은 **간접적** 개입, 가격규제(상한제)가 직접적 개입
- ⚠️ "정(+) 외부효과 = 과다생산" → 정(+)은 **과소**생산, 부(-)가 과다생산
- ⚠️ "부(-) 외부효과: 사적비용 > 사회적비용" → 부(-) 외부효과 시 사적비용 **<** 사회적비용 (오염 등 비용이 사회에 전가)
- ⚠️ "NIMBY = PIMFY 혼동" → NIMBY = 부(-) 외부효과(혐오시설), PIMFY = 정(+) 외부효과(선호시설)
- ⚠️ "공공재 = 과다생산" → 공공재는 무임승차 문제로 **과소생산** → 정부가 직접 공급
