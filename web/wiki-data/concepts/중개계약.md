# 중개계약제도

## 1. 중개계약의 의의
- **중개계약**: 개업공인중개사와 의뢰인 간의 계약 (≠ 거래계약)
- **거래계약**: 의뢰인 양 당사자 간의 계약

## 2. 일반중개계약 (법 제22조)

### 특징
- 의뢰인이 **다수의 개업공인중개사에게 중개 의뢰 가능**
- 가장 먼저 중개를 성사시킨 개업공인중개사가 보수를 받음
- 우리나라에서 **가장 많이 이용**되는 형태

### 일반중개계약서 작성
- 의뢰인이 작성을 **요청할 수 있음** (의무 아님)
- 개업공인중개사는 요청을 **거절할 수 있음**
- 유효기간: **3개월** (다른 약정 가능)

### 기재사항 (법 제22조)
1. 중개대상물의 위치 및 규모
2. **거래예정가격**
3. 거래예정가격에 대한 **중개보수**
4. 기타 준수사항

## 3. 전속중개계약 (법 제23조)

### 정의
중개의뢰인이 **특정한 개업공인중개사에게 한정하여** 중개를 의뢰하는 계약

### 개업공인중개사의 의무

| 의무 | 내용 | 위반 시 |
|---|---|---|
| 전속중개계약서 사용 | 국토교통부 정한 서식 사용, **3년간 보존** | 업무정지 |
| 물건정보 공개 | 체결일로부터 **7일 이내** 거래정보망 또는 일간신문에 공개 | 임의 등록취소/업무정지 |
| 공개 통지 | 공개 후 **지체없이** 의뢰인에게 문서 통지 | |
| 업무처리상황 통지 | **2주일에 1회 이상** 문서로 통지 | |
| 확인·설명 의무 | 성실히 이행 | |

- 의뢰인이 비공개 요청 시 → **공개 금지** (위반 시 등록취소 가능)

### 의뢰인의 의무

| 상황 | 의뢰인 부담 |
|---|---|
| 유효기간 내 타 개업공인중개사에 의뢰하여 **거래한** 경우 | 중개보수 상당 **위약금** |
| 개업공인중개사를 배제하고 **소개받은 상대방과 직접거래** | 중개보수 상당 **위약금** |
| **스스로 발견한** 상대방과 거래 | 중개보수의 **50% 범위 내**에서 소요 비용 (사회통념상 상당한 비용) |

### 전속중개계약의 유효기간
- 원칙: **3개월**
- 당사자 간 다른 약정 가능 (영 제20조 제1항)

## 4. 부동산거래정보망 (법 제24조)

### 정의
개업공인중개사 상호 간에 중개대상물의 정보를 교환하는 정보통신 시스템

### 거래정보사업자 지정 요건 (규칙 제15조)
- 전국 **500명 이상** 가입신청 개업공인중개사
- **2개 이상** 시·도에서 각 **30명 이상** 가입
- 정보처리기사 **1인 이상** 확보
- 공인중개사 **1인 이상** 확보
- 적정 컴퓨터 설비 확보

### 운영규정
- 지정 후 **3개월 이내** 운영규정 제정 → 국토교통부장관 승인
- 지정 후 **1년 이내** 거래정보망 설치·운영

### 거래정보사업자의 의무
- 의뢰받은 정보만 공개, **차별적 공개 금지**
- 위반 시: 지정취소 + 1년 이하 징역 또는 1천만원 이하 벌금

### 개업공인중개사의 의무
- 정보를 **거짓으로 공개 금지**
- 거래완성 시 **지체없이** 거래정보사업자에게 통보
- 위반 시: 업무정지
