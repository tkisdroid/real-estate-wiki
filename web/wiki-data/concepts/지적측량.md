# 지적측량

## 1. 지적측량의 의의

지적측량이란 토지를 지적공부에 등록하거나 지적공부에 등록된 경계점을 지표상에 복원하기 위하여 필지의 경계 또는 좌표와 면적을 정하는 측량을 말한다.

## 2. 지적측량의 구분

| 구분 | 내용 | 세부 |
|------|------|------|
| **기초측량** | 지적측량의 기준이 되는 측량 | 지적삼각측량, 지적삼각보조측량, 지적도근측량 |
| **세부측량** | 기초측량 성과를 기초로 필지의 경계·면적 결정 | 신규등록측량, 등록전환측량, 분할측량, 경계복원측량, 지적현황측량, 지적확정측량 등 |

## 3. 지적기준점

| 종류 | 내용 | 도면 표시 |
|------|------|-----------|
| **지적삼각점** | 지적측량의 최상위 기준점 | 직경 3mm 원 안에 십자선 |
| **지적삼각보조점** | 삼각점의 보조 기준점 | 직경 3mm 원 안을 검은색 채색 |
| **지적도근점** | 세부측량의 기준점 | 직경 2mm 원 |

## 4. 지적측량의 방법

### 측량방법의 종류

| 방법 | 내용 |
|------|------|
| **평판측량** (전자평판측량 포함) | 평판 위에서 도해법으로 측량 |
| **경위의측량** | 경위의(트랜시트)를 사용하여 각도·거리 측정 |
| **전파기·광파기측량** | 전파 또는 광파를 이용한 거리측정 |
| **사진측량** | 항공사진 이용 |
| **위성측량** | GPS 등 위성을 이용한 측량 |

### 기초측량별 측량방법

| 기초측량 | 사용 가능한 방법 |
|----------|----------------|
| **지적삼각점측량** | 경위의, 전파기·광파기, 위성, 국토교통부장관 승인방법 |
| **지적삼각보조점측량** | 경위의, 전파기·광파기, 위성, 국토교통부장관 승인방법 |
| **지적도근점측량** | 경위의, 전파기·광파기, 위성, 국토교통부장관 승인방법 (도선법·교회법·다각망도선법) |
| **세부측량** | 경위의, **평판**, 위성, **전자평판** |

### 지적기준점측량의 절차

```
계획의 수립 → 준비 → 현지답사 → 선점 → 조표 → 관측 → 계산과 성과표 작성
```

## 5. 지적측량의 의뢰

- **의뢰자**: 토지소유자 또는 이해관계인
- **의뢰처**: 지적측량수행자 (한국국토정보공사 등)
- 지적측량수행자는 측량을 의뢰받으면 측량기간, 측량수수료 등을 기재한 서면을 의뢰인에게 발급

## 5. 지적측량 적부심사

지적측량성과에 대하여 이의가 있는 자가 지적측량의 정확성 여부를 심사청구하는 절차.

### 적부심사 절차

```
지적측량 적부심사 청구 (지적소관청에 청구)
    ↓
지적소관청이 지방지적위원회에 회부
    ↓
지방지적위원회 심의·의결 (90일 이내)
    ↓ (불복 시)
중앙지적위원회에 재심사 청구 (의결서 받은 날부터 90일 이내)
    ↓
중앙지적위원회 심의·의결
    ↓
지적소관청이 등록사항 정정 또는 측량성과 수정
```

### 적부심사청구의 제한
- 지방지적위원회의 의결서를 받은 날부터 **90일** 이내에 재심사를 청구하지 아니하거나
- 중앙지적위원회의 의결이 있는 경우에는
- 해당 지적측량성과에 대하여 **다시 적부심사청구를 할 수 없다**

## 6. 축척변경

지적도에 등록된 경계점의 정밀도를 높이기 위하여 작은 축척을 큰 축척으로 변경하는 것.

- **예**: 1/1,200 → 1/500
- 축척변경 시행지역의 지번부여: 지적확정측량과 동일한 방법 적용
- 축척변경으로 결번이 생기면 결번대장에 기재하되, 결번된 지번을 **다시 사용할 수 있다**

## 핵심 암기 포인트

- **지적측량 구분**: 기초측량(지적삼각·삼각보조·도근측량) + 세부측량(신규등록·등록전환·분할·경계복원·지적현황·지적확정측량 등)
- **지적기준점 3종**: 지적삼각점(최상위, 직경3mm원+십자선), 지적삼각보조점(직경3mm원+검은색채색), 지적도근점(직경2mm원)
- **적부심사 절차**: 지적소관청 청구 → 지방지적위원회(90일 이내) → 불복 시 중앙지적위원회 재심사(의결서 받은 날부터 90일 이내)
- **적부심사 제한**: 지방위 의결서 받은 날부터 90일 이내 재심사 미청구 또는 중앙위 의결이 있으면 다시 적부심사 불가
- **축척변경**: 작은 축척 → 큰 축척 (예: 1/1,200 → 1/500)
- **축척변경 결번**: 결번대장에 기재하되 결번 지번을 **다시 사용 가능**

## OX 퀴즈

**Q1.** 지적측량은 기초측량과 세부측량으로 구분된다. → **O**

**Q2.** 지적도근점은 지적측량의 최상위 기준점이다. → **X** (최상위 기준점은 지적삼각점. 도근점은 세부측량의 기준점)

**Q3.** 지적측량 적부심사는 중앙지적위원회에 직접 청구한다. → **X** (지적소관청에 청구 → 지방지적위원회에 회부)

**Q4.** 지방지적위원회 의결에 불복하는 경우 재심사 청구기한은 60일 이내이다. → **X** (의결서 받은 날부터 90일 이내)

**Q5.** 축척변경은 큰 축척을 작은 축척으로 변경하는 것이다. → **X** (작은 축척을 큰 축척으로 변경하여 정밀도를 높이는 것)

**Q6.** 축척변경으로 결번이 생긴 지번은 다시 사용할 수 없다. → **X** (축척변경·지적확정측량·지번변경 시에는 결번 지번 재사용 가능)

## 함정 노트

- ⚠️ **지적기준점 도면 표시**: 삼각점(원+십자선) vs 삼각보조점(원+검은색채색) vs 도근점(작은 원) — 표시 방법 혼동 주의
- ⚠️ **적부심사 기한 90일**: 지방위 심의기한(90일)과 재심사 청구기한(90일)이 동일 — 어디에 적용되는 90일인지 구분
- ⚠️ **결번 지번 재사용**: 원칙은 재사용 불가이지만, 지적확정측량·축척변경·지번변경 시에는 재사용 가능 — 예외를 반드시 암기
