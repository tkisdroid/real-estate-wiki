# 담보물권

채권의 담보를 목적으로 하는 물권. **유치권, 질권, 저당권**이 있다.

## 담보물권의 통유성

| 성질 | 내용 |
|------|------|
| **부종성** | 피담보채권 없으면 성립 안됨, 소멸하면 담보물권도 소멸 |
| **수반성** | 피담보채권 이전 시 담보물권도 함께 이전 |
| **불가분성** | 채권 전부 변제 시까지 목적물 전부에 효력 |
