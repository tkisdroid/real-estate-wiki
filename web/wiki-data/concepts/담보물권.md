# 담보물권

채권의 담보를 목적으로 하는 물권. **유치권, 질권, 저당권**이 있다.

## 담보물권의 통유성

| 성질 | 내용 |
|------|------|
| **부종성** | 피담보채권 없으면 성립 안됨, 소멸하면 담보물권도 소멸 |
| **수반성** | 피담보채권 이전 시 담보물권도 함께 이전 |
| **불가분성** | 채권 전부 변제 시까지 목적물 전부에 효력 |

## 담보물권의 통유성 상세

### 부종성
- **성립에서의 부종성**: 피담보채권 없으면 담보물권 성립 **불가** (예외: 근저당권 - 장래 채권 담보 가능)
- **소멸에서의 부종성**: 피담보채권 소멸 → 담보물권도 소멸
- **처분에서의 부종성**: 피담보채권과 분리하여 담보물권만 별도 양도 **불가** (제361조)
- 근저당권의 확정 전: 부종성 **완화** → 피담보채무의 소멸·이전이 근저당권에 영향 없음

### 수반성
- 피담보채권이 양도되면 담보물권도 함께 이전
- 채권양도의 대항요건(통지·승낙)을 갖추면 담보물권 이전의 부기등기 없이도 제3자에게 대항 가능 (대판 75다1478)

### 불가분성
- 채권의 **전부** 변제를 받을 때까지 목적물 **전부**에 대하여 권리 행사 가능
- 채권의 일부가 변제되어도 담보물의 **일부 반환 의무 없음**
- 저당물이 분할되어도 각 부분에 저당권 존속

### 물상대위성
- **인정**: 질권, 저당권 (지급 전 압류 필요)
- **부정**: 유치권
