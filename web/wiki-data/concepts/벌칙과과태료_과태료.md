# 과태료 (행정질서벌)

## 1. 500만원 이하 과태료 (법 제51조 제2항)

| 번호 | 사유 | 부과권자 |
|---|---|---|
| ① | **부당 표시·광고** (개업공인중개사) | 등록관청 |
| ② | 모니터링 **자료제출 요구 위반** (정보통신서비스 제공자) | 국토교통부장관 |
| ③ | 모니터링 결과 **조치 요구 위반** (정보통신서비스 제공자) | 국토교통부장관 |
| ④ | 거래정보망 **운영규정 위반** | 국토교통부장관 |
| ⑤ | **확인·설명 의무 위반** (불이행·부실·근거자료 미제시) | 등록관청 |
| ⑥ | **연수교육 미이수** | **시·도지사** |
| ⑦ | 지도·감독 불응 (거래정보사업자·협회) | 국토교통부장관 |
| ⑧ | 공제사업 **운용실적 미공시** | 국토교통부장관 |
| ⑨ | 공제업무 **개선명령 미이행** | 국토교통부장관 |
| ⑩ | 협회 임원 징계·해임 요구 미이행, 시정명령 미이행 | 국토교통부장관 |
| ⑪ | **중개보조원 고지의무 위반** (중개보조원 및 개업공인중개사) | 등록관청 |

### 중개보조원 고지의무 위반 특칙
- 개업공인중개사가 해당 업무에 관하여 **상당한 주의와 감독을 게을리하지 않은 경우** → 과태료 부과 제외

## 2. 100만원 이하 과태료 (법 제51조 제3항)

| 번호 | 사유 | 부과권자 |
|---|---|---|
| ① | **등록증 등 게시의무** 위반 | 등록관청 |
| ② | **명칭 사용** 위반 (공인중개사사무소, 부동산중개 문자 미사용) | 등록관청 |
| ③ | 옥외광고물 **성명 표기** 위반 (미표기 또는 거짓 표기) | 등록관청 |
| ④ | 표시·광고 시 **명시사항** 위반 | 등록관청 |
| ⑤ | **이전신고** 미이행 | 등록관청 |
| ⑥ | **휴업·폐업·재개·기간변경** 신고 미이행 | 등록관청 |
| ⑦ | 보증 관련 **설명·사본교부** 위반 | 등록관청 |
| ⑧ | 자격증·등록증 **반납** 미이행 | **시·도지사** / 등록관청 |

## 3. 부과권자 정리

| 대상 | 부과권자 |
|---|---|
| **정보통신서비스 제공자** 관련 | **국토교통부장관** |
| **거래정보사업자** 관련 | **국토교통부장관** |
| **협회** 관련 (운용실적, 개선명령, 징계 등) | **국토교통부장관** |
| **연수교육** 미이수 | **시·도지사** |
| **자격증 반납** 미이행 | **시·도지사** |
| **등록증 반납** 미이행 | **등록관청** |
| 그 외 개업공인중개사 관련 | **등록관청** |

### 부과권자 구분 핵심
- **국토교통부장관**: 정보통신서비스 제공자, 거래정보사업자, 협회
- **시·도지사**: 연수교육, 자격증 반납
- **등록관청**: 개업공인중개사 대부분

## 4. 감경·가중

### 기준
- 개별기준의 **1/2 범위**에서 가중·감경 가능
- 가중해도 **상한(500만원/100만원) 초과 불가**

### 감경 제한
- **과태료 체납자**는 감경 **불가**

## 5. 500만원 vs 100만원 분류 핵심

| 500만원 (중대 위반) | 100만원 (경미 위반) |
|---|---|
| 부당 표시·광고 | 등록증 게시 위반 |
| 확인·설명 의무 위반 | 명칭 사용 위반 |
| 연수교육 미이수 | 성명 표기 위반 |
| 중개보조원 고지의무 위반 | 표시·광고 명시사항 위반 |
| 정보통신서비스 제공자 관련 | 이전신고 미이행 |
| 거래정보망 운영규정 위반 | 휴폐업 신고 미이행 |
| 공제사업·협회 관련 | 보증 설명·교부 위반 |
| | 자격증·등록증 반납 미이행 |

## 핵심 암기 포인트

1. **500만원**: 부당광고, 확인설명, 연수교육, 보조원고지, 정보통신·거래정보·협회, 공제
2. **100만원**: 게시, 명칭, 성명표기, 광고명시, 이전신고, 휴폐업신고, 보증설명교부, 반납
3. 부과권자: 국토교통부장관(정보통신·거래정보·협회), 시·도지사(연수교육·자격증반납), **등록관청**(나머지)
4. 감경: 1/2 범위, 상한 초과 불가, **체납자 감경 불가**
5. 연수교육 미이수: **시·도지사**가 부과 (등록관청 아님)
6. 보증 미설명·미교부: **100만원** (500만원 아님)
7. 확인·설명 위반: **500만원**

## OX 퀴즈

**Q1.** 확인·설명 의무 위반 시 100만원 이하의 과태료가 부과된다.
> **X** — 확인·설명 의무 위반 시 **500만원 이하** 과태료이다.

**Q2.** 연수교육 미이수에 대한 과태료는 등록관청이 부과한다.
> **X** — 연수교육 미이수 과태료는 **시·도지사**가 부과한다.

**Q3.** 과태료 감경은 개별기준의 1/2 범위에서 가능하다.
> **O** — 개별기준의 **1/2 범위**에서 가중·감경 가능하다.

**Q4.** 과태료 체납자도 감경을 받을 수 있다.
> **X** — 과태료 **체납자는 감경 불가**이다.

**Q5.** 손해배상책임에 관한 설명·사본교부 위반 시 500만원 이하 과태료가 부과된다.
> **X** — 보증 관련 설명·사본교부 위반은 **100만원 이하** 과태료이다.

**Q6.** 협회 관련 과태료의 부과권자는 등록관청이다.
> **X** — 협회 관련 과태료는 **국토교통부장관**이 부과한다.

**Q7.** 과태료를 가중하여도 법정 상한액을 초과할 수 없다.
> **O** — 가중해도 **상한(500만원/100만원) 초과 불가**이다.

**Q8.** 중개보조원 고지의무 위반 시 개업공인중개사가 상당한 주의와 감독을 한 경우에도 과태료가 부과된다.
> **X** — 상당한 주의와 감독을 게을리하지 않은 경우 **과태료 부과 제외**.

## 함정 노트

| 함정 포인트 | 정확한 내용 |
|---|---|
| 확인설명 위반 → 100만원 | 확인설명 위반 → **500만원** |
| 보증 설명교부 → 500만원 | 보증 설명교부 → **100만원** |
| 연수교육 → 등록관청 | 연수교육 → **시·도지사** |
| 자격증 반납 → 등록관청 | 자격증 반납 → **시·도지사** |
| 등록증 반납 → 시·도지사 | 등록증 반납 → **등록관청** |
| 체납자 감경 가능 | 체납자 **감경 불가** |
| 가중 시 상한 초과 가능 | 가중해도 **상한 초과 불가** |
| 협회 → 등록관청 | 협회 → **국토교통부장관** |
