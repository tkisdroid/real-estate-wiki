# 건축법 총론

건축법은 건축물의 대지·구조·설비 기준 및 용도 등을 정하여 건축물의 안전·기능·환경 및 미관을 향상시키기 위한 법률이다.

## 건축행위 5종 (건축의 정의)

### 1. 신축
- 건축물이 없는 대지(기존 건축물이 철거·멸실된 대지 포함)에 새로 건축물을 축조하는 것
- 부속건축물만 있는 대지에 새로 주된 건축물을 축조하는 것 포함 (개축·재축은 제외)

### 2. 증축
- 기존 건축물이 있는 대지에서 건축물의 **건축면적·연면적·층수 또는 높이**를 늘리는 것

### 3. 개축
- 기존 건축물의 전부 또는 일부(**내력벽·기둥·보·지붕틀 중 셋 이상** 포함)를 해체하고 그 대지에 **종전과 같은 규모**의 범위에서 다시 축조하는 것

### 4. 재축
- 건축물이 **천재지변이나 그 밖의 재해**로 멸실된 경우 다음 요건을 모두 갖추어 다시 축조하는 것
  - 연면적 합계는 **종전 규모 이하**
  - 동수·층수 및 높이가 모두 종전 규모 이하이거나, 초과하는 경우 법령에 모두 적합할 것

### 5. 이전
- 건축물의 **주요구조부를 해체하지 아니하고** 같은 대지의 다른 위치로 옮기는 것

## 대수선

증축·개축·재축에 해당하지 아니하는 것으로서 다음의 어느 하나에 해당하는 것:

| 대상 | 기준 |
|------|------|
| **내력벽** | 증설·해체하거나 벽면적 **30㎡ 이상** 수선·변경 |
| **기둥** | 증설·해체하거나 **3개 이상** 수선·변경 |
| **보** | 증설·해체하거나 **3개 이상** 수선·변경 |
| **지붕틀** | 증설·해체하거나 **3개 이상** 수선·변경 (한옥 서까래 제외) |
| **방화벽·방화구획** | 바닥·벽을 증설·해체·수선·변경 |
| **주계단·피난계단·특별피난계단** | 증설·해체·수선·변경 |
| **미관지구 외부형태** | 건축물의 외부형태(담장 포함) 변경 |
| **경계벽** | 다가구주택 가구간·다세대주택 세대간 경계벽 증설·해체·수선·변경 |
| **외벽 마감재료** | 증설·해체하거나 벽면적 **30㎡ 이상** 수선·변경 |

## 용도변경

### 시설군 분류 (9개군)

| 시설군 | 속하는 용도 |
|--------|------------|
| ①자동차 관련 시설군 | 자동차 관련 시설 |
| ②산업 등 시설군 | 운수·창고·공장·위험물저장·자원순환·묘지·장례시설 |
| ③전기통신시설군 | 방송통신·발전시설 |
| ④문화집회시설군 | 문화및집회·종교·위락·관광휴게시설 |
| ⑤영업시설군 | 판매·운동·숙박시설, 제2종근린(다중생활시설) |
| ⑥교육및복지시설군 | 의료·교육연구·노유자·수련·야영장시설 |
| ⑦근린생활시설군 | 제1종·제2종 근린생활시설(다중생활시설 제외) |
| ⑧주거업무시설군 | 단독·공동주택, 업무시설, 교정및군사시설 |
| ⑨그 밖의 시설군 | 동물및식물 관련 시설 |

### 용도변경 방법

| 변경방향 | 절차 |
|----------|------|
| **상위군**으로 변경 | **허가** |
| **하위군**으로 변경 | **신고** |
| 같은 시설군 내 변경 | **건축물대장 변경신청** |

- 용도변경 허가·신고 시 바닥면적 합계 **100㎡ 이상**: 사용승인 규정 준용
- 용도변경 허가대상 바닥면적 합계 **500㎡ 이상**: 건축사 설계 규정 준용

## 건축허가·건축신고

### 건축허가권자
- **원칙**: 특별자치시장·특별자치도지사 또는 **시장·군수·구청장**
- **예외**: 21층 이상 또는 연면적 10만㎡ 이상 → **특별시장·광역시장**

### 건축신고대상 (허가 대신 신고)
- 바닥면적 합계 **85㎡ 이내**의 증축·개축·재축
- 관리·농림·자연환경보전지역에서 연면적 **200㎡ 미만**, **3층 미만** 건축물의 건축
- 연면적 **200㎡ 미만**, **3층 미만** 건축물의 대수선
- 연면적 합계 **100㎡ 이하**인 건축물
- 높이 **3m 이하** 범위에서 증축하는 건축물

### 건축허가의 취소 (기속취소)
- 허가일부터 **2년**(공장 승인 공장은 3년) 이내 공사 미착수
- 정당한 사유 시 **1년** 범위에서 착수기간 연장 가능

### 사전승인 (도지사 승인)
- 시장·군수가 **21층 이상** 또는 **연면적 10만㎡ 이상** 건축 허가 시

## 건축물의 대지 및 도로

### 도로의 정의
- 보행과 자동차 통행이 가능한 너비 **4m 이상**의 도로
- 막다른 도로 길이별 최소 너비: 10m 미만 → **2m**, 10~35m → **3m**, 35m 이상 → **6m** (비도시 읍면 4m)

### 대지와 도로의 관계
- **원칙**: 건축물의 대지는 **2m 이상** 도로에 접할 것
- **강화**: 연면적 합계 **2,000㎡** 이상(공장 3,000㎡) → 너비 **6m** 이상 도로에 **4m** 이상 접할 것

### 건축선
- **원칙**: 대지와 도로의 경계선
- **예외**: 소요너비(4m)에 못 미치는 도로 → 중심선에서 소요너비의 **1/2** 수평거리만큼 후퇴
- **지정건축선**: 도시지역에서 **4m 이하** 범위에서 별도 지정 가능

### 대지의 분할제한

| 용도지역 | 최소면적 |
|----------|---------|
| 주거지역 | **60㎡** |
| 상업지역 | **150㎡** |
| 공업지역 | **150㎡** |
| 녹지지역 | **200㎡** |
| 기타 | **60㎡** |

## 높이제한 (일조권)

### 전용·일반주거지역의 일조 확보
- 높이 **10m 이하** 부분: 인접 대지경계선에서 **1.5m** 이상
- 높이 **10m 초과** 부분: 인접 대지경계선에서 해당 건축물 높이의 **1/2** 이상

### 공동주택의 일조 확보 (일반·중심상업지역 제외)
- 채광방향 인접 대지경계선까지 수평거리의 **2배** 이하 (근린상업·준주거 **4배**)
- 인동거리: 채광창 벽면에서 건축물 높이의 **0.5배** 이상 (도시형생활주택 0.25배)
- 남향 건축물 높은 쪽 **0.4배**, 낮은 쪽 **0.5배** 이상
- 측벽과 측벽: **4m** 이상, 측벽과 채광창 없는 벽면: **8m** 이상
- 동지 기준 9시~15시 사이 **2시간** 이상 일조 확보 시 인동거리 완화 가능

### 높이제한 배제
- **2층 이하**로서 높이 **8m 이하**인 건축물: 조례로 일조 높이제한 적용 배제 가능

## 건축물의 층수·높이 산정

- **층수 불산입**: 옥상부분(승강기탑 등) 수평투영면적 합계가 건축면적의 **1/8** 이하(공동주택 전용 85㎡ 이하 1/6), 지하층
- **층수 불명확**: 높이 **4m**마다 1층으로 산정
- **건축물 높이**: 지표면~상단 (1층 전체 필로티: 필로티 층고 제외)
- **옥상돌출물**: 수평투영면적 건축면적의 1/8 이하이면서 **12m** 넘는 부분만 높이 산입

## 핵심 암기 포인트

- **건축 5종**: 신축·증축·개축·재축·이전
- **개축**: 내력벽·기둥·보·지붕틀 중 **셋 이상** 해체 후 종전 규모로 재축조
- **재축**: 천재지변 등 **재해**로 멸실 → 종전 규모 이하
- **이전**: 주요구조부 **해체 없이** 같은 대지 내 이동
- **대수선 내력벽**: **30㎡ 이상**, 기둥·보·지붕틀: **3개 이상**
- **용도변경**: 상위군 → 허가, 하위군 → 신고, 같은 군 → 대장 변경
- **건축허가권자**: 시장·군수·**구청장** (개발행위허가는 시장·군수)
- **건축신고**: 85㎡ 이내 증축, 관리·농림·자연환경보전 200㎡ 미만 3층 미만
- **대지 접도**: 원칙 **2m**, 강화(2,000㎡↑) **6m 도로에 4m**
- **건축선 후퇴**: 4m 미달 도로 → 중심선에서 **1/2** 후퇴
- **일조권**: 10m 이하 **1.5m**, 10m 초과 높이의 **1/2**
- **공동주택 인동거리**: 채광창 벽면~건축물 높이의 **0.5배**
- **허가 취소**: 허가일부터 **2년** 미착공 시 (공장 3년)

## OX 퀴즈

**Q1.** 건축물의 주요구조부를 해체하지 아니하고 같은 대지의 다른 위치로 옮기는 것은 "이전"에 해당한다. → **O**

**Q2.** 개축이란 내력벽·기둥·보·지붕틀 중 둘 이상이 포함되는 경우를 해체하고 다시 축조하는 것이다. → **X** (셋 이상이 포함되는 경우)

**Q3.** 재축은 화재로 멸실된 건축물을 종전 규모 이하로 다시 축조하는 경우에도 해당한다. → **O** (천재지변이나 그 밖의 재해 포함)

**Q4.** 건축물의 용도를 하위군에서 상위군으로 변경하려면 건축물대장 변경신청을 하면 된다. → **X** (상위군 변경은 허가 대상)

**Q5.** 건축허가권자는 원칙적으로 시장·군수·구청장이다. → **O**

**Q6.** 건축물의 대지는 원칙적으로 4m 이상 도로에 접하여야 한다. → **X** (2m 이상 접하여야 한다)

**Q7.** 전용주거지역에서 높이 10m를 초과하는 건축물은 인접 대지경계선에서 해당 높이의 1/3 이상 띄어야 한다. → **X** (1/2 이상)

**Q8.** 보를 2개 변경하는 것은 대수선에 해당한다. → **X** (보는 3개 이상 수선·변경)

**Q9.** 내력벽의 벽면적을 30㎡ 수선하는 것은 대수선에 해당한다. → **O** (30㎡ 이상)

**Q10.** 2층 이하로서 높이 8m 이하인 건축물은 조례로 일조 높이제한을 적용하지 않을 수 있다. → **O**

## 비교 정리

### 건축 5종 비교

| 구분 | 신축 | 증축 | 개축 | 재축 | 이전 |
|------|------|------|------|------|------|
| 대지 | 건축물 없는 대지 | 기존 건축물 있는 대지 | 기존 건축물 대지 | 재해 멸실 대지 | 같은 대지 |
| 특징 | 새로 축조 | 면적·높이·층수 늘림 | 해체 후 같은 규모 | 재해 후 종전이하 | 구조부 해체 없이 이동 |
| 해체여부 | - | - | 셋 이상 해체 | - | 해체 없음 |

### 건축허가 vs 건축신고

| 구분 | 건축허가 | 건축신고 |
|------|---------|---------|
| 권한자 | 시장·군수·구청장 | 시장·군수·구청장 |
| 대상 | 원칙적 건축·대수선 | 85㎡ 이내 증·개·재축 등 |
| 효력 | - | 신고일부터 1년 이내 미착공 시 효력 상실 |

## 함정 노트

- ⚠️ **개축 vs 재축**: 개축은 "해체 후 종전 규모", 재축은 "재해로 멸실 후 종전 이하" → 사유(해체 vs 재해)와 규모(같은 vs 이하) 구분
- ⚠️ **대수선 기둥·보**: "3개 이상" 수선·변경이 기준. 2개는 대수선 아님
- ⚠️ **허가권자 혼동**: 건축허가 = 시장·군수·**구청장**, 개발행위허가 = 시장·**군수** (구청장 없음)
- ⚠️ **용도변경 방향**: 상위군(규제 강화) → **허가**, 하위군(규제 완화) → **신고**. 방향 주의
- ⚠️ **접도 2m vs 4m**: 대지 접도는 **2m**, 도로의 최소 너비는 **4m** → 혼동 주의
- ⚠️ **일조권 10m 기준**: 10m 이하 1.5m, 10m 초과 1/2 → "10m"는 건축물 높이 기준
