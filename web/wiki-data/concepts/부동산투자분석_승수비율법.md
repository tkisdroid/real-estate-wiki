# 승수법과 비율법(수익률법)

> 📖 상위 문서: [[부동산투자분석]]

## 1. 어림셈법의 의의

- 소득이득 중에서 **한 해의 소득**을 기준으로 투자분석하는 방법
- **화폐의 시간가치 고려하지 않음** (자본이득도 미고려)
- **승수법**과 **수익률법**으로 구분, 서로 **역수관계**

| 승수법 | ↔ 역수 | 수익률법 |
|--------|--------|---------|
| 몇 **배**인가? (회수기간) | | 몇 **%**인가? (이익률) |

## 2. 승수법 (회수기간법)

| 지표 | 산식 | 판단 |
|------|------|------|
| **총소득승수(GIM)** | **총투자액** / 총소득(PGI 또는 EGI) | 낮을수록 유리 |
| **순소득승수(NIM)** (=자본회수기간) | **총투자액** / NOI | 낮을수록 유리 |
| **세전현금수지승수** | **지분투자액** / BTCF | 낮을수록 유리 |
| **세후현금수지승수** | **지분투자액** / ATCF | 낮을수록 유리 |

### 주의사항

- 총소득승수·순소득승수 = **총투자액** 기준
- 세전·세후현금수지승수 = **지분투자액** 기준
- 승수값이 **작을수록** 자본회수기간 **짧음** → **유리**
- 동일 투자안: 순소득승수 < 총소득승수 (NOI < 총소득이므로)

## 3. 수익률법 (비율법)

| 지표 | 산식 | 판단 |
|------|------|------|
| **총자산회전율** | 총소득 / 총자산(총투자) | 높을수록 유리 |
| **종합자본환원율(Ro)** | NOI / **총투자액**(부동산가격) | 높을수록 유리 |
| **지분배당률(Re)** | BTCF / **지분투자액** | 높을수록 유리 |
| **세후수익률** | ATCF / **지분투자액** | 높을수록 유리 |

### 종합자본환원율과 저당-지분 관계

$$R_o = R_m \times L + R_e \times (1-L)$$

- L = 대부비율(LTV), Rm = 저당상수, Re = 지분배당률
- 종합자본환원율의 역수 = **순소득승수**

## 4. 재무비율

| 지표 | 산식 | 핵심 판단 |
|------|------|----------|
| **저당상수(Rm)** | DS / 저당대출잔금 | 원리금균등상환 매기 상환비율 |
| **부채감당률(DCR)** | NOI / DS | **1.0 이상**이면 부채 감당 가능 |
| **채무불이행률(DR)** (=손익분기율) | (OE+DS) / EGI | **1.0 이하**이면 채무이행 가능 |
| **영업경비율(OER)** | OE / EGI(또는 PGI) | 재무관리 상태 지표 |

### 부채감당률(DCR) 상세

- DCR > 1: NOI가 DS 감당하고 **남음** → 양호
- DCR = 1: NOI = DS → 세전현금수지(BTCF) = **0**, 지분배당률도 **0**
- DCR < 1: NOI가 DS 감당 **부족** → 위험
- 대출기관은 DCR을 **높이는 것**이 유리 (채무불이행 위험 낮춤)

## 핵심 암기 포인트

- 어림셈법 = 화폐의 시간가치 **미고려**, 승수법과 수익률법은 **역수관계**
- 총소득승수·순소득승수 = **총투자액** 기준 / 세전·세후 승수 = **지분투자액** 기준
- 승수 작을수록 유리 / 수익률 높을수록 유리
- Ro = **Rm x L + Re x (1-L)** / Ro의 역수 = **순소득승수**
- 부채감당률(DCR) = **NOI / DS**, **1.0 이상** = 부채 감당 가능
- DCR = 1이면 BTCF = **0**, 지분배당률 = **0**

## OX 퀴즈

**Q1.** 총소득승수는 화폐의 시간가치를 고려한 방법이다. → **X** (화폐의 시간가치 **미고려**)

**Q2.** 총소득승수는 총투자액을 세후현금흐름으로 나눈 값이다. → **X** (총투자액을 **총소득**으로 나눈 값)

**Q3.** 순소득승수는 지분투자액을 순영업소득으로 나눈 값이다. → **X** (**총투자액**을 NOI로 나눈 값. 지분투자액 아님)

**Q4.** 종합자본환원율의 역수는 순소득승수이다. → **O**

**Q5.** 부채감당률이 1보다 작으면 순영업소득으로 원리금 지불능력이 충분하다. → **X** (1보다 작으면 **부족**, 충분한 것은 1보다 **클 때**)

**Q6.** 승수법에서 승수값이 작을수록 자본회수기간이 길어진다. → **X** (작을수록 회수기간 **짧아짐**)

**Q7.** 자본환원율은 화폐의 시간가치를 고려하지 않는다. → **O**

**Q8.** 동일한 투자안에서 순소득승수가 총소득승수보다 크다. → **X** (NOI < 총소득이므로 순소득승수가 **작음**)

## 함정 노트

- ⚠️ "총소득승수 = 시간가치 고려" → **미고려** (어림셈법)
- ⚠️ "순소득승수 = 지분투자액/NOI" → **총투자액**/NOI (지분투자액은 세전·세후 승수)
- ⚠️ "승수 작을수록 불리" → **유리** (회수기간 짧음)
- ⚠️ "DCR < 1 = 충분" → **부족** (1 이상이어야 감당 가능)
- ⚠️ "총투자수익률 = BTCF/지분투자액" → 총투자수익률 = **NOI/총투자액**, BTCF/지분 = **지분배당률**
- ⚠️ "순소득승수 > 총소득승수" → 순소득승수 < 총소득승수 (분모인 NOI < 총소득)
