# 중력모형 (레일리·컨버스·허프)

> 📖 상위 문서: [[입지이론]]

## 1. 중력모형의 의의

- 두 물체 간의 인력은 **거리의 제곱에 반비례**하고, **질량의 크기에 비례**한다는 자연과학의 중력법칙을 상권분석에 적용
- 크리스탈러의 중심지이론이 **중심지의 형성과정**을 설명하는 데 비해, 중력모형은 **중심지 간의 상호작용 요인**을 설명

## 2. 레일리(W. Reilly)의 소매인력법칙

### 유인력 공식

$$유인력 = \frac{크기(인구수\ 또는\ 매장면적)}{거리^2}$$

- 유인력은 상점의 **크기에 비례**, **거리의 제곱에 반비례**

### 분기점의 위치

- 고차중심지(큰 도시)와 저차중심지(작은 도시)가 서로 마주할 때
- **분기점**은 **저차중심지(작은 도시)에 가까운 곳**에서 결정
- A도시가 B도시보다 크다면 → 분기점은 **B도시(작은 도시)**에 가까이 형성
- 큰 도시의 인력이 더 멀리까지 미치기 때문

### 구매지향비율

- 두 도시의 **인구에 비례**하고, 분기점에서 두 도시에 이르는 **거리의 제곱에 반비례**

## 3. 컨버스(P.D. Converse)의 분기점 모형

$$상권분기점(A로부터의\ 거리) = \frac{A와\ B\ 사이\ 거리}{1 + \sqrt{\frac{B도시\ 인구}{A도시\ 인구}}}$$

- 레일리의 소매인력법칙을 **실용적으로 변환**하여 상권 분기점의 구체적 위치를 산정
- **결정론적** 접근 (확률 아님)

## 4. 허프(D. Huff)의 확률모형

### 특성

- **확률론적** 접근 (레일리·컨버스는 **결정론적**)
- 소비자가 특정 점포에 갈 **확률** = f(거리, 경쟁점포 수, 점포 크기)

### 마찰계수의 차이

| 구분 | 레일리·컨버스 | 허프 |
|------|-------------|------|
| 마찰계수 | **'2'로 고정** | **가변적** |
| 접근 | 결정론적 | 확률론적 |

### 마찰계수의 특성

- 교통조건이 **나쁠수록** → 마찰계수 **큼**
- 교통조건이 **좋을수록** → 마찰계수 **작음**
- **전문품**(보석, 가구 등)은 일상용품보다 마찰계수 **작음** (멀어도 가려 함)
- **일상용품**(편의품)은 마찰계수 **큼** (가까이서 구입)

## 5. 넬슨(Nelson)의 소매입지이론

- 점포입지의 **8가지 원칙** 제시
- 특히 **양립성**(compatibility): 서로 다른 인접 점포 간 **고객 교환** 효과 강조

## 핵심 암기 포인트

- 레일리: 유인력 = **크기 / 거리²**, 분기점은 **작은 도시**에 가까이 형성
- 컨버스: 분기점 구체적 위치 산정 공식, **결정론적** 접근
- 허프: **확률론적** 모형, 마찰계수 **가변적** (레일리·컨버스는 '2'로 고정)
- 마찰계수: 교통 나쁘면 **큼**, 전문품은 일상용품보다 **작음**

## OX 퀴즈

**Q1.** 레일리의 소매인력법칙에서 상권분기점은 큰 도시에 가까운 곳에 형성된다. → **X** (**작은 도시(저차중심지)**에 가까운 곳에 형성)

**Q2.** 레일리의 소매인력법칙에서 유인력은 상점의 크기에 반비례한다. → **X** (크기에 **비례**, 거리의 제곱에 **반비례**)

**Q3.** 허프 모형의 마찰계수는 레일리·컨버스와 동일하게 '2'로 고정된다. → **X** (허프만 **가변적**, 레일리·컨버스는 '2' 고정)

**Q4.** 허프 모형은 확률론적 접근 방법이다. → **O**

**Q5.** 마찰계수는 교통조건이 나쁠수록 작아진다. → **X** (교통조건 나쁠수록 마찰계수 **커짐**)

**Q6.** 전문품의 마찰계수는 일상용품보다 크다. → **X** (전문품은 일상용품보다 마찰계수 **작음**)

**Q7.** 컨버스의 분기점 모형은 확률론적 접근이다. → **X** (**결정론적** 접근, 확률론적은 **허프**)

## 함정 노트

- ⚠️ "분기점은 큰 도시 가까이" → **작은 도시** 가까이 형성 (큰 도시의 인력이 더 멀리까지 미침)
- ⚠️ "허프·레일리 마찰계수 동일" → 레일리·컨버스는 **2로 고정**, 허프만 **가변적**
- ⚠️ "마찰계수: 교통 나쁘면 작음" → 교통 나쁠수록 **큼**
- ⚠️ "전문품 마찰계수 > 일상용품" → 전문품 < 일상용품 (전문품은 멀어도 갈 의향)
- ⚠️ "중력모형 = 중심지 형성과정 설명" → 중력모형은 **중심지 간 상호작용** 설명, 형성과정은 **크리스탈러**
