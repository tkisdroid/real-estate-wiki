# 경계와 면적

## 1. 경계

### 경계의 의의
경계란 필지를 구획하는 선으로서 지적도나 임야도에 등록한 것 또는 경계점좌표등록부에 좌표로 등록한 것을 말한다.

### 경계의 의의 (상세)
- **경계점**: 필지를 구획하는 선의 굴곡점으로서 지적도·임야도에 도해 형태로 등록하거나 경계점좌표등록부에 좌표 형태로 등록하는 점

### 경계결정의 원칙

| 원칙 | 내용 |
|------|------|
| **경계국정주의** | 경계는 국가가 결정하여 지적공부에 등록 |
| **경계직선주의** (경계실형주의) | 경계점과 경계점 사이를 직선(최단거리)으로 연결하여 등록 |
| **경계불가분의 원칙** | 경계는 유일무이한 것으로서 어느 한쪽 필지에만 전속하지 않고 인접토지에 공통으로 작용 (너비 없음) |
| **축척종대의 원칙** | 동일 경계가 축척이 다른 도면에 각각 등록된 경우 **축척이 큰 도면**의 경계에 따름 |
| **부동성의 원칙** | 법령 절차에 따라 결정·등록된 경계는 토지이동 사유 없이 임의 변경 불가 |

### 지상경계의 결정 기준 (5가지)

| 상황 | 경계 |
|------|------|
| 1. 연접 토지 간 **높낮이 차이 없는** 경우 | 구조물 등의 **중앙** |
| 2. 연접 토지 간 **높낮이 차이 있는** 경우 | 구조물 등의 **하단부** |
| 3. 도로·구거 등에 **절토(땅깎기)된** 부분이 있는 경우 | 경사면의 **상단부** |
| 4. 토지가 **해면·수면에 접하는** 경우 | **최대만조위·최대만수위**가 되는 선 |
| 5. **공유수면매립지** 중 제방 등을 토지에 편입 등록하는 경우 | **바깥쪽 어깨부분** |

> 지상경계의 구획을 형성하는 구조물 등의 **소유자가 다른** 경우에는 위 1~3 규정에도 불구하고 **소유권에 따라** 경계를 결정

## 2. 면적

### 면적의 의의
면적이란 지적공부에 등록한 필지의 수평면상 넓이를 말한다.

### 면적의 단위
- **㎡** (제곱미터)를 사용
- **1평 = 3.3㎡** (참고)

### 면적측정 대상
세부측량 시 면적을 측정하는 경우:
1. 지적공부의 복구·신규등록·등록전환·분할 및 축척변경
2. 등록된 면적 또는 경계를 정정하는 경우
3. 도시개발사업 등의 토지이동에 따라 토지표시를 새로 결정하는 경우
4. 경계복원측량·지적현황측량에 면적측정이 수반되는 경우

> **면적측정 불필요**: 합병, 지목변경, 지번변경, 행정구역변경, 위치정정

### 면적의 계산 및 끝수처리

| 축척 | 등록면적 단위 | 끝수처리 기준 |
|------|-------------|--------------|
| 1/600, 경계점좌표등록부 | **0.1㎡** | 0.05㎡ 기준 |
| 1/500~1/2,400 | **1㎡** | 0.5㎡ 기준 |
| 1/3,000~1/6,000 | **1㎡** | 0.5㎡ 기준 |

### 5사5입법 (끝수처리 상세 방법)

모든 면적의 결정 및 방위각·종횡선·거리 계산에 적용:

**1㎡ 단위 등록 (1/500~1/6,000)**:
- 0.5㎡ **미만** → **버림**
- 0.5㎡ **초과** → **올림**
- 0.5㎡ **일 때** → 구하려는 끝자리 숫자가 **0 또는 짝수이면 버리고**, **홀수이면 올림**
- 1필지 면적이 1㎡ 미만 → **1㎡로** 함

**0.1㎡ 단위 등록 (1/600, 경계점좌표등록부)**:
- 0.05㎡ **미만** → **버림**
- 0.05㎡ **초과** → **올림**
- 0.05㎡ **일 때** → 구하려는 끝자리 숫자가 **0 또는 짝수이면 버리고**, **홀수이면 올림**
- 1필지 면적이 0.1㎡ 미만 → **0.1㎡로** 함

**적용 예시 (0.1㎡ 단위)**:

| 산출면적 | 등록면적 | 이유 |
|----------|----------|------|
| 123.34 | 123.3 | 0.04 < 0.05 → 버림 |
| 123.35 | 123.4 | 0.05이고 앞자리 3(홀수) → 올림 |
| 123.36 | 123.4 | 0.06 > 0.05 → 올림 |
| 123.45 | 123.4 | 0.05이고 앞자리 4(짝수) → 버림 |
| 123.451 | 123.5 | 0.051 > 0.05 → 올림 |

> **산출면적**: 모든 경우 1/1,000㎡까지 계산하여 1/10㎡ 단위로 정함

> **시험 포인트**: 큰 축척(1/600)과 경계점좌표등록부는 0.1㎡ 단위, 나머지는 1㎡ 단위. **5사5입법**은 반올림과 다름 — 정확히 0.5 또는 0.05일 때 짝수는 버리고 홀수는 올림

### 면적의 측정
- 원칙적으로 경계점의 좌표에 의하여 전자계산조직으로 계산
- 경계점좌표등록부 비치지역 외의 지역에서는 도해법에 의함

## 3. 축척과 도면 포용면적의 관계

지적도의 도곽 크기: 가로 40cm, 세로 30cm

| 축척 | 지상거리 (세로×가로) | 지상면적 |
|------|---------------------|----------|
| 1/500 | 150m × 200m | 3만㎡ |
| 1/600 | 200m × 250m | 5만㎡ |
| 1/1,000 | 300m × 400m | 12만㎡ |
| 1/1,200 | 400m × 500m | 20만㎡ |
| 1/2,400 | 800m × 1,000m | 80만㎡ |
| 1/3,000 | 1,200m × 1,500m | 180만㎡ |
| 1/6,000 | 2,400m × 3,000m | 720만㎡ |

> **시험 포인트**: 축척 = 도상거리 / 지상거리. 1/1,200이면 도면 1cm = 지상 12m

## 핵심 암기 포인트

- **경계**: 필지를 구획하는 선 (지적도·임야도 등록 또는 경계점좌표등록부 좌표)
- **경계결정 5원칙**: 경계국정주의, 경계직선주의, 경계불가분의 원칙, 축척종대의 원칙, 부동성의 원칙
- **지상경계 결정 기준**: ①높낮이 없으면 중앙, ②높낮이 있으면 하단부, ③절토면 상단부, ④해면·수면 최대만조위·최대만수위, ⑤매립지 제방 바깥쪽 어깨
- **소유자 다른 경우**: ①~③에 불구하고 소유권에 따라 결정
- **면적 단위**: ㎡ (수평면상 넓이). 1평 = 400/121㎡(약 3.3㎡)
- **면적 끝수처리**: 1/600·경계점좌표등록부 = **0.1㎡**, 나머지 = **1㎡**
- **5사5입법**: 0.5(0.05)미만 버림, 초과 올림, 정확히 0.5(0.05)일 때 짝수 버리고 홀수 올림
- **면적측정 불필요 경우**: 합병, 지목변경, 지번변경, 행정구역변경, 위치정정
- **도곽 크기**: 가로 40cm × 세로 30cm
- **주요 축척별 지상면적**: 1/500 = 3만㎡, 1/1,200 = 20만㎡, 1/6,000 = 720만㎡
- **축척종대의 원칙**: 동일 경계가 다른 축척 도면에 등록 시 축척이 **큰** 도면에 따름

## OX 퀴즈

**Q1.** 축척 1/500 지적도의 면적은 1㎡ 단위로 산출한다. → **X** (1/500은 0.1㎡ 단위)

**Q2.** 축척 1/3,000 지적도의 면적은 1㎡ 단위로 산출한다. → **X** (1/3,000은 10㎡ 단위)

**Q3.** 경계점좌표등록부 비치지역의 면적은 0.1㎡ 단위로 산출한다. → **O**

**Q4.** 경계는 지상에 표시된 경계선 자체를 말한다. → **X** (지적도·임야도에 등록한 것 또는 경계점좌표등록부에 좌표로 등록한 것)

**Q5.** 축척 1/1,200 지적도에서 도면 1cm는 지상 120m에 해당한다. → **X** (지상 12m에 해당)

## 함정 노트

- ⚠️ **면적 끝수처리 구분**: 큰 축척(1/500, 1/600) = 0.1㎡, 중간 축척(1/1,000~1/2,400) = 1㎡, 작은 축척(1/3,000, 1/6,000) = 10㎡ — 축척 크기와 면적 단위의 관계를 혼동하면 안 됨
- ⚠️ **경계점좌표등록부**: 축척이 아닌 좌표이지만 면적 끝수는 0.1㎡ — 별도 암기 필요
- ⚠️ **축척 계산**: 1/1,200에서 1cm = 12m (120m로 자릿수 실수하는 함정 빈출)
