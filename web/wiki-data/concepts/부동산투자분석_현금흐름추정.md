# 현금흐름(수지)의 추정

> 📖 상위 문서: [[부동산투자분석]]

## 1. 의의

- 부동산 투자안의 수익과 위험을 측정하기 위해서는 먼저 **현금흐름(수지)이 추정**되어야 함
- 현금흐름 = **영업 현금흐름(소득이득)** + **매각 현금흐름(자본이득)**
- 예상**보유기간**에 국한하여 추정

## 2. 영업소득(소득이득) 산출과정

### 단계별 산출 흐름

```
Ⅰ. 가능총소득(PGI) ─── 만실 시 최대 임대소득
    (-) 공실 및 불량부채(대손충당금)
    (+) 기타소득(영업외수입: 주차료, 자판기 등)
    ─────────────────────────
Ⅱ. 유효총소득(EGI)
    (-) 영업경비(OE)
    ─────────────────────────
Ⅲ. 순영업소득(NOI)
    (-) 부채서비스액(DS: 원금+이자 상환액)
    ─────────────────────────
Ⅳ. 세전현금흐름(BTCF)
    (-) 영업소득세
    ─────────────────────────
Ⅴ. 세후현금흐름(ATCF)
```

### 각 단계 상세

| 항목 | 내용 |
|------|------|
| **PGI(가능총소득)** | 단위당 임대료 x 임대단위 수. 만실(100% 임대) 가정 최대 소득 |
| **공실·대손충당금** | 현재 공실 없어도 **최소 공실률 반드시 계상** |
| **기타소득** | 임대 외 수입 (주차료, 관리수입, 자판기, 전용선 임대료 등) |
| **EGI(유효총소득)** | PGI - 공실·대손충당금 + 기타소득 |
| **OE(영업경비)** | 부동산 운영 비용 |
| **NOI(순영업소득)** | EGI - OE |
| **DS(부채서비스액)** | 매기 원금상환 + 이자지급 |
| **BTCF(세전현금흐름)** | NOI - DS. **지분투자자** 귀속 세전소득 |
| **ATCF(세후현금흐름)** | BTCF - 영업소득세 |

### 영업경비(OE) 포함/불포함 항목

| 포함하는 항목 | **포함하지 않는** 항목 |
|-------------|---------------------|
| 보험료 | **공실 및 대손충당금** |
| 재산세 | **부채서비스액(DS)** |
| 유지·관리비 | **영업소득세** |
| 광고비 | **자본이득세** |
| 계약서비스 | **건물의 감가상각비** |
| 수선비 | **건물 취득세 및 등록세** |
| 비품 감가상각비 | **소유자 봉급·개인 업무비** |
| 편익시설사용료 | |

> **핵심 주의**: 영업경비에 **감가상각비(건물), 대출상환액(DS), 소득세 불포함!**

## 3. 매각소득(자본이득) 산출과정

```
Ⅰ. 매도가격
    (-) 매도비용
    ─────────────────────────
Ⅱ. 순매도액
    (-) 미상환저당잔금
    ─────────────────────────
Ⅲ. 세전지분복귀액
    (-) 자본이득세
    ─────────────────────────
Ⅳ. 세후지분복귀액
```

## 4. 영업소득세 계산

| NOI 기준 출발 | BTCF 기준 출발 |
|-------------|--------------|
| NOI | BTCF |
| (+) 대체충당금 | (+) 대체충당금 |
| (-) 이자지급분 | (+) 원금상환분 |
| (-) 감가상각분 | (-) 감가상각분 |
| = 과세대상소득 | = 과세대상소득 |
| x 세율 = 영업소득세 | x 세율 = 영업소득세 |

### 세금 공제 여부

- **이자지급분**: 세금 **공제** O
- **원금상환분**: 세금 공제 **X** (부동산 가치 상승이므로)
- **감가상각분**: 세금 **공제** O (개인 주택은 비공제)
- **대체충당금**: 세금 공제 **X** (자본적 지출비)

## 5. 중요 관계

- **자기자본으로만 투자**: NOI = BTCF (DS가 없으므로)
- 과세대상 소득이 적자가 아니고 과세 대상이라면: **BTCF > ATCF**
- 매각 시 미상환 대출잔액 있다면: **세전 매각현금흐름 < 총 매각대금**

---

## 핵심 암기 포인트

- 산출 순서: **PGI → EGI → NOI → BTCF → ATCF**
- PGI = **만실** 최대 임대소득 / EGI = PGI - 공실 + 기타소득
- NOI = EGI - **영업경비(OE)** / BTCF = NOI - **DS** / ATCF = BTCF - **소득세**
- 영업경비 **불포함**: 감가상각비, DS, 소득세, 자본이득세
- 자기자본만 투자 시: **NOI = BTCF**
- 이자: 세금공제 **O** / 원금상환: 세금공제 **X**

## OX 퀴즈

**Q1.** 영업경비에는 임대소득에 대한 소득세가 포함되어야 한다. → **X** (소득세는 영업경비에 **불포함**)

**Q2.** 부채서비스액은 유효총소득을 산정하는 데 필요한 항목이다. → **X** (DS는 EGI 산정에 불필요. NOI에서 차감하여 BTCF 산정 시 필요)

**Q3.** 세전현금흐름은 순영업소득에서 부채서비스액을 차감한 소득이다. → **O**

**Q4.** 순영업소득은 세전현금흐름과 동일할 수 없다. → **X** (자기자본만 투자 시 **동일 가능**)

**Q5.** 회수불가능한 임대료수입은 영업경비에 포함하여 NOI를 산정한다. → **X** (회수불가능 임대료(대손충당금)는 **EGI 산정 시** PGI에서 차감)

**Q6.** 세후현금흐름은 세전현금흐름에서 영업소득세를 차감한 소득이다. → **O**

**Q7.** 세전현금흐름은 순영업소득에 부채서비스액을 가산한 소득이다. → **X** (**차감**한 소득)

**Q8.** 과세대상 소득이 적자가 아니고 과세대상이라면 BTCF는 ATCF보다 크다. → **O**

## 함정 노트

- ⚠️ "영업경비에 소득세 포함" → 소득세는 **불포함** (DS, 감가상각비, 자본이득세도 불포함)
- ⚠️ "DS = EGI 산정 항목" → DS는 EGI 산정과 무관, **BTCF 산정 시** 차감
- ⚠️ "BTCF = NOI + DS" → BTCF = NOI **-** DS (가산이 아니라 **차감**)
- ⚠️ "NOI ≠ BTCF 항상" → 자기자본만 투자하면 DS=0이므로 **NOI = BTCF**
- ⚠️ "대손충당금 = 영업경비" → 대손충당금은 **EGI 산정 시 PGI에서 차감**, 영업경비 아님
- ⚠️ "원금상환분 세금공제 가능" → 원금상환분은 세금공제 **불가** (가치 상승분이므로)
