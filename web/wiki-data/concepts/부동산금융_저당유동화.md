# 주택저당채권 유동화 (MBS)

> 📖 상위 문서: [[부동산금융론]]

## 1. 의의

- 비현금자산인 **주택저당채권을 담보 또는 기초**로 하여 새로운 증권을 발행하여 **현금화**하는 수단
- 자본시장의 자금을 간접적으로 **부동산시장으로** 유입
- 유동화 조건: **대출이자율 > 저당수익률 > 요구수익률**

## 2. 저당시장의 구조

| 구분 | 1차 저당시장 | 2차 저당시장 |
|------|-------------|-------------|
| 별명 | **주택담보대출시장** | **주택자금공급시장** |
| 기능 | 저당채권이 **형성**되는 시장 | 저당채권이 **유동화**되는 시장 |
| 참여자 | 대출수요자 + 금융기관 | 대출기관 + **SPC** + 투자자 |
| 역할 | 대출 실행 | 1차 시장에 **자금 공급** |

- **SPC(특별목적회사)**: 주택저당증권 발행을 목적으로 설립. 대출기관과 자본시장을 **연계**
- 우리나라: **한국주택금융공사**가 SPC 역할

## 3. 유동화증권(MBS)의 종류

### 종합 비교표

| 구분 | MPTS(지분형) | MBB(채권형) | MPTB(혼합형) | CMO(혼합형) |
|------|-------------|-------------|-------------|-------------|
| **저당채권 소유권** | **투자자** | 발행자 | 발행자 | 발행자 |
| **원리금수취권** | 투자자 | **발행자** | **투자자** | **투자자** |
| **조기상환위험** | **투자자** | 발행자 | **투자자** | **투자자** |
| **채무불이행위험** | **투자자** | 발행자 | **발행자** | **발행자** |
| **콜방어기능** | **없다** | **있다** | **없다** | **있다** |

### MPTS(Mortgage Pass-Through Securities) - 지분형 MBS

- 차입자의 원리금상환액이 투자자에게 **그대로 전달(pass through)**
- 소유권·원리금수취권 **모두 투자자**
- **모든 위험**(채무불이행·조기상환) = **투자자 부담**
- 콜방어기능 **없음**
- 발행액 = 주택저당채권 총액 (같음)

### MBB(Mortgage Backed Bond) - 채권형 MBS (이중담보)

- 저당채권을 **담보**로 발행기관이 채권 발행 → 투자자에게 매각
- 소유권·원리금수취권 **모두 발행자** 보유
- **모든 위험** = **발행자 부담** → 투자자에게 **안정적**
- 콜방어기능 **있음** (만기전 변제하지 않는 약속)
- **초과담보** 제공 필요 (발행액 < 저당채권 총액)
- MPTS보다 현금흐름이 **안정적**이고 확실성이 큼

### MPTB(Mortgage Pay-Through Bond) - 혼합형 MBS (전이)

- MBB + MPTS의 **혼합** 형태
- 소유권 = **발행자**, 원리금수취권 = **투자자**
- 채무불이행위험 = **발행자**, 조기상환위험 = **투자자**
- 콜방어기능 **없음**

### CMO(Collateralized Mortgage Obligation) - 혼합형 MBS (트렌치)

- MPTB의 **단점 보완**, 다양한 **트렌치(tranche)** 구성
- 만기별로 그룹을 나누어 **상이한 이자율** 적용, **원금 지급 순서** 차등
- 소유권 = **발행자**, 원리금수취권 = **투자자**
- 채무불이행위험 = **발행자**, 조기상환위험 = **투자자**
- 콜방어기능 **있음** (이전 트렌치 원금 지급 완료 전까지 다음 트렌치 원금 미지급)

## 4. MBS 발행 효과

- 주택금융시장과 **자본시장의 연결**
- 금융기관의 **유동성 확보** → 추가 대출 여력
- **장기·저금리** 주택자금 공급 활성화
- 주택구입 기회 **증가**, 자가소유가구 비중 **증가**
- 대출기관의 **재무상태 건전성** 증가

## 핵심 암기 포인트

- MPTS(지분형): 소유권·원리금 **투자자**, 모든 위험 **투자자**, 콜방어 **없음**
- MBB(채권형): 소유권·원리금 **발행자**, 모든 위험 **발행자**, 콜방어 **있음**, **초과담보**
- MPTB(혼합): 소유권 **발행자**, 원리금 **투자자**, 채무불이행=발행자, 조기상환=투자자
- CMO(혼합): MPTB와 동일 구조 + **트렌치** 구분, 콜방어 **있음**
- 콜방어: MBB(**있음**), CMO(**있음**) / MPTS(**없음**), MPTB(**없음**)

## OX 퀴즈

**Q1.** MPTS의 투자자는 조기상환위험을 부담하지 않는다. → **X** (MPTS는 **모든 위험 투자자** 부담)

**Q2.** MBB는 지분형 증권이다. → **X** (**채권형** 증권. 지분형은 MPTS)

**Q3.** MBB의 투자자는 주택저당채권 집합물에 대한 소유권을 갖는다. → **X** (MBB 소유권은 **발행자**. 투자자가 소유권 보유는 MPTS)

**Q4.** MBB 발행자는 초과담보를 제공하는 것이 일반적이다. → **O**

**Q5.** CMO는 트렌치별로 적용되는 이자율이 동일하다. → **X** (트렌치별로 **상이한** 이자율)

**Q6.** MPTB의 주택저당채권 소유권은 투자자에게 이전된다. → **X** (MPTB 소유권은 **발행자**. 투자자는 원리금수취권)

**Q7.** CMO는 콜방어기능이 없다. → **X** (CMO는 콜방어기능 **있음**)

**Q8.** MBS 발행으로 자가소유가구 비중이 감소한다. → **X** (**증가**)

## 함정 노트

- ⚠️ "MBB: 지분형 증권" → MBB는 **채권형**, 지분형은 **MPTS**
- ⚠️ "MBB 소유권 = 투자자" → MBB 소유권·원리금수취권 모두 **발행자** (투자자 갖는 것은 MPTS)
- ⚠️ "MPTB 소유권 = 투자자" → MPTB 소유권 = **발행자**, 원리금수취권만 투자자 (제35회, 감평 2020 출제)
- ⚠️ "MPTB 채무불이행위험 = 투자자" → MPTB 채무불이행위험 = **발행자** (제32회 출제)
- ⚠️ "CMO 조기상환위험 = 발행자" → CMO 조기상환위험 = **투자자** (제15회 출제)
- ⚠️ "콜방어: MPTS 있음, MBB 없음" → **반대**. MBB·CMO만 콜방어 있음
