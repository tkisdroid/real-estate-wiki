# 양도소득세 - 세율

## 1. 기본세율 (2년 이상 보유, 8단계 초과누진)

| 과세표준 | 세율 | 누진공제 |
|----------|:----:|----------|
| 1,400만원 이하 | **6%** | - |
| 1,400만원 초과 ~ 5,000만원 이하 | **15%** | 126만원 |
| 5,000만원 초과 ~ 8,800만원 이하 | **24%** | 576만원 |
| 8,800만원 초과 ~ 1.5억원 이하 | **35%** | 1,544만원 |
| 1.5억원 초과 ~ 3억원 이하 | **38%** | 1,994만원 |
| 3억원 초과 ~ 5억원 이하 | **40%** | 2,594만원 |
| 5억원 초과 ~ 10억원 이하 | **42%** | 3,594만원 |
| 10억원 초과 | **45%** | 6,594만원 |

> **8단계**: 6-15-24-35-38-40-42-45 (소득세 기본세율과 동일)

## 2. 단기보유 세율

### 주택·조합원입주권

| 보유기간 | 세율 |
|----------|:----:|
| **1년 미만** | **70%** |
| **1년 이상 ~ 2년 미만** | **60%** |
| 2년 이상 | 기본세율 (6~45%) |

### 분양권

| 보유기간 | 세율 |
|----------|:----:|
| **1년 미만** | **70%** |
| **1년 이상** | **60%** |

> **분양권**: 보유기간과 무관하게 2년 이상이어도 **60%** (기본세율 적용 없음)

### 토지·건물 (주택 외)

| 보유기간 | 세율 |
|----------|:----:|
| **1년 미만** | **50%** |
| **1년 이상 ~ 2년 미만** | **40%** |
| 2년 이상 | 기본세율 |

## 3. 미등기양도자산

| 항목 | 내용 |
|------|------|
| **세율** | **70%** (보유기간 무관) |
| **비과세·감면** | 배제 |
| **장기보유특별공제** | 배제 |
| **양도소득기본공제** | 배제 |
| **개산공제율** | **0.3%** (일반 3%의 1/10) |

### 미등기 제외 자산 (미등기로 보지 않는 경우)
- 장기할부조건으로 **등기 불가능**한 자산
- 법률·법원결정에 의해 **등기 불가능**한 자산
- 1세대1주택 **무허가건물** 등

## 4. 비사업용토지

| 항목 | 세율 |
|------|:----:|
| 2년 이상 보유 | 기본세율 **+ 10%** |

> 비사업용토지: 재촌·자경하지 않는 농지, 비업무용 토지 등

## 5. 조정대상지역 내 다주택 중과세

| 구분 | 중과세율 | 장기보유특별공제 |
|------|:--------:|:---------------:|
| 1세대 **2주택** | 기본세율 **+ 20%** | 배제 |
| 1세대 **3주택 이상** | 기본세율 **+ 30%** | 배제 |

> **한시적 중과배제**: 2022.5.10 ~ 2024.5.9 양도분은 기본세율 적용 + 장기보유특별공제 허용 (시행령 개정에 따라 기간 변동 가능)

## 6. 세율 비교 종합표

| 자산 | 보유기간 | 세율 |
|------|----------|:----:|
| **미등기양도** | 기간 무관 | **70%** |
| 주택 1년 미만 | 1년 미만 | **70%** |
| 분양권 1년 미만 | 1년 미만 | **70%** |
| 주택 1~2년 | 1~2년 | **60%** |
| 분양권 1년 이상 | 1년 이상 | **60%** |
| 토지·건물 1년 미만 | 1년 미만 | **50%** |
| 토지·건물 1~2년 | 1~2년 | **40%** |
| 비사업용토지 | 2년 이상 | 기본세율 **+ 10%** |
| 조정 2주택 | 2년 이상 | 기본세율 **+ 20%** |
| 조정 3주택+ | 2년 이상 | 기본세율 **+ 30%** |
| 일반 | 2년 이상 | **기본세율** (6~45%) |

## 7. 세율 적용 우선순위

복수 세율이 적용되는 경우 → **높은 세율** 적용

## 핵심 암기 포인트

- 기본세율 8단계: **6-15-24-35-38-40-42-45%**
- **미등기 70%**: 모든 불이익(비과세 배제, 공제 배제, 개산공제 0.3%)
- **주택 단기**: 1년 미만 70%, 1~2년 60%
- **분양권**: 1년 미만 70%, 1년 이상 60% (2년 이상도 60%)
- **비사업용토지**: 기본세율 **+ 10%**
- **다주택 중과**: 2주택 +20%, 3주택+ +30%
- 기본세율 최고구간: **10억 초과 45%**

## OX 퀴즈

**Q1.** 미등기양도자산의 세율은 50%이다. → **X** (70%)

**Q2.** 분양권을 3년 보유 후 양도하면 기본세율이 적용된다. → **X** (1년 이상은 60%)

**Q3.** 주택을 1년 6개월 보유 후 양도하면 세율은 60%이다. → **O** (1~2년 미만)

**Q4.** 비사업용토지의 세율은 기본세율에 20%를 가산한다. → **X** (10% 가산)

**Q5.** 기본세율의 최저세율은 6%이다. → **O**

**Q6.** 조정대상지역 내 3주택자는 기본세율에 20%를 가산한다. → **X** (30% 가산)

**Q7.** 미등기양도자산에는 양도소득기본공제가 적용된다. → **X** (배제)

**Q8.** 토지·건물(주택 외)을 6개월 보유 후 양도하면 세율은 70%이다. → **X** (50%, 70%는 주택)

## 함정 노트

- **주택 vs 토지·건물 단기세율**: 주택=70%/60%, 토지·건물(주택 외)=50%/40%. 혼동 주의
- **분양권 60% 고정**: 분양권은 1년 이상이면 보유기간과 무관하게 **60%**. 기본세율 적용 없음
- **미등기 70%와 주택 1년 미만 70%**: 세율은 동일하나 **불이익 범위**가 다름 (미등기는 모든 공제 배제)
- **중과세 + 비사업용토지**: 동시 적용 시 **높은 세율** 적용
- **누진공제 활용**: 산출세액 = 과세표준 x 세율 - 누진공제. 계산문제에서 반드시 사용
