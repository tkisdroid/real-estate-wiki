# 지대이론

지대이론은 토지의 지대(임대료)가 왜, 어떻게 발생하는지를 설명하는 이론이다. 공인중개사 시험에서 각 학자별 지대 발생 원인, 한계지 지대 발생 여부, 지대의 성격(잉여 vs 비용), 곡물가격과의 인과관계가 핵심 출제 포인트이다.

## 세부 내용
| 이론 | 학자 | 핵심 키워드 | 링크 |
|------|------|-------------|------|
| 차액지대설 | 리카도 | 비옥도 차이, 한계지 지대 미발생, 잉여 | [[지대이론_리카도]] |
| 위치지대설 | 튀넨 | 고립국, 수송비, 동심원, 집약도 | [[지대이론_튀넨]] |
| 절대지대설 | 마르크스 | 토지소유, 한계지 지대 발생, 비용 | [[지대이론_마르크스]] |
| 입찰지대이론 | 알론소 | 입찰지대곡선, 효용극대화, 도시 토지이용 | [[지대이론_알론소]] |
| 준지대설·경제적 지대 | 마샬 | 준지대, 전용수입, 경제적 지대, 헤이그 | [[지대이론_마샬]] |

## 핵심 비교표

### 지대이론 종합 비교

| 구분 | 리카도 | 튀넨 | 마르크스 | 알론소 |
|------|--------|------|----------|--------|
| **저서/연도** | 경제학과 과세의 원리(1817) | 고립국(1826) | - | - |
| **지대 결정 요인** | 비옥도 차이 | 거리(수송비) | 토지소유 | 입찰지대(수송비) |
| **비옥도 가정** | 차이 있음 | **동일** | 무관 | - |
| **한계지 지대** | **미발생** | **미발생** | **발생** | **미발생** |
| **지대 성격** | 잉여 | 잉여 | **비용** | 잉여 |
| **곡물가격 관계** | 가격→지대 | 가격→지대 | **지대→가격** | - |

### 지대 공식 비교

| 학자 | 공식 |
|------|------|
| 리카도 | 지대 = 매상고 - 생산비 |
| 튀넨 | 지대 = 매상고 - 생산비 - **수송비** |
| 알론소 | 지대 = 매상고 - 생산비 - **운송비** |
| 헤이그 | 지대 = **마찰비용 - 교통비** |

### 준지대·경제적 지대·전용수입

| 개념 | 내용 |
|------|------|
| **준지대** | 인공 고정생산요소(기계·기구)의 소득, **단기**에만 발생 |
| **전용수입** | 현재 용도에 머물도록 하는 **최소 지급액** = 기회비용 |
| **경제적 지대** | 전용수입을 **초과하는** 소득 = 생산자잉여 |

### 공급탄력성과 경제적 지대/전용수입

| 공급탄력성 | 경제적 지대 | 전용수입 |
|-----------|------------|---------|
| 완전비탄력적 | 전액 | 0 |
| 완전탄력적 | 0 | 전액 |

## 이론 간 영향 관계

```
리카도(1817) ──재편성──→ 마샬(준지대설)
    │
    ├──보완──→ 마르크스(절대지대설)
    │
튀넨(1826) ──도시적용──→ 알론소(입찰지대이론)
    │
    └──영향──→ 버제스(동심원이론)
```
