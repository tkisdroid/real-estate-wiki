# 납세의무의 성립·확정·소멸

## 1. 납세의무의 성립

세법이 정한 과세요건이 충족됨으로써 추상적 납세의무가 발생하는 시점이다.

### 주요 세목별 성립시기

| 세목 | 성립시기 |
|------|----------|
| [[취득세]] | 과세물건을 **취득하는 때** |
| [[등록면허세]] (등록) | 재산권 등을 **등기·등록하는 때** |
| [[재산세]] | **과세기준일 (6월 1일)** |
| 소방분 지역자원시설세 | **과세기준일 (6월 1일)** |
| 지방교육세 | 본세의 납세의무 **성립시** |
| [[종합부동산세]] | **과세기준일 (6월 1일)** |
| 소득세·법인세 | **과세기간이 끝나는 때** |
| 상속세 | **상속이 개시되는 때** |
| 증여세 | 재산을 **증여에 의해 취득하는 때** |
| 농어촌특별세 | **본세의 납세의무 성립시** |

## 2. 납세의무의 확정

추상적으로 성립한 납세의무를 구체적으로 확정하는 절차이다.

| 확정방법 | 내용 | 해당 세목 |
|----------|------|-----------|
| **신고주의** | 납세의무자의 **신고**로 확정 | [[취득세]], [[등록면허세]], [[양도소득세]], 지방소득세 등 |
| **부과주의** | 과세관청의 **결정**으로 확정 | [[재산세]], [[종합부동산세]], 상속세, 증여세 등 |
| **자동확정** | 성립과 **동시에** 확정 | 인지세, 원천징수 소득세, 특별징수 지방소득세 |

> **신고하는 조세**를 신고하지 않거나 과세표준이 법령에 어긋나는 경우 → 과세관청이 **결정·경정**

## 3. 납세의무의 소멸

### 소멸사유

| 구분 | 소멸사유 | 비고 |
|------|----------|------|
| 조세채권의 **실현** | ① 납부(납입) | 본래·연대·제2차 납세의무자 포함 |
| | ② 충당 | 환급금을 미납세액에 직권 상계 |
| 조세채권의 **미실현** | ③ 부과의 취소 | 처음부터 부과처분이 없었던 것으로 봄 |
| | ④ **부과권 제척기간** 만료 | 중단·정지 **없음** |
| | ⑤ **징수권 소멸시효** 완성 | 중단·정지 **있음** |

### 소멸사유가 **아닌** 것

- 납세의무자의 **사망** → 상속인에게 승계
- 법인의 **합병** → 합병법인에게 승계
- **부과철회** → 재산 발견 시 취소하고 징수 가능
- **결손처분** → 다른 재산 취득 시 취소하고 체납처분 가능

### 제척기간

**부과할 수 있는 권리의 존속기간** (중단·정지 없음)

| 사유 | 일반국세 | 상속·증여세 | 지방세 |
|------|----------|-------------|--------|
| **부정행위로 포탈** | 10년 (역외: 15년) | 15년 | 10년 (역외: 15년) |
| **무신고** | 7년 (역외: 10년) | 15년 | 7년 (역외: 10년) |
| **기타** | 5년 (역외: 7년) | 10년 | 5년 (역외: 7년) |

**기산일**:
- 신고하는 조세 ([[취득세]], [[등록면허세]], [[양도소득세]]): **신고기한의 다음 날**
- 결정하는 조세 ([[재산세]], [[종합부동산세]]): **납세의무성립일** (과세기준일)

### 소멸시효

**징수할 수 있는 권리의 불행사 기간** (중단·정지 있음)

| 세액 | 기간 |
|------|------|
| 지방세 5,000만원 미만 / 국세 5억원 미만 | **5년** |
| 지방세 5,000만원 이상 / 국세 5억원 이상 | **10년** |

**기산일**:
- 신고하는 조세: **신고납부기한의 다음 날**
- 결정하는 조세: **고지에 따른 납부기한의 다음 날**

### 소멸시효의 중단·정지

| 중단사유 | 정지사유 |
|----------|----------|
| 납세고지 | 분납기간 |
| 독촉(납부최고) | 징수유예기간 |
| 교부청구 | 체납처분유예기간 |
| 압류 | 연부연납기간 |
| | 사해행위 취소소송·채권자대위소송 진행 기간 |
| | 체납자 국외 6개월 이상 체류기간 |

> **중단**: 그때까지의 시효기간 **효력 상실**, 새로 시작
> **정지**: 진행을 일시 **멈추고**, 정지기간 종료 후 잔여기간 계속

### 제척기간과 소멸시효 비교

| 구분 | 제척기간 | 소멸시효 |
|------|----------|----------|
| 대상 | **부과권** | **징수권** |
| 기간 | 5, 7, 10, 15년 | 5, 10년 |
| 기산일 | 부과할 수 있는 날 | 징수권 행사할 수 있는 날 |
| 중단·정지 | **없음** | **있음** |

## 핵심 암기 포인트

- **성립시기**: 취득세=취득 시, 등록면허세=등기·등록 시, 재산세·종부세=6월 1일, 소득세=과세기간 종료, 상속세=상속개시, 증여세=증여취득 시
- **확정방법**: 신고주의(취득세·등록면허세·양도소득세), 부과주의(재산세·종부세·상속세·증여세), 자동확정(인지세·원천징수)
- **소멸사유 5가지**: 납부, 충당, 부과의 취소, 부과권 제척기간 만료, 징수권 소멸시효 완성
- **소멸사유 아닌 것**: 사망(상속인 승계), 합병(합병법인 승계), 부과철회, 결손처분
- **제척기간**: 부정행위 10년(상속·증여 15년), 무신고 7년(상속·증여 15년), 기타 5년(상속·증여 10년). 중단·정지 없음
- **제척기간 기산일**: 신고조세 = 신고기한 다음 날, 결정조세 = 납세의무성립일
- **소멸시효**: 5천만원(국세 5억원) 미만 = 5년, 이상 = 10년. 중단·정지 있음
- **소멸시효 중단사유**: 납세고지, 독촉(납부최고), 교부청구, 압류
- **소멸시효 정지사유**: 분납기간, 징수유예, 체납처분유예, 연부연납, 소송기간, 국외 6개월 이상 체류

## OX 퀴즈

**Q1.** 부과권의 제척기간에는 중단·정지 사유가 있다. → **X** (없음. 소멸시효에만 있음)

**Q2.** 납세의무자의 사망은 납세의무의 소멸사유이다. → **X** (상속인에게 승계)

**Q3.** 결손처분은 납세의무의 소멸사유이다. → **X** (재산 발견 시 취소하고 체납처분 가능)

**Q4.** 제척기간과 소멸시효의 대상은 동일하다. → **X** (제척기간=부과권, 소멸시효=징수권)

**Q5.** 무신고 시 일반국세의 부과 제척기간은 10년이다. → **X** (7년. 10년은 부정행위)

**Q6.** 징수권 소멸시효의 중단사유에 압류가 포함된다. → **O**

**Q7.** 재산세의 부과 제척기간 기산일은 신고기한의 다음 날이다. → **X** (재산세는 부과주의이므로 납세의무성립일=과세기준일)

## 함정 노트

- ⚠️ **제척기간 vs 소멸시효**: 제척기간=부과권(중단·정지 없음), 소멸시효=징수권(중단·정지 있음) — 반대 적용 금지
- ⚠️ **제척기간 기산일**: 신고조세(취득세·양도소득세 등)는 신고기한 다음 날, 결정조세(재산세·종부세)는 성립일 — 구분 필수
- ⚠️ **소멸사유가 아닌 것**: 사망·합병·부과철회·결손처분 — 소멸처럼 보이지만 소멸사유 아님
- ⚠️ **중단 vs 정지**: 중단 = 시효 리셋(새로 시작), 정지 = 일시 멈춤(잔여기간 계속) — 효과가 다름
