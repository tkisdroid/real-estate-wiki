# 부동산 금융론

부동산 금융론은 부동산 관련 자금의 조달·운용·회수에 관한 이론이다. 저당대출의 상환방식, 저당채권의 유동화, 간접투자(REITs), 프로젝트파이낸싱, 주택연금 등을 다룬다.

## 비교 정리

### 상환방식 비교

| 구분 | 원리금균등 | 원금균등 | 체증식 |
|------|-----------|---------|--------|
| 매기 상환액 | **일정** | **체감** | **체증** |
| 초기 부담 | 중간 | **가장 큼** | **가장 작음** |
| 총이자부담 | 중간 | **가장 작음** | **가장 큼** |

### MBS 종류 비교

| 구분 | MPTS | MBB | MPTB | CMO |
|------|------|-----|------|-----|
| 유형 | 지분형 | 채권형 | 혼합형 | 혼합형 |
| 소유권 | 투자자 | 발행자 | 발행자 | 발행자 |
| 원리금수취권 | 투자자 | 발행자 | 투자자 | 투자자 |
| 콜방어 | 없음 | **있음** | 없음 | **있음** |

### 대출 규제 비교

| 규제 | 기준 | 범위 |
|------|------|------|
| LTV | 담보가치 대비 | 대출금 |
| DTI | 연소득 대비 | 해당 대출 상환액 |
| DSR | 연소득 대비 | **모든 대출** 상환액 (가장 엄격) |

## 2. 부동산금융의 기본개념

- 부동산금융: 부동산 취득·개발에 필요한 자금을 융통하는 것
- **간접금융**(금융기관 경유)이 더 발달
- 감가상각비 및 대출금 이자에 대한 **세금감면** 혜택
- 주택금융이 부동산금융의 주종

### 주택금융 구분

| 구분 | 내용 | 목적 |
|------|------|------|
| **주택소비금융** | 주택 담보로 구입·개량 자금 융자 | **소비자** 금융 |
| **주택개발금융** | 건설업자에게 건축자금 대출 | **공급자** 금융 |

### 부채금융 vs 지분금융

| 부채금융(타인자본 조달) | 지분금융(자기자본 조달) |
|------------------------|----------------------|
| 주택저당채권담보부채권(MBB) | 부동산 신디케이션 |
| 주택저당증권(MBS) | 조인트벤처 |
| 주택상환사채 | **부동산투자신탁(REITs)** |
| 자산유동화증권(ABS) | 부동산간접투자펀드 |
| ABCP | 공모에 의한 증자 |
| **프로젝트금융(PF)** | - |
| 주택저당대출 | - |

### 메자닌 금융 (지분금융 + 부채금융 혼합)

- **전환사채**, 자산매입조건부대출, **신주인수권부사채**, 우선주, **후순위대출**

### 저당위험

| 위험 | 내용 |
|------|------|
| **채무불이행위험** | 차입자의 원리금 상환의무 미이행. 시간 경과 → 위험 **낮아짐** |
| **조기상환위험** | 시장금리 < 약정금리 시 차입자가 만기 전 상환. 대출자 이자수익 손실 |
| **유동성위험** | 장기 대출채권의 현금화 어려움 → **저당유동화** 필요 |

### 고정금리 vs 변동금리

| 구분 | 고정금리 | 변동금리 |
|------|---------|---------|
| 금리 | 전체기간 **동일** | 기준금리에 연동 변동 |
| **대출취급 당시 금리** | **높음** | **낮음** |
| 인플레 위험 부담 | **대출자** | **차입자** |
| 채무불이행위험 | 상대적 낮음 | 상대적 **높음** |
| 조기상환위험 | 시장금리 하락 시 발생 | 적음 |

- 변동금리: **대출금리 = 기준금리 + 가산금리 - 우대금리**
- 기준금리: CD(양도성예금증서), **COFIX**(은행자금조달비용 반영)
- 시장금리 **상승 시** 금리조정주기 **짧을수록** → 대출자 유리 (위험을 차입자에게 신속 전가)
