# 중개대상물 확인·설명의무 (허브)

## 개요

개업공인중개사는 부동산거래 전문가로서 중개대상물의 각종 사항을 조사·확인하여 **권리를 취득하고자 하는 의뢰인**에게 설명할 의무가 있다(법 제25조). 이 의무는 확인·설명사항(내용)과 확인·설명서(문서)로 구분된다.

## 설명사항 vs 확인설명서 비교

| 구분 | 설명사항 | 확인설명서 |
|---|---|---|
| **성격** | 구두·서면 설명 의무 | 서면 작성·교부·보관 의무 |
| **시기** | 중개의뢰 후 ~ 중개완성 전 | 거래계약 체결 시 작성 |
| **상대방** | 권리취득 의뢰인(매수인, 임차인 등) | 거래당사자 **쌍방**에게 교부 |
| **위반 제재** | 500만원 이하 과태료 | 업무정지 (서명날인·교부·보관 위반) |
| **근거자료** | 반드시 제시 | 서식에 기재 |

## 주요 판례 요약

| 판례 | 내용 |
|---|---|
| 대판 2002.2.5 | **무료 중개**라도 확인·설명의무 감경 안 됨 |
| 대판 1993.5.11 | **매도인의 진정한 권리자** 확인의무 있음 |
| 수원지법 1997.8.26 | 잔금 지급 전 **등기사항증명서 재확인** 의무 |
| 서울남부지원 2000.2.11 | 선순위 임차인 미설명 → 손해배상 |
