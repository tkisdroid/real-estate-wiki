# 행정처분 - 개업공인중개사 등록취소 (법 제38조)

## 1. 개요

| 항목 | 내용 |
|------|------|
| **처분 대상** | 개업공인중개사 |
| **처분청** | **등록관청** |
| **처분 성격** | **절대적 + 임의적** 혼합 |
| **청문** | 대상 **O** (사망·해산 제외) |

## 2. 절대적 등록취소 사유 (법 제38조 제1항) — "취소하여야 한다"

| 번호 | 사유 | 3년 결격사유 적용 |
|------|------|:-:|
| 1 | 개인 **사망** / 법인 **해산** | **X** (청문 대상도 아님) |
| 2 | **거짓·부정** 등록 | **O** |
| 3 | **결격사유** 해당 (법인 임원·사원 **2개월** 내 해소 시 제외) | **X** (결격사유 자체 기간만 적용) |
| 4 | **이중등록** (2개 이상 중개사무소 개설등록) | **O** |
| 5 | **이중소속** (다른 개업공인중개사의 소속·보조원·사원·임원) | **O** |
| 6 | 등록증 **양도·대여** (성명·상호 사용하여 업무 하게 한 경우 포함) | **O** |
| 7 | **업무정지 기간 중 업무** 수행 또는 자격정지 소속공인중개사에게 업무 시킨 경우 | **O** |
| 8 | 1년 내 **2회 업무정지** 후 다시 업무정지 사유 행위 | **O** |
| 9 | **중개보조원 채용제한** 위반 (공인중개사+소속공인중개사 합산 수의 5배 초과) | **O** |

### 3년 결격사유 적용 여부 정리
- **적용됨**: 2, 4, 5, 6, 7, 8, 9번
- **적용 안 됨**: 1번(사망·해산), 3번(결격사유 해당 — 별도 결격기간)

### 결격사유 해당 시 세부 사항
- 피성년후견인·피한정후견인
- 파산선고
- 금고 이상 실형·집행유예
- 자격 취소
- **이 법 위반 벌금형** 선고
- 법인 사원·임원 결격 → **2개월 내 해소** 시 등록취소 제외

## 3. 임의적 등록취소 사유 (법 제38조 제2항) — "취소할 수 있다"

| 번호 | 사유 | 3년 결격사유 적용 |
|------|------|:-:|
| 1 | **등록기준 미달** | **X** |
| 2 | **2 이상 사무소** 설치 | O |
| 3 | **임시 중개시설물** 설치 | O |
| 4 | **법인 겸업** 위반 (법 제14조 외 겸업) | O |
| 5 | **6개월 초과 휴업** (무신고) | O |
| 6 | 전속중개계약 **정보공개 위반** (비공개 요청 무시 등) | O |
| 7 | 거래계약서 **거짓기재** / **이중계약서** 작성 | O |
| 8 | **업무보증 미설정** 후 업무 개시 | O |
| 9 | **금지행위** 위반 (법 제33조 제1항) | O |
| 10 | 1년 내 **3회** 업무정지 또는 과태료 후 다시 해당 행위 | O |
| 11 | 고용인 **결격사유 2개월** 내 미해소 | O |

## 4. 절대적 vs 임의적 등록취소 비교표

| 비교 항목 | 절대적 등록취소 | 임의적 등록취소 |
|-----------|:-------------:|:-------------:|
| **재량** | 없음 (취소**하여야**) | 있음 (취소**할 수 있다**) |
| **업무정지 대체** | 불가 | **가능** (업무정지로 대체 가능) |
| **등록기준 미달** | - | 해당 |
| **금지행위** | - | 해당 |
| **이중등록** | 해당 | - |
| **양도·대여** | 해당 | - |

## 5. 등록취소 후 조치

| 항목 | 내용 |
|------|------|
| **등록증 반납** | 취소일부터 **7일 이내** 등록관청에 반납 |
| **미반납 과태료** | **100만원** 이하 과태료 |
| **결격사유 기간** | 등록취소일부터 **3년** (적용 대상 사유에 한함) |
| **법인 해산 시** | 대표자이었던 자가 7일 이내 반납 |

## 6. 등록취소 vs 업무정지 관계

- 임의적 등록취소 사유 → **업무정지**로 대체 가능 (법 제39조)
- 절대적 등록취소 사유 → 업무정지로 대체 **불가**
- 1년 내 **2회** 업무정지 후 다시 사유 → **절대적 등록취소**
- 1년 내 **3회** 업무정지·과태료 후 다시 사유 → **임의적 등록취소**

## 핵심 암기 포인트

- 처분청: **등록관청** (자격취소는 시·도지사)
- 절대적 9개 + 임의적 11개 사유
- **사망·해산**: 절대적이나 3년 결격사유 X, 청문 대상 X
- **결격사유 해당**: 법인 임원·사원은 **2개월** 해소 유예
- **등록기준 미달**: 임의적, 3년 결격사유 X
- **2회 업무정지 → 절대 등록취소** / **3회 업무정지·과태료 → 임의 등록취소**
- 반납: **7일** 이내, 미반납 **100만원** 과태료
- 결격사유: **3년**

## OX 퀴즈

**Q1.** 등록취소는 시·도지사가 한다. → **X** (등록관청)

**Q2.** 개업공인중개사의 사망은 절대적 등록취소 사유이다. → **O**

**Q3.** 사망으로 등록취소된 경우 3년의 결격사유가 적용된다. → **X** (사망·해산은 3년 적용 안 됨)

**Q4.** 등록기준에 미달하게 된 경우 반드시 등록을 취소하여야 한다. → **X** (임의적, "할 수 있다")

**Q5.** 법인 임원이 결격사유에 해당하면 즉시 등록취소된다. → **X** (2개월 내 해소하면 등록취소 제외)

**Q6.** 임의적 등록취소 사유는 업무정지로 대체할 수 있다. → **O**

**Q7.** 1년 내 2회 업무정지 후 다시 사유가 있으면 임의적 등록취소이다. → **X** (절대적 등록취소)

**Q8.** 등록취소 시 청문을 실시하여야 한다. → **O** (단, 사망·해산 제외)

## 함정 노트

- **2회 vs 3회**: 1년 내 2회 업무정지 후 → **절대적** 등록취소, 1년 내 3회 업무정지·과태료 후 → **임의적** 등록취소. 횟수에 따라 절대/임의가 다름
- **거짓기재·이중계약서**: 절대적 등록취소가 아닌 **임의적** 등록취소 사유 (자격정지에서는 6개월)
- **금지행위**: **임의적** 등록취소 사유 (절대적이 아님)
- **결격사유 해당 → 3년 적용 X**: 결격사유 자체의 기간이 적용될 뿐, 등록취소로 인한 3년 추가 적용은 없음
- **업무보증 미설정**: **임의적** 등록취소. "업무보증 미설정=절대적"이라고 착각하지 않도록 주의
