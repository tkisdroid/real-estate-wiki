# 재산세 토지분류

[[재산세]] 과세대상 토지는 **분리과세**, **종합합산**, **별도합산** 세 가지로 구분되며, 이 분류에 따라 세율과 과세방법이 달라진다. [[종합부동산세]] 과세 여부도 이 분류에 따라 결정된다.

## 1. 분류 체계

| 유형 | 과세방법 | 세율 | 종합부동산세 |
|------|----------|------|-------------|
| **분리과세** | 개별 토지별 분리 | 차등비례세율 (0.07%, 0.2%, 4%) | **과세 제외** |
| **종합합산** | 소유자별 시·군·구별 합산 | 초과누진세율 (0.2~0.5%) | 5억원 초과 시 과세 |
| **별도합산** | 소유자별 시·군·구별 합산 | 초과누진세율 (0.2~0.4%) | 80억원 초과 시 과세 |

## 2. 분리과세대상 토지

### 저율 분리과세 (0.07%)

#### 농지 (전·답·과수원)

**개인 소유 농지**:
- 과세기준일 현재 **실제 영농에 사용**
- 도시지역: **개발제한구역 + 녹지지역**만 분리과세
- **주거·상업·공업지역** 내 농지 → 종합합산과세

**법인·단체 소유 농지** (원칙: 종합합산):

분리과세 예외:
- 농업법인이 소유하고 실제 영농 사용 농지
- 한국농어촌공사가 농가 공급 목적 소유 농지
- 사회복지사업자 소유 농지 (1990.5.31 이전부터 소유)
- 법인이 **매립·간척**으로 취득하여 실제 영농 사용 농지
- **종중** 소유 농지 (1989.12.31 이전부터 소유)

#### 목장용지

- 개인 또는 법인의 **축산용** 토지
- 도시지역: 개발제한구역·녹지지역만 해당
- **기준면적 이내**: 분리과세 / **초과분**: 종합합산
- 도시지역 목장용지: 1989.12.31 이전부터 소유에 한정

#### 임야

- 산림의 보호·육성을 위한 임야
- **종중** 소유 임야 (1989.12.31 이전부터 소유)

### 중률 분리과세 (0.2%)

- **공장용지** (산업집적활성화법에 따른 공장의 부속토지, 기준면적 이내)
- **주택용·산업용 공급 목적** 토지 (택지공사 등)

### 고율 분리과세 (4%)

- **골프장용** 토지 (회원제 골프장)
- **고급오락장용** 토지

### 기타 분리과세

- 개발제한구역 지정 이전에 부지취득 완료된 공장 부속토지
- 국가·지방자치단체 지원 특정목적 사업용 토지
- 에너지·자원 공급, 방송·통신·교통 기반시설용 토지
- 국토 효율적 이용을 위한 개발사업용 토지

## 3. 종합합산과세대상 토지

**분리과세·별도합산과세 이외**의 모든 토지:

- **나대지** (건축물 없는 토지)
- 잡종지
- 영농에 사용되지 않는 농지
- **도시지역 주거·상업·공업지역** 내 농지
- 기준면적 초과 **목장용지**
- 기준면적 초과 **공장용지** 부속토지
- 기준면적 초과 **별도합산** 건축물 부속토지
- **별장**의 부속토지
- 주택 부속토지 중 주택 건물에 대한 건축물 기준면적 초과분

> **종합합산과세 = 나머지 토지** (분리·별도 아닌 것)

## 4. 별도합산과세대상 토지

**일반건축물의 부속토지** 등 사업용 토지:

- **일반건축물** (사무실, 점포, 공장 등)의 부속토지 (기준면적 이내)
- 건축물 시가표준액이 해당 토지 시가표준액의 **2% 이상**
- 가설건축물의 부속토지 (존속기간 **3년 이상**)

**기준면적**: 용도지역에 따라 건축물 연면적의 일정 배율

| 용도지역 | 기준면적 배율 |
|----------|---------------|
| 주거지역 | 건축물 연면적의 **5배** |
| 상업지역 | 건축물 연면적의 **3배** |
| 공업지역 | 건축물 연면적의 **7배** |
| 녹지지역 | 건축물 연면적의 **7배** |
| 비도시지역 | 건축물 연면적의 **7배** |

> 기준면적 **이내** → 별도합산 / **초과** → 종합합산

## 5. 분류 판단 플로우

```
토지
├── 분리과세 대상? (농지·목장용지·임야·공장용지·골프장 등)
│   ├── YES → 분리과세 (0.07% / 0.2% / 4%)
│   └── NO ↓
├── 일반건축물의 부속토지? (기준면적 이내)
│   ├── YES → 별도합산과세 (0.2~0.4%)
│   └── NO ↓
└── 종합합산과세 (0.2~0.5%)
```

## 6. 종합부동산세와의 연계

| 토지 유형 | 종합부동산세 |
|-----------|-------------|
| 분리과세대상 | **과세 제외** |
| 종합합산 | 전국 합산 **5억원** 초과 시 과세 |
| 별도합산 | 전국 합산 **80억원** 초과 시 과세 |

## 핵심 암기 포인트

- **분류 체계**: 분리과세(차등비례, 종부세 제외) / 종합합산(초과누진, 5억 초과 종부세) / 별도합산(초과누진, 80억 초과 종부세)
- **분리과세 저율 0.07%**: 농지(전·답·과수원), 목장용지(기준면적 이내), 임야
- **분리과세 중률 0.2%**: 공장용지(기준면적 이내), 주택용·산업용 공급 목적 토지
- **분리과세 고율 4%**: 골프장(회원제), 고급오락장용 토지
- **농지 분리과세 조건**: 개인, 실제 영농, 도시지역은 개발제한구역+녹지지역만 (주거·상업·공업지역 → 종합합산)
- **종합합산**: 나대지, 잡종지, 비영농 농지, 도시 주거·상업·공업 내 농지, 기준면적 초과분, 별장 부속토지
- **별도합산**: 일반건축물 부속토지(기준면적 이내). 건축물 시가표준액이 토지의 2% 이상
- **기준면적 배율**: 주거 5배, 상업 3배, 공업·녹지·비도시 7배
- **종합합산 = 나머지**: 분리과세도 별도합산도 아닌 토지

## OX 퀴즈

**Q1.** 분리과세대상 토지도 종합부동산세 과세대상이다. → **X** (과세 제외)

**Q2.** 도시지역 주거지역 내 농지는 분리과세대상이다. → **X** (종합합산과세대상. 도시지역은 개발제한구역+녹지지역만 분리과세)

**Q3.** 별도합산과세대상 토지에서 기준면적을 초과하면 초과분은 분리과세한다. → **X** (종합합산과세)

**Q4.** 별도합산 기준면적의 상업지역 배율은 5배이다. → **X** (3배. 주거가 5배)

**Q5.** 나대지는 별도합산과세대상이다. → **X** (종합합산과세대상)

**Q6.** 골프장용 토지의 재산세율은 0.07%이다. → **X** (4%)

**Q7.** 별장의 부속토지는 분리과세대상이다. → **X** (종합합산과세대상)

**Q8.** 법인 소유 농지는 원칙적으로 분리과세한다. → **X** (원칙 종합합산, 예외적으로 농업법인 등만 분리과세)

## 함정 노트

- ⚠️ **농지 분리과세 조건**: 개인 + 실제 영농 + 도시지역이면 개발제한구역·녹지지역만 — 3가지 조건 모두 충족 필요
- ⚠️ **기준면적 배율**: 주거 5배, 상업 3배, 공업·녹지·비도시 7배 — 상업이 가장 작음(3배)
- ⚠️ **종합합산 = 잔여토지**: 분리과세도 별도합산도 아닌 나머지 전부 — 분류 판단 시 '나머지'임을 기억
- ⚠️ **종부세 연계**: 분리과세 = 종부세 제외, 종합합산 = 5억 초과, 별도합산 = 80억 초과 — 공제금액 혼동 주의
