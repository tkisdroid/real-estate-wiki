# 부동산 관련 부가세

부동산 관련 조세에는 본세에 부가하여 징수되는 **농어촌특별세**, **지방교육세** 등이 있다.

## 1. 농어촌특별세 (국세, 목적세)

농어촌 산업기반시설 확충, 농어촌 지역개발사업에 필요한 재원 조달 목적

| 본세 | 농어촌특별세 |
|------|-------------|
| [[취득세]] | 과세표준 × **2%**(표준세율) × **10%** |
| [[취득세]] 감면 | 감면세액의 **20%** |
| [[등록면허세]] 감면 | 감면세액의 **20%** |
| [[종합부동산세]] | 납부세액의 **20%** |
| [[양도소득세]] 감면 | 감면세액의 **20%** |

> 농어촌특별세는 **신고불성실가산세 미적용** (납부지연가산세만 적용)

## 2. 지방교육세 (지방세, 목적세)

지방교육재정에 필요한 재원 확보 목적

| 본세 | 지방교육세 |
|------|-----------|
| [[취득세]] (표준세율) | 과세표준 × **(표준세율 - 2%)** × **20%** |
| [[취득세]] (유상거래 주택) | 과세표준 × **해당세율 × 50%** × **20%** |
| [[등록면허세]] | 납부세액의 **20%** |
| [[재산세]] | 납부세액의 **20%** (도시지역분 제외) |

> 지방교육세도 **신고불성실가산세 미적용** (납부지연가산세만 적용)

## 3. 지역자원시설세 (소방분, 지방세, 목적세)

소방사무에 소요되는 비용 충당 목적

- 과세대상: 건축물 및 선박
- 과세기준일: **6월 1일** ([[재산세]]와 동일)
- 납기: [[재산세]]와 같을 때 재산세 고지서에 **병기하여 고지** 가능
- **물납 불가** ([[재산세]] 물납 시에도 별도)

## 4. 개인 지방소득세 (양도소득에 대한)

| 항목 | 내용 |
|------|------|
| 과세주체 | 특별시·광역시·시·군 |
| 독립세 여부 | **독립세** |
| 세율 | 소득세 세율의 **10분의 1** |
| 소액징수면제 | 세액 **2,000원 미만** 미징수 |
| 신고기한 | [[양도소득세]] 예정·확정 신고기한 + **2개월** |

## 5. 세목별 부가세 관계 정리

| 본세 | 농어촌특별세 | 지방교육세 |
|------|-------------|-----------|
| [[취득세]] | ○ (2%분의 10%) | ○ ((세율-2%)분의 20%) |
| [[등록면허세]] | × (감면시만 20%) | ○ (납부세액의 20%) |
| [[재산세]] | × | ○ (납부세액의 20%) |
| [[종합부동산세]] | ○ (납부세액의 20%) | × |
| [[양도소득세]] | × (감면시만 20%) | × |

## 6. 부가세 신고·납부

- 본세를 **신고·납부**할 때 부가세를 **함께** 신고·납부
- 본세를 **보통징수** 할 때 부가세를 **함께** 부과·징수

## 핵심 암기 포인트

- **농어촌특별세**: 국세, 목적세. 취득세(2%분의 10%), 종부세(납부세액 20%), 감면세액(20%)
- **지방교육세**: 지방세, 목적세. 취득세((세율-2%)분의 20%), 등록면허세(납부세액 20%), 재산세(납부세액 20%, 도시지역분 제외)
- **지역자원시설세(소방분)**: 과세기준일 6/1, 재산세와 병기 고지 가능, 물납 불가
- **개인 지방소득세(양도분)**: 독립세, 세율 = 소득세의 1/10, 소액면제 2,000원 미만, 신고기한 = 양도소득세+2개월
- **부가세 관계표**: 취득세(농특세O·지방교육세O), 등록면허세(감면시만 농특세·지방교육세O), 재산세(지방교육세O·농특세X), 종부세(농특세O·지방교육세X), 양도소득세(감면시만 농특세·지방교육세X)
- **가산세**: 농어촌특별세·지방교육세 모두 신고불성실가산세 미적용 (납부지연가산세만)

## OX 퀴즈

**Q1.** 재산세에 농어촌특별세가 부과된다. → **X** (재산세에는 농어촌특별세 미부과. 지방교육세만 부과)

**Q2.** 종합부동산세에 지방교육세가 부과된다. → **X** (종부세에는 농어촌특별세만 부과. 지방교육세 미부과)

**Q3.** 양도소득세에는 항상 농어촌특별세가 부과된다. → **X** (감면 시에만 감면세액의 20% 부과)

**Q4.** 취득세의 지방교육세는 과세표준 × 표준세율 × 20%이다. → **X** (과세표준 × (표준세율-2%) × 20%)

**Q5.** 지역자원시설세(소방분)는 재산세 물납 시 함께 물납된다. → **X** (물납 불가)

**Q6.** 개인 지방소득세(양도분)의 신고기한은 양도소득세와 동일하다. → **X** (양도소득세 신고기한에 2개월을 더한 날까지)

**Q7.** 농어촌특별세에는 무신고가산세가 적용된다. → **X** (신고불성실가산세 미적용)

## 함정 노트

- ⚠️ **세목별 부가세 조합**: 취득세(농특세+지방교육세), 재산세(지방교육세만), 종부세(농특세만), 양도소득세(감면시만 농특세) — 각각 다르므로 표로 암기
- ⚠️ **취득세 지방교육세 산식**: (세율-2%)분의 20%이지, 세율 자체의 20%가 아님 — 2%를 빼는 것을 잊으면 오답
- ⚠️ **지방소득세 신고기한**: 양도소득세 + 2개월 — '같은 기한'이 아님
- ⚠️ **도시지역분 재산세**: 지방교육세 산정 시 도시지역분은 제외 — 빠뜨리면 과대 계산
