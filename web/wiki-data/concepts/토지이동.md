# 토지이동

토지이동이란 토지의 표시(지번·지목·면적·경계·좌표)가 변경되어 이를 **지적공부에 반영**하는 것을 말한다.

## 비교 정리

### 토지이동 종류 비교표

| 종류 | 내용 | 측량 | 등기촉탁 | 신청기한 |
|------|------|:----:|:--------:|:--------:|
| **신규등록** | 미등록토지 새로 등록 | 필요 | **불가** | 60일 |
| **등록전환** | 임야→토지대장 | 필요 | 가능 | 60일 |
| **분할** | 1필지→2필지+ | 필요* | 가능 | 60일** |
| **합병** | 2필지+→1필지 | **불요** | 가능 | 60일*** |
| **지목변경** | 지목 변경 | **불요** | 가능 | 60일 |

> *합병 전 경계대로 분할 시 불요, **용도변경 시, ***의무합병 시

### 측량 필요 여부 비교

| 측량 필요 | 측량 불요 |
|----------|----------|
| 신규등록, 등록전환, 분할, 축척변경 | **합병**, **지목변경** |

### 토지이동 신청권자

| 원칙 | 예외 |
|------|------|
| **토지소유자** | 지적소관청의 **직권** (등록강제주의) |
| | 사업시행자 (도시개발사업 등) |
| | 대위 신청 (채권자, 공공사업 시행자 등) |

## 세부 내용
- **[[토지이동_신규등록|신규등록]]**: 공유수면매립·미등록토지, 60일 신청, 등기촉탁 불가, 창설적 효력
- **[[토지이동_등록전환|등록전환]]**: 임야→토지대장, 면적환산과 오차허용범위, 등록전환측량 필수
- **[[토지이동_분할|분할]]**: 1필지→2필지+, 지번부여 규칙, 분할제한·최소면적, 용도변경 시 60일
- **[[토지이동_합병|합병]]**: 2필지+→1필지, 합병 제한사유 10가지, 합병 가능 등기 5종, 측량 불요
- **[[토지이동_지목변경|지목변경]]**: 60일 신청, 영속성 원칙, 측량 불요, 취득세 간주취득
