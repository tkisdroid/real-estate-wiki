# 경제기반분석 (수출기반모형)

> 📖 상위 문서: [[입지이론]]

## 1. 기반산업과 비기반산업

| 구분 | 기반산업 | 비기반산업 |
|------|---------|-----------|
| 정의 | 지역에서 생산된 제품·서비스가 **지역 밖으로 수출** | 지역 내에서 생산된 제품·서비스가 **지역주민에 의해 소비** |
| 역할 | 지역경제 **성장 동력** | 지역 내 서비스 |

- 경제기반분석에서는 지역경제를 구성하는 산업을 **기반산업**과 **비기반산업**으로 구분
- 지역의 경제규모는 **수출량이 많을수록** 커짐

## 2. 경제기반승수(수출승수)

### 공식

$$경제기반승수 = \frac{총고용인구}{기반산업의\ 총고용인구} = \frac{1}{기반비율} = \frac{1}{1 - 비기반비율}$$

- 총고용인구 = 기반부문 고용인구 + 비기반부문 고용인구

### 의미

- 지역의 기반산업 **한 단위**가 지역경제에 미치는 **영향**을 측정
- 지역수출량의 변화가 지역경제에 미치는 영향 측정 가능

### 비기반부문과의 관계

| 변화 | 경제기반승수 |
|------|-------------|
| **비기반부문** 비율 **상승** | 승수 **증가** (비례) |
| **기반부문** 비율 **상승** | 승수 **감소** (반비례) |

- **비기반부문과 경제기반승수는 비례관계**
- **기반부문과 경제기반승수는 반비례관계**

### 계산 예시

- 총고용 10,000명, 비기반부문 6,000명인 경우
- 기반부문 = 10,000 - 6,000 = 4,000명
- 경제기반승수 = 10,000 / 4,000 = **2.5**
- 의미: 기반부문 1명 증가 → 지역 전체 **2.5명** 고용 증가 (기반 1명 + 비기반 1.5명)

## 3. 입지계수(Locational Quotient, LQ)

### 공식

$$LQ = \frac{특정산업이\ 지역에서\ 차지하는\ 비율}{특정산업이\ 전국에서\ 차지하는\ 비율}$$

### 해석

| LQ 값 | 의미 | 결과 |
|-------|------|------|
| **LQ > 1** | 지역 **특화** 산업 | 잉여분을 지역 외부로 **수출** |
| **LQ < 1** | **비특화** 산업 | 타지역으로부터 **수입** |
| **LQ = 1** | 전국과 동일 비율 | **자급자족** |

- 입지계수는 지역경제를 구성하는 산업의 **지역 특화도**를 측정하는 방법
- 산업의 **상대적인 집중도**에 의한 것

---

## 핵심 암기 포인트

- 기반산업 = 지역 밖으로 **수출** / 비기반산업 = 지역 내 **소비**
- 경제기반승수 = **총고용인구 / 기반산업 고용인구**
- **비기반부문↑** → 경제기반승수 **증가** (비례)
- **기반부문↑** → 경제기반승수 **감소** (반비례)
- 입지계수(LQ) > 1: **특화** 산업(수출) / < 1: 비특화(수입) / = 1: 자급자족

## OX 퀴즈

**Q1.** 경제기반승수는 비기반부문 비율이 증가하면 감소한다. → **X** (비기반부문 비율↑ → 경제기반승수 **증가**)

**Q2.** 기반부문 비율이 상승하면 경제기반승수는 증가한다. → **X** (기반부문↑ → 승수 **감소**)

**Q3.** 입지계수(LQ)가 1보다 크면 해당 산업은 지역의 특화산업이다. → **O**

**Q4.** 경제기반승수 = 기반산업 고용인구 / 총고용인구이다. → **X** (**총고용인구 / 기반산업 고용인구** 분자·분모 반대)

**Q5.** 입지계수가 1보다 작은 산업은 잉여분을 지역 외부로 수출한다. → **X** (LQ < 1이면 비특화 → 타지역에서 **수입**. 수출은 LQ > 1)

**Q6.** 비기반산업이란 지역 생산물이 지역 외부로 수출되는 산업을 말한다. → **X** (수출은 **기반산업**, 비기반산업은 **지역 내 소비**)

**Q7.** 비기반부문과 경제기반승수는 반비례관계에 있다. → **X** (**비례**관계)

## 함정 노트

- ⚠️ "기반부문↑ → 경제기반승수↑" → 기반부문↑이면 승수 **감소**, **비**기반부문↑이면 승수 증가
- ⚠️ "경제기반승수 = 기반/총" → **총/기반** (분자·분모 혼동 주의)
- ⚠️ "LQ > 1 = 수입" → LQ > 1은 **특화산업으로 수출**, LQ < 1이 수입
- ⚠️ "기반산업 = 지역 내 소비" → 기반산업은 **지역 밖 수출**, 지역 내 소비는 비기반산업
