# 포트폴리오 이론

## 1. 평균-분산법 (지배원리)

- 수익성 지표: **기대수익률**, 위험 지표: **분산(표준편차)**

### 선택 기준

| 상황 | 기대수익률 | 표준편차(위험) | 선택 |
|------|-----------|---------------|------|
| 1 | A = B | A < B | **A** (위험 작은 것) |
| 2 | A > B | A = B | **A** (수익 큰 것) |
| 3 | A > B | A < B | **A** (수익 크고 위험 작음) |
| 4 | A > B | A > B | **판단 불가** → 변이계수법 활용 |

### 변이계수법

$$변이계수 = \frac{표준편차}{기대수익률}$$

- **단위수익률당 위험도** 측정
- 변이계수가 **낮을수록** 유리한 투자안
- 평균-분산법으로 판단 불가 시 활용

## 2. 포트폴리오의 위험 구조

$$총위험 = 체계적\ 위험 + 비체계적\ 위험$$

| 구분 | 체계적 위험 | 비체계적 위험 |
|------|-----------|-------------|
| **분산투자** | 제거 **불가능** | 제거 **가능** |
| **원인** | 인플레이션, 경기변동, 금리 변화 | 개별 투자안 고유 위험 |
| **영향** | 모든 투자안에 **동일 방향** | 투자안별 **상이한 방향** |
| **관리** | 감수 | 포트폴리오 구성으로 관리 |

> **포트폴리오 효과**: 투자안 수를 늘릴수록 **비체계적 위험**이 감소하여 기대수익이 안정되는 현상.

## 3. 상관계수와 포트폴리오 효과

상관계수: 포트폴리오 수익률간의 방향성 관계 (-1 ≤ r ≤ +1)

| 상관계수 | 의미 | 포트폴리오 효과 |
|----------|------|----------------|
| **r = +1** | 완전 정(+)의 상관 (동일 방향·크기) | 효과 **없음** (비체계적 위험 전혀 감소 안 됨) |
| **r = 0** | 무상관 | 효과 있음 |
| **r = -1** | 완전 부(-)의 상관 (반대 방향·동일 크기) | 효과 **최대** (비체계적 위험 **완전 제거** 가능) |

- 상관계수가 **-1에 가까울수록** 포트폴리오 효과 **크다**
- 상관계수가 **+1이 아닌 한** 포트폴리오 효과는 항상 발생
- 수익률이 **상이한(반대) 추세**를 보이는 자산에 분산투자하는 것이 유리

## 4. 포트폴리오의 기대수익률

$$포트폴리오\ 기대수익률 = \sum(개별자산\ 기대수익률 \times 투자비중)$$

- 개별자산 기대수익률의 **가중평균**
- 총투자금액 크기, 운영주체(개인/법인), **상관계수와 무관**

## 5. 효율적 전선 (Efficient Frontier)

- 동일 위험에서 **최고 수익률** 또는 동일 수익에서 **최소 위험**을 나타내는 투자대안의 집합
- **우상향** (더 높은 수익률 → 더 많은 위험 감수 필요)
- 효율적 전선 **위의** 포트폴리오가 아래의 모든 포트폴리오를 지배

## 6. 무차별곡선과 최적 포트폴리오

- **무차별곡선**: 투자자의 **위험에 대한 태도** 표현
- **보수적 투자자** (위험회피 강함): 기울기 **가파름** → 위험 적은 투자안 선택
- **공격적 투자자** (위험회피 약함): 기울기 **완만** → 수익 높은 투자안 선택
- **최적 포트폴리오** = 효율적 전선과 무차별곡선의 **접점**

## 핵심 암기 포인트

- 변이계수 = **표준편차 / 기대수익률** (낮을수록 유리)
- 총위험 = **체계적 위험 + 비체계적 위험**
- 체계적 위험: 분산투자로 제거 **불가능** (인플레, 경기변동, 금리)
- 비체계적 위험: 분산투자로 제거 **가능** (개별 투자안 고유)
- 상관계수 r = +1: 포트폴리오 효과 **없음** / r = -1: 효과 **최대** (비체계적 위험 완전 제거)
- r ≠ +1이면 포트폴리오 효과 **항상 발생**
- 포트폴리오 기대수익률 = 개별자산 기대수익률의 **가중평균** (상관계수와 **무관**)
- 효율적 전선: **우상향**, 동일 위험에서 최고 수익 or 동일 수익에서 최소 위험
- 최적 포트폴리오 = 효율적 전선과 무차별곡선의 **접점**
- 보수적 투자자: 무차별곡선 기울기 **가파름** / 공격적 투자자: 기울기 **완만**

## OX 퀴즈

**Q1.** 체계적 위험은 분산투자를 통해 제거할 수 있다. → **X** (체계적 위험은 분산투자로 제거 **불가능**, 비체계적 위험만 제거 가능)

**Q2.** 상관계수가 +1이면 포트폴리오 효과가 최대이다. → **X** (+1이면 효과 **없음**, **-1**이면 최대)

**Q3.** 포트폴리오의 기대수익률은 상관계수에 의해 영향을 받는다. → **X** (기대수익률은 상관계수와 **무관**, 개별자산 기대수익률의 가중평균)

**Q4.** 변이계수가 높을수록 유리한 투자안이다. → **X** (변이계수가 **낮을수록** 유리)

**Q5.** 상관계수가 +1이 아닌 한 포트폴리오 효과는 항상 발생한다. → **O**

**Q6.** 보수적 투자자의 무차별곡선 기울기는 공격적 투자자보다 완만하다. → **X** (보수적 투자자가 기울기 **가파름**, 공격적이 완만)

**Q7.** 효율적 전선은 우하향하는 형태를 가진다. → **X** (효율적 전선은 **우상향**)

## 함정 노트

- ⚠️ "체계적 위험 = 분산 가능" → 분산 가능한 것은 **비체계적** 위험, 체계적 위험은 분산 **불가**
- ⚠️ "r = +1 → 포트폴리오 효과 최대" → r = +1은 효과 **없음**, r = **-1**이 효과 최대
- ⚠️ "포트폴리오 기대수익률이 상관계수에 영향받음" → 기대수익률은 상관계수와 **무관**, 위험(분산)만 영향받음
- ⚠️ "보수적 = 기울기 완만" → 보수적(위험회피 강함)은 기울기 **가파름**, 공격적이 완만
- ⚠️ "변이계수 높음 = 유리" → 변이계수는 단위수익률당 위험도이므로 **낮을수록** 유리
