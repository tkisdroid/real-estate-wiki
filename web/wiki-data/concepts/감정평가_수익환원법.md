# 수익환원법 (수익방식)

> 📖 상위 문서: [[감정평가3방식]]

## 1. 의의

- 장래 기대되는 **순수익을 환원(자본화)**하여 시산가액(수익가액) 산정
- 대상물건의 **수익성**에 착안한 감정평가 방식
- 기본 원칙: **예측의 원칙, 수익배분의 원칙**
- **수익용 부동산**에 적합

## 2. 직접자본환원법

### 기본 공식

$$수익가액 = \frac{NOI(순영업소득)}{R(환원율)}$$

- 한 해의 NOI를 환원율로 나누어 산정
- **화폐의 시간가치를 고려하지 않음** (단년도 기준)

## 3. 할인현금수지분석법 (DCF법)

### 기본 공식

$$수익가액 = \sum_{t=1}^{n}\frac{NOI_t}{(1+Y)^t} + \frac{복귀가치}{(1+Y)^n}$$

- Y = **할인율(수익률)**
- 복귀가치 = 보유기간 종료 시 **처분가격(매도가격)**
- **화폐의 시간가치를 고려** (여러 해의 소득 반영)
- 소득이득과 자본이득 **모두 고려**

## 4. 환원율(R)과 할인율(Y)의 관계

| 순수익의 변화 | R과 Y의 관계 |
|-------------|-------------|
| 순수익 **일정** | R = Y |
| 순수익 **증가** | R < Y (환원율이 할인율보다 **작음**) |
| 순수익 **감소** | R > Y (환원율이 할인율보다 **큼**) |

### 직관적 이해

- 순수익 증가 예상 → 미래 수익이 더 크므로 → 현재 수익가액 높아야 → 환원율(분모) 작아져야 → **R < Y**
- 순수익 감소 예상 → 미래 수익이 더 작으므로 → 현재 수익가액 낮아져야 → 환원율(분모) 커져야 → **R > Y**

## 5. 환원율 산정방법

| 방법 | 내용 |
|------|------|
| **시장추출법** | 유사 거래사례에서 추출 (NOI/거래가격) |
| **조성법(적산법)** | 무위험률 + 위험할증률 + 관리비 등 합산 |
| **저당-지분비율법(밴드오브인베스트먼트법)** | Rm x L + Re x (1-L) |
| **엘우드법** | 저당·지분 분석 확장 |
| **투자결합법(토지잔여법/건물잔여법)** | 토지환원율과 건물환원율의 가중평균 |

### 저당-지분비율법

$$R_o = R_m \times L + R_e \times (1-L)$$

- L = 대부비율(LTV), Rm = 저당상수, Re = 지분배당률

## 6. 유효총수입(EGI) 산출과정

```
가능총소득(PGI) - 공실 및 불량부채 + 기타소득 = 유효총소득(EGI)
유효총소득(EGI) - 영업경비(OE) = 순영업소득(NOI)
```

## 7. 수익분석법 (임대료 평가)

$$수익임대료 = 기초가액 \times 환원율 + 필요경비$$

---

## 핵심 암기 포인트

- 수익가액 = **NOI / 환원율(R)** (직접자본환원법)
- DCF: 화폐의 시간가치 **고려**, 복귀가치 포함
- 순수익 일정: **R = Y** / 증가: **R < Y** / 감소: **R > Y**
- 환원율 산정: 시장추출법, 조성법, 저당-지분비율법, 엘우드법, 투자결합법
- Ro = **Rm x L + Re x (1-L)**
- 수익방식 원리 = **수익성**, 원칙 = **예측·수익배분**
- 자본환원율은 화폐의 시간가치를 **고려하지 않음** (단년도)

## OX 퀴즈

**Q1.** 직접자본환원법에서 수익가액 = NOI x 환원율이다. → **X** (NOI **/** 환원율)

**Q2.** 순수익이 증가할 때 환원율(R)은 할인율(Y)보다 크다. → **X** (증가할 때 R **< Y**)

**Q3.** DCF법은 화폐의 시간가치를 고려하지 않는다. → **X** (DCF법은 화폐의 시간가치를 **고려**)

**Q4.** 자본환원율은 화폐의 시간가치를 고려하지 않는다. → **O** (단년도 NOI/R)

**Q5.** 환원율의 조성법은 무위험률에 위험할증률 등을 합산하는 방법이다. → **O**

**Q6.** 수익환원법으로 산정한 시산가액을 적산가액이라 한다. → **X** (**수익가액**. 적산가액은 원가법)

**Q7.** 순수익이 일정할 때 환원율과 할인율은 같다. → **O**

**Q8.** 시장추출법은 유사 거래사례에서 환원율을 추출하는 방법이다. → **O**

## 함정 노트

- ⚠️ "수익가액 = NOI x R" → NOI **/** R (곱이 아니라 나눗셈)
- ⚠️ "순수익 증가 → R > Y" → 순수익 증가 시 R **< Y** (증가면 환원율이 작아야)
- ⚠️ "수익환원법 시산가액 = 적산가액" → **수익가액** (적산가액은 원가법, 비준가액은 비교법)
- ⚠️ "DCF = 시간가치 미고려" → DCF는 시간가치 **고려** (할인현금수지분석법)
- ⚠️ "자본환원율 = 시간가치 고려" → 자본환원율은 시간가치 **미고려** (단년도 기준)
