# 손해배상책임과 업무보증

## 개요

공인중개사법은 개업공인중개사의 귀책사유로 발생하는 거래당사자의 피해를 최소화하기 위해 **손해배상책임제도**와 **업무보증설정의무**를 두고 있다. 업무보증은 개업공인중개사의 배상능력 부재 시 보증기관이 대신 배상하는 제도이다.

## 보증설정 vs 배상책임 비교

| 구분 | 보증설정 | 배상책임 |
|---|---|---|
| **성격** | 사전적 예방·담보 | 사후적 손해전보 |
| **시기** | 업무 개시 **전** 설정 | 손해 발생 **후** 배상 |
| **방법** | 보증보험/공제/공탁 택1 | 보증기관 지급 또는 직접 배상 |
| **대상** | 재산상 손해 | 재산상 손해 (정신적 손해 제외) |
| **보증금액(개인)** | 2억원 이상 | 실제 손해액 |
| **보증금액(법인)** | 4억원 이상 + 분사무소당 2억원 | 실제 손해액 |
