# 마르크스(K. Marx)의 절대지대설

> 📖 상위 문서: [[지대이론]]

## 1. 의의

- **토지의 소유 자체**가 지대 발생 요인
- 열등지라 해도 소유자는 존재하며, **토지소유자의 요구**에 의해 발생하는 지대를 **절대지대(absolute rent)**라 함
- 절대지대는 **비옥도와 아무런 관계 없이** 결정
- 리카도·튀넨의 차액지대설로 **설명 불가능한 부분을 보완**

## 2. 핵심 내용

### 한계지에서도 지대 발생

- **차액지대설의 한계**: 한계지에서 지대가 왜 존재하는지 설명 불가
- 마르크스는 **한계지에서도 절대지대가 발생**하는 근거를 제시

### 지대의 구성

| 위치 | 존재하는 지대 |
|------|---------------|
| 한계지 안쪽 (우등지) | **차액지대 + 절대지대** |
| 한계지 및 한계지 밖 | **절대지대만** 존재 |

- 절대지대란 **차액지대를 초과한 부분**이라고도 할 수 있음

### 지대와 곡물가격의 관계

- 지대는 생산물가격에 영향을 미치는 **비용**
- **지대가 곡물가격을 결정**한다 (리카도와 **정반대**)
- 리카도: 곡물가격 → 지대 (지대는 결과) / 마르크스: **지대 → 곡물가격** (지대는 원인)

## 3. 리카도와의 결정적 차이

| 구분 | 리카도 (차액지대설) | 마르크스 (절대지대설) |
|------|---------------------|----------------------|
| 지대 발생 원인 | 비옥도 차이 | **토지소유 자체** |
| 비옥도와의 관계 | 비옥도가 핵심 | 비옥도 **무관** |
| 한계지 지대 | **미발생** | **발생** (절대지대) |
| 지대의 성격 | 잉여 (초과이윤) | **비용** |
| 인과관계 | 곡물가격 → 지대 | **지대 → 곡물가격** |
| 가격 구성요인 여부 | 아님 | **구성요인** |

## 4. 절대지대와 차액지대의 관계

```
[우등지]           [열등지/한계지]        [한계지 밖]
┌──────────┐      ┌──────────┐       ┌──────────┐
│ 차액지대  │      │          │       │          │
├──────────┤      │          │       │          │
│ 절대지대  │      │ 절대지대  │       │ 절대지대  │
└──────────┘      └──────────┘       └──────────┘
```

## 핵심 암기 포인트

- 마르크스 = **절대지대설**
- 지대 발생 요인 = **토지소유 자체** (비옥도 **무관**)
- **한계지에서도 지대 발생** → 차액지대설의 한계 보완
- 한계지 안쪽: **차액지대 + 절대지대** / 한계지·한계지 밖: **절대지대만**
- 지대는 **비용** → **지대가 곡물가격을 결정** (리카도와 반대)
- 리카도: 가격→지대(잉여) / 마르크스: 지대→가격(비용)

## OX 퀴즈

**Q1.** 마르크스의 절대지대설에서 지대는 생산물가격의 결과이다. → **X** (마르크스에서 지대는 가격의 **원인(비용)**. 가격의 결과는 **리카도**)

**Q2.** 마르크스에 의하면 한계지에서는 지대가 발생하지 않는다. → **X** (한계지에서도 **절대지대 발생**)

**Q3.** 마르크스의 절대지대는 비옥도에 따라 결정된다. → **X** (비옥도와 **무관**, **토지소유 자체**가 원인)

**Q4.** 마르크스에 의하면 한계지 안쪽에서는 차액지대와 절대지대가 함께 존재한다. → **O**

**Q5.** 마르크스의 절대지대설에서 지대가 곡물가격을 결정한다. → **O**

**Q6.** 절대지대는 차액지대를 초과한 부분이라고도 할 수 있다. → **O**

**Q7.** 마르크스에 의하면 지대는 잉여(초과이윤)의 성격을 갖는다. → **X** (지대는 **비용**의 성격. 잉여는 **리카도**)

## 함정 노트

- ⚠️ "마르크스: 한계지 지대 미발생" → 한계지에서도 **절대지대 발생** (차액지대설과 핵심 차이)
- ⚠️ "마르크스: 비옥도가 지대 결정" → 비옥도 **무관**, **토지소유 자체**가 원인
- ⚠️ "마르크스: 곡물가격이 지대를 결정" → **지대가 곡물가격을 결정** (리카도와 혼동 주의)
- ⚠️ "마르크스: 지대는 잉여" → 지대는 **비용** (잉여라 본 것은 리카도·튀넨)
- ⚠️ "한계지 밖에서는 차액지대 + 절대지대" → 한계지 및 한계지 밖에서는 **절대지대만** 존재
