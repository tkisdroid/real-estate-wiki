# 중개사무소 개설등록

## 1. 등록의 의의 및 성격

### 등록의 의의
- 중개사무소 개설등록이란 등록관청에 비치된 **중개사무소개설등록대장**에 등재함으로써 이루어지는 행정행위
- 등록은 낮은 의미의 허가로서, 허가적 성격이 있으나 재량 없이 기속적으로 처리

### 등록의 성격 (4가지)

| 성격 | 내용 |
|---|---|
| **기속적** | 요건 충족 시 등록관청의 재량 없이 반드시 처리 |
| **일신전속적** | 대물적이 아닌 **대인적** 성격 → 양도·대여 금지 |
| **1인 1등록주의** | 이중등록 금지 (위반 시 절대 등록취소 + 1년/1천만원) |
| **1인 1사무소 원칙** | 2개 이상 사무소 설치 금지 (법인 분사무소만 예외) |

### 등록의 효과
- 부동산 중개업을 **적법하게 영위**할 수 있는 지위 취득
- 단, 등록만으로는 업무 개시 불가 → **업무보증 설정 후** 업무 개시
- 등록통지를 받은 후 업무를 개시하면 이미 등록 상태이므로 **무등록업자에는 해당하지 않음**
- 등록을 받지 않고 중개업을 한 경우 처벌을 받으나, 그 행위의 **사법상 효력까지 부정되지는 않음**

## 2. 등록할 수 있는 자

### 개인
- **공인중개사** 자격을 갖춘 자 (소속공인중개사 제외)
- 소속공인중개사는 이중소속 금지 → 등록 불가

### 법인
- 상법상 회사 또는 협동조합기본법상 **협동조합** (사회적협동조합 **제외**)
- 특수법인(자산관리공사 등)도 등록 필요하나, 법인 등록기준은 적용 안 됨

### 주의
- 부동산전시업이나 인터넷을 통한 중개업무도 **반드시 개설등록** 필요 (국토교통부 유권해석 2000.9.26)

## 3. 등록관청

**시장·군수·구청장** (사무소를 두고자 하는 지역 관할)

| 구분 | 설명 |
|---|---|
| **자치구** | 특별시·광역시의 구 → 구청장이 등록관청 |
| **비자치구(행정구)** | 시에서 행정편의상 분구된 구 (수원시 장안구, 성남시 분당구 등) → **시장**이 등록관청 |
| **특별자치도 행정시** | 제주시, 서귀포시 등 → 행정시장이 등록관청 |

## 4. 등록 기준 (영 제13조)

### 공인중개사인 개업공인중개사

| 요건 | 내용 | 비고 |
|---|---|---|
| **실무교육** | 등록신청일 전 **1년 이내** 실무교육(실무수습 포함) 이수 | 폐업 후 1년 내 재등록 시 면제 |
| **사무소 확보** | 건축물대장 기재 건물 (**가설건축물대장 제외**) | 준공검사·사용승인 받은 건물 포함 |
| | 소유·전세·임대차·사용대차 등으로 사용권 확보 | 무허가건물 불가 |

### 법인인 개업공인중개사 (특수법인 적용 안 됨)

| 요건 | 내용 |
|---|---|
| **법인 형태** | 상법상 회사 또는 협동조합기본법상 협동조합 (사회적협동조합 **제외**) |
| **자본금** | **5천만원 이상** |
| **목적** | 법 제14조 업무만 영위 목적 |
| **대표자** | **공인중개사**이어야 함 |
| **임원·사원** | 대표자 제외 **1/3 이상** 공인중개사 |
| **실무교육** | 대표자·임원·사원 **모두** 등록신청일 전 1년 이내 이수 |
| **사무소** | 건축물대장 기재 건물 확보 |

### 등록기준 미달 효과
- 등록 전 미달 → 등록 반려
- 등록 후 미달 → **등록취소** 또는 **업무정지** 처분 가능 (법 제38조 제2항)

## 5. 등록 절차

### 제출서류 (규칙 제5조)
1. 개설등록신청서
2. 실무교육수료확인증 사본
3. 여권용 사진
4. 사무소 확보 증명서류 (임대차계약서 또는 건물등기사항증명서 등)
5. 결격사유 해당 없음 증명서류 (외국인만)

### 등록 처리
- 등록관청은 신청일로부터 **7일 이내** 등록 → 서면 통지
- 업무보증 설정 확인 후 → **등록증 교부**
- 등록사항을 다음달 **10일까지** 공인중개사협회에 통지

### 등록 반려 사유
1. 등록의 결격사유에 해당하는 경우
2. 개설등록 기준에 적합하지 않은 경우
3. 기타 법령에 따른 제한에 위반되는 경우

## 6. 이중등록·이중소속 금지

### 이중등록 (법 제12조 제1항)
- 개업공인중개사가 **2중**으로 등록받는 것 금지
- 폐업신고 절차 미완료 상태에서 다른 곳 등록 → 이중등록 해당
- 위반 시: **절대 등록취소** + 1년 이하 징역 또는 1천만원 이하 벌금

### 이중소속 (법 제12조 제2항)
- 개업공인중개사가 다른 법인의 사원·임원, 다른 개업공인중개사의 소속공인중개사 등이 되는 것
- 소속공인중개사가 다른 법인의 사원·임원이 되는 것도 해당
- **절대적으로 금지** (법 제12조 제2항)
- 위반 시: 개업공인중개사 → **절대 등록취소**, 소속공인중개사 → **자격정지** (모두 1년/1천만원)

## 7. 등록증 양도·대여 금지 (법 제19조)

| 대상 | 제재 |
|---|---|
| 양도·대여한 자 | **절대 등록취소** + 1년/1천만원 |
| 양수·대여받은 자 / 알선자 | 1년/1천만원 |

- 타인에는 **가족도 포함**
- 다른 사람의 성명·상호를 사용하여 중개업무를 하는 것도 해당

## 8. 업무개시 관련 세부사항

### 등록 후 업무 미개시 시 제재
| 기간 | 제재 |
|---|---|
| 등록 후 **3개월** 초과 업무 미개시 | 휴업신고 의무 위반 → **100만원 이하 과태료** |
| 등록 후 **6개월** 초과 업무 미개시 | 정당한 사유 없이 6개월 초과 휴업 → 등록취소 또는 업무정지 |

### 등록신청 상태에서 업무 개시
- 아직 등록이 되지 않은 상태 → **무등록 중개업** 해당 (3년/3천만원)

## 9. 종별 변경 (규칙 제4조)

### 등록신청서 재제출
- 개업공인중개사가 **종별을 달리하여** 업무하고자 할 때 → 등록신청서 **재제출**
- 변동사항 없는 서류는 재제출 **불요** (실무교육수료확인증, 사무소확보 증명 등)
- 종전 등록증 **반납**
- 별도 **폐업신고 불요** (국토교통부 1999.9.3)

### 부칙 개업공인중개사의 종별변경 특례 (규칙 제5조 제4항)
- 공인중개사 자격 취득 후 **동일 관할구역 내**에서 공인중개사인 개업공인중개사로 계속 → **등록증 재교부 신청**만으로 충분
- 동일 관할구역 **밖** 또는 법인으로 변경 → 등록신청서 **재제출** 필요

## 10. 수수료 납부 정리 (법 제47조)

| 행위 | 수수료 결정 기관 |
|---|---|
| 자격시험 응시 | **시·도 조례** (국토교통부장관 시행 시 국토교통부장관 결정·공고) |
| 자격증 재교부 | **시·도 조례** |
| 개설등록 신청 | **시·군·구 조례** |
| 등록증 재교부 | **시·군·구 조례** |
| 분사무소 설치신고 | **시·군·구 조례** |
| 휴업·폐업 신고 등 | **납부 불요** |

- 관할 **내** 이전 시 등록증 변경교부 → 수수료 **불요**
- 관할 **외** 이전 시 등록증 재교부 → 수수료 **납부**

## 11. 등록의 효력 소멸

### 소멸하는 경우
- 개인 개업공인중개사 **사망** / 법인 **해산** → 자연 소멸
- **폐업신고** 수리 → 소멸
- **등록취소** 처분 → 소멸

### 소멸하지 않는 경우
- 휴업신고·업무정지처분 시
- 이전신고 미이행 시
- 결격사유 해당 시 (등록취소 처분이 있어야 소멸)
- 폐업신고 미수리 상태
- 법인 대표자 사망 시 (대표자 변경으로 해결)

## 핵심 암기 포인트

1. 등록 성격 4가지: **기속적, 일신전속적, 1인1등록, 1인1사무소**
2. 등록관청: **시장·군수·구청장** (비자치구는 시장)
3. 법인 자본금: **5천만원 이상**, 임원·사원 공인중개사 비율: **1/3 이상**
4. 실무교육: 등록신청일 전 **1년 이내** (폐업 후 1년 내 재등록 시 면제)
5. 등록 처리: 신청 후 **7일 이내**, 등록사항 협회 통지: 다음달 **10일까지**
6. 가설건축물대장 건물은 사무소로 **사용 불가**
7. 이중등록·이중소속 → 모두 **절대 등록취소**
8. 등록증 양도·대여 → **절대 등록취소**

## OX 퀴즈

**Q1.** 중개사무소 개설등록은 등록관청의 재량에 의해 허가되는 재량적 성격을 가진다.
> **X** — 개설등록은 요건 충족 시 재량 없이 처리하는 **기속적** 성격이다.

**Q2.** 법인인 개업공인중개사의 자본금 요건은 3천만원 이상이다.
> **X** — 법인의 자본금 요건은 **5천만원 이상**이다.

**Q3.** 비자치구(행정구)의 경우 구청장이 등록관청이 된다.
> **X** — 비자치구(행정구)의 경우 **시장**이 등록관청이 된다.

**Q4.** 개업공인중개사가 등록통지를 받은 후 업무보증을 설정하지 않고 업무를 개시하면 무등록업자에 해당한다.
> **X** — 이미 등록한 상태이므로 무등록업자에 해당하지 않으나, 업무보증 미설정으로 **등록취소 및 업무정지** 처분 가능.

**Q5.** 폐업 후 1년 이내에 재등록하는 경우에도 실무교육을 다시 이수하여야 한다.
> **X** — 폐업 후 **1년 이내** 재등록 시 실무교육 면제 (법 제33조 제1항 단서).

**Q6.** 법인의 대표자뿐만 아니라 임원·사원도 등록신청일 전 1년 이내에 실무교육을 이수해야 한다.
> **O** — 법인의 경우 대표자·임원·사원 **모두** 실무교육 이수 필요.

**Q7.** 개업공인중개사가 폐업신고를 완료하지 않은 상태에서 다른 곳에 등록하면 이중등록에 해당한다.
> **O** — 폐업신고 미완료 상태에서 다른 곳 등록 시 이중등록 해당.

**Q8.** 가설건축물대장에 기재된 건물에 중개사무소를 개설할 수 있다.
> **X** — 가설건축물대장에 기재된 건물은 중개사무소로 **사용 불가**.

**Q9.** 법인인 개업공인중개사의 대표자는 공인중개사가 아니어도 된다.
> **X** — 법인의 대표자는 반드시 **공인중개사**이어야 한다.

**Q10.** 법인의 임원·사원 중 공인중개사의 비율은 대표자를 포함하여 1/3 이상이어야 한다.
> **X** — **대표자를 제외**하고 임원·사원의 1/3 이상이 공인중개사이어야 한다.

**Q11.** 등록관청은 신청일로부터 14일 이내에 등록을 처리해야 한다.
> **X** — 신청일로부터 **7일 이내**에 등록을 처리해야 한다.

**Q12.** 사회적협동조합도 중개법인으로 등록할 수 있다.
> **X** — 사회적협동조합은 **제외**되며, 일반 협동조합만 가능하다.

**Q13.** 등록을 받지 않고 중개업을 한 경우 그 거래행위는 사법상 무효이다.
> **X** — 처벌은 받으나 거래행위의 **사법상 효력까지 부정되지는 않는다**.

**Q14.** 등록사항은 다음달 15일까지 공인중개사협회에 통지해야 한다.
> **X** — 다음달 **10일까지** 통지해야 한다.

**Q15.** 이중소속 위반 시 소속공인중개사는 자격이 취소된다.
> **X** — 소속공인중개사는 자격**정지** 처분을 받는다 (자격취소가 아님).

**Q16.** 등록증을 가족에게 대여하는 것은 등록증 대여에 해당하지 않는다.
> **X** — 타인에는 **가족도 포함**되므로 등록증 대여에 해당하며 절대 등록취소 사유이다.

**Q17.** 법인 대표자가 사망하면 법인의 개설등록 효력이 소멸한다.
> **X** — 대표자 변경으로 해결 가능하며, 등록 효력은 **소멸하지 않는다**.

**Q18.** 개업공인중개사가 휴업신고를 하면 등록의 효력이 소멸한다.
> **X** — 휴업신고 시 등록 효력은 **소멸하지 않는다**. 등록취소 처분이 있어야 소멸한다.

## 함정 노트

| 함정 포인트 | 정확한 내용 |
|---|---|
| 등록 = 업무개시 | 등록만으로 업무 개시 불가, **업무보증 설정 후** 개시 |
| 비자치구 → 구청장 | 비자치구(행정구)는 **시장**이 등록관청 |
| 사회적협동조합도 가능 | 사회적협동조합은 **제외**, 일반 협동조합만 가능 |
| 법인 대표자 사망 → 등록소멸 | 대표자 변경하면 됨, **등록 효력 소멸 안 됨** |
| 등록을 안 받고 중개 → 계약 무효 | 처벌은 받으나 **사법상 효력까지 부정되지 않음** |
| 부동산전시업 → 등록 불필요 | 인터넷·부동산전시업도 **반드시 등록 필요** |
| 폐업신고 후 업무 → 무등록 아님 | 폐업신고 **수리** 후 업무하면 무등록업자 |
