# 토지이동 - 신규등록

## 1. 의의

신규등록이란 **새로 조성된 토지**와 **지적공부에 등록되어 있지 아니한 토지**를 지적공부에 등록하는 것을 말한다.

## 2. 대상

| 대상 | 예시 |
|------|------|
| **새로 조성된 토지** | 공유수면매립지, 간척지 |
| **지적공부에 등록되어 있지 않은 토지** | 미등록토지 |

## 3. 신청

- **신청권자**: 토지소유자
- **신청기한**: 사유 발생일부터 **60일 이내** 지적소관청에 신청
- **거짓 신청 시**: **1년 이하 징역** 또는 **1천만원 이하 벌금**

### 첨부서류
1. 법원의 **확정판결서** 정본 또는 사본
2. 공유수면 관리 및 매립에 관한 법률에 의한 **준공검사확인증** 사본
3. 도시계획구역 토지를 지방자치단체 명의로 등록 시 기획재정부장관과 **협의문서** 사본
4. 그 밖에 **소유권을 증명**할 수 있는 서류 사본

> 해당 서류를 지적소관청이 관리하는 경우 → **지적소관청의 확인**으로 서류 제출 갈음 가능

## 4. 지번부여

- **원칙**: 인접토지의 **본번에 부번**을 붙여서 부여
- 인접토지의 본번이 없거나 부번 부여가 부적합한 경우: 해당 지번부여지역의 **최종 본번 다음 순번**

## 5. 처리절차

| 항목 | 내용 |
|------|------|
| **등기촉탁** | **불가** (선등록·후등기 원칙상 등기부가 아직 없음) |
| **소유자 결정** | 지적소관청이 **직접 조사**하여 등록 |
| **대장 개설** | 새로운 토지대장(또는 임야대장) 개설 |

> 신규등록은 토지이동 중 유일하게 **등기촉탁 불가** (등기부 미존재)

## 6. 토지등록의 효력

신규등록은 토지등록의 효력 중 **창설적 효력**에 해당한다.
- 지적공부에 등록됨으로써 토지(부동산)로서의 법적 지위가 **창설**됨

## 핵심 암기 포인트

- 대상: 새로 조성된 토지 + 미등록토지
- 신청기한: **60일** 이내
- 지번부여: 인접토지 **본번에 부번**
- **등기촉탁 불가**: 등기부가 아직 없으므로
- 소유자: 지적소관청이 **직접 조사**하여 등록
- **창설적 효력**: 등록에 의해 토지로서의 법적 지위 발생

## OX 퀴즈

**Q1.** 신규등록 시 등기촉탁을 하여야 한다. → **X** (등기부 미존재로 등기촉탁 불가)

**Q2.** 신규등록은 사유 발생일부터 30일 이내에 신청해야 한다. → **X** (60일 이내)

**Q3.** 신규등록하는 토지의 소유자는 토지소유자가 직접 신고한다. → **X** (지적소관청이 직접 조사하여 등록)

**Q4.** 공유수면매립지는 신규등록의 대상이다. → **O**

**Q5.** 신규등록 시 지번은 인접토지의 본번에 부번을 붙여 부여한다. → **O**

## 함정 노트

- **등기촉탁 불가**: 다른 토지이동(등록전환·분할·합병 등)은 등기촉탁이 가능하나, 신규등록만 **불가**. "선등록·후등기" 원칙
- **60일**: 신규등록·등록전환·분할(용도변경 시)·합병(의무합병)·지목변경 모두 **60일** 이내 신청. 토지이동 신청기한의 공통점
- **소유자 결정**: 신규등록 시 소유자는 소유자의 신고가 아니라 **지적소관청이 직접 조사**. 이 점이 다른 토지이동과 다름
