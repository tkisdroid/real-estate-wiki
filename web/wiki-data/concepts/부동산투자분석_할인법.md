# 할인현금수지분석법 (NPV·IRR·PI)

> 📖 상위 문서: [[부동산투자분석]]

## 1. 의의

- 장래 기대되는 현금수입과 지출을 **현재가치로 할인**하고 비교하여 투자 판단
- **화폐의 시간가치를 고려**
- 여러 해의 소득 기준 → **소득이득과 자본이득 모두** 고려

## 2. 순현재가치법 (NPV)

### 공식

$$NPV = \sum_{t=1}^{n}\frac{CF_t}{(1+r)^t} - 초기투자액$$

$$= 현금유입의\ 현가 - 현금유출의\ 현가$$

### 할인율

- NPV법에서 할인율 = **요구수익률** (사전에 알아야 적용 가능)
- 투자자마다 요구수익률이 다르므로 동일 현금흐름이라도 **NPV가 다를 수 있음**

### 의사결정기준

| 상황 | 기준 |
|------|------|
| **독립적** 투자안 | NPV ≥ 0: **채택** / NPV < 0: **기각** |
| **상호배타적** 투자안 | NPV > 0인 것 중 **가장 큰** 것 선택 |

### NPV법의 장점

- 부의 극대화 기준에서 **우수** (가치가산원리 적용)
- 비정상적 현금흐름에도 **적용 가능**

## 3. 내부수익률법 (IRR)

### 공식

$$NPV = 0이\ 되는\ 할인율 = IRR$$

$$\sum_{t=1}^{n}\frac{CF_t}{(1+IRR)^t} = 초기투자액$$

- IRR = NPV를 **0**으로 만드는 할인율
- IRR = 수익성지수(PI)를 **1**로 만드는 할인율

### 의사결정기준

| 상황 | 기준 |
|------|------|
| **독립적** 투자안 | IRR ≥ 요구수익률: **채택** / IRR < 요구수익률: **기각** |
| **상호배타적** 투자안 | IRR > 요구수익률인 것 중 **가장 큰** 것 선택 |

### IRR법의 한계

- **복수의 IRR** 존재 가능 (비정상적 현금흐름 시)
- **가치가산원리 미적용** (부의 극대화 보장 못함)

## 4. 수익성지수법 (PI, 편익비용비율)

### 공식

$$PI = \frac{현금유입의\ 현재가치\ 합}{현금유출의\ 현재가치\ 합(초기투자액)}$$

### 의사결정기준

| 상황 | 기준 |
|------|------|
| **독립적** 투자안 | PI ≥ 1: **채택** / PI < 1: **기각** |
| **상호배타적** 투자안 | PI > 1인 것 중 **가장 큰** 것 선택 |

### PI와 NPV의 관계

- NPV > 0 ↔ PI > 1 ↔ IRR > 요구수익률 (방향 일치)
- NPV = 0 ↔ PI = 1 ↔ IRR = 요구수익률
- PI = 투자안의 **상대적 수익성** (규모 다른 투자안 비교 용이)

## 5. NPV법 vs IRR법 비교

| 구분 | NPV법 | IRR법 |
|------|-------|-------|
| **할인율** | **요구수익률** 사용 | - |
| **재투자수익률 가정** | **요구수익률** | **IRR** |
| **상호배타적 투자안** | **우월** (일관된 결과) | 결과 상이 가능 |
| **비정상적 현금흐름** | 적용 가능 | **복수해** 발생 가능 |
| **가치가산원리** | **적용** | 미적용 |
| **부의 극대화** | **보장** | 보장 못함 |

> **결론**: NPV법과 IRR법의 결과가 상이할 때는 **NPV법이 우월**하다는 것이 일반적 견해.

## 6. 투자규모 차이 시

- 투자규모에 차이가 있는 상호배타적 투자안의 경우
- **NPV법과 수익성지수법**을 통한 의사결정이 **달라질 수 있음**
- 부의 극대화 = NPV법 / 효율성 = 수익성지수법

## 핵심 암기 포인트

- NPV: 현금유입 현가 - 현금유출 현가, 할인율 = **요구수익률**, **> 0이면 채택**
- IRR: NPV = **0**이 되는 할인율 = PI가 **1**이 되는 할인율
- PI: 유입 현가 / 유출 현가, **> 1이면 채택** (= NPV > 0)
- 재투자수익률: NPV = **요구수익률** / IRR = **IRR 자체** → NPV가 더 합리적
- 상호배타적: **NPV법 우월**, 가치가산원리 적용
- IRR: **복수해** 발생 가능 (비정상적 현금흐름)

## OX 퀴즈

**Q1.** 순현재가치(NPV)는 투자자의 내부수익률로 할인한 값이다. → **X** (투자자의 **요구수익률**로 할인)

**Q2.** 내부수익률은 순현가를 0보다 작게 하는 할인율이다. → **X** (순현가를 **0으로** 만드는 할인율)

**Q3.** 수익성지수(PI)가 0보다 크면 투자타당성이 있다. → **X** (PI는 **1보다 크면** 투자 채택, 0이 아님)

**Q4.** 재투자율 가정에서 NPV법보다 IRR법이 더 합리적이다. → **X** (**NPV법**이 더 합리적)

**Q5.** 투자자산의 현금흐름에 따라 복수의 IRR이 존재할 수 있다. → **O**

**Q6.** NPV법은 가치가산원리가 적용되나 IRR법은 적용되지 않는다. → **O**

**Q7.** 순현재가치가 0인 투자안의 수익성지수는 0이다. → **X** (NPV = 0이면 PI = **1**)

**Q8.** 동일한 현금흐름이라도 투자자의 요구수익률에 따라 NPV가 달라질 수 있다. → **O**

## 함정 노트

- ⚠️ "NPV 할인율 = IRR" → NPV는 **요구수익률**, IRR은 NPV=0 되는 율
- ⚠️ "IRR = NPV를 0보다 작게" → 0**으로 만드는** 할인율
- ⚠️ "PI > 0이면 채택" → PI > **1**이면 채택 (0이 아닌 1이 기준)
- ⚠️ "NPV = 0 → PI = 0" → NPV = 0이면 PI = **1**
- ⚠️ "IRR이 NPV보다 합리적" → **NPV가 IRR보다** 합리적 (재투자율, 가치가산원리)
- ⚠️ "IRR = 수익성지수를 0으로" → IRR = PI를 **1**로 만드는 할인율
