# 공인중개사협회 (법 제41~43조)

## 1. 협회의 성격

| 항목 | 내용 |
|---|---|
| 법적 성격 | **비영리사단법인** |
| 설립방식 | **인가주의** (국토교통부장관 인가) |
| 설립 | **임의설립주의** ("설립할 수 있다") |
| 가입 | **임의가입주의** (가입 자유, 미가입 시 제재 없음) |
| 가입 자격 | **개업공인중개사만** (공인중개사는 가입 불가) |

## 2. 설립 요건 (법 제41조 제3항, 영 제30조)

| 요건 | 내용 |
|---|---|
| 발기인 | 회원 **300인 이상** |
| 창립총회 | 개업공인중개사 **600인 이상** 출석 |
| 지역 요건 | 서울 **100인 이상**, 광역시·도 각 **20인 이상** |
| 의결 | 출석한 개업공인중개사 **과반수** 동의 |
| 설립등기 | 주된 사무소 소재지에서 등기 |

## 3. 조직

| 조직 | 설치 장소 | 신고 대상 |
|---|---|---|
| 지부 | 시·도 | 시·도지사 |
| 지회 | 시·군·구 | 등록관청 |

- 최고의결기관: **총회**
- 총회 의결 내용: **지체없이** 국토교통부장관에게 보고

## 4. 협회의 업무

### 고유업무 (영 제31조)
1. 회원의 **품위유지** 업무
2. 부동산중개제도 **연구·개선**
3. 회원의 **자질향상** 지도·교육·연수
4. 회원의 **윤리헌장** 제정 및 실천
5. 부동산 **정보제공**
6. **공제사업** (비영리, 상호부조 목적)
7. 기타 설립목적 달성 업무

### 수탁업무
- 교육 업무 (시·도지사로부터 위탁)
- 공인중개사 시험 업무 (시험시행기관으로부터 위탁)

## 5. 공제사업 (법 제42조)

### 목적
- 개업공인중개사의 **손해배상 보장**을 위한 사업

### 공제사업 범위 (영 제34조)
1. 손해배상책임 보장을 위한 **공제기금 조성** 및 공제금 지급
2. 공제사업 **부대업무**

### 공제규정
- 공제사업 시 **공제규정 제정** → 국토교통부장관 **승인** 필요 (변경 시에도)
- 공제규정에 포함될 사항:
  - 공제사업의 범위
  - 공제계약의 내용
  - 공제금의 징수 및 지급
  - 공제기금의 조성·관리·운용
  - 기타 사업 운영에 필요한 사항

### 운용실적 공시
- 협회는 매 사업연도 종료 후 **3개월 이내** 공제사업 운용실적을 공시
- 미공시 시: **500만원 이하 과태료**

### 감독
- 국토교통부장관은 공제업무의 건전한 운용을 위하여 **개선명령** 가능
- 불이행 시: **500만원 이하 과태료**
- 협회 **임원에 대한 징계·해임 요구** 가능
