# 행정처분 - 업무정지 (법 제39조)

## 1. 개요

| 항목 | 내용 |
|------|------|
| **처분 대상** | 개업공인중개사 |
| **처분청** | **등록관청** |
| **처분 성격** | **임의적 (명할 수 있다)** — 재량 있음 |
| **기간 범위** | **6개월** 범위 내 |
| **법인 특례** | 법인 또는 **분사무소별**로 업무정지 가능 |
| **청문** | 대상 **X** (의견제출 기회만 부여) |
| **핵심 기능** | **임의적 등록취소 대체** 가능 |

## 2. 업무정지 사유

| 번호 | 사유 | 비고 |
|------|------|------|
| 1 | 결격사유 있는 고용인(소속공인중개사·중개보조원)을 둔 경우 | 2개월 내 해소 시 제외 |
| 2 | **인장등록** 미이행 또는 미등록 인장 사용 | |
| 3 | **전속중개계약서** 미사용 또는 계약서 미보존 | |
| 4 | 거래정보망에 정보 **거짓공개** 또는 거래완성 사실 **미통보** | |
| 5 | **확인·설명서** 미교부 또는 미보존 | |
| 6 | 확인·설명서에 **서명·날인** 미이행 | |
| 7 | **거래계약서** 미작성·미교부·미보존 | |
| 8 | 거래계약서에 **서명·날인** 미이행 | |
| 9 | 등록관청의 지도·감독 **명령 불응** | |
| 10 | **임의적 등록취소** 사유에 해당하는 경우 (등록취소 **대신** 업무정지) | 핵심 |
| 11 | 이 법의 명령·처분 위반 | |

## 3. 등록취소 대체 기능

### 핵심 원리
- **임의적 등록취소** 사유가 있는 경우 → 등록취소 **대신** 업무정지 가능
- **절대적 등록취소** 사유 → 업무정지로 대체 **불가** (반드시 등록취소)

### 업무정지 → 등록취소 격상 구조

```
1회 업무정지 → (1년 이내) → 2회 업무정지 → (다시 사유) → 절대적 등록취소
1회 업무정지/과태료 → ... → 3회 업무정지/과태료 → (다시 사유) → 임의적 등록취소
```

## 4. 업무정지 기간 중 업무수행 시

업무정지 기간 중 중개업무를 수행 → **절대적 등록취소** 사유 (법 제38조 제1항 제7호)

## 5. 업무정지 기준

업무정지의 세부 기준은 **국토교통부령**(규칙 별표 2)으로 정한다.

| 위반행위 | 1차 | 2차 | 3차 이상 |
|----------|-----|-----|---------|
| 인장등록·사용 위반 | 업무정지 3개월 | 업무정지 6개월 | 등록취소 |
| 확인·설명서 미교부·미보존 | 업무정지 3개월 | 업무정지 6개월 | 등록취소 |
| 거래계약서 미작성·미교부·미보존 | 업무정지 3개월 | 업무정지 6개월 | 등록취소 |
| 지도·감독 명령 불응 | 업무정지 6개월 | 등록취소 | - |

> 가중·경감: **1/2 범위** 내 가중·경감 가능 (가중해도 6개월 초과 불가)

## 6. 자격정지와 업무정지 비교

| 비교 항목 | 자격정지 | 업무정지 |
|-----------|---------|---------|
| **대상** | 소속공인중개사 | 개업공인중개사 |
| **처분청** | 시·도지사 | 등록관청 |
| **성격** | 임의적 | 임의적 |
| **기간** | 6개월 범위 | 6개월 범위 |
| **청문** | X | X |
| **등록취소 대체** | - | **가능** |
| **위반 시** | 자격취소 | 등록취소 |

## 핵심 암기 포인트

- 처분청: **등록관청** (등록취소와 동일)
- **임의적 처분**: "명할 수 **있다**"
- 기간: **6개월** 범위 내 (자격정지와 동일)
- **등록취소 대체 가능**: 임의적 등록취소 사유 → 업무정지로 대체 가능
- 법인은 **분사무소별** 업무정지 가능
- 업무정지 중 업무 → **절대적 등록취소**
- 1년 내 **2회** 업무정지 후 재위반 → **절대적 등록취소**

## OX 퀴즈

**Q1.** 업무정지는 시·도지사가 처분한다. → **X** (등록관청)

**Q2.** 절대적 등록취소 사유를 업무정지로 대체할 수 있다. → **X** (임의적 사유만 대체 가능)

**Q3.** 법인인 개업공인중개사에 대하여는 분사무소별로 업무정지를 명할 수 있다. → **O**

**Q4.** 업무정지기간은 최대 1년이다. → **X** (6개월)

**Q5.** 업무정지 기간 중 업무를 하면 임의적 등록취소 사유이다. → **X** (절대적 등록취소 사유)

**Q6.** 업무정지 시 청문을 실시해야 한다. → **X** (의견제출 기회만 부여)

**Q7.** 1년 내 2회 업무정지 후 다시 업무정지 사유 행위를 하면 등록이 취소되어야 한다. → **O** (절대적 등록취소)

## 함정 노트

- **업무정지 vs 자격정지**: 개업공인중개사=업무정지, 소속공인중개사=자격정지. 동일한 위반행위(인장미등록 등)에 대해 대상에 따라 처분 종류가 다름
- **등록취소 대체**: "임의적 등록취소 사유 = 업무정지로 대체 가능"이 핵심. 절대적은 대체 불가
- **2회 vs 3회 구별**: 2회 업무정지 후 재위반 = 절대적 등록취소, 3회 업무정지·과태료 후 재위반 = 임의적 등록취소
- **분사무소별 업무정지**: 법인 전체가 아닌 해당 분사무소만 정지 가능한 점이 출제 포인트
- **청문 대상 구분**: 자격취소=O, 등록취소=O(사망해산 제외), 자격정지=X, 업무정지=X
