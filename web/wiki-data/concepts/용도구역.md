# 용도구역

[[용도지역]] 및 [[용도지구]]의 제한을 강화하거나 완화하여 시가지의 무질서한 확산 방지, 계획적 토지이용 도모 등을 위해 **도시·군관리계획으로 결정**하는 지역.

## 용도구역의 종류

### 1. 개발제한구역 (그린벨트)
| 항목 | 내용 |
|------|------|
| 지정권자 | **국토교통부장관** |
| 목적 | 도시 무질서 확산 방지, 자연환경 보전, 건전한 생활환경 확보, 보안상 도시개발 제한 |
| 행위제한 | **개발제한구역의 지정 및 관리에 관한 특별조치법**으로 별도 규정 |

### 2. 도시자연공원구역
| 항목 | 내용 |
|------|------|
| 지정권자 | **시·도지사 또는 대도시 시장** |
| 목적 | 도시 자연환경·경관 보호, 여가·휴식공간 제공, 도시지역 내 식생 양호 산지 개발제한 |
| 행위제한 | 별도 법률로 정함 |

### 3. 수산자원보호구역
| 항목 | 내용 |
|------|------|
| 지정권자 | **해양수산부장관** |
| 목적 | 수산자원 보호·육성 |
| 대상 | 공유수면이나 인접 토지 |
| 건축제한 | **수산자원관리법** 규정에 따름 |

### 4. 시가화조정구역
| 항목 | 내용 |
|------|------|
| 지정권자 | **시·도지사** (예외: 국토교통부장관 - 국가계획 연계 시) |
| 목적 | 도시지역과 주변의 무질서한 시가화 방지, 계획적·단계적 개발 도모 |
| 유보기간 | **5년 이상 20년 이내** |
| 실효 | 유보기간 끝난 날의 **다음 날** 자동 실효 (별도 고시 불요) |

#### 시가화조정구역 내 행위
- **원칙**: 도시·군계획사업 시행 유보
- **허가 대상**: 농림어업 시설, 주택 증축(기존 포함 100㎡ 이하), 마을공동시설 등
- 관리용 건축물: 기존 포함 **33㎡ 이하**
- 축사: 1가구당 기존 포함 **300㎡ 이하** (허가 거부 불가)

### 5. 도시혁신구역 (신설)
| 항목 | 내용 |
|------|------|
| 지정권자 | 공간재구조화계획 결정권자 |
| 목적 | 창의적·혁신적 도시공간 개발 |
| 특례 | **특별건축구역** 의제 |
| 계획 | 도시혁신계획 (공간재구조화계획으로 결정) |

### 6. 복합용도구역 (신설)
| 항목 | 내용 |
|------|------|
| 지정권자 | 공간재구조화계획 결정권자 |
| 목적 | 주거·상업·산업·교육·문화·의료 등 융복합 공간 조성 |
| 특례 | **특별건축구역** 의제 |

### 7. 도시·군계획시설입체복합구역
- 도시·군계획시설 준공 후 **10년** 경과 시 등 지정 가능
- 건폐율: 해당 용도지역 최대한도의 **150%** 이하
- 용적률: 해당 용도지역 최대한도의 **200%** 이하
- 높이: 가로구역 제한의 150%, 일조권 제한의 200% 이하

## 시험 핵심 포인트
- 개발제한구역 결정권자: **국토교통부장관**
- 시가화조정구역 유보기간: **5~20년**, 실효는 **자동** (다음 날)
- 도시혁신구역·복합용도구역: **특별건축구역** 자동 의제

## 관련 개념

## 핵심 암기 포인트

- **개발제한구역 지정권자**: **국토교통부장관**
- **도시자연공원구역 지정권자**: **시·도지사 또는 대도시 시장**
- **수산자원보호구역 지정권자**: **해양수산부장관**
- **시가화조정구역 지정권자**: **시·도지사** (예외: 국토교통부장관)
- **시가화조정구역 유보기간**: **5년 이상 20년 이내**
- **시가화조정구역 실효**: 유보기간 끝난 날의 **다음 날** 자동 실효
- **도시혁신구역·복합용도구역**: **특별건축구역** 의제
- **입체복합구역 건폐율**: 해당 용도지역의 **150%**, 용적률 **200%**
- **시가화조정구역 축사**: 1가구당 기존 포함 **300㎡ 이하** (허가 거부 불가)
- **관리용 건축물**: 기존 포함 **33㎡ 이하**

## OX 퀴즈

**Q1.** 개발제한구역의 지정권자는 시·도지사이다. → **X** (국토교통부장관)

**Q2.** 시가화조정구역의 유보기간은 3년 이상 10년 이내이다. → **X** (5년 이상 20년 이내)

**Q3.** 시가화조정구역은 유보기간이 끝나면 별도의 해제 고시 없이 자동 실효된다. → **O** (다음 날 자동 실효)

**Q4.** 수산자원보호구역의 지정권자는 국토교통부장관이다. → **X** (해양수산부장관)

**Q5.** 도시혁신구역은 특별건축구역으로 의제된다. → **O**

**Q6.** 시가화조정구역 내에서 축사를 건축하려는 경우 허가를 거부할 수 있다. → **X** (1가구당 300㎡ 이하 축사는 허가 거부 불가)

**Q7.** 도시·군계획시설입체복합구역의 건폐율은 해당 용도지역 최대한도의 200% 이하이다. → **X** (건폐율 150%, 용적률이 200%)

## 함정 노트

- ⚠️ **지정권자 혼동**: 개발제한구역=**국토교통부장관**, 도시자연공원구역=**시·도지사**, 수산자원보호구역=**해양수산부장관**, 시가화조정구역=**시·도지사**. 각각 다르므로 구분 필수
- ⚠️ **5~20년**: 시가화조정구역 유보기간. "3~10년", "5~10년" 등은 오답
- ⚠️ **자동 실효**: 시가화조정구역은 별도 고시 **없이** 자동 실효. "해제 고시 필요"는 오답
- ⚠️ **150% vs 200%**: 입체복합구역 건폐율은 **150%**, 용적률은 **200%**. 숫자 바꿔 출제 주의
- ⚠️ **의제 규정**: 도시혁신구역·복합용도구역 → 특별건축구역 의제. "지구단위계획구역 의제"와 혼동 주의
- ⚠️ **300㎡ vs 33㎡**: 시가화조정구역 내 축사 **300㎡**, 관리용 건축물 **33㎡**
