# 토지이동 - 분할

## 1. 의의

분할이란 지적공부에 등록된 **1필지**를 **2필지 이상**으로 나누어 등록하는 것을 말한다.

## 2. 분할 신청 사유

### 토지소유자의 임의 분할

관계법령에 따라 개발행위 허가 등의 대상인 경우에는 **허가를 받은 후** 분할 신청 가능

| 사유 |
|------|
| **소유권 이전**(매매 등)을 위하여 필요한 경우 |
| 토지이용상 **불합리한 지상경계**를 시정하기 위한 경우 |

### 의무적 분할

- 지적공부에 등록된 **1필지의 일부**가 형질변경 등으로 **용도가 변경**된 경우
- 용도 변경일부터 **60일 이내** 분할 신청 (지목변경 신청서도 함께 제출)

## 3. 분할에 따른 지번부여

### 원칙
- 분할 후 **1필지**는 분할 전 지번 유지
- 나머지 필지는 **본번의 최종 부번 다음 순번**

### 예외 (우선 지번 부여)
- 주거·사무실 등 **건축물이 있는 필지**에는 분할 전 지번을 **우선** 부여

### 예시
```
분할 전: 10번지 (1필지)
분할 후: 10번지 + 10-5번지 (10번지의 최종 부번이 4인 경우)

건축물이 있는 토지 = 10번지 (원래 지번 우선 배정)
```

## 4. 분할의 처리절차

| 절차 | 내용 |
|------|------|
| **측량** | 분할측량 실시 (원칙) |
| **측량 불요** | 합병된 토지를 **합병 전 경계대로** 분할하는 경우 |
| **대장** | 분할 후 새 대장에 **분할 당시의 지목** 등록 |
| **지목 변경** | 1필지 일부 용도 변경으로 분할한 경우 → 새로운 지목 등록 |
| **면적** | 분할측량 후 새로 측정된 면적 등록 (분할 전 면적과 **증감 없도록**) |

### 면적 결정 특례
- 합병된 토지를 합병 전 경계대로 분할 → 합병 전 각 필지의 면적을 분할 후 면적으로 함

## 5. 분할 제한

### 개발행위허가에 의한 제한
- 관계법령에 따라 분할이 **개발행위 허가** 등의 대상인 경우 → 허가를 받아야 분할 신청 가능

### 국토계획법상 분할 제한 (최소면적)

| 용도지역 | 최소 분할면적 |
|----------|-------------|
| 녹지지역 | **200㎡** |
| 관리지역 | **200㎡** (보전·생산·계획) |
| 농림지역 | **200㎡** |
| 자연환경보전지역 | **200㎡** |
| 도시지역 내 주거·상업·공업지역 | **60㎡** |

> 관계법령에 따른 분할의 경우에는 최소면적 제한 적용 배제 가능

## 핵심 암기 포인트

- **1필지 → 2필지 이상**
- 지번부여: 1필지는 원래 지번, 나머지는 **본번의 최종 부번 다음 순번**
- 건축물 있는 필지에 **원래 지번 우선** 부여
- 면적: 분할 전과 **증감 없도록** 결정
- 합병 전 경계대로 분할 → **측량 불요**
- 용도변경 분할: **60일** 이내 + 지목변경 신청서 동시 제출

## OX 퀴즈

**Q1.** 분할 시 모든 필지에 새로운 지번이 부여된다. → **X** (1필지는 분할 전 지번 유지)

**Q2.** 합병된 토지를 합병 전의 경계대로 분할하는 경우 지적측량이 필요하다. → **X** (측량 불요)

**Q3.** 분할 후 면적의 합은 분할 전 면적과 동일하여야 한다. → **O** (증감 없도록)

**Q4.** 건축물이 있는 필지에 분할 전 지번을 우선 부여한다. → **O**

**Q5.** 1필지 일부의 용도 변경 시 60일 이내에 분할을 신청해야 한다. → **O**

**Q6.** 녹지지역에서의 최소 분할면적은 60㎡이다. → **X** (200㎡, 60㎡는 주거·상업·공업지역)

## 함정 노트

- **지번부여 "최종 부번 다음 순번"**: 본번이 10이고 최종 부번이 3이면, 분할 시 10-4, 10-5 등으로 부여
- **건축물 우선 지번**: 분할 전 지번은 건축물이 있는 필지에 **우선** 배정. 없으면 면적이 큰 쪽 등 규칙 적용
- **합병 전 경계대로 분할**: 이 경우만 측량 불요. 일반 분할은 반드시 **분할측량** 필요
- **면적 증감 없음**: 측량 결과 면적이 달라지더라도 분할 전후 합계가 동일하도록 조정
