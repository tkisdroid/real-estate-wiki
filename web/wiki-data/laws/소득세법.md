# 소득세법 (양도소득세)

소득세법 중 **양도소득세** 부분은 개인이 토지·건물 등 자산을 양도함으로써 발생하는 소득에 대하여 과세하는 규정이다. 매년 **4~6문항**이 출제되며, 계산문제가 포함되는 최고 난이도 영역이다.

## 1. 양도소득세의 위치와 특징

### 소득세법 체계 속 양도소득세

소득세법은 개인의 소득을 종합소득·퇴직소득·양도소득으로 구분하여 과세한다. 양도소득세는 **분류과세**로서 다른 소득과 합산하지 않고 별도로 과세한다.

```
소득세
├── 종합소득 (이자·배당·사업·근로·연금·기타)
├── 퇴직소득
└── 양도소득 ← 부동산세법 시험 범위
```

### 핵심 특징

| 특징 | 내용 |
|------|------|
| **국세** | 세무서장이 관할 |
| **직접세** | 납세의무자 = 담세자 |
| **신고납부** | 예정신고(2개월) + 확정신고(다음해 5월) |
| **개인만 과세** | 법인은 법인세로 과세 |
| **초과누진세율** | 기본세율 8단계 (6~45%) |
| **분류과세** | 종합소득과 합산 불가 |
| **실현과세** | 양도 시점에 과세 (미실현 이익 불과세) |
| **양도차손 통산 불가** | 다른 소득과 손익통산 안 됨 |

## 2. 과세대상 자산

| 분류 | 과세대상 |
|------|----------|
| **토지·건물** | 모든 토지와 건물 |
| **부동산에 관한 권리** | 지상권, 전세권, 등기된 부동산임차권 |
| **부동산취득권리** | 조합원입주권, 분양권 |
| **기타 자산** | 영업권(사업용 고정자산과 함께 양도 시), 특정시설물 이용권 등 |
| **주식 등** | 상장·비상장 주식 |

## 3. 양도의 개념

**양도** = 자산의 등기·등록에 관계없이 매도, 교환, 현물출자 등으로 **유상 사실상 이전** (소득세법 제88조)

### 양도 해당 vs 비해당

| 양도 해당 | 양도 비해당 |
|-----------|------------|
| 매매, 교환, 현물출자 | 신탁재산 위탁자→수탁자 이전 |
| 대물변제, 수용, 경매 | 환지처분(지목·면적 불변) |
| 부담부증여 채무인수액 | 도시개발법상 환지 |

## 4. 양도소득세 계산구조

```
양도가액 (실지거래가액)
- 취득가액 (실지거래가액)
- 필요경비 (자본적지출액, 양도비)
─────────────────────────
= 양도차익
- 장기보유특별공제
─────────────────────────
= 양도소득금액
- 양도소득기본공제 (연 250만원)
─────────────────────────
= 양도소득과세표준
× 세율
─────────────────────────
= 양도소득 산출세액
```

## 5. 비과세

### 1세대1주택 비과세 (제89조)

| 요건 | 내용 |
|------|------|
| 보유기간 | **2년 이상** |
| 거주요건 | 조정대상지역: **2년 이상 거주** |
| 고가주택 | 양도가액 **12억원 초과** 시 초과분만 과세 |
| 일시적 2주택 | 종전주택 1년 이상 보유 후 신규취득 → **3년**(조정 2년) 이내 종전주택 양도 |

### 기타 비과세
- 농지의 교환·분합
- 8년 이상 자경농지 (조세특례제한법)

> 상세 내용: [[양도소득세_비과세]]

| 과세표준 | 세율 |
|----------|------|
| 1,400만원 이하 | **6%** |
| 5,000만원 이하 | **15%** |
| 8,800만원 이하 | **24%** |
| 1.5억원 이하 | **35%** |
| 3억원 이하 | **38%** |
| 5억원 이하 | **40%** |
| 10억원 이하 | **42%** |
| 10억원 초과 | **45%** |

### 특수 세율

| 구분 | 세율 |
|------|------|
| 주택 1년 미만 보유 | **70%** |
| 주택 1~2년 보유 | **60%** |
| 분양권 1년 미만 | **70%** |
| 분양권 1년 이상 | **60%** |
| 미등기양도자산 | **70%** |
| 비사업용토지 | 기본세율 **+10%** |
| 조정지역 2주택 | 기본세율 **+20%** |
| 조정지역 3주택+ | 기본세율 **+30%** |

| 구분 | 기한 |
|------|------|
| 일반 양도 | 양도일 속하는 달의 말일부터 **2개월** |
| 부담부증여 채무인수액 | 양도일 속하는 달의 말일부터 **3개월** |

- 예정신고: 양도소득세 예정신고기한 + **2개월**

## 9. 양도소득 계산의 특례

### 이월과세 (배우자·직계존비속)

| 항목 | 내용 |
|------|------|
| 적용 대상 | **10년 이내** 증여받은 자산 양도 |
| 취득가액 | **증여자의 취득가액** |
| 보유기간 | **증여자의 취득일**부터 기산 |
| 납세의무자 | **수증자** |
| 연대납세의무 | **없음** |
| 증여세 처리 | 필요경비에 **산입** |

### 부당행위계산부인 (우회양도)

| 항목 | 내용 |
|------|------|
| 적용 대상 | 특수관계인(배우자·직계존비속 **제외**)에게 증여 후 **10년 이내** 양도 |
| 판단 기준 | 수증자(증여세+양도세) < 증여자 직접 양도 시 양도세 |
| 납세의무자 | **증여자** |
| 연대납세의무 | **있음** |
| 증여세 처리 | **비과세** |

> 상세 내용: [[양도소득세_이월과세_부당행위]]

## 10. 미등기양도자산의 불이익

미등기양도자산에 대하여는 5가지 세제상 불이익이 적용된다:

1. **비과세·감면 배제**
2. **장기보유특별공제 배제**
3. **양도소득기본공제 배제**
4. **세율 70% 중과**
5. **필요경비 개산공제 0.3%** (일반 3%)

## 핵심 암기 포인트

- **양도소득세**: 국세, 직접세, 신고납부, 개인만 과세, 분류과세, 초과누진세율(8단계)
- **기본세율 8단계**: 6%→15%→24%→35%→38%→40%→42%→45%
- **1세대1주택 비과세**: 2년 보유(조정 거주 추가), 고가주택 12억 초과분만 과세
- **일시적 2주택**: 종전 1년 이상 보유 후 신규 → 3년(조정 2년) 이내 종전 양도
- **장기보유특별공제**: 일반 매년 2%(한도 30%), 1세대1주택 보유4%+거주4%=최대 80%
- **단기보유 세율**: 주택 1년 미만 70%, 1~2년 60%
- **미등기양도**: 70%, 장특공제·기본공제·비과세 모두 배제, 개산공제 0.3%
- **예정신고**: 양도일 속하는 달 말일부터 2개월(부담부증여 3개월)
- **확정신고**: 다음해 5/1~5/31
- **이월과세**: 배우자·직계존비속, 10년 내 증여, 납세의무자=수증자, 연대의무 없음
- **부당행위계산부인**: 특수관계인(배우자·직계 제외), 10년 내 양도, 납세의무자=증여자, 연대의무 있음

## OX 퀴즈

**Q1.** 양도소득세는 법인에게도 부과된다. → **X** (개인에게만 과세. 법인은 법인세로 과세)

**Q2.** 양도소득세는 종합소득과 합산하여 과세된다. → **X** (분류과세로서 종합소득과 합산하지 않는다)

**Q3.** 소득세법상 양도란 유상이전뿐 아니라 무상이전도 포함한다. → **X** (양도는 유상 사실상 이전만 해당. 무상이전은 증여)

**Q4.** 1세대1주택 비과세 요건 중 보유기간은 3년 이상이다. → **X** (2년 이상. 조정지역은 2년 거주 추가)

**Q5.** 고가주택의 기준은 양도가액 15억원 초과이다. → **X** (12억원 초과)

**Q6.** 양도소득세 예정신고 기한은 양도일이 속하는 달의 말일부터 3개월이다. → **X** (2개월. 부담부증여만 3개월)

**Q7.** 이월과세와 부당행위계산부인 모두 납세의무자는 수증자이다. → **X** (이월과세=수증자, 부당행위계산부인=증여자)

## 함정 노트

- ⚠️ **양도 해당 여부**: 신탁재산 이전, 환지처분은 양도 아님. "대물변제, 경매"는 양도에 해당 — 사례별 구분 필수
- ⚠️ **예정신고 2개월 vs 3개월**: 일반양도 2개월, 부담부증여 채무인수액 3개월. "3개월"로 통일하는 함정 주의
- ⚠️ **이월과세 vs 부당행위계산부인 혼동**: 대상자(배우자·직계 vs 배우자·직계 제외 특수관계인), 납세의무자(수증자 vs 증여자), 연대의무(없음 vs 있음), 증여세(필요경비 산입 vs 비과세) — 4가지 차이점 반드시 암기
- ⚠️ **미등기양도 70% vs 1년 미만 70%**: 같은 세율이지만 미등기는 추가로 장특공제·기본공제·비과세 모두 배제되므로 불이익이 훨씬 큼
- ⚠️ **장기보유특별공제 한도**: 일반 30%(15년) vs 1세대1주택 80%(보유40%+거주40%) — 한도 혼동 빈출
- ⚠️ **양도차손 통산 불가**: 부동산 양도차손을 사업소득 등 다른 소득에서 공제 불가
- ⚠️ **지방소득세 신고기한**: 양도소득세 신고기한에 "2개월을 더한 날"까지 — 별도 기한임을 주의

## 주요 조문 원문

> 출처: 소득세법 [법률 제20610호, 2025. 2. 18., 일부개정]

### 제88조 (정의)

"양도"란 자산에 대한 등기 또는 등록에 관계없이 매도, 교환, 법인에 대한 현물출자 등으로 인하여 그 자산이 **유상으로 사실상 이전**되는 것을 말한다.

> 시험 포인트: "등기·등록에 관계없이" — 미등기라도 양도 해당. "유상 사실상 이전" — 무상(증여)은 양도 아님.

---

### 제89조 (비과세 양도소득) 제1항 제3호

1세대가 1주택(고가주택 제외)을 보유하는 경우로서 **2년 이상** 보유한 것을 양도하는 경우 양도소득세를 과세하지 아니한다. 다만, 대통령령으로 정하는 **조정대상지역에 있는 주택**의 경우에는 해당 주택에서 **거주한 기간이 2년 이상**인 것으로 한정한다.

> 시험 포인트: 비과세 핵심 요건 = 1세대 + 1주택 + 2년 보유(조정은 2년 거주 추가). 고가주택(12억 초과)은 초과분만 과세.

---

### 제95조 (양도소득금액)

② 장기보유특별공제액은 토지 또는 건물의 보유기간이 **3년 이상**인 경우에 양도차익에 보유기간별 공제율을 곱하여 계산한다.

> 시험 포인트: 장기보유특별공제는 보유기간 3년 이상부터 적용. 1세대1주택은 보유+거주 각각 연 4% (합계 최대 80%).

---

### 제104조 (세율)

양도소득에 대한 소득세는 해당 과세기간의 양도소득과세표준에 다음의 세율을 적용하여 계산한 금액을 그 세액으로 한다.

1. **기본세율**: 6%, 15%, 24%, 35%, 38%, 40%, 42%, 45% (8단계 초과누진)
2. **1년 미만** 보유 토지·건물: **70%** (주택·조합원입주권·분양권)
3. **1~2년** 보유 주택: **60%**
4. **미등기양도자산**: **70%**
5. **비사업용토지**: 기본세율 + **10%**

> 시험 포인트: 세율 구간별 정확한 암기 필수. 특히 단기보유·미등기 세율(70%, 60%)은 매년 출제.

---

### 제110조 (확정신고)

거주자는 해당 과세기간의 양도소득금액이 있는 경우에는 그 양도소득과세표준을 그 과세기간의 **다음 연도 5월 1일부터 5월 31일까지** 납세지 관할 세무서장에게 신고하여야 한다.

> 시험 포인트: 확정신고 = 다음해 5/1~5/31. 예정신고를 한 경우에도 2건 이상 양도 등은 확정신고 필요.
