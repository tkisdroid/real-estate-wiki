# 부동산 실권리자명의 등기에 관한 법률 (부동산실명법)

## 목적

부동산에 관한 소유권 기타 물권을 **실체적 권리관계에 부합하도록** 실권리자 명의로 등기하게 함으로써 부동산등기제도를 악용한 투기·탈세·탈법행위를 방지하고 부동산 거래의 정상화와 부동산 가격의 안정을 도모

## 명의신탁약정의 효력

### 원칙: 무효 (제4조)

- **명의신탁약정**은 무효
- 명의신탁약정에 의한 **등기**로 이루어진 부동산 물권변동도 무효

## 명의신탁의 유형별 효과

### 1. 2자간 등기명의신탁 (신탁자-수탁자)

| 항목 | 효과 |
|------|------|
| 명의신탁약정 | **무효** |
| 물권변동 | **무효** (소유권은 신탁자에게 남음) |
| 수탁자의 처분 | 제3자가 선의인 경우 **유효** |

- 신탁자가 수탁자에게 직접 이전등기 → 신탁약정 무효 → 소유권은 **신탁자에게**
- 수탁자 명의의 등기는 **원인무효**

### 2. 3자간 등기명의신탁 (매도인-신탁자-수탁자)

| 항목 | 효과 |
|------|------|
| 명의신탁약정 | **무효** |
| 매도인-수탁자 간 물권변동 | **무효** |
| 매도인-신탁자 간 매매계약 | **유효** |

- 매도인으로부터 수탁자에게 직접 이전등기한 경우
- 물권은 **매도인에게** 남아 있음
- 신탁자는 매도인에 대한 **채권적 청구권**(소유권이전등기청구권) 보유
- 매도인이 부동산실명법 위반 사실을 알지 못한 경우 → 매도인은 수탁자에 대해 **등기 말소 후 신탁자에게** 이전등기 의무

### 3. 계약명의신탁 (매도인과 수탁자가 계약)

| 항목 | 효과 |
|------|------|
| 명의신탁약정 | **무효** |
| 매도인-수탁자 간 매매계약 | **무효** |
| 물권변동 | **무효** |

- 수탁자가 자기 명의로 매도인과 매매계약을 체결
- 매매계약 자체가 **무효** → 소유권은 **매도인에게** 남아 있음
- 다만, **매도인이 선의**인 경우 수탁자에 대하여 물권변동의 무효를 주장할 수 없음 (제4조 제3항)

## 명의신탁 유형별 비교

| 구분 | 2자간 | 3자간 | 계약명의신탁 |
|------|-------|-------|-------------|
| **소유권 귀속** | 신탁자 | 매도인 | 매도인 |
| **매매계약 효력** | - | 유효(신탁자-매도인) | **무효** |
| **등기 효력** | 무효 | 무효 | 무효 (선의 매도인 보호) |

## 수탁자의 제3자 처분

### 2자간 명의신탁에서 수탁자의 처분
- 소유권이 **신탁자**에게 있으므로 수탁자는 **무권리자**
- 수탁자가 제3자에게 처분 → 제3자가 **선의**인 경우 유효하게 소유권 취득 (제4조 제3항 유추)
- 선의의 제3자가 소유권을 유효하게 취득하면 → 신탁자의 물권적 청구권은 더 이상 인정 안됨 (대판 2010다89814)

### 3자간 명의신탁에서 수탁자의 처분
- 소유권이 **매도인**에게 있으므로 수탁자는 무권리자
- 수탁자의 처분에 대하여 제3자가 **선의**이면 유효 취득 가능

### 계약명의신탁에서 수탁자의 처분
- 매도인이 **선의**인 경우 → 매도인은 수탁자에 대하여 물권변동의 무효 주장 **불가** → 수탁자가 유효하게 소유권 취득한 것과 같은 결과
- 매도인이 **악의**인 경우 → 소유권은 매도인에게 → 수탁자는 무권리자

## 예외 (적용 배제)

1. **종중·배우자·종교단체** 명의의 등기 (조세포탈·강제집행면탈 목적 아닌 경우)
2. **신탁법·자본시장법**에 따른 신탁재산의 등기
3. 채무의 변제를 담보하기 위한 **양도담보**

## 제재

### 과징금 (제5조)

- 부동산 가액의 **30%** 범위 내에서 과징금 부과
- 명의신탁약정의 **당사자**(신탁자)에게 부과

### 이행강제금 (제6조)

- 과징금 부과 후 **1년이 경과**해도 실명등기 등의 의무를 이행하지 않으면
- 부동산 가액의 **10~20%** 범위 내 이행강제금 부과
- **매년 1회** 부과 가능

### 형사처벌 (제7조)

- 명의신탁약정을 한 자: **5년 이하 징역 또는 2억원 이하 벌금**
- 명의수탁자: **3년 이하 징역 또는 1억원 이하 벌금**

## 핵심 암기 포인트

- **명의신탁약정**: 무효 (제4조)
- **2자간 명의신탁**: 소유권은 **신탁자**에게, 등기는 원인무효
- **3자간 명의신탁**: 소유권은 **매도인**에게, 매매계약(신탁자-매도인)은 **유효**
- **계약명의신탁**: 소유권은 **매도인**에게, 매매계약 자체가 **무효**, 다만 매도인 선의 시 물권변동 무효 주장 불가
- **적용 배제**: 종중·배우자·종교단체(조세포탈·강제집행면탈 목적 아닌 경우), 신탁법·자본시장법 신탁, 양도담보
- **과징금**: 부동산 가액 **30%** 범위, **신탁자**에게 부과
- **이행강제금**: 과징금 부과 후 **1년** 경과해도 미이행 시, 가액의 **10~20%**, **매년 1회**
- **형사처벌**: 신탁자 5년 이하 징역/2억원 이하 벌금, 수탁자 3년 이하 징역/1억원 이하 벌금

## OX 퀴즈

**Q1.** 3자간 명의신탁에서 소유권은 신탁자에게 귀속된다. → **X** (3자간 명의신탁에서 소유권은 매도인에게 남아 있다. 신탁자에게 귀속되는 것은 2자간 명의신탁이다)

**Q2.** 계약명의신탁에서 매도인과 수탁자 사이의 매매계약은 유효하다. → **X** (계약명의신탁에서 매매계약 자체가 무효이다. 매매계약이 유효한 것은 3자간 명의신탁이다)

**Q3.** 배우자 명의로 부동산을 등기하는 것은 항상 부동산실명법 위반이다. → **X** (종중·배우자·종교단체 명의의 등기는 조세포탈·강제집행면탈 목적이 아닌 경우 적용이 배제된다)

**Q4.** 과징금은 명의수탁자에게 부과된다. → **X** (과징금은 신탁자에게 부과된다)

**Q5.** 3자간 명의신탁에서 신탁자와 매도인 사이의 매매계약은 무효이다. → **X** (매매계약은 유효하다. 무효인 것은 명의신탁약정과 수탁자 명의의 물권변동이다)

**Q6.** 채무변제를 담보하기 위한 양도담보에도 부동산실명법이 적용된다. → **X** (양도담보는 적용 배제 대상이다)

**Q7.** 이행강제금은 과징금 부과 후 즉시 부과할 수 있다. → **X** (과징금 부과 후 1년이 경과해도 의무를 이행하지 않는 경우에 부과한다)

**Q8.** 계약명의신탁에서 매도인이 선의인 경우에도 물권변동은 무효이다. → **X** (매도인이 선의인 경우 수탁자에 대하여 물권변동의 무효를 주장할 수 없다)

## 함정 노트

⚠️ "3자간 명의신탁 소유권 = 신탁자" → 3자간은 **매도인**에게 남음. 신탁자에게 귀속되는 것은 **2자간**

⚠️ "계약명의신탁 매매계약 = 유효" → 계약명의신탁은 매매계약 **무효**. 매매계약 유효는 **3자간**

⚠️ "과징금 = 수탁자에게 부과" → **신탁자**에게 부과

⚠️ "3자간에서 신탁자의 권리 = 물권적 청구권" → 신탁자는 매도인에 대한 **채권적 청구권**(소유권이전등기청구권)을 보유

⚠️ "양도담보 = 부동산실명법 적용" → 양도담보는 **적용 배제** 대상

## 주요 판례

- **대판 2002.9.6, 2002다35157** — 부동산실명법 시행 전 명의신탁은 유예기간 내 실명등기를 하지 않으면 **유예기간 경과 시 명의신탁약정과 등기가 무효**가 된다.
- **대판 1998.12.11, 98다43250** — 유예기간 경과 후 수탁자 명의 소유권이전등기는 **원인무효로 말소되어야 하며**, 수탁자의 상속인도 소유권을 주장할 수 없다.
- **대판 2002.12.26, 2000다21123** — 3자간 등기명의신탁에서 매도인과 명의수탁자 사이의 **등기 자체는 무효이나**, 매도인이 매수인(신탁자)에게 이전의무를 부담하는 것은 별개의 문제이다.
- **대판 2013.2.28, 2010다89814** — 양자간 등기명의신탁에서 명의수탁자가 신탁부동산을 처분하여 **제3자가 유효하게 소유권을 취득한 경우**, 신탁자의 물권적 청구권은 더 이상 인정되지 않는다.
- **대판 2004.6.25, 2004다6764** — 3자간 등기명의신탁에서 수탁자가 자진하여 신탁자에게 소유권이전등기를 해주는 경우, **실체관계에 부합하는 유효한 등기**이다.
