# 공인중개사 시험 출제경향 종합분석 (15~30회, 2005~2019)

## 1. 시험 난이도 변화 (초기 vs 최근)

### 시기별 난이도 종합 비교

| 시기 | 회차 | 평균 난이도 | 특징 |
|------|------|-------------|------|
| 초기 | 15~19회 (2005~2008) | 중~중상 | 기본 이론 및 법조문 암기 중심 |
| 중기 | 20~24회 (2009~2013) | 중상~상 | 판례 적용, 계산 심화, 법령 개정 반영 |
| 후기 | 25~28회 (2014~2017) | 상 | 융합형 문제 일반화, 감정평가 규칙 개정 반영 |
| 최근 | 29~30회 (2018~2019) | 상~최상 | 복합 계산, 최신 판례, 법령 즉시 반영 |

### 난이도 상승의 주요 원인

1. **계산 문제의 복잡화**: 단순 공식 대입 → 다단계 산출 과정 요구
2. **판례 문제의 심화**: 법조문 직접 출제 → 최신 판례의 결론과 논리 적용
3. **법령 개정의 빈번화**: 안정적 출제 → 매년 개정사항을 즉시 반영
4. **선지 구성의 정교화**: 명확한 오답 → 그럴듯한 오답으로 변화
5. **융합형 문제의 증가**: 단일 주제 → 2~3개 주제 결합

### 과목별 난이도 변화

```
부동산학개론:  ■■■□□ (초기) → ■■■■■ (최근)  [감정평가+투자분석 계산 심화]
민법:          ■■■■□ (초기) → ■■■■■ (최근)  [판례 문제 비중 급증]
공인중개사법:  ■■■□□ (초기) → ■■■■□ (최근)  [법령 개정 반영 가속]
공시법+세법:   ■■■□□ (초기) → ■■■■■ (최근)  [세법 복잡화, 중과세 강화]
공법:          ■■■■□ (초기) → ■■■■□ (최근)  [비교적 안정적]
```

### TOP 20 분석 시사점
- **감정평가**와 **투자분석**은 매회 빠짐없이 출제되는 최고 빈출 주제
- **민법** 과목에서는 의사표시, 대리, 저당권이 3대 핵심 주제
- **특별법**(주택임대차보호법, 상가건물임대차보호법, 부동산실명법)은 매회 고정 출제
- **세법** 중 양도소득세는 가장 중요하며, 취득세가 그 다음
- **공법** 중 국토계획법과 건축법이 양대 핵심

## 4. 최근 5년 신규 출제 주제 (25~30회, 2014~2019)

### 부동산학개론
- 감정평가 규칙 개정 관련 용어 (시장가치, 기준시점 등)
- 정률법에 의한 감가수정 (잔가율 적용)
- 공시지가기준법의 상승식 개별요인비교
- 비율임대차(percentage lease)
- 메자닌 금융(mezzanine financing) 세부 유형
- 비율분석 복합 계산 (3개 비율 동시 산출)
- 금융위험(financial risk) 개념
- REITs 설립자본금 변경 (자기관리 5억원→10억원 등)

### 민법 및 민사특별법
- 성년후견제도 (구 금치산/한정치산 폐지)
- 불공정한 법률행위(민법 제104조) 심화 판례
- 복대리 세부 규정 (대리인 사망시 복대리권 소멸)
- 기한이익상실특약의 법적 성격 (정지조건부 추정)
- 내용증명우편 도달 추정
- 통정허위표시와 채권자취소권의 관계
- 상가건물임대차보호법 계약갱신요구권 기간 확대 (5년→10년)

### 공인중개사법
- 부동산거래신고 기한 단축 (60일→30일)
- 확인설명서 양식 개정
- 포상금 제도

### 세법
- 다주택자 양도소득세 중과
- 조정대상지역 관련 규제
- 종합부동산세 세율 인상

## 5. 합격 전략 제안

### 전략 1: 과목별 시간 배분 최적화

| 과목 | 권장 학습 비중 | 근거 |
|------|:---:|------|
| 부동산학개론 | 25% | 계산문제 비중 높고 난이도 상승 추세, 감정평가 필수 |
| 민법 및 민사특별법 | 30% | 판례 학습에 많은 시간 필요, 범위 방대 |
| 공인중개사법령 및 중개실무 | 15% | 법조문 중심, 상대적으로 안정적 |
| 부동산공시법령 및 세법 | 15% | 양도소득세 중심 집중 학습 |
| 부동산공법 | 15% | 국토계획법+건축법 중심 |

### 전략 2: 반드시 정복해야 할 핵심 주제

#### 부동산학개론 필수 주제 (이것만은 반드시)
1. **NOI 산정 과정**: 가능총소득 → 공실차감 → 기타소득 가산 → 유효총소득 → 영업경비 차감 → NOI
2. **감정평가 3방식**: 원가법(재조달원가-감가수정), 거래사례비교법(사정보정+시점수정+지역요인+개별요인), 수익환원법(NOI/환원율)
3. **화폐의 시간가치 6함수**: 각 계수의 의미와 역수 관계
4. **저당상환 3방식**: 원금균등/원리금균등/체증식의 차이
5. **부동산투자회사(REITs)**: 자기관리/위탁관리/기업구조조정 유형별 특징

#### 민법 필수 주제 (이것만은 반드시)
1. **의사표시 5유형**: 비진의(원칙 유효), 통정허위(무효), 착오(취소가능), 사기(취소가능), 강박(취소가능)
2. **무권대리와 표현대리**: 추인/최고/철회권의 요건
3. **법정지상권 3유형**: 민법 제366조, 관습법상, 입목법상
4. **주택임대차보호법**: 대항력 취득시기(익일), 우선변제권, 소액임차인 보호
5. **점유취득시효**: 20년/10년, 중간처분시 효과

### 전략 3: 시기별 기출문제 활용법

| 시기 | 활용 목적 | 학습 방법 |
|------|-----------|-----------|
| 초기(15~19회) | 기본개념 확인 | 빠르게 풀어보고 부족한 기본기 보완 |
| 중기(20~24회) | 응용력 배양 | 오답 선지까지 꼼꼼히 분석 |
| 후기(25~28회) | 현행 법령 적응 | 현행 법령과 대조하며 변경사항 확인 |
| 최근(29~30회) | 실전 감각 확보 | 실제 시험과 동일한 조건(시간 제한)으로 풀이 |

### 전략 4: 법령 개정 대응 전략

기출문제 학습시 가장 주의할 점은 **법령 개정으로 인해 정답이 바뀐 문제**이다. 다음 분야는 특히 주의가 필요하다:

1. **공인중개사법**: 부동산거래신고 기한, 중개보수 한도, 교육 의무 등 수시 개정
2. **세법**: 양도소득세 세율, 종합부동산세 과세기준, 취득세 감면 등 매년 변경
3. **주택임대차보호법**: 보증금 기준액, 소액임차인 범위 등 주기적 변경
4. **상가건물임대차보호법**: 계약갱신요구권 기간, 보증금 한도 등 개정
5. **도시정비법**: 재건축/재개발 절차 수시 개정

### 전략 5: 계산 문제 대비 공식 정리

#### 부동산투자 관련
```
자기자본수익률(ROE) = 세전현금흐름(BTCF) / 지분투자액
총투자수익률(ROI) = 순영업소득(NOI) / 총투자액
부채감당률(DCR) = NOI / 부채서비스액(DS)
채무불이행률 = (영업경비 + DS) / 유효총소득(EGI)
대부비율(LTV) = 융자액 / 부동산가치
순소득승수(NIM) = 총투자액 / NOI
```

#### 감정평가 관련
```
[원가법] 적산가액 = 재조달원가 - 감가수정액
  정액법: 매년감가액 = (재조달원가 - 잔존가치) / 내용연수
  정률법: 기준시점 가치 = 재조달원가 x (잔가율)^경과연수

[수익환원법] 수익가액 = NOI / 환원율(R)

[거래사례비교법] 비준가격 = 사례가격 x 사정보정 x 시점수정 x 지역요인비교 x 개별요인비교

[공시지가기준법] 대상토지 가격 = 비교표준지 공시지가 x 시점수정 x 지역요인비교 x 개별요인비교 x 기타요인보정
```
