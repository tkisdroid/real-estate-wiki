# 종합부동산세법

종합부동산세법은 고액의 부동산 보유자에 대하여 [[재산세]]에 추가로 과세하는 **국세**(보유세)를 규정한 법률이다. 2005년 시행되었으며, 매년 **1~2문항** 출제된다. [[재산세]]와의 비교 출제가 빈출이다.

## 하위 개념 페이지

| 페이지 | 주요 내용 |
|--------|----------|
| [[종합부동산세]] | 개요, 과세대상, 세율, 세액공제, 세부담상한, 부과·징수 전반 |

## 1. 법률의 목적 및 위치

### 보유세 이원화 체계

종합부동산세는 부동산 보유에 대한 과세를 **재산세**(지방세)와 **종합부동산세**(국세)로 이원화한 체계의 일부이다.

```
부동산 보유
├── 1차 과세: 재산세 (지방세, 시·군·구)
│   └── 모든 부동산 보유자 대상
└── 2차 과세: 종합부동산세 (국세, 세무서장)
    └── 공시가격 합계가 공제금액 초과 시 추가 과세
```

### 주요 특징

| 특징 | 내용 |
|------|------|
| **국세** | 세무서장이 부과·징수 |
| **보유세** | 과세기준일(6월 1일) 현재 보유 기준 |
| **정부부과과세** | 원칙 정부부과 (선택 신고납부 가능) |
| **초과누진세율** | 개인: 누진세율, 법인: 비례세율 |
| **이중과세 방지** | 재산세액 공제 |
| **세부담 상한** | 전년 대비 150% (법인 미적용) |
| **물납 불가** | 재산세와 다르게 물납 불가 |

## 2. 과세대상 및 공제금액

### 주택

과세기준일(6월 1일) 현재 주택분 재산세 납세의무자로서, 국내 소재 주택의 공시가격 합계액이 공제금액을 초과하는 자에게 과세한다.

| 구분 | 공제금액 |
|------|----------|
| **1세대1주택자** (단독명의) | **12억원** |
| **일반** (개인) | **9억원** |
| **법인** | **0원** (공제 없음) |

### 토지

| 구분 | 공제금액 |
|------|----------|
| **종합합산과세대상** 토지 | **5억원** |
| **별도합산과세대상** 토지 | **80억원** |

> **분리과세대상 토지**: 종합부동산세 **과세 제외**

## 3. 과세표준

| 과세대상 | 산식 | 공정시장가액비율 |
|----------|------|-----------------|
| **주택** | (공시가격 합계 - 공제금액) × **60%** | 60% |
| **종합합산 토지** | (공시가격 합계 - 5억원) × **100%** | 100% |
| **별도합산 토지** | (공시가격 합계 - 80억원) × **100%** | 100% |

## 4. 세율

### 주택 (개인)

| 과세표준 | 세율 |
|----------|------|
| 3억원 이하 | **0.5%** |
| 3억~6억원 | 150만원 + 초과액의 **0.7%** |
| 6억~12억원 | 360만원 + 초과액의 **1.0%** |
| 12억~25억원 | 960만원 + 초과액의 **1.3%** |
| 25억~50억원 | 2,650만원 + 초과액의 **1.5%** |
| 50억~94억원 | 6,400만원 + 초과액의 **2.0%** |
| 94억원 초과 | 1억5,200만원 + 초과액의 **2.7%** |

### 주택 (법인)

| 구분 | 세율 |
|------|------|
| **2주택 이하** | **2.7%** (비례세율) |
| **3주택 이상** | **5.0%** (비례세율) |

### 종합합산과세대상 토지

| 과세표준 | 세율 |
|----------|------|
| 15억원 이하 | **1%** |
| 15억~45억원 | 1,500만원 + 초과액의 **2%** |
| 45억원 초과 | 7,500만원 + 초과액의 **3%** |

### 별도합산과세대상 토지

| 과세표준 | 세율 |
|----------|------|
| 200억원 이하 | **0.5%** |
| 200억~400억원 | 1억원 + 초과액의 **0.6%** |
| 400억원 초과 | 2억2천만원 + 초과액의 **0.7%** |

## 5. 세액계산 흐름

### 주택 (1세대1주택자)

```
① (공시가격 합계 - 12억원) × 60% = 과세표준
② 과세표준 × 세율 = 종합부동산세액
③ ② - 재산세 공제액 = 산출세액
④ ③ - 연령별 공제 - 장기보유공제 = 최종 납부세액
⑤ 세부담 상한 적용 (전년 대비 150%)
```

### 재산세액 공제 (이중과세 방지)

종합부동산세 과세표준에 해당하는 금액에 대하여 주택분 재산세로 부과된 세액을 공제한다.

## 6. 1세대1주택자 세액공제

### 연령별 공제 (만 60세 이상, 단독소유)

| 연령 | 공제율 |
|------|--------|
| 만 60세 이상~65세 미만 | **20%** |
| 만 65세 이상~70세 미만 | **30%** |
| 만 70세 이상 | **40%** |

### 장기보유공제 (5년 이상 보유, 단독소유)

| 보유기간 | 공제율 |
|----------|--------|
| 5년 이상~10년 미만 | **20%** |
| 10년 이상~15년 미만 | **40%** |
| 15년 이상 | **50%** |

> 연령별 공제 + 장기보유공제 **합계 한도: 80%**

### 1세대1주택으로 보는 경우

| 유형 | 조건 |
|------|------|
| **일시적 2주택** | 신규주택 취득 후 과세기준일 현재 **3년** 미경과 |
| **상속주택** | 상속개시일부터 **5년** 미경과, 또는 지분율 40% 이하, 또는 지분율 상당 공시가격 6억원(수도권 밖 3억원) 이하 |
| **지방 저가 주택** | 수도권 밖, 공시가격 **3억원 이하** |

### 공동명의 1주택 특례

- 부부 공동명의 1주택이고 다른 주택 없는 경우
- 신청 시 1세대1주택자로 보아 **12억원 공제** + 연령별·장기보유 공제 적용
- 신청기간: **9월 16일 ~ 9월 30일**

## 7. 세부담 상한

| 구분 | 상한 |
|------|------|
| 주택 (개인) | 직전 연도 총세액의 **150%** |
| 토지 | 직전 연도 총세액의 **150%** |
| **법인** | 세부담 상한 **미적용** |

> 총세액 = 재산세 + 종합부동산세

## 8. 부과·징수

| 항목 | 내용 |
|------|------|
| 과세기준일 | 매년 **6월 1일** |
| 납세지 | 개인: 주소지(거소지), 법인: 본점 소재지 |
| 부과·징수기간 | **12월 1일 ~ 12월 15일** |
| 납세고지서 발부 | 납부기간 개시 **5일 전**까지 |

### 분할납부

| 납부세액 | 분납금액 |
|----------|----------|
| **500만원 초과~1,000만원 이하** | 500만원 초과분 |
| **1,000만원 초과** | 세액의 **50% 이하** |

분납기한: 납부기한 경과 후 **6개월 이내**

### 부가세: 농어촌특별세

종합부동산세 납부세액의 **20%**

## 9. 재산세와 종합부동산세 비교

| 항목 | [[재산세]] | 종합부동산세 |
|------|----------|--------------|
| 세목 | **지방세** (구세) | **국세** |
| 과세주체 | 시·군·구 | 세무서장 |
| 과세기준일 | 6월 1일 | 6월 1일 |
| 과세대상 | 토지, 건축물, 주택, 선박, 항공기 | **주택, 토지** (종합·별도합산만) |
| 과세방법 | **보통징수** | **정부부과** (선택 신고납부) |
| 세율 | 누진 + 비례 | **초과누진** (법인 비례) |
| 물납 | **가능** (1,000만원 초과) | **불가** |
| 분납 기준 | **250만원** 초과 | **500만원** 초과 |
| 분납 기한 | 납부기한 후 **3개월** | 납부기한 후 **6개월** |
| 부가세 | 지방교육세(20%), 지역자원시설세 | 농어촌특별세(20%) |
| 세부담상한 | 150% (분리과세 제외) | 150% (법인 미적용) |

## 10. 법인 과세 특례

법인 소유 주택에 대하여는 개인과 완전히 다른 체계가 적용된다:

| 항목 | 개인 | 법인 |
|------|------|------|
| 공제금액 | 9억(일반)/12억(1세대1주택) | **0원** |
| 세율 | 초과누진 (0.5~2.7%) | 비례세율 (2.7%/5.0%) |
| 세액공제 | 연령별·장기보유 | **없음** |
| 세부담 상한 | 150% | **미적용** |

## 핵심 암기 포인트

- **국세**, 보유세, 과세기준일 **6월 1일**, 부과·징수 **12/1~12/15**
- **주택 공제금액**: 1세대1주택 **12억**, 일반 **9억**, 법인 **0원**
- **토지 공제금액**: 종합합산 **5억**, 별도합산 **80억**, 분리과세 = 과세 제외
- **공정시장가액비율**: 주택 **60%**, 토지 **100%**
- **주택 개인 세율**: 0.5%~2.7% (7단계 초과누진)
- **주택 법인 세율**: 2주택 이하 **2.7%**, 3주택+ **5.0%** (비례세율)
- **토지 종합합산 세율**: 1%~3%, **별도합산 세율**: 0.5%~0.7%
- **1세대1주택 세액공제**: 연령별(60세 20%, 65세 30%, 70세 40%) + 장기보유(5년 20%, 10년 40%, 15년 50%), 합계 한도 **80%**
- **세부담 상한**: **150%** (법인 미적용)
- **분할납부**: 500만원 초과~1,000만원 이하 → 500만원 초과분, 1,000만원 초과 → 50% 이하, 기한 **6개월**
- **부가세**: 농어촌특별세 = 납부세액의 **20%**
- **공동명의 특례**: 9/16~9/30 신청 → 12억 공제 + 연령별·장기보유 공제
- **물납 불가** (재산세만 물납 가능)

## OX 퀴즈

**Q1.** 종합부동산세는 지방자치단체가 부과하는 지방세이다. → **X** (국세이다. 세무서장이 부과)

**Q2.** 종합부동산세의 과세기준일은 매년 1월 1일이다. → **X** (6월 1일)

**Q3.** 1세대1주택자의 종합부동산세 주택 공제금액은 9억원이다. → **X** (12억원. 일반이 9억원)

**Q4.** 법인 소유 주택의 공제금액은 9억원이다. → **X** (0원 — 공제 없음)

**Q5.** 분리과세대상 토지도 종합부동산세 과세대상이다. → **X** (과세 제외. 종합합산·별도합산 토지만 과세)

**Q6.** 종합부동산세의 분할납부 기한은 납부기한 경과 후 3개월 이내이다. → **X** (6개월 이내. 재산세가 3개월)

**Q7.** 종합부동산세도 부동산으로 물납할 수 있다. → **X** (물납 불가. 재산세만 물납 가능)

## 함정 노트

- ⚠️ **공제금액 숫자 혼동**: 1세대1주택 12억 vs 일반 9억 vs 법인 0원, 종합합산토지 5억 vs 별도합산토지 80억 — 가장 빈출되는 함정
- ⚠️ **공정시장가액비율**: 주택 **60%** vs 토지 **100%** — 같은 세목 내에서도 다르므로 혼동 주의
- ⚠️ **분납 기준·기한**: 종부세 500만원 초과/6개월 vs 재산세 250만원 초과/3개월 — 금액과 기한 모두 다름
- ⚠️ **물납**: 재산세 = 가능, 종부세 = 불가 — 둘 다 보유세이지만 물납 여부가 다름
- ⚠️ **법인 과세 특례**: 공제 0원 + 비례세율 + 세액공제 없음 + 세부담상한 미적용 — 개인과 완전히 다른 체계
- ⚠️ **세부담 상한 150%**: 개인에게만 적용, 법인은 미적용 — "법인에도 적용"이라는 함정 주의
- ⚠️ **과세기준일**: 재산세와 동일하게 6월 1일이지만, "1월 1일"이라는 함정 선지 빈출
- ⚠️ **재산세액 공제**: 이중과세 방지를 위해 재산세를 공제하는 것이지, 재산세 납부 면제가 아님

## 주요 조문 원문

> 출처: 종합부동산세법 [법률 제20609호, 2025. 2. 18., 일부개정]

### 제1조 (목적)

이 법은 고액의 부동산 보유자에 대하여 종합부동산세를 부과하여 부동산보유에 대한 **조세부담의 형평성을 제고**하고, 부동산의 가격안정을 도모함으로써 **지방재정의 균형발전**과 **국민경제의 건전한 발전**에 이바지함을 목적으로 한다.

> 시험 포인트: 목적 3가지 = 조세부담 형평성, 부동산 가격안정, 지방재정 균형발전.

---

### 제7조 (주택에 대한 과세표준)

① 주택에 대한 종합부동산세의 과세표준은 납세의무자별로 주택의 **공시가격을 합산한 금액**에서 **9억원**(1세대1주택자는 **12억원**)을 공제한 금액에 **공정시장가액비율**을 곱한 금액으로 한다.

② 공정시장가액비율은 **100분의 60**으로 한다.

> 시험 포인트: 과세표준 = (공시가격 합계 - 공제금액) × 60%. 1세대1주택 공제 12억, 일반 9억.

### 제9조 (주택에 대한 세액)

③ 1세대1주택자로서 일정 요건을 충족하는 자에 대하여는 다음의 **세액공제**를 적용한다.

1. **연령별 공제**: 만 60세 이상(20%), 만 65세 이상(30%), 만 70세 이상(40%)
2. **장기보유 공제**: 5년 이상(20%), 10년 이상(40%), 15년 이상(50%)
3. 합계 공제율 **한도**: 산출세액의 **80%**

> 시험 포인트: 연령별 + 장기보유 합계 한도 80%. 연령·보유기간별 공제율 정확히 암기.

### 제10조 (세부담의 상한)

① 종합부동산세의 납부세액(주택분)이 해당 주택에 대한 **직전 연도 총세액상당액의 100분의 150**을 초과하는 경우에는 그 초과하는 세액은 없는 것으로 본다. 다만, **법인**에 대하여는 이를 적용하지 아니한다.

> 시험 포인트: 세부담 상한 150%. 법인은 미적용 — 핵심 함정 포인트.

---

### 제20조 (분할납부)

① 납세의무자는 종합부동산세의 납부세액이 다음 각 호의 금액을 초과하는 경우 해당 호에서 정하는 바에 따라 납부기한이 경과한 날부터 **6개월 이내**에 분할납부할 수 있다.

1. **500만원 초과 1,000만원 이하**: 그 세액에서 500만원을 뺀 금액
2. **1,000만원 초과**: 그 세액의 **100분의 50 이하**의 금액

> 시험 포인트: 분납 기준 500만원(재산세 250만원과 구분), 분납기한 6개월(재산세 3개월과 구분).
