# 부동산 투자이론

## 1. 부동산투자의 장점

- **인플레이션 헷징** (구매력 보호): 실물자산 투자 → 화폐가치 손실 방지
- **소득이득**(임대수입) + **자본이득**(매매차익) 기대
- **절세효과**(tax shelter): 감가상각비, 차입이자 절세혜택
- **지렛대(레버리지) 효과** 기대

## 2. 레버리지 효과 (Leverage Effect)

타인자본(저당) 이용 시 **부채비율의 증감이 자기자본수익률에 미치는 효과**.

### 유형

| 유형 | 조건 | 결과 |
|------|------|------|
| **정(+)의 레버리지** | 종합수익률 > 저당수익률 | 지분수익률 > 종합수익률 |
| **부(-)의 레버리지** | 종합수익률 < 저당수익률 | 지분수익률 < 종합수익률 |
| **중립(0) 레버리지** | 종합수익률 = 저당수익률 | 지분수익률 = 종합수익률 |

### 공식

$$지분수익률 = 종합수익률 + (종합수익률 - 저당수익률) \times 부채비율$$

$$부채비율 = \frac{타인자본(저당)}{자기자본(지분)}$$

> **주의**: 대부비율(LTV)↑ → 레버리지효과 커지나 **위험도 증가**. 부(-)의 레버리지를 정(+)으로 바꾸려면 **이자율을 낮추어야** 함 (부채비율 조정으로는 불가).

## 3. 부동산투자의 위험 유형

| 위험 | 내용 | 제거 방법 |
|------|------|----------|
| **사업상 위험** | 시장위험, 운영위험, 위치적 위험 | - |
| **금융적 위험** | 대출 시 금리인상·채무불이행 가능성 | 자기자본 100% 투자 |
| **법률적 위험** | 세제·감가상각·건폐율 등 변경 | - |
| **유동성 위험** | 현금전환 과정에서의 가치손실 | - |
| **인플레 위험** | 예상 못한 인플레로 실질수익률 저하 | - |

- 위험 측정 척도: **분산** 또는 **표준편차** (클수록 위험 큼)

## 4. 수익률의 종류

| 수익률 | 다른 명칭 | 의미 |
|--------|----------|------|
| **기대수익률** | 예상수익률, 내부수익률, 객관적 수익률 | 예상수입·지출 기반 계산, 가중평균의 합 |
| **요구수익률** | 최소수익률, 외부수익률, 주관적 수익률, 기회비용 | 투자하기 위해 충족되어야 할 **최소한**의 수익률 |
| **실현수익률** | 실제수익률, 사후수익률, 역사적 수익률 | 투자 후 현실적으로 달성된 수익률 |

### 요구수익률 구성

$$요구수익률 = 무위험률 + 위험할증률(해당자산 + 인플레)$$

- **무위험률**: 국채 실질이자율, 모든 투자자가 기본적으로 요구하는 최저수익률
- **위험할증률**: 위험에 대한 대가, 투자자 위험혐오도에 따라 다름
- **피셔효과**: 예상 인플레이션을 요구수익률에 반영
- **체계적 위험** 증대 → 요구수익률 **상승**

### 투자의사결정

| 조건 | 결정 | 이후 과정 |
|------|------|----------|
| 기대수익률 > 요구수익률 | **투자** | 수요↑ → 시장가치↑ → 기대수익률↓ → 균형 |
| 기대수익률 < 요구수익률 | **기각** | 수요↓ → 시장가치↓ → 기대수익률↑ → 균형 |
| 투자가치 > 시장가치 | **투자** | 차액 = 투자자 초과이익 |

## 5. 투자가치와 시장가치

$$투자가치 = \frac{투자에\ 대한\ 영구소득}{요구수익률}$$

- **투자가치**: 투자자가 지불할 용의가 있는 **최대비용** (할인율 = 요구수익률)
- **시장가치**: 공개경쟁시장에서 성립될 가능성이 가장 많은 가격 = 실제 지불 비용

## 6. 위험관리 방법

| 방법 | 내용 |
|------|------|
| **회피** | 위험한 투자자산을 투자대상에서 **제외** (국채 투자 등) |
| **전가** | 다른 경제주체에게 위험 이전 (보험, 변동금리, 물가연동 임대계약) |
| **축소(통제)** | 기대수익률 하향조정, 요구수익률 상향조정, 민감도 분석, 포트폴리오 |

## 핵심 암기 포인트

- 지분수익률 = **종합수익률 + (종합수익률 - 저당수익률) × 부채비율**
- 부채비율 = **타인자본(저당) / 자기자본(지분)**
- 정(+) 레버리지: 종합수익률 **>** 저당수익률 → 지분수익률 > 종합수익률
- 부(-) 레버리지: 종합수익률 **<** 저당수익률 → 지분수익률 < 종합수익률
- 요구수익률 = **무위험률 + 위험할증률**
- 기대수익률 > 요구수익률 → **투자** / 투자가치 > 시장가치 → **투자**
- 투자가치 = **영구소득 / 요구수익률** (할인율 = 요구수익률)
- 위험 측정: **분산** 또는 **표준편차** (클수록 위험 큼)
- 위험 축소 방법: 기대수익률 **하향**조정, 요구수익률 **상향**조정
- 부(-) 레버리지를 정(+)으로 바꾸려면 **이자율을 낮추어야** 함 (부채비율 조정 불가)

## OX 퀴즈

**Q1.** 종합수익률이 저당수익률보다 높으면 부(-)의 레버리지 효과가 발생한다. → **X** (종합 > 저당이면 **정(+)의** 레버리지)

**Q2.** 부(-)의 레버리지를 정(+)으로 바꾸려면 부채비율을 높여야 한다. → **X** (**이자율을 낮추어야** 함, 부채비율 조정으로는 불가)

**Q3.** 기대수익률이 요구수익률보다 높으면 투자를 기각한다. → **X** (기대 > 요구이면 **투자 채택**)

**Q4.** 요구수익률은 투자하기 위해 충족되어야 할 최소한의 수익률이다. → **O**

**Q5.** 금융적 위험은 자기자본 100%로 투자하면 제거할 수 있다. → **O**

**Q6.** 기대수익률은 외부수익률이라고도 한다. → **X** (기대수익률 = **내부수익률**(객관적), 요구수익률이 외부수익률(주관적))

**Q7.** 위험의 축소 방법으로 기대수익률을 상향조정하는 방법이 있다. → **X** (기대수익률 **하향**조정 또는 요구수익률 **상향**조정)

## 함정 노트

- ⚠️ "부채비율↑ → 부(-) 레버리지를 정(+)으로 전환" → 부채비율 변경은 전환 불가, **이자율 인하**만 가능
- ⚠️ "기대수익률 = 주관적 수익률" → 기대수익률은 **객관적**(내부), 요구수익률이 **주관적**(외부)
- ⚠️ "요구수익률 = 내부수익률" → 요구수익률은 **외부**수익률(기회비용), 내부수익률은 기대수익률
- ⚠️ "LTV↑ = 위험 감소" → LTV↑ = 레버리지 효과 커지나 **위험도 증가**
- ⚠️ "위험 축소: 기대수익률 상향" → 기대수익률은 **하향**, 요구수익률은 **상향** 조정
