# 등록의 결격사유 (법 제10조)

## 1. 결격사유의 의의

### 취지
- 개업공인중개사는 국민의 소중한 재산을 다루는 직업이므로, 개설등록에 일정한 제한을 가하는 것
- 결격사유는 **개업공인중개사**뿐만 아니라 **소속공인중개사**, **중개보조원**, **법인의 임원·사원**(분사무소 책임자 포함)에게도 적용

### 관계기관 조회
- 등록관청은 결격사유 해당 여부를 확인하기 위해 **관계 기관에 조회** 가능

### 결격사유의 효과
- 결격사유 해당 → 개설등록 불가, 법인 임원·중개보조원·소속공인중개사 종사 불가
- 등록 후 결격사유 해당 → 등록관청은 **절대 등록취소** (단, 등록취소로 인한 3년 기간 적용 안 됨)

## 2. 결격사유 11종 상세

### (1) 제한능력자

#### 미성년자 (만 19세 미만)
- 미성년자는 개업공인중개사가 될 수 없으며, 중개업 종사 불가
- **혼인에 의한 성년의제 불인정**: 민법상 혼인으로 성년으로 의제되더라도, 공인중개사법에서는 **인정하지 않음**
- 법정대리인의 동의를 받아도 결격사유에서 벗어나지 못함

#### 피성년후견인
- 사무처리 능력이 **지속적으로 결여**된 사람에 대해 가정법원이 성년후견개시 심판
- 법정대리인의 동의가 있더라도 언제나 취소 가능한 행위능력 없는 자
- 등록취소 시 **3년 기간 적용 안 됨** → 취소선고를 받으면 즉시 재등록 가능

#### 피한정후견인
- 사무처리 능력이 **부족**한 사람에 대해 가정법원이 한정후견개시 심판
- 선고를 받지 않은 사무처리 능력 부족자는 결격사유자가 **아님**
- 등록취소 시 **3년 기간 적용 안 됨** → 취소선고를 받으면 즉시 재등록 가능

#### 피특정후견인
- **결격사유에 해당하지 않음** (일시적 후원 또는 특정 사무에 관한 후원)

### (2) 파산자로서 복권되지 않은 자
- 파산선고를 받고 **복권되지 않은 자**는 개업공인중개사 불가
- 복권 방법: 면책결정, 파산선고 취소, **파산선고 후 10년 경과**(당연복권)
- 등록취소 시 **3년 기간 적용 안 됨** → 복권되면 즉시 재등록 가능

### (3) 금고 이상 실형
- **집행종료(만기출소)** 또는 **집행면제**일로부터 **3년** 경과하지 않은 자

| 상황 | 결격사유 해제 시점 |
|---|---|
| 만기출소 | 출소일로부터 **3년** |
| 가석방 | 잔여형기 경과 후 + **3년** (예: 잔여 2년 → 총 5년) |
| 특별사면(집행면제) | 사면일로부터 **3년** |
| 일반사면 | 형선고 효력 소멸 → **결격사유 해당 안 됨** |
| 형의 시효완성 | 시효완성일로부터 **3년** |

- 등록취소 시 **3년 기간 적용 안 됨** (형벌 자체의 3년과 별도)

### (4) 금고 이상 집행유예
- 유예기간 **만료일**로부터 **2년** 경과하지 않은 자
- 집행유예 기간 중에는 당연히 결격

### (5) 자격취소
- **공인중개사 자격취소**일로부터 **3년** 경과하지 않은 자

### (6) 자격정지 기간 중
- 공인중개사 **자격정지** 기간 동안은 결격

### (7) 등록취소
- 등록취소일로부터 **3년** 경과하지 않은 자
- **예외 (3년 적용 안 되는 경우)**:
  - 사망·해산으로 인한 등록취소
  - 결격사유(제한능력자·파산자·형벌)로 인한 등록취소
  - 등록기준 미달로 인한 등록취소

### (8) 업무정지 중 폐업한 자
- 업무정지 기간이 경과할 때까지 결격

### (9) 업무정지 사유 발생 당시 법인 임원·사원
- 업무정지 기간이 경과할 때까지 결격

### (10) 공인중개사법 위반 300만원 이상 벌금
- **이 법 위반**으로 300만원 이상 벌금 **선고일**로부터 **3년** 경과하지 않은 자
- 다른 법률 위반 벌금은 해당 안 됨

### (11) 법인의 임원·사원 중 결격사유자
- 법인의 임원·사원 중 결격사유자가 있으면 법인 자체가 결격
- **2개월 내** 해소 시 등록 유지

## 3. 결격사유 기간 총정리

| 결격사유 | 기간 | 기산점 |
|---|---|---|
| 미성년자 | 만 19세까지 | - |
| 피성년후견인/피한정후견인 | 선고 기간 | 취소선고 시 해소 |
| 파산자 | 복권 시까지 | 10년 경과 시 당연복권 |
| 금고 이상 실형 | **3년** | 집행종료/면제일 |
| 금고 이상 집행유예 | **2년** | 유예기간 만료일 |
| 자격취소 | **3년** | 취소일 |
| 자격정지 | 정지기간 | - |
| 등록취소 | **3년** | 취소일 (일부 예외) |
| 업무정지 중 폐업 | 업무정지기간 | - |
| 이 법 위반 벌금 300만원 이상 | **3년** | 선고일 |

### 기간 계산 주의사항
- 결격사유 기간은 **초일 산입** (오전 0시부터 시작하므로)

## 4. 법인의 결격사유 특칙

### 법인 자체의 결격
- 대표자·임원·사원 중 결격사유 해당 → 법인 결격사유 해당 → 절대 등록취소
- 단, **2개월 내** 결격사유를 해소하면 등록취소 안 됨

### 소속공인중개사·중개보조원의 결격
- 소속공인중개사·중개보조원이 결격사유 해당 → 개업공인중개사가 **2개월 내** 해소
- 미해소 시 → 개업공인중개사 **업무정지** 처분

## 5. 양벌규정과 결격사유

- 양벌규정으로 받은 **벌금형** → **결격사유에 해당 안 됨** (법 제50조)
- 개업공인중개사가 상당한 주의와 감독을 게을리하지 않은 경우 → 양벌규정 미적용

## 핵심 암기 포인트

1. 결격사유는 개업공인중개사 + **소속공인중개사 + 중개보조원 + 법인 임원·사원** 모두 적용
2. 피특정후견인은 결격사유에 **해당하지 않음**
3. 혼인에 의한 성년의제 → 공인중개사법에서 **불인정**
4. 금고 이상 실형: 집행종료/면제일부터 **3년**
5. 금고 이상 집행유예: 유예기간 만료일부터 **2년**
6. 등록취소: 취소일부터 **3년** (사망·해산·결격사유·등록기준미달 취소는 적용 안 됨)
7. 이 법 위반 벌금 **300만원 이상**: 선고일부터 **3년**
8. 법인 임원·사원 결격 해소 기한: **2개월**
9. 기간 계산: **초일 산입**
10. 양벌규정 벌금 → **결격사유 해당 안 됨**

## OX 퀴즈

**Q1.** 피특정후견인은 공인중개사법상 결격사유에 해당한다.
> **X** — 피특정후견인은 결격사유에 **해당하지 않는다**.

**Q2.** 미성년자가 혼인하여 성년으로 의제되면 개업공인중개사가 될 수 있다.
> **X** — 공인중개사법은 혼인에 의한 성년의제를 **인정하지 않는다**.

**Q3.** 금고 이상의 집행유예를 받은 자는 유예기간 만료일로부터 3년이 경과해야 결격사유에서 벗어난다.
> **X** — 유예기간 만료일로부터 **2년**이 경과하면 결격사유에서 벗어난다.

**Q4.** 결격사유로 인한 등록취소의 경우에도 취소일로부터 3년간 결격사유가 적용된다.
> **X** — 결격사유(제한능력자·파산자 등)로 인한 등록취소는 **3년 기간이 적용되지 않는다**.

**Q5.** 가석방된 자의 결격사유 해제 시점은 가석방일로부터 3년이다.
> **X** — 가석방의 경우 **잔여형기 경과 후** 3년 추가이다 (예: 잔여형기 2년 + 3년 = 5년).

**Q6.** 양벌규정에 의하여 개업공인중개사에게 부과된 벌금형은 결격사유에 해당하지 않는다.
> **O** — 양벌규정으로 받은 벌금형은 결격사유에 해당하지 않는다.

**Q7.** 법인의 임원 중 결격사유에 해당하는 자가 있으면 즉시 등록이 취소된다.
> **X** — **2개월 내** 해소하면 등록이 취소되지 않는다.

**Q8.** 다른 법률 위반으로 300만원 이상의 벌금을 선고받은 자도 결격사유에 해당한다.
> **X** — **이 법(공인중개사법)** 위반으로 300만원 이상 벌금일 때만 결격사유이다.

**Q9.** 일반사면을 받은 경우에도 결격사유 기간 3년이 적용된다.
> **X** — 일반사면은 형선고 효력이 소멸되므로 결격사유에 **해당하지 않는다**.

**Q10.** 파산선고 후 10년이 경과하면 복권 판결 없이도 결격사유에서 벗어난다.
> **O** — 파산선고 후 10년 경과 시 **당연복권**되어 결격사유에서 벗어난다.

## 함정 노트

| 함정 포인트 | 정확한 내용 |
|---|---|
| 피특정후견인도 결격 | 피특정후견인은 결격사유 **아님** |
| 혼인 → 성년의제 인정 | 공인중개사법에서 **불인정** |
| 집행유예도 3년 | 집행유예는 유예기간 만료일부터 **2년** |
| 가석방 후 3년 | 가석방은 **잔여형기 경과 + 3년** |
| 등록취소 → 항상 3년 | 사망·해산·결격사유·등록기준미달 취소 시 **3년 미적용** |
| 다른 법 벌금도 결격 | **이 법** 위반 벌금 300만원 이상만 결격 |
| 양벌규정 벌금도 결격 | 양벌규정 벌금은 결격사유 **아님** |
| 법인 임원 결격 → 즉시 취소 | **2개월** 해소 기한 있음 |
| 일반사면 → 3년 적용 | 일반사면은 형선고 효력 소멸 → **결격 아님** |
| 특별사면 → 결격 아님 | 특별사면은 집행면제일부터 **3년** 적용 |
