# 부동산 수요·공급 이론

## 1. 수요의 기본개념

- **수요**: 일정기간 동안 재화를 구매하고자 하는 욕구
- **수요량**: 구매력을 가진 소비자가 일정기간 동안 구입하고자 하는 **최대수량**
- 수요는 **유효수요**(구매력 동반), **유량(flow)** 개념, **사전적** 개념

### 유량변수 vs 저량변수

| 유량(flow) 변수 | 저량(stock) 변수 |
|-----------------|------------------|
| 일정**기간** 명시 필요 | 일정**시점** 명시 필요 |
| 임대료, 급여, 주택거래량, 저축, 신규주택공급량, NOI, 투자, 국민총생산 | 자산, 부채, 자본, 국부, 재고, 인구수, 가격, 주택보급률, 통화량 |

## 2. 수요·공급의 법칙

- **수요의 법칙**: 가격↑ → 수요량↓ (역의 관계, 우하향 곡선)
- **공급의 법칙**: 가격↑ → 공급량↑ (정의 관계, 우상향 곡선)

### 수요량의 변화 vs 수요의 변화

| 구분 | 수요량의 변화 | 수요의 변화 |
|------|---------------|-------------|
| 원인 | **해당 상품 가격** 변화 | 가격 이외 요인 변화 |
| 그래프 | 수요곡선 **위의 점 이동** | 수요곡선 **자체 이동** |

### 수요 변동 요인

소득 증가, 인구 증가, 대체재 가격 상승, 보완재 가격 하락, 선호도 증가, 금리 하락 → **수요 증가(우측 이동)**

### 부동산 수요의 특징

| 특징 | 내용 |
|------|------|
| 수요자 수 중시 | 수요된 **수량보다 수요자 수**가 더 큰 비중 |
| **비소멸성 수요** | 기존 수요량이 새로운 **공급**으로 전환 가능 |
| 차별화된 수요 | 개별성으로 부동산 종류별 상이한 탄력성 |
| **파생수요** | 토지는 생산재로서 소비재 수요에서 **파생** (직접수요도 가능: 주거용, 공원, 주차장 등) |
| 전문지식 요구 | 구매절차 특수, 자금축적 장기간 소요 |

### 공급 변동 요인

생산기술 향상, 생산요소 가격 하락, 정부 규제 완화 → **공급 증가(우측 이동)**

## 3. 균형의 결정 및 변동

### 수요·공급 동시 변동 시 균형

| 변동 | 균형가격 | 균형거래량 |
|------|----------|-----------|
| 수요↑, 공급↑ | **불확정** | **반드시 증가** |
| 수요↑, 공급↓ | **반드시 상승** | **불확정** |
| 수요↓, 공급↑ | **반드시 하락** | **불확정** |
| 수요↓, 공급↓ | **불확정** | **반드시 감소** |

## 4. 가격탄력성

### 수요의 가격탄력성

$$수요의\ 가격탄력성 = \left|\frac{수요량의\ 변화율}{가격의\ 변화율}\right|$$

| 크기 | 구분 | 수요곡선 모양 |
|------|------|---------------|
| ε = 0 | 완전비탄력적 | 수직선 |
| 0 < ε < 1 | 비탄력적 | 급경사 |
| ε = 1 | 단위탄력적 | 직각쌍곡선 |
| 1 < ε < ∞ | 탄력적 | 완만한 경사 |
| ε = ∞ | 완전탄력적 | 수평선 |

### 가격탄력성과 총수입

| 수요 탄력성 | 가격 상승 시 | 가격 하락 시 |
|-------------|-------------|-------------|
| 탄력적 (ε>1) | 총수입 **감소** | 총수입 **증가** |
| 단위탄력적 (ε=1) | 총수입 **불변** | 총수입 **불변** |
| 비탄력적 (ε<1) | 총수입 **증가** | 총수입 **감소** |

### 수요 가격탄력성 결정요인

- **대체재의 유무 및 다소** (가장 중요): 대체재 많을수록 탄력적
- 재화의 용도: 다양할수록 탄력적
- 기간: 장기일수록 탄력적
- 가계소득 대비 비중: 클수록 탄력적
- 생활필수품 < 사치품 (사치품이 더 탄력적)
- 부동산은 일반재화보다 **비탄력적**, 주거용 > 상업·공업용 (주거용이 더 탄력적)

### 탄력성과 균형점 이동

- **수요가 탄력적**일수록: 공급 변화 시 가격 **덜** 변하고, 거래량 **더** 변함
- **공급이 비탄력적**일수록: 수요 증가 시 가격이 **더 크게 상승**

## 5. 소득탄력성

$$수요의\ 소득탄력성 = \frac{수요량의\ 변화율}{소득의\ 변화율}$$

| 소득탄력성 | 재화 분류 |
|-----------|----------|
| **> 0 (+)** | **정상재** (소득↑ → 수요↑) |
| **= 0** | **중간재** (소득 무관) |
| **< 0 (-)** | **열등재** (소득↑ → 수요↓) |

> **주의**: 탄력적/비탄력적 판단은 항상 '1' 기준, 정상재/열등재 판단은 '0' 기준

## 6. 교차탄력성

$$수요의\ 교차탄력성 = \frac{해당\ 재화의\ 수요량\ 변화율}{다른\ 재화의\ 가격\ 변화율}$$

| 교차탄력성 | 관계 |
|-----------|------|
| **> 0 (+)** | **대체재** |
| **= 0** | **독립재** |
| **< 0 (-)** | **보완재** |

## 핵심 암기 포인트

- 수요의 가격탄력성 = |수요량 변화율 / 가격 변화율|
- 탄력성 판단 기준: **1** (탄력적/비탄력적), 소득탄력성 판단 기준: **0** (정상재/열등재)
- ε = 0: 완전비탄력적(**수직선**) / ε = ∞: 완전탄력적(**수평선**)
- 소득탄력성 > 0: **정상재** / < 0: **열등재** / = 0: **중간재**
- 교차탄력성 > 0: **대체재** / < 0: **보완재** / = 0: **독립재**
- 유량변수: 임대료, 급여, 주택거래량, NOI, 투자, 국민총생산
- 저량변수: 자산, 부채, 자본, 가격, 인구수, 주택보급률, 통화량
- 수요↑ + 공급↑ → 균형가격 **불확정**, 거래량 **반드시 증가**
- 탄력적 수요 + 가격 상승 → 총수입 **감소** / 비탄력적 수요 + 가격 상승 → 총수입 **증가**

## OX 퀴즈

**Q1.** 주택보급률은 유량(flow) 변수이다. → **X** (주택보급률은 **저량(stock)** 변수)

**Q2.** 수요의 가격탄력성이 비탄력적일 때 가격을 인상하면 총수입이 증가한다. → **O**

**Q3.** 소득탄력성이 양(+)이면 열등재이다. → **X** (양이면 **정상재**, 음이면 열등재)

**Q4.** 교차탄력성이 음(-)이면 두 재화는 대체재 관계이다. → **X** (음이면 **보완재**, 양이면 대체재)

**Q5.** 해당 상품의 가격 변화로 수요곡선 자체가 이동한다. → **X** (가격 변화는 수요곡선 **위의 점 이동**, 곡선 자체 이동은 가격 이외 요인)

**Q6.** 수요와 공급이 동시에 증가하면 균형거래량은 반드시 증가한다. → **O**

**Q7.** 부동산은 일반재화보다 가격탄력성이 탄력적이다. → **X** (부동산은 일반재화보다 **비탄력적**)

**Q8.** 대체재가 많을수록 수요의 가격탄력성은 비탄력적이다. → **X** (대체재 많을수록 **탄력적**)

## 함정 노트

- ⚠️ "가격 변화 → 수요곡선 이동" → 가격 변화는 곡선 **위의 점 이동**(수요량의 변화), 곡선 자체 이동(수요의 변화)은 가격 **이외** 요인
- ⚠️ "탄력성 판단 기준 혼동" → 탄력적/비탄력적은 **1** 기준, 정상재/열등재는 **0** 기준
- ⚠️ "임대료는 저량변수" → 임대료는 일정**기간** 개념이므로 **유량** 변수
- ⚠️ "가격은 유량변수" → 가격은 일정**시점** 개념이므로 **저량** 변수
- ⚠️ "수요·공급 동시 증가 시 가격도 반드시 상승" → 가격은 **불확정**, 거래량만 반드시 증가
