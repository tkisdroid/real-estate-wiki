# 개업공인중개사의 의무

## 1. 기본윤리 (법 제29조)

### 공정한 업무처리의무 (법 제29조 제1항)
- **품위유지**: 전문직업인으로서의 교양과 상식 (선언적 의무)
- **신의성실**: 공정하게 중개 관련 업무 수행
- **선관주의의무**: 법에 규정은 없으나 **판례로 확정**
  - 중개 완성 후 **거래 완성 시까지** 선관주의 의무 존속
- 적용 대상: 개업공인중개사, 소속공인중개사 (중개보조원은 **미적용**)
- 위반 시: **업무정지**

### 비밀준수의무 (법 제29조 제2항)
- 업무상 알게 된 비밀 **누설 금지**
- 업무를 **떠난 후에도** 적용
- 적용 대상: 개업공인중개사, 소속공인중개사, 중개보조원 모두
- 위반 시: **1년 이하 징역 또는 1천만원 이하 벌금** (반의사불벌죄)
- 예외: 법률상 고지의무, 거래상대방 보호, 본인 승낙

## 2. 인장등록 (법 제16조)

### 등록 대상·시기
| 대상 | 시기 | 등록할 곳 |
|---|---|---|
| 개업공인중개사 | **업무 개시 전** (개설등록신청 시 가능) | 사무소 소재지 등록관청 |
| 소속공인중개사 | **업무 개시 전** (고용신고 시 가능) | 사무소 소재지 등록관청 |
| 법인(주된사무소) | 개설등록신청 시 | 주된사무소 등록관청 |
| 법인(분사무소) | 설치신고 시 | **주된사무소** 등록관청 |

### 인장 규격
- 개인: 성명이 나타난 인장, **가로·세로 7mm 이상 30mm 이내**
- 법인: 상업등기규칙에 의한 법인 인장

### 인장 변경
- 변경 시 **7일 이내** 변경 등록 (단, 중개행위 시에는 반드시 변경 인장 등록 후 사용)

### 등록 인장 사용 의무
- 거래계약서, 확인·설명서, 중개계약서, 등록관청 제출 서면 등에 사용
- **중개보조원이 인장 사용** → 등록증 대여행위 (등록취소)
- 위반 시: 개업공인중개사 → **업무정지**, 소속공인중개사 → **자격정지**

## 3. 겸업제한 (법 제14조)

### 법인의 겸업 가능 업무 (법 제14조 제1항, 제2항)
| 번호 | 겸업 가능 업무 |
|---|---|
| ① | 상업용 건축물 및 주택의 **임대관리** 등 부동산 관리대행 |
| ② | 부동산의 이용·개발 및 거래에 관한 **상담** |
| ③ | 개업공인중개사 대상 중개업 **경영기법·정보 제공** |
| ④ | 상업용 건축물 및 주택의 **분양대행** |
| ⑤ | 도배·이사업체 소개 등 주거이전 **부수 용역 알선** |
| ⑥ | 경매·공매 대상 부동산의 **권리분석·취득알선·매수신청대리** |

- 법인이 이외의 업무를 겸업한 경우: **임의 등록취소** 또는 업무정지

## 4. 중개사무소 명칭·성명표기 (법 제18조)

### 명칭 의무
- 개업공인중개사: **"공인중개사사무소"** 또는 **"부동산중개"** 문자 사용
- 부칙 개업공인중개사: "공인중개사사무소" 명칭 **사용 불가** (위반 시 100만원 이하 과태료)
- 아닌 자의 유사명칭 사용: **1년 이하 징역 또는 1천만원 이하 벌금**

### 성명 표기
- 옥외광고물(벽면이용간판, 돌출간판, 옥상간판)에 개업공인중개사(법인 대표자) **성명 표기**
- 위반 시: **100만원 이하 과태료**
- 철거명령 불이행 → **행정대집행** 가능

## 5. 게시의무 (법 제17조)
- 중개사무소등록증, 중개보수·실비 요율표 등 **사무소에 게시**
- 위반 시: **100만원 이하 과태료**

## 6. 휴업·폐업 신고 (법 제21조)

### 휴업신고
| 항목 | 내용 |
|---|---|
| 대상 | **3개월 초과** 휴업 시 |
| 시기 | **사전신고** (하고자 할 때) |
| 첨부서류 | 휴업신고서 + 등록증 |
| 기간 | **6개월 초과 불가** (부득이한 사유 시 기간변경 가능) |
| 위반 | 미신고 3개월 초과 → 100만원 이하 과태료 |
| | 미신고 6개월 초과 → 임의 등록취소/업무정지 |

### 폐업신고
- **사전신고** (하고자 할 때)
- 위반 시: 100만원 이하 과태료
- 사망·해산 시: 폐업신고 의무 없음

### 재개업
- 업무재개신고서 제출 → 등록증 즉시 반환
- 미신고 재개업: 100만원 이하 과태료

## 7. 중개대상물 표시·광고 (법 제18조의2)

### 명시사항
- 중개사무소 명칭, 소재지, 연락처, 등록번호
- 개업공인중개사 성명 (법인 대표자 성명)
- **중개보조원 관련 사항은 명시 금지**

### 인터넷 광고 추가 명시
- 소재지, 면적, 가격, 종류, 거래형태
- 건축물: 총 층수, 사용승인일, 방향, 방·욕실 수, 입주가능일, 주차대수, 관리비

### 부당 표시·광고 금지 (법 제18조의2 제4항)
- 존재하지 않는 중개대상물 광고 → **500만원 이하 과태료**
- 거짓·과장 표시·광고 → 500만원 이하 과태료
- 실제 중개 의사 없는 광고 → 500만원 이하 과태료
