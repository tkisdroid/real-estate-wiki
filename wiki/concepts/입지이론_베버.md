# 베버(A. Weber)의 최소비용이론

> 📖 상위 문서: [[입지이론]]

## 1. 의의

- **공업입지**에 관한 이론을 **최초로** 전개한 경제학자
- 공업입지에 영향을 미치는 요인: **수송비 → 노동비 → 집적경제** (순서대로 고려)
- 수송비가 최소되는 지점을 먼저 조사 → 임금의 지역적 차이에 의한 수정 → 마지막으로 집적이익에 의한 최적지점 변화를 **단계적으로** 고찰

## 2. 최소운송비 원리

### 보편원료와 국지(편재)원료

| 구분 | 보편원료 | 국지(편재)원료 |
|------|---------|---------------|
| 정의 | 어느 장소에서나 구할 수 있는 원료 | 특정 장소에서만 생산되는 원료 |
| 예시 | 물, 공기 | 광물자원 |
| 수송비 차이 | 상대적으로 크지 않음 | 지역에 따라 **크게 차이** |
| 특성 | 100% 사용 시 시장 입지 → 총수송비 '0' | 입지 결정에 큰 영향 |

### 원료지수와 입지중량

$$원료지수 = \frac{국지원료중량}{제품중량}$$

$$입지중량 = 국지원료중량 + 제품중량 = 원료지수 + 1$$

### 원료지수에 따른 입지유형

| 원료지수 | 입지유형 | 산업 특성 | 입지중량 |
|----------|---------|----------|---------|
| **> 1** | **원료지향적** | 중량감소산업 | > 2 |
| **< 1** | **시장지향적** | 중량증가산업 | < 2 |
| **= 1** | **입지자유형** | 순수원료 | = 2 |

## 3. 공업입지 형태

| 유형 | 특징 | 예시 |
|------|------|------|
| **원료지향형** | 국지원료(편재원료) 많이 사용, 원료 부패성, 중량감소산업 | 시멘트, 제련, 통조림, 냉동 |
| **시장지향형** | 제품 부피/무게 증가, 제품 부패성, 소비자 접촉 필요 | 청량음료, 맥주, 제약, 제빵 |
| **노동지향형** | 노동비 비중 높음, 수송비 상대적으로 적을 때 | 섬유, 피복, 신발, 가전전자 |
| **집적지향형** | 기술연관성 높음, 전·후방 연계성 큰 업종 | 자동차 |
| **중간지점지향형** | 대량 원료 수입, 총운송비 최소 적환지점 | 수송적환지점 산업 |
| **자유입지형** | 부가가치 엄청나게 큼, 수송비 비중 극소 | 고도 기술집약산업 |

### 원료지향형 vs 시장지향형 비교

| 구분 | 원료지향형 | 시장지향형 |
|------|----------|----------|
| 수송비 | 국지원료수송비 > 제품수송비 | 국지원료수송비 < 제품수송비 |
| 산업 | 중량감소산업 | 중량증가산업 |
| 원료지수 | > 1 | < 1 |
| 입지중량 | > 2 | < 2 |
| 부패성 | 부패하기 쉬운 **원료** 이용 | 부패하기 쉬운 **제품** 생산 |

## 4. 노동지향론과 등비용선

### 노동지향론

- 노동비가 저렴한 지역이 최소운송비 지점보다 유리할 수 있음
- 조건: **노동비 절감분 > 추가 수송비** → 최적입지 이동

### 등비용선(Isodapane)

- 최소수송비 지점으로부터 기업이 입지를 바꿀 경우, **추가적인 수송비 부담액이 동일한 지점**을 연결한 곡선
- 주의: 등비용선은 **수송비**의 추가 부담액이 동일한 선 (노동비 아님)

## 5. 집적경제에 따른 최적지점의 변화

- 서로 다른 기업들이 한 지점에 **집적**함으로써 생산비용 절감 → 집적이익 최대 시 최소수송비 지점 이동 가능
- **집적이익 > 수송비·노동비 증가분** → 공장들이 해당 지점으로 모임
- **집적 불이익** 발생 시 → 공장들 다시 **분산**

## 핵심 암기 포인트

- 베버 = **최소비용이론**, 공업입지 이론 **최초** 전개
- 입지 고려 순서: **수송비 → 노동비 → 집적경제** (수송비가 가장 먼저!)
- 원료지수 = **국지원료중량 / 제품중량**
- 입지중량 = **국지원료중량 + 제품중량** (= 원료지수 + 1)
- 원료지수 > 1: **원료지향적** / < 1: **시장지향적** / = 1: **입지자유형**
- 등비용선 = 추가적 **수송비** 부담액 동일 지점 연결 (노동비 아님!)
- 원료지향형: 시멘트·제련·통조림 / 시장지향형: 청량음료·맥주·제약·제빵

## OX 퀴즈

**Q1.** 베버의 최소비용이론에서 입지 고려 순서는 집적경제 → 노동비 → 수송비이다. → **X** (**수송비 → 노동비 → 집적경제** 순서)

**Q2.** 원료지수가 1보다 크면 시장지향적 입지가 유리하다. → **X** (원료지수 > 1이면 **원료지향적**, < 1이면 시장지향적)

**Q3.** 보편원료를 많이 사용하는 공장은 원료지향형 입지를 선호한다. → **X** (**국지(편재)원료**를 많이 사용하는 공장이 원료지향형)

**Q4.** 등비용선은 최소수송비 지점에서 추가적인 수송비 부담액이 동일한 지점을 연결한 곡선이다. → **O**

**Q5.** 등비용선은 운송비와 노동비가 동일한 지점을 연결한 곡선이다. → **X** (**수송비**의 추가 부담액만 해당)

**Q6.** 베버의 입지 고려 요인 중 수송비가 가장 중요한 요소이다. → **O**

**Q7.** 제품 중량이 국지원료 중량보다 큰 제품을 생산하는 공장은 원료지향적 입지를 선호한다. → **X** (제품중량 > 국지원료중량이면 원료지수 < 1 → **시장지향적**)

**Q8.** 자동차산업은 기술연관성이 높아 집적지향적 산업에 해당한다. → **O**

## 함정 노트

- ⚠️ "입지 고려 순서: 집적경제가 먼저" → **수송비가 가장 먼저**, 그 다음 노동비, 마지막 집적경제
- ⚠️ "원료지수 > 1 = 시장지향" → 원료지수 > 1은 **원료지향적**, < 1이 시장지향적
- ⚠️ "보편원료 많이 사용 = 원료지향형" → 보편원료 100% → 시장 입지(총수송비 0). **국지원료** 많이 사용해야 원료지향형
- ⚠️ "등비용선 = 수송비 + 노동비 동일" → **수송비만** 해당 (제34회 출제)
- ⚠️ "각 요인이 최소가 되는 지점에 입지" → 각 요인이 최소가 아니라 **종합적으로 고려**하여 최소비용 지점 결정 (제24회)
