# 중개보수 산정방법

## 1. 보수 청구권

### 발생
- 중개계약에 의하여 발생 (보수특약 없어도 발생, 대판 1968.7.24)
- **중개완성**을 정지조건으로 행사 가능

### 행사 요건
1. **중개계약**이 존재
2. 중개 **계약기간 내**
3. **거래계약이 성립**
4. 중개활동과 거래 성립 간 **인과관계** 존재

### 지급시기
- 약정에 따르되, 약정 없으면 **거래대금 지급이 완료된 날**

### 소멸
- 개업공인중개사의 **고의·과실**로 거래가 무효·취소·해제된 경우 → **보수청구권 없음** (법 제32조 제1항)
- 당사자 사정으로 합의해제·채무불이행 → 보수청구권 **영향 없음**

## 2. 거래유형별 산정 기준

| 거래유형 | 산정 기준 |
|---|---|
| **매매** | 거래가격 x 요율 |
| **교환** | **큰 금액** 중개대상물 가액 x 요율 |
| 임대차 (월차임 없음) | 전세보증금 x 요율 |
| 임대차 (월차임 있음) | **(보증금 + 월차임 x 100)** x 요율 |
| 위 계산액이 **5천만원 미만**인 경우 | **(보증금 + 월차임 x 70)** x 요율 |
| **분양권 전매** | 실제거래가격(기납입금 + 프리미엄) x 요율 |
| 동일 대상물 2이상 거래 | **매매계약 거래금액만** 적용 |

### 월세 환산 계산 예시

**예시 1**: 보증금 3,000만원, 월차임 50만원
- 보증금 + 월차임 x 100 = 3,000 + 5,000 = 8,000만원
- 5천만원 이상이므로 → **8,000만원** 기준 요율 적용

**예시 2**: 보증금 1,000만원, 월차임 30만원
- 보증금 + 월차임 x 100 = 1,000 + 3,000 = 4,000만원
- 5천만원 미만이므로 → **(보증금 + 월차임 x 70)** 재계산
- 1,000 + 30 x 70 = 1,000 + 2,100 = **3,100만원** 기준 요율 적용

## 3. 쌍방 부담

- 의뢰인 **쌍방**으로부터 각각 받음 (규칙 제20조 제1항, 제4항)
- 매도인·매수인 각각에게 같은 요율 적용하여 받음

## 4. 부가가치세

| 과세 유형 | 부가세 징수 |
|---|---|
| **일반과세자** | 중개보수와 **별도**로 부가가치세 징수 **가능** |
| **간이과세자** | 부가가치세 **징수 불가** (초과수수로 처벌) |
| **미등록자** | 부가가치세 **징수 불가** (초과수수로 처벌) |

- 부가가치세법상 부가가치세를 별도 징수하는 것은 공인중개사법 위반이 **아님** (법제처 유권해석 2006.9.25)

## 5. 초과수수 금지

- 법정 보수 초과 수수 → **1년 이하 징역 또는 1천만원 이하 벌금**
- 사례·증여 등 **어떠한 명목**으로도 초과 금품 수수 금지
- 의뢰인이 **스스로** 초과 금액을 준 경우에도 위반
- 초과 수수 후 **되돌려준 경우**에도 위반 (수수 시점에 죄 성립)
- 초과 부분은 **무효** → 반환 의무

## 6. 실비 (법 제32조 제2항)

| 실비 종류 | 청구 대상 |
|---|---|
| 권리관계 확인 비용 (공부 열람·발급 등) | **매도·임대 등 권리이전 의뢰인** |
| 계약금 등 반환채무이행 보장 비용 | **권리취득 의뢰인** |

### 실비 주의사항
- 실비를 과다하게 청구할 경우 → **금지행위 위반** (1년 이하 징역 또는 1천만원 이하 벌금)

## 7. 순가중개

- 미리 매도·매수가격을 의뢰인이 제시 → 이를 초과한 금액으로 거래 시 초과분을 보수로 정하는 방식
- 법정 중개보수 이내라면 위반 아님
- **법정 보수 초과 시** 금지행위 위반

## 8. 주요 판례

| 판례 | 핵심 내용 |
|---|---|
| 대판 1968.7.24 | 보수특약 없어도 매매 성립 시 보수청구권 발생 |
| 대판 1956.4.12 | 중개 미성립 시 노력 비율에 상당한 보수 청구 불가 |
| 대판 1991.4.9 | 위임계약 해지 시 기대 보수 손해배상 청구 불가 |
| 대판 1990.11.13 | 사례금 명목 초과수수도 유죄 |
| 대판 1999.2.9 | 수고비 명목 초과수수도 위반 |
| 대판 2004.11.12 | 당좌수표 교부 시점에 죄 성립 (부도·반환 무관) |
| 대구지법 1987.10.30 | 확인설명 게을리하여 해제 시 보수 반환 + 손해배상 |

## 핵심 암기 포인트

1. 보수 청구: **중개완성** 정지조건, 4가지 요건 충족
2. 교환: **큰 금액** 기준
3. 월세 환산: **보증금 + 월차임 x 100** (5천만원 미만 시 **x 70**)
4. **쌍방**으로부터 각각 수수
5. 부가세: **일반과세자**만 별도 징수 가능
6. 간이과세자·미등록자 부가세 징수 → **초과수수 처벌**
7. 초과수수: 1년/1천만원, 어떤 명목이든 금지
8. 실비: 권리관계 확인 → **권리이전 의뢰인**, 반환보장 → **권리취득 의뢰인**
9. 고의·과실로 무효·취소·해제 → 보수청구권 **없음**
10. 합의해제·채무불이행 → 보수청구권 **영향 없음**

## OX 퀴즈

**Q1.** 교환의 경우 두 중개대상물 가액의 평균을 기준으로 보수를 산정한다.
> **X** — 교환은 **큰 금액**의 중개대상물 가액을 기준으로 산정한다.

**Q2.** 보증금 2,000만원, 월차임 20만원인 경우 월세 환산액은 4,000만원이다.
> **X** — 2,000 + 20 x 100 = 4,000만원이나, 5천만원 미만이므로 **2,000 + 20 x 70 = 3,400만원**을 기준으로 한다.

**Q3.** 간이과세자인 개업공인중개사가 중개보수와 별도로 부가가치세를 징수하면 초과수수에 해당한다.
> **O** — 간이과세자가 부가가치세를 별도 징수하면 **초과수수**에 해당한다.

**Q4.** 당사자 사정으로 거래가 합의해제된 경우 개업공인중개사의 보수청구권은 소멸한다.
> **X** — 당사자 사정으로 합의해제된 경우 보수청구권에 **영향 없다**.

**Q5.** 개업공인중개사의 과실로 거래가 취소된 경우에도 보수를 받을 수 있다.
> **X** — 개업공인중개사의 고의·과실로 무효·취소·해제 시 **보수 받을 수 없다**.

**Q6.** 의뢰인이 자발적으로 법정 보수를 초과하여 지급한 경우에는 초과수수에 해당하지 않는다.
> **X** — 의뢰인이 스스로 준 경우에도 **초과수수에 해당**한다.

**Q7.** 초과수수한 보수를 나중에 돌려주면 위반이 아니다.
> **X** — 수수 **시점에 죄가 성립**하므로 되돌려줘도 위반이다 (대판 2004.11.12).

**Q8.** 권리관계 확인 비용(실비)은 매수의뢰인에게 청구한다.
> **X** — 권리관계 확인 비용은 **매도·임대 등 권리이전 의뢰인**에게 청구한다.

## 함정 노트

| 함정 포인트 | 정확한 내용 |
|---|---|
| 교환 → 평균 기준 | 교환은 **큰 금액** 기준 |
| 월세 환산 항상 x 100 | 5천만원 미만 시 **x 70** 재계산 |
| 간이과세자도 부가세 가능 | 간이과세자·미등록자 부가세 징수 → **초과수수** |
| 합의해제 → 보수 소멸 | 합의해제 → 보수 **영향 없음** |
| 자발적 초과 지급 → 적법 | 자발적이어도 **초과수수 위반** |
| 초과 후 반환 → 면책 | 수수 **시점**에 죄 성립, 반환 무관 |
| 실비 → 매수인 청구 | 권리확인 실비 → **권리이전** 의뢰인 |
| 보수특약 없으면 청구 불가 | 특약 없어도 매매 성립 시 **청구 가능** |
