# 직접거래·쌍방대리 금지 (법 제33조 제1항 제6호)

## 1. 직접거래 금지

### 의의
- 개업공인중개사등이 중개의뢰인의 물건을 **직접 매수**하거나, **자기 물건을 의뢰인에게 매도**하는 행위
- 중개인이 거래 당사자가 되는 것을 금지

### 직접거래에 해당하는 경우
- 의뢰인의 물건을 자기가 직접 매수
- 자기 물건을 의뢰인에게 직접 매도
- **부부간 거래**도 직접거래에 해당 (대판 2021.9.3)
- 개업공인중개사가 토지를 매수한 다음 **미등기 전매**하여 차익 취득한 경우

### 직접거래에 해당하지 않는 경우
- **다른 개업공인중개사의 의뢰**로 거래 → 직접거래 아님
- 업으로 하지 않는 일회성 매매는 매매업 금지와 별개

### 법적 성격: 단속규정
- **단속규정**으로 거래행위 자체는 **무효 아님** (대판 2017.2.3)
- 처벌은 받으나, 사법상 거래의 효력은 유지

## 2. 쌍방대리 금지

### 의의
- 매도인·매수인 **양측을 동시에 대리**하는 행위
- 중개인이 양쪽의 이익을 동시에 대리하면 공정한 중개가 불가능

### 민법과의 차이
| 구분 | 민법 | 공인중개사법 |
|---|---|---|
| 쌍방대리 | 본인의 **허락 시 가능** (민법 제124조) | 본인의 허락이 있어도 **절대 금지** |
| 위반 효과 | 무권대리 | 형벌 + 행정처분 |

### 판례
- 개업공인중개사가 매도인의 대리인으로서 매수인까지 대리하여 계약 체결 → 쌍방대리 금지 위반

## 3. 위반 시 제재

| 위반 | 형벌 | 행정처분(개업) | 행정처분(소속) |
|---|---|---|---|
| 직접거래 | **3년/3천만원** | 임의 등록취소/업무정지 | 자격정지 |
| 쌍방대리 | **3년/3천만원** | 임의 등록취소/업무정지 | 자격정지 |

- 고용인(소속공인중개사·중개보조원) 위반 시 → **양벌규정** 적용 (개업공인중개사에게 벌금형)

## 4. 주요 판례

| 판례 | 핵심 내용 |
|---|---|
| 대판 2017.2.3 | 직접거래 금지는 **단속규정** → 거래행위 자체 **무효 아님** |
| 대판 2021.9.3 | **부부간 거래**도 직접거래에 해당 |
| 대판 1991.12.24 | 매도의뢰가격을 숨기고 높은 가격으로 매도 후 차액 취득 → 금지행위 + 불법행위 |
| 대판 1992.6.23 | 토지 매수 후 미등기 전매하여 차익 취득 → 투기조장 + 직접거래 |

## 핵심 암기 포인트

1. 직접거래: 의뢰인 물건 직접 매수 또는 자기 물건 의뢰인에게 매도
2. 부부간 거래 → **직접거래 해당** (대판 2021.9.3)
3. 다른 개업공인중개사 의뢰로 거래 → **직접거래 아님**
4. 직접거래는 **단속규정** → 거래 자체는 무효 아님
5. 쌍방대리: 본인 허락 있어도 **절대 금지** (민법과 차이)
6. 형벌: **3년/3천만원** (가장 무거운 벌칙 그룹)
7. 행정처분: **임의 등록취소** 또는 업무정지

## OX 퀴즈

**Q1.** 직접거래 금지 규정을 위반한 거래행위 자체는 무효이다.
> **X** — 직접거래 금지는 **단속규정**으로, 거래행위 자체는 **무효가 아니다** (대판 2017.2.3).

**Q2.** 개업공인중개사의 배우자가 의뢰인의 물건을 매수하는 것은 직접거래에 해당하지 않는다.
> **X** — **부부간 거래**도 직접거래에 해당한다 (대판 2021.9.3).

**Q3.** 쌍방대리는 매도인과 매수인 모두의 동의가 있으면 허용된다.
> **X** — 공인중개사법상 쌍방대리는 본인의 허락이 있어도 **절대 금지**이다.

**Q4.** 다른 개업공인중개사의 의뢰로 물건을 거래한 경우 직접거래에 해당한다.
> **X** — 다른 개업공인중개사의 의뢰로 거래한 것은 **직접거래에 해당하지 않는다**.

**Q5.** 직접거래·쌍방대리 위반 시 1년 이하 징역 또는 1천만원 이하 벌금에 처한다.
> **X** — **3년 이하** 징역 또는 **3천만원 이하** 벌금이다.

**Q6.** 소속공인중개사가 직접거래를 한 경우 그를 고용한 개업공인중개사에게 양벌규정이 적용된다.
> **O** — 고용인 위반 시 양벌규정 적용으로 개업공인중개사에게 벌금형이 부과된다.

## 함정 노트

| 함정 포인트 | 정확한 내용 |
|---|---|
| 직접거래 → 무효 | 직접거래는 **단속규정** → 거래 **유효** |
| 부부 → 직접거래 아님 | 부부간 거래 → **직접거래 해당** |
| 쌍방대리 → 동의 시 허용 | 쌍방대리 → **절대 금지** (동의 무관) |
| 다른 중개사 의뢰 → 직접거래 | 다른 중개사 의뢰 → **직접거래 아님** |
| 직접거래 → 1년/1천만원 | 직접거래 → **3년/3천만원** |
| 민법과 동일 | 민법은 허락 시 가능, 공인중개사법은 **절대 금지** |
