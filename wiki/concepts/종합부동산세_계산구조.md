# 종합부동산세 - 계산구조

## 1. 종합부동산세 개요

종합부동산세는 2005년 신설된 **국세**로, 고액의 부동산 보유자에 대하여 [[재산세]]와 별도로 부과하는 보유세이다.

| 특징 | 내용 |
|------|------|
| **과세기준일** | 매년 **6월 1일** (재산세와 동일) |
| **과세대상** | 주택 + 토지 (건축물·선박·항공기는 제외) |
| **세율구조** | 초과누진세율 (인세 성격) |
| **과세방식** | 원칙: 정부부과과세 / 예외: 선택적 신고납부 |
| **부과·징수기간** | 매년 **12월 1일~12월 15일** |
| **분할납부** | 가능 |
| **세부담 상한** | 있음 (150%) |

## 2. 주택에 대한 종합부동산세

### 납세의무자

| 구분 | 공시가격 합계액 기준 | 납세의무 |
|------|-------------------|---------|
| **개인 (일반)** | 9억원 초과 | 있음 |
| **개인 (1세대1주택자)** | 12억원 초과 | 있음 |
| **법인** | 0원 초과 (공제 없음) | 있음 |

### 과세표준

| 구분 | 과세표준 |
|------|---------|
| **개인 (일반)** | (주택 공시가격 합계액 - **9억원**) x 공정시장가액비율(**60%**) |
| **개인 (1세대1주택)** | (주택 공시가격 합계액 - **12억원**) x 공정시장가액비율(**60%**) |
| **법인** | 주택 공시가격 합계액 x 공정시장가액비율(**60%**) |

> 공정시장가액비율 범위: 100분의 60 ~ 100분의 100 (현행 **60%**)

### 세율 (개인)

#### 2주택 이하 소유

| 과세표준 | 세율 |
|---------|------|
| 3억원 이하 | 0.5% |
| 3억원 초과 ~ 6억원 이하 | 150만원 + 3억원 초과분의 0.7% |
| 6억원 초과 ~ 12억원 이하 | 360만원 + 6억원 초과분의 1.0% |
| 12억원 초과 ~ 25억원 이하 | 960만원 + 12억원 초과분의 1.3% |
| 25억원 초과 ~ 50억원 이하 | 2,650만원 + 25억원 초과분의 1.5% |
| 50억원 초과 ~ 94억원 이하 | 6,400만원 + 50억원 초과분의 2.0% |
| 94억원 초과 | 1억5,200만원 + 94억원 초과분의 2.7% |

#### 3주택 이상 소유

| 과세표준 | 세율 |
|---------|------|
| 3억원 이하 | 0.5% |
| 3억원 초과 ~ 6억원 이하 | 150만원 + 3억원 초과분의 0.7% |
| 6억원 초과 ~ 12억원 이하 | 360만원 + 6억원 초과분의 1.0% |
| 12억원 초과 ~ 25억원 이하 | 960만원 + 12억원 초과분의 2.0% |
| 25억원 초과 ~ 50억원 이하 | 3,560만원 + 25억원 초과분의 3.0% |
| 50억원 초과 ~ 94억원 이하 | 1억1,060만원 + 50억원 초과분의 4.0% |
| 94억원 초과 | 2억8,660만원 + 94억원 초과분의 5.0% |

#### 법인 세율 (비례세율)

| 구분 | 세율 |
|------|------|
| **2주택 이하** | **2.7%** |
| **3주택 이상** | **5.0%** |

### 주택분 세액 계산 흐름

```
① 과세표준 x 세율 = 종합부동산세액
② (-) 주택분 재산세 부과액 (이중과세 공제)
③ (-) 연령별 세액공제 (1세대1주택 단독소유만)
④ (-) 장기보유기간별 세액공제 (1세대1주택 단독소유만)
⑤ = 실제 산출세액
⑥ 세부담상한 적용 (전년도 총세액의 150% 초과분은 없는 것으로 봄)
```

### 1세대1주택자(단독소유) 세액공제

#### 연령별 공제율

| 연령 | 공제율 |
|------|-------|
| 만 **60세** 이상 ~ 만 65세 미만 | **20%** |
| 만 **65세** 이상 ~ 만 70세 미만 | **30%** |
| 만 **70세** 이상 | **40%** |

#### 장기보유기간별 공제율

| 보유기간 | 공제율 |
|---------|-------|
| **5년** 이상 ~ 10년 미만 | **20%** |
| **10년** 이상 ~ 15년 미만 | **40%** |
| **15년** 이상 | **50%** |

> 연령별 공제율 + 보유기간별 공제율 합계 **최대 80%** 한도 내에서 중복 적용 가능

## 3. 토지에 대한 종합부동산세

### 과세구분 및 공제금액

| 구분 | 공제금액 | 공정시장가액비율 |
|------|---------|---------------|
| **종합합산과세대상** | **5억원** | **100%** |
| **별도합산과세대상** | **80억원** | **100%** |
| **분리과세대상** | 종합부동산세 **과세 제외** | - |

### 종합합산과세대상 세율

| 과세표준 | 세율 |
|---------|------|
| 15억원 이하 | 1.0% |
| 15억원 초과 ~ 45억원 이하 | 1,500만원 + 15억원 초과분의 2.0% |
| 45억원 초과 | 7,500만원 + 45억원 초과분의 3.0% |

### 별도합산과세대상 세율

| 과세표준 | 세율 |
|---------|------|
| 200억원 이하 | 0.5% |
| 200억원 초과 ~ 400억원 이하 | 1억원 + 200억원 초과분의 0.6% |
| 400억원 초과 | 2억2,000만원 + 400억원 초과분의 0.7% |

### 토지분 세부담 상한

해당 연도 재산세액 + 종합(별도)합산세액 > 직전 연도 총세액의 **150%** → 초과분은 없는 것으로 봄

## 4. 부과·징수

| 구분 | 내용 |
|------|------|
| **원칙** (정부부과과세) | 관할세무서장이 세액 결정하여 **12월 1일~12월 15일** 부과·징수 |
| **예외** (신고납세) | 납세의무자가 과세표준·세액을 **12월 1일~12월 15일**에 신고납부 가능 |
| **분할납부** | 납부세액 250만원 초과 시, 일부를 납부기한 경과 후 6개월 이내 분납 가능 |

### 분할납부

| 납부세액 | 분할납부 가능 세액 |
|---------|-----------------|
| **500만원 이하** | 250만원 초과 금액 |
| **500만원 초과** | 세액의 **50% 이하** |

## 5. 재산세와 종합부동산세 비교

| 구분 | 재산세 | 종합부동산세 |
|------|--------|------------|
| **세목** | 지방세 (시·군·구) | 국세 |
| **과세대상** | 주택, 토지, 건축물, 선박, 항공기 | 주택, 토지 |
| **과세방식** | 정부부과 | 정부부과 (선택적 신고납부) |
| **공정시장가액비율** | 주택 60%, 토지·건축물 70% | 주택 60%, 토지 100% |
| **세부담상한** | 직전연도 세액의 105~150% | 직전연도 총세액의 **150%** |
| **세액공제** | 없음 | 1세대1주택 연령별+보유기간별 공제 |

## 핵심 암기 포인트

- 과세기준일: **6월 1일**
- 주택분 공제: 일반 **9억원**, 1세대1주택 **12억원**, 법인 **0원**
- 토지분 공제: 종합합산 **5억원**, 별도합산 **80억원**
- 공정시장가액비율: 주택 **60%**, 토지 **100%**
- 1세대1주택 세액공제 합계 한도: **80%**
- 세부담상한: **150%** (법인은 적용 안 함)
- 부과·징수: **12월 1일~12월 15일**
- 법인 세율: 2주택 이하 **2.7%**, 3주택 이상 **5.0%** (비례세율)

**Q1.** 종합부동산세의 과세기준일은 매년 1월 1일이다. → **X** (6월 1일, 재산세와 동일)

**Q2.** 1세대1주택자의 종합부동산세 과세표준은 공시가격 합계액에서 9억원을 공제한다. → **X** (1세대1주택자는 12억원 공제)

**Q3.** 법인이 소유한 주택에 대해서도 종합부동산세 공제금액이 적용된다. → **X** (법인은 공제금액 0원)

**Q4.** 주택분 공정시장가액비율은 현행 60%이다. → **O**

**Q5.** 토지분 종합합산과세의 공정시장가액비율은 60%이다. → **X** (토지분은 100%)

**Q6.** 1세대1주택자의 연령별 공제율과 장기보유기간별 공제율은 합계 100%까지 중복 적용할 수 있다. → **X** (합계 80% 한도)

**Q7.** 법인의 주택분 종합부동산세에는 세부담상한이 적용된다. → **X** (법인은 세부담상한 적용 안 됨)

**Q8.** 건축물은 종합부동산세 과세대상이다. → **X** (주택과 토지만 과세대상, 건축물·선박·항공기는 재산세만 과세)

**Q9.** 별도합산과세대상 토지의 공제금액은 80억원이다. → **O**

**Q10.** 종합부동산세는 매년 12월 1일부터 12월 15일까지 부과·징수한다. → **O**

## 함정 노트

- ⚠️ **1세대1주택 12억원 vs 일반 9억원**: 1세대1주택자만 12억원 공제, 일반은 9억원. 법인은 0원
- ⚠️ **공정시장가액비율 주택 vs 토지**: 주택 **60%**, 토지 **100%** — 반드시 구분
- ⚠️ **법인의 특례**: 법인은 공제금액 0원, 비례세율 적용, 세액공제 없음, 세부담상한 없음
- ⚠️ **세액공제 합계 한도 80%**: 연령별(최대40%) + 보유기간별(최대50%) = 최대 80% (단순합산 90%가 아님)
- ⚠️ **세부담상한 150% 균일**: 주택은 2주택 이하·3주택 이상 모두 150%로 동일 (법인 제외)
- ⚠️ **분리과세대상 토지는 종합부동산세 과세 제외**: 종합합산·별도합산만 종합부동산세 대상
