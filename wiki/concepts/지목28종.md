# 지목 28종

## 1. 지목의 의의

지목이란 토지의 **주된 용도**에 따라 토지의 종류를 구분하여 지적공부에 등록한 것을 말한다. 우리나라는 **용도지목주의**를 채택하고 있으며, 현재 **28종**으로 구분한다.

## 2. 지목설정의 원칙

| 원칙 | 내용 |
|------|------|
| **1필1목 원칙** (단식지목주의) | 1필지마다 하나의 지목을 설정 |
| **주용도추종의 원칙** | 1필지가 둘 이상의 용도로 활용시 주된 용도에 따라 설정 |
| **영속성의 원칙** (일시변경불변) | 토지가 일시적·임시적 용도로 사용되는 때에는 지목 불변경 |
| **사용목적추종의 원칙** | 개발사업 준공 후 사용목적에 따라 지목 설정 |

## 4. 차문자(次文字)로 표기하는 지목 (4개)

두문자와 다른 지목이 있어 **두 번째 글자**로 표기하는 지목:
- 공**장**용지 (장) - '공'은 공원이 사용
- 주**차**장 (차) - '주'는 주유소용지가 사용
- **천** (하천) - '하'가 아닌 '천' 사용
- **원** (유원지) - '유'는 유지가 사용

## 6. 주요 시험 포인트

- **전과 답의 구분**: 물을 직접 이용하느냐의 여부 (경작방법에 따라 결정)
- **주거용 건축물의 부지**: 과수원·목장용지 안이라도 주거용 건축물 부지는 **'대'**
- 도로: 현행법은 2필지 이상의 **'대'**에 한정하지 않고 2필지 이상에 진입하는 통로로 규정
- **고가도로·지하도로**: 평면지적(2차원)을 채택하므로 '도로'로 지목 설정 **불가**
- 자연공원법상 국립·도립·군립공원: 원칙적으로 **'임야'**
- 묘지공원: **'묘지'** (나머지 도시공원은 '공원')
- 원상회복 조건 돌/흙 캐내는 곳: 잡종지로 변경 **불가** (영속성의 원칙)

## 함정 노트

- ⚠️ **차문자 4개 혼동**: 장(공장용지)·차(주차장)·천(하천)·원(유원지) — '공·주·하·유'와 혼동하지 말 것
- ⚠️ **전 vs 답**: 핵심 구분은 '물의 직접 이용 여부'. 연·미나리 = 답(물 직접 이용)
- ⚠️ **영속성의 원칙**: 일시적·임시적 용도 변경 시 지목 불변경. '원상회복 조건 채석장'도 변경 불가
- ⚠️ **도로 요건 변경**: 현행법은 2필지 이상의 '대'가 아닌 2필지 이상에 진입하는 통로로 확대
- ⚠️ **묘지공원 vs 도시공원**: 묘지공원은 '묘지', 나머지 도시공원은 '공원' — 혼동 빈출
