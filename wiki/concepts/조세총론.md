# 조세총론

## 1. 조세의 개념 및 특징

**조세**란 국가 또는 지방자치단체가 재정경비에 충당하기 위하여 법률이 정하는 바에 따라 과세요건을 충족하는 자연인이나 법인에게 **특별한 반대급부 없이** 법률적·강제력에 의해 부과·징수하는 경제적 부담(금전급부)을 말한다.

### 조세의 특징

| 특징 | 내용 |
|------|------|
| 과세주체 | 국가(국세) 또는 지방자치단체(지방세) |
| 부과목적 | 재정수입 조달 + 사회·경제정책적 목적 |
| 과세근거 | **조세법률주의** (헌법 제38조, 제59조) |
| 비대가성 | 직접적·개별적 반대급부 없음 |
| 납부방법 | **금전납부 원칙**, 예외적으로 물납·분납 허용 |

### 물납·분납 가능 여부

| 구분 | 물납 | 분할납부 |
|------|------|----------|
| 지방세 | [[재산세]] (도시지역분 포함) | [[재산세]] (도시지역분 포함) 등 |
| 국세 | 상속세 | [[종합부동산세]], [[양도소득세]] 등 |

## 2. 조세의 분류

### 과세주체에 따른 분류

**국세** (13개 세목): 소득세, 법인세, 상속세, 증여세, [[종합부동산세]], 부가가치세, 농어촌특별세, 교육세 등

**지방세** (11개 세목): [[취득세]], [[등록면허세]], [[재산세]], 지방소득세, 지방소비세, 지방교육세, 지역자원시설세, 자동차세, 주민세, 담배소비세, 레저세

### 부동산 활동에 따른 분류

| 구분 | 취득 단계 | 보유 단계 | 양도 단계 |
|------|-----------|-----------|-----------|
| 국세 | 농어촌특별세, 인지세 | [[종합부동산세]], 종합소득세 | [[양도소득세]] |
| 지방세 | [[취득세]], [[등록면허세]] | [[재산세]], 지역자원시설세 | - |

### 직접세와 간접세

- **직접세** (납세자 = 담세자): [[취득세]], [[등록면허세]], [[재산세]], 소득세, [[종합부동산세]] 등 대부분의 조세
- **간접세** (납세자 ≠ 담세자): 부가가치세, 인지세, 개별소비세 등

### 보통세와 목적세

- **보통세**: 용도를 특정하지 않는 조세 (소득세, [[취득세]], [[재산세]] 등)
- **목적세**: 특별한 목적에 충당하는 조세 (농어촌특별세, 교육세, 지방교육세, 지역자원시설세)

### 인세와 물세

- **인세**: 조세부담능력 고려, 합산과세, **초과누진세율** 적용 (소득세, [[종합부동산세]], [[재산세]] 일부)
- **물세**: 조세부담능력 불문, 개별과세, **비례세율** 적용 ([[취득세]], [[등록면허세]], [[재산세]])

## 3. 과세요건

납세의무 성립에 필요한 법률상의 요건: **과세대상, 납세의무자, 과세표준, 세율**

### 세율의 분류

| 분류 기준 | 종류 | 내용 | 해당 세목 |
|-----------|------|------|-----------|
| 표시단위 | 정률세율 | 백분비(%)로 표시 | 대부분의 조세 |
| | 정액세율 | 화폐단위(원)로 표시 | 등록면허세 일부 |
| 과세표준 크기 | 비례세율 | 과세표준과 무관하게 고정 | [[취득세]], 법인세 등 |
| | 초과누진세율 | 과세표준 증가시 세율 상승 | [[재산세]] 일부, [[양도소득세]] |

## 4. 납세의무의 성립·확정·소멸

### 납세의무 성립시기

| 세목 | 성립시기 |
|------|----------|
| [[취득세]] | 과세물건을 **취득하는 때** |
| [[등록면허세]] (등록) | 재산권 등을 **등기·등록하는 때** |
| [[재산세]] | **과세기준일 (6월 1일)** |
| [[종합부동산세]] | **과세기준일 (6월 1일)** |
| [[양도소득세]] (소득세) | **과세기간이 끝나는 때** |

### 납세의무 확정

| 확정방법 | 내용 | 해당 세목 |
|----------|------|-----------|
| **신고주의** | 납세의무자의 신고로 확정 | [[취득세]], [[등록면허세]], [[양도소득세]] |
| **부과주의** | 과세관청의 결정으로 확정 | [[재산세]], [[종합부동산세]], 상속세 |
| **자동확정** | 성립과 동시에 확정 | 인지세, 원천징수 소득세 |

### 납세의무 소멸사유

1. **납부** (납입)
2. **충당** (환급금을 미납세액에 직권충당)
3. **부과의 취소**
4. **부과권 제척기간 만료**
5. **징수권 소멸시효 완성**

### 제척기간 (부과할 수 있는 기간)

| 사유 | 일반국세 | 상속·증여세 | 지방세 |
|------|----------|-------------|--------|
| 부정행위로 포탈 | **10년** | **15년** | **10년** |
| 무신고 | **7년** | **15년** | **7년** |
| 기타 | **5년** | **10년** | **5년** |

### 소멸시효 (징수할 수 있는 기간)

| 구분 | 5천만원(국세 5억원) 미만 | 5천만원(국세 5억원) 이상 |
|------|--------------------------|--------------------------|
| 지방세 | **5년** | **10년** |
| 국세 | **5년** | **10년** |

- 소멸시효는 **중단·정지 사유**가 있음 (제척기간과 차이)
- 중단사유: 납세고지, 독촉, 교부청구, 압류
- 정지사유: 분납기간, 징수유예기간, 체납처분유예기간 등

## 5. 조세채권의 우선순위

**원칙**: 조세 > 일반채권

### 조세우선권의 제한 (배당순위)

| 순위 | 내용 |
|------|------|
| **0순위** | 체납처분비 (강제집행·경매비용) |
| **1순위** | 소액임차보증금, 최종 3월분 임금·최종 3년분 퇴직금·재해보상금 |
| **2순위** | **당해 재산에 부과된 조세** ([[종합부동산세]], 상속세, 증여세, [[재산세]], 자동차세 등) |
| **3순위** | 저당권 등 피담보채권 및 기타 조세 (법정기일 기준) |

> **시험 핵심**: 당해 재산에 부과된 조세([[재산세]], [[종합부동산세]], 상속세, 증여세)는 저당권 설정시기와 **관계없이** 항상 피담보채권보다 우선 징수된다.

## 6. 가산세

### 신고불성실가산세

| 종류 | 세율 |
|------|------|
| 일반 무신고 | 무신고납부세액의 **20%** |
| 부정 무신고 | 무신고납부세액의 **40%** |
| 일반 과소신고 | 과소신고분 납부세액의 **10%** |
| 부정 과소신고 | 부정과소신고분 납부세액의 **40%** |

### 납부지연가산세

- **1일당 0.022%** (10만분의 22)
- 지방세: 납부하지 아니한 세액의 **75%** 한도
- 납부고지 후: **3%** + 매 1개월마다 **0.75%** (지방세 60개월, 국세 5년 한도)
- 체납세액 **45만원 미만**(지방세) / **150만원 미만**(국세)인 경우 월별 가산세 미적용

### 수정신고에 따른 가산세 감면

| 수정신고 시기 | 감면율 |
|--------------|--------|
| 1개월 이내 | **90%** |
| 1~3개월 | **75%** |
| 3~6개월 | **50%** |
| 6개월~1년 | **30%** |
| 1년~1년6개월 | **20%** |
| 1년6개월~2년 | **10%** |

## 7. 조세 구제제도

### 불복청구 절차

- **국세**: 이의신청(임의) → 심사청구 또는 심판청구(택일, 필수) → 행정소송
- **지방세**: 이의신청(임의) → 심사청구 또는 심판청구(택일) → 행정소송
- 청구기간: 처분을 안 날부터 **90일 이내**
- 행정소송: 결정통지를 받은 날부터 **90일 이내**

## 핵심 암기 포인트

- **국세 13개, 지방세 11개** 세목
- **취득 단계**: 취득세·등록면허세(지방세), 농어촌특별세·인지세(국세)
- **보유 단계**: 재산세·지역자원시설세(지방세), 종합부동산세·종합소득세(국세)
- **양도 단계**: 양도소득세(국세)
- **납세의무 성립시기**: 취득세 = 취득하는 때, 등록면허세 = 등기·등록하는 때, 재산세·종부세 = 6월 1일, 양도소득세 = 과세기간 종료 시
- **확정방법**: 신고주의(취득세·등록면허세·양도소득세), 부과주의(재산세·종부세·상속세·증여세), 자동확정(인지세·원천징수)
- **소멸사유 5가지**: 납부, 충당, 부과의 취소, 부과권 제척기간 만료, 징수권 소멸시효 완성
- **제척기간**: 부정행위 10년(상속·증여 15년), 무신고 7년(상속·증여 15년), 기타 5년(상속·증여 10년)
- **소멸시효**: 5천만원(국세 5억원) 미만 = 5년, 이상 = 10년
- **가산세**: 무신고 20%(부정 40%), 과소신고 10%(부정 40%), 납부지연 1일당 0.022%
- **수정신고 감면**: 1개월 내 90%, 3개월 75%, 6개월 50%, 1년 30%, 1년6개월 20%, 2년 10%
- **조세우선순위**: 체납처분비 → 소액임차·임금 → 당해세(재산세·종부세·상속세·증여세) → 저당권·기타조세
- **불복절차**: 이의신청(임의) → 심사·심판(택일) → 행정소송 (각 90일 이내)

## OX 퀴즈

**Q1.** 취득세의 납세의무는 과세기준일(6월 1일)에 성립한다. → **X** (취득세는 과세물건을 취득하는 때 성립. 6월 1일은 재산세·종부세)

**Q2.** 재산세는 신고납부하는 조세이다. → **X** (재산세는 부과주의 — 과세관청의 결정으로 확정, 보통징수)

**Q3.** 납세의무자의 사망은 납세의무의 소멸사유이다. → **X** (상속인에게 승계되므로 소멸사유가 아님)

**Q4.** 부과권의 제척기간에는 중단·정지 사유가 있다. → **X** (제척기간에는 중단·정지 없음. 소멸시효에만 있음)

**Q5.** 당해 재산에 부과된 조세(재산세·종부세)는 저당권 설정시기와 관계없이 저당권보다 우선 징수된다. → **O**

**Q6.** 물납은 모든 세목에 허용된다. → **X** (재산세(지방세)와 상속세(국세)에만 물납 가능)

**Q7.** 부정 무신고 시 가산세는 무신고납부세액의 20%이다. → **X** (40%이다. 일반 무신고가 20%)

**Q8.** 수정신고를 1개월 이내에 하면 가산세의 75%를 감면받는다. → **X** (1개월 이내 90%, 1~3개월이 75%)

## 함정 노트

- ⚠️ **성립시기 혼동**: 취득세 = 취득 시, 재산세·종부세 = 6월 1일, 양도소득세 = 과세기간 종료 — 각각 다르므로 구분 필수
- ⚠️ **제척기간 vs 소멸시효**: 제척기간은 부과권(중단·정지 없음), 소멸시효는 징수권(중단·정지 있음) — 반대로 적용 금지
- ⚠️ **당해세의 범위**: 재산세·종합부동산세·상속세·증여세만 해당. 취득세·양도소득세는 당해세가 아님
- ⚠️ **가산세 세율**: 일반 무신고 20% vs 부정 무신고 40%, 일반 과소신고 10% vs 부정 과소신고 40% — 숫자 혼동 주의
