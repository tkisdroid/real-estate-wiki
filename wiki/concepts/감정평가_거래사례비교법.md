# 거래사례비교법 (비교방식)

> 📖 상위 문서: [[감정평가3방식]]

## 1. 기본 공식

$$비준가액 = 거래사례가격 \times 사정보정 \times 시점수정 \times 지역요인\ 비교 \times 개별요인\ 비교$$

- 비교방식은 대상물건의 **시장성**에 착안한 감정평가 방식
- 기본 원칙: **대체의 원칙**
- **일반 거래가 활발한** 부동산에 적합

## 2. 사례 선정 요건

거래사례비교법 적용 시 비교 가능한 적절한 **거래사례**를 선정해야 함

### 사례 선정 기준

1. 대상부동산과 **용도**가 동일하거나 유사
2. 대상부동산과 **인근지역** 또는 **유사지역**의 사례
3. **정상적인 거래**에 해당하는 사례 (특수사정 배제)
4. 거래시점이 기준시점과 **가까운** 사례

## 3. 4단계 보정·비교 과정

### 1단계: 사정보정

| 항목 | 내용 |
|------|------|
| **목적** | 거래사례에 포함된 **특수한 사정**을 제거 |
| **사유** | 급매, 친인척 거래, 강제집행, 특수 동기 등 |
| **방향** | 정상가격 수준으로 보정 |

- 사정보정이 필요 없는 정상거래 = 보정계수 **1.0**

### 2단계: 시점수정

| 항목 | 내용 |
|------|------|
| **목적** | 거래시점과 기준시점 간의 **가격 변동** 보정 |
| **사유** | 시간 경과에 따른 부동산 시세 변화 |
| **기준** | 지가변동률, 생산자물가지수 등 활용 |

### 3단계: 지역요인 비교

| 항목 | 내용 |
|------|------|
| **목적** | 사례부동산 소재 지역과 대상부동산 소재 지역의 **차이** 보정 |
| **요인** | 교통, 환경, 행정, 접근성 등 지역적 특성 |
| **방법** | 인근지역과 유사지역의 지역 격차 보정 |

### 4단계: 개별요인 비교

| 항목 | 내용 |
|------|------|
| **목적** | 사례부동산과 대상부동산의 **개별적 차이** 보정 |
| **요인** | 면적, 형상, 접면도로, 방위, 층수, 구조 등 |

## 4. 보정 순서의 중요성

```
거래사례가격 → ①사정보정 → ②시점수정 → ③지역요인 비교 → ④개별요인 비교 → 비준가액
```

- **순서 암기**: **사(사정보정) → 시(시점수정) → 지(지역요인) → 개(개별요인)**

## 5. 임대사례비교법 (임대료 평가)

- 임대사례를 비교하여 **비준임대료** 산정
- 보정 과정은 거래사례비교법과 동일한 4단계 적용

## 6. 개별성과의 관계

- 부동산의 **개별성**은 감정평가 시 **개별분석의 필요성**을 요구
- 개별성으로 인해 거래사례비교법 적용이 **어려울 수 있음**

## 핵심 암기 포인트

- 비준가액 = 거래사례가격 x **사정보정 x 시점수정 x 지역요인 x 개별요인**
- 순서: **사 → 시 → 지 → 개** (사시지개)
- 사정보정: **특수 사정** 제거 / 시점수정: **가격 변동** 보정
- 지역요인: **지역 차이** 보정 / 개별요인: **개별적 차이** 보정
- 비교방식의 원리 = **시장성**, 원칙 = **대체의 원칙**
- 시산가액 = **비준가액**

## OX 퀴즈

**Q1.** 거래사례비교법의 보정 순서는 시점수정 → 사정보정 → 지역요인 → 개별요인이다. → **X** (**사정보정 → 시점수정** → 지역요인 → 개별요인)

**Q2.** 사정보정은 시간의 경과에 따른 가격 변동을 보정하는 것이다. → **X** (가격 변동 보정은 **시점수정**. 사정보정은 **특수한 사정** 제거)

**Q3.** 거래사례비교법은 수익성에 착안한 감정평가 방식이다. → **X** (**시장성**에 착안. 수익성은 수익환원법)

**Q4.** 거래사례비교법으로 산정한 시산가액을 비준가액이라 한다. → **O**

**Q5.** 지역요인 비교는 대상부동산과 사례부동산의 개별적 차이를 보정하는 것이다. → **X** (개별적 차이 보정은 **개별요인 비교**. 지역요인은 **지역 차이**)

**Q6.** 부동산의 개별성은 거래사례비교법의 적용을 용이하게 한다. → **X** (개별성은 비교를 **어렵게** 함)

## 함정 노트

- ⚠️ "보정 순서: 시점수정이 먼저" → **사정보정이 먼저** (사→시→지→개)
- ⚠️ "사정보정 = 가격변동 보정" → 가격변동은 **시점수정**, 사정보정은 **특수사정 제거**
- ⚠️ "지역요인 = 개별적 차이" → 지역요인은 **지역 차이**, 개별적 차이는 **개별요인**
- ⚠️ "비교방식 원리 = 비용성" → 비교방식 = **시장성**, 원가방식이 비용성
- ⚠️ "개별성 → 거래사례비교법 용이" → 개별성으로 인해 적용이 **어려워짐**
