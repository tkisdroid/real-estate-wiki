# 양도소득세 비과세

[[양도소득세]]의 비과세 규정은 시험에서 가장 자주 출제되는 부분으로, 1세대1주택 비과세가 핵심이다.

## 1. 1세대1주택 비과세 (소득세법 제89조)

### 기본 요건

1. **1세대**가 국내에 **1주택**을 보유
2. **2년 이상** 보유 (조정대상지역 내 취득: 2년 이상 **거주** 추가)
3. 양도하는 경우

### 1세대의 범위

거주자 및 **배우자** (사실혼 포함) + 같은 주소에서 생계를 같이 하는 가족 (직계존비속, 형제자매 등)

> **배우자가 없어도 1세대로 보는 경우**: 만 30세 이상, 또는 배우자 사망·이혼, 또는 일정 소득 이상으로 독립세대 구성

### 고가주택 과세

양도가액이 **12억원을 초과**하는 주택은 12억원 **초과분**에 대해서만 양도소득세 과세

**고가주택 양도차익 계산**:

```
과세대상 양도차익 = 양도차익 × (양도가액 - 12억원) / 양도가액
```

### 주택 수 판단

| 경우 | 주택 수 |
|------|---------|
| 다가구주택 | **1주택** (구분등기 시 각각 1주택) |
| 다세대주택 (구분등기) | **각각 1주택** |
| 겸용주택 (주거면적 50% 이상) | 전체를 **1주택**으로 봄 |
| 조합원입주권·분양권 | **주택 수에 포함** |

## 2. 일시적 2주택 비과세

### 기본 요건

1. 1세대1주택자가 종전주택 취득 후 **1년 이상** 경과한 후 신규주택 취득
2. 신규주택 취득일부터 **3년 이내** 종전주택 양도
3. 조정대상지역 내 취득: **2년 이내** 종전주택 양도

### 특수한 일시적 2주택

| 유형 | 요건 |
|------|------|
| **상속주택 + 일반주택** | 일반주택을 먼저 양도하면 비과세 (상속주택은 주택 수 제외) |
| **혼인합가** | 혼인으로 2주택 → **5년 이내** 1주택 양도 시 비과세 |
| **동거봉양합가** | 60세 이상 직계존속 동거봉양 목적 합가 → **10년 이내** 양도 시 비과세 |
| **이농주택** | 영농 목적 취득 후 귀농 → 일반주택 양도 시 비과세 |
| **문화재주택** | 문화재보호법상 문화재 주택은 주택 수 제외 |

## 3. 농지 관련 비과세

### 농지의 교환·분합

- 경작 농지의 **교환**으로 발생하는 소득: 비과세
- 경작 농지의 **분합**(토지 합병·분할)으로 발생하는 소득: 비과세

### 8년 이상 자경농지 (조세특례제한법)

- **8년 이상** 직접 경작한 농지 양도 시 양도소득세 **감면** (100%, 연간 한도 2억원, 5년간 한도 3억원)

## 4. 비과세 배제 (비과세 적용 불가)

다음의 경우 1세대1주택 비과세 등이 **적용되지 않는다**:

1. **미등기양도자산** (무허가건물 등 등기불능 자산 제외)
2. 약정에 의하여 취득자에게 **주택신축판매업자**가 양도하는 경우

## 5. 비과세 vs 감면 비교

| 구분 | 비과세 | 감면 |
|------|--------|------|
| 근거법 | 소득세법 | 조세특례제한법 |
| 신고의무 | 예정신고 **면제** | 예정신고 **필요** |
| 농어촌특별세 | **없음** | 감면세액의 **20%** |
| 양도소득기본공제 | 공제항목에 **미포함** | 과세되므로 **공제 가능** |

## 핵심 암기 포인트

- **1세대1주택 비과세 요건**: 1세대 + 1주택 + 2년 이상 보유(조정지역 거주 추가) + 양도
- **고가주택**: 양도가액 **12억원** 초과 → 초과분만 과세. 과세양도차익 = 양도차익 × (양도가액-12억)/양도가액
- **1세대 구성**: 거주자 + 배우자(사실혼 포함) + 같은 주소 생계 같이 하는 가족
- **배우자 없이 1세대**: 만 30세 이상, 배우자 사망·이혼, 일정 소득 이상
- **일시적 2주택**: 종전 1년 이상 보유 후 신규 취득 → 3년 이내(조정 2년) 종전 양도
- **상속주택 + 일반주택**: 일반주택 먼저 양도 시 비과세 (상속주택 수 제외)
- **혼인합가**: 5년 이내 양도
- **동거봉양합가**: 60세 이상 직계존속, 10년 이내 양도
- **다가구주택**: 1주택 (구분등기 시 각각 1주택)
- **겸용주택**: 주거면적 50% 이상 → 전체 1주택
- **8년 자경농지**: 감면 100%, 연간 2억·5년간 3억 한도
- **비과세 배제**: 미등기양도자산, 주택신축판매업자 양도
- **비과세 vs 감면**: 비과세 = 예정신고 면제·농특세 없음, 감면 = 예정신고 필요·농특세 20%

## OX 퀴즈

**Q1.** 1세대1주택 비과세 요건 중 보유기간은 3년 이상이다. → **X** (2년 이상)

**Q2.** 고가주택 기준은 양도가액 9억원 초과이다. → **X** (12억원 초과)

**Q3.** 일시적 2주택 비과세를 위해서는 신규주택 취득일부터 2년 이내에 종전주택을 양도해야 한다. → **X** (3년 이내. 조정대상지역만 2년)

**Q4.** 혼인합가로 2주택이 된 경우 10년 이내에 1주택 양도 시 비과세이다. → **X** (5년 이내. 10년은 동거봉양합가)

**Q5.** 다가구주택을 구분등기하지 않으면 1주택으로 본다. → **O**

**Q6.** 8년 이상 자경농지 양도 시 양도소득세가 비과세된다. → **X** (비과세가 아닌 '감면'이다 — 조세특례제한법)

**Q7.** 미등기양도자산에도 1세대1주택 비과세가 적용된다. → **X** (비과세 배제)

**Q8.** 비과세 주택 양도 시 예정신고를 해야 한다. → **X** (비과세는 예정신고 면제. 감면은 신고 필요)

## 함정 노트

- ⚠️ **고가주택 기준**: 12억원 (과거 9억원에서 변경) — 구 기준으로 출제하는 함정
- ⚠️ **일시적 2주택 기간**: 일반 3년, 조정지역 2년 — 혼동 주의
- ⚠️ **혼인합가 5년 vs 동거봉양 10년**: 기간 혼동 빈출
- ⚠️ **비과세 vs 감면**: 자경농지는 '비과세'가 아닌 '감면'. 농어촌특별세 유무, 신고 의무 유무가 다름
- ⚠️ **1세대1주택 거주 요건**: 조정대상지역에서 취득한 주택만 2년 거주 요건 추가. 비조정지역은 보유만으로 충분
