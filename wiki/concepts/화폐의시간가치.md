# 화폐의 시간가치 (Time Value of Money)

## 1. 개요

같은 금액이라도 현금흐름 실현 시점에 따라 가치가 다르게 평가되는 것. 부동산투자에서 여러 기간에 걸친 현금흐름을 비교하기 위한 필수 개념.

## 2. 6대 자본환원계수

### 체계

| | 현재가치 계산 | 미래가치 계산 |
|---|---|---|
| **일시불** | 일시불의 현가계수 | 일시불의 내가계수 |
| **연금** | 연금의 현가계수 | 연금의 내가계수 |
| **파생** | 저당상수 (연금현가의 역수) | 감채기금계수 (연금내가의 역수) |

### 각 계수의 의미와 공식

| 계수 | 의미 | 공식 |
|------|------|------|
| **일시불의 내가계수** | 현재 1원 → n년 후 가치 | $(1+r)^n$ |
| **일시불의 현가계수** | n년 후 1원 → 현재 가치 | $\frac{1}{(1+r)^n}$ |
| **연금의 내가계수** | 매년 말 1원 적립 → n년 후 총액 | $\frac{(1+r)^n - 1}{r}$ |
| **감채기금계수** | n년 후 1원 마련 위한 매년 적립액 | $\frac{r}{(1+r)^n - 1}$ |
| **연금의 현가계수** | 매년 말 1원 수취 → 현재가치 합계 | $\frac{(1+r)^n - 1}{r(1+r)^n}$ |
| **저당상수** | 현재 1원 차입 → 매년 상환액 | $\frac{r(1+r)^n}{(1+r)^n - 1}$ |

### 역수 관계

- 일시불의 현가계수 ↔ 일시불의 내가계수 (**역수**)
- 연금의 현가계수 ↔ **저당상수** (**역수**)
- 연금의 내가계수 ↔ **감채기금계수** (**역수**)

## 3. 활용 사례

### 일시불의 현가계수

- "5년 후 받을 1억원의 현재가치는?" → 1억 × 일시불의 현가계수

### 연금의 현가계수

- "매년 1,000만원씩 10년간 받을 임대료의 현재가치 합계는?" → 1,000만원 × 연금의 현가계수

### 저당상수 (Mortgage Constant)

- "1억원 대출 시 매년 상환액은?" → 1억 × 저당상수
- **부채서비스액(DS) = 대출잔금 × 저당상수**
- 저당상수 = 저당이자율 + 감채기금계수

### 감채기금계수 (Sinking Fund Factor)

- "10년 후 1억원 마련을 위해 매년 적립해야 할 금액은?" → 1억 × 감채기금계수

## 4. 주요 관계

- **이자율 상승** → 현가계수 **감소** → 현재가치 **하락**
- **이자율 상승** → 내가계수 **증가** → 미래가치 **상승**
- **기간 증가** → 현가계수 **감소**, 내가계수 **증가**
- 기간이 **무한대** → 연금의 현가계수 = **1/r** (영구연금)

## 5. 영구연금 (Perpetuity)

$$영구연금의\ 현재가치 = \frac{매기\ 수익}{이자율(할인율)} = \frac{NOI}{R}$$

- 감정평가의 **직접자본환원법**과 동일 구조
- 부동산의 영속성 특성과 관련

## 핵심 암기 포인트

- 6대 계수: 일시불(현가·내가), 연금(현가·내가), 파생(저당상수·감채기금계수)
- 일시불의 내가계수: **(1+r)^n** / 현가계수: **1/(1+r)^n**
- 연금의 현가계수 ↔ **저당상수** (역수) / 연금의 내가계수 ↔ **감채기금계수** (역수)
- 저당상수 = **저당이자율 + 감채기금계수**
- DS(부채서비스액) = **대출잔금 × 저당상수**
- 이자율 상승 → 현가계수 **감소** → 현재가치 **하락**
- 이자율 상승 → 내가계수 **증가** → 미래가치 **상승**
- 기간 무한대 → 연금의 현가계수 = **1/r** (영구연금)
- 영구연금의 현재가치 = **NOI / R** = 직접자본환원법과 동일 구조

## OX 퀴즈

**Q1.** 연금의 현가계수의 역수는 감채기금계수이다. → **X** (연금의 현가계수의 역수는 **저당상수**, 감채기금계수는 연금의 **내가**계수의 역수)

**Q2.** 이자율이 상승하면 일시불의 현가계수는 증가한다. → **X** (이자율 상승 → 현가계수 **감소**)

**Q3.** 저당상수는 현재 1원을 차입했을 때 매년 상환해야 할 금액이다. → **O**

**Q4.** 부채서비스액(DS)은 대출잔금에 감채기금계수를 곱하여 산출한다. → **X** (DS = 대출잔금 × **저당상수**)

**Q5.** 기간이 무한대이면 연금의 현가계수는 0에 수렴한다. → **X** (기간 무한대 → 연금의 현가계수 = **1/r**)

**Q6.** 일시불의 현가계수와 일시불의 내가계수는 역수 관계이다. → **O**

**Q7.** 이자율이 상승하면 미래가치는 하락한다. → **X** (이자율 상승 → 내가계수 증가 → 미래가치 **상승**)

## 함정 노트

- ⚠️ "연금의 현가계수 ↔ 감채기금계수" → 연금의 현가계수의 역수는 **저당상수**, 감채기금계수는 연금의 **내가**계수의 역수
- ⚠️ "DS = 대출잔금 × 감채기금계수" → DS = 대출잔금 × **저당상수**
- ⚠️ "이자율↑ → 현재가치↑" → 이자율↑이면 현가계수↓이므로 현재가치 **하락**
- ⚠️ "기간 무한대 → 연금현가계수 = 0" → 기간 무한대이면 **1/r**로 수렴 (0이 아님)
- ⚠️ "저당상수 = 저당이자율" → 저당상수 = 저당이자율 + **감채기금계수**
