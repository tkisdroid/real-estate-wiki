# 부동산 투자분석 기법

부동산 투자분석은 투자의 타당성을 평가하기 위한 기법으로, 화폐의 시간가치 고려 여부에 따라 어림셈법(승수법·비율법)과 할인현금수지분석법(NPV·IRR·PI)으로 나뉜다.

## 비교 정리

### 시간가치 고려 여부

| 미고려 (어림셈법) | 고려 (할인현금수지분석법) |
|------------------|------------------------|
| 승수법 (총소득승수, 순소득승수 등) | **순현재가치법(NPV)** |
| 수익률법 (종합자본환원율, 지분배당률 등) | **내부수익률법(IRR)** |
| 단순회수기간법 | **수익성지수법(PI)** |
| 회계적이익률법 | 현가회수기간법 |

### NPV vs IRR vs PI 비교

| 구분 | NPV | IRR | PI |
|------|-----|-----|-----|
| 채택 기준 | > 0 | > 요구수익률 | > 1 |
| 재투자수익률 | 요구수익률 | IRR | 요구수익률 |
| 상호배타적 우열 | **우월** | 결과 상이 가능 | 규모 차이 시 상이 |
| 복수해 문제 | 없음 | 있음 | 없음 |

### 현금흐름 산출 단계

```
PGI → (-공실+기타) → EGI → (-OE) → NOI → (-DS) → BTCF → (-소득세) → ATCF
```
