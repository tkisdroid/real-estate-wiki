# 경매·공매의 권리분석과 취득알선 실무

## 1. 경매와 공매의 비교

| 구분 | 경매 | 공매 |
|---|---|---|
| 근거법 | 민사집행법 | 국세징수법 등 |
| 실시 기관 | **법원** | 한국자산관리공사(캠코) 등 |
| 목적 | 채권자의 채권 회수 | 체납세금 징수 등 |
| 절차 | 경매신청→개시결정→배당→낙찰 | 공매공고→입찰→낙찰→대금납부 |

## 2. 법인의 겸업 업무 (법 제14조 제1항 제6호)

법인인 개업공인중개사가 할 수 있는 경매·공매 관련 업무:
1. 경매·공매 대상 부동산에 대한 **권리분석**
2. **취득의 알선**
3. **매수신청 또는 입찰신청의 대리**

## 3. 공인중개사의 매수신청대리인 등록

### 근거
- 민사집행법 및 대법원 규칙
- 개업공인중개사는 **매수신청대리인**으로 등록하여 경매 입찰 대리 가능

### 등록 절차
- 관할 법원에 매수신청대리인 등록 신청
- 공인중개사 자격증, 개설등록증 등 첨부

## 4. 경매 절차 개요

| 단계 | 내용 |
|---|---|
| 1. 경매신청 | 채권자가 법원에 신청 |
| 2. 경매개시결정 | 법원이 개시결정, 압류 등기 촉탁 |
| 3. 배당요구종기 | 채권자의 배당요구 마감 |
| 4. 매각기일 공고 | 최저매각가격 결정, 물건명세서 공개 |
| 5. 입찰 | 매수희망자 입찰 |
| 6. 매각허가결정 | 최고가 매수인에 대해 매각 허가 |
| 7. 대금납부 | 매수인이 대금 납부 |
| 8. 소유권이전등기 | 법원의 촉탁에 의한 등기 |
| 9. 배당 | 채권자에 대한 배당 실시 |

## 5. 경매 권리분석 핵심

### 말소기준권리
- 경매로 소멸하는 권리와 인수하는 권리를 구분하는 기준
- 일반적으로 **압류등기, (근)저당권, 담보가등기** 중 가장 먼저 설정된 것

### 소멸하는 권리
- 말소기준권리 이후에 설정된 전세권, 저당권, 임차권 등

### 인수하는 권리
- 말소기준권리 이전에 설정된 용익물권 (지상권, 지역권 등)
- 대항력 있는 임차인 (말소기준권리보다 선순위)
- 법정지상권, 유치권 등

## 6. 매수신청대리 가능 주체 (법 제14조 제2항)

| 주체 | 매수신청대리 가능 |
|---|---|
| 법인인 개업공인중개사 | **O** |
| 공인중개사인 개업공인중개사 | **O** |
| 부칙 개업공인중개사 | **X** (변호사법 위반) |

### 업무 범위 구분
| 업무 | 법원 등록 필요 |
|---|---|
| 권리분석 | **불요** |
| 취득 알선 | **불요** |
| **매수신청 대리** (민사집행법) | **법원에 등록 필요** + 감독 |
| 공매 입찰 대리 (국세징수법) | **불요** |

### 중개업과 겸업의 차이
| 구분 | 중개업 | 겸업(경매대리 등) |
|---|---|---|
| 적용 법률 | **공인중개사법** | 대법원 규칙 등 (공인중개사법 미적용) |
| 인장 사용 | 등록 인장 사용 | 등록 인장 불요 (매수신청대리 등록 시 별도) |
| 보수 | 법정 중개보수 | 대법원 규칙에 의한 수수료 |
| 서명날인 | 법정 의무 | 공인중개사법상 의무 아님 |
