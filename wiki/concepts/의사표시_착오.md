# 착오 (제109조)

## 의의

의사와 표시의 불일치를 **표의자 자신이 모르는** 경우. 표의자가 알았더라면 그러한 의사표시를 하지 않았을 것인 경우이다.

## 취소요건

### 1. 법률행위 내용의 중요부분의 착오일 것
- "중요부분"이란 착오가 없었더라면 그 의사표시를 하지 않았으리라고 인정되는 경우
- **객관적으로** 보통인도 그 착오가 없었더라면 그 의사표시를 하지 않았을 것이라고 인정되어야

### 2. 표의자에게 중대한 과실이 없을 것
- **중대한 과실**: 표의자의 직업·행위의 종류·목적 등에 비추어 보통 요구되는 주의를 **현저히 결한 것**
- 중대한 과실의 **입증책임**: **상대방** (법률행위의 유효를 주장하는 자)
- 예외: 표의자에게 중대한 과실이 있더라도 상대방이 **착오를 알고 이용**한 경우에는 취소 **허용** (신의칙)

## 착오의 유형

### 표시상의 착오 (표시행위의 착오)
- 표시행위 자체에 대한 착오
- 예: 1000만원을 **100만원**으로 잘못 적은 경우
- 중요부분의 착오에 해당하면 취소 가능

### 내용의 착오
- 표시행위의 의미에 대한 착오
- 예: 달러를 엔으로 착각

### 법률의 착오
- 법률의 존재 또는 효과에 대한 오인
- 중요부분의 착오에 해당할 수 있음

### 동기의 착오
- **원칙: 취소 불가** (동기는 법률행위의 내용이 아니므로)
- **예외: 취소 가능** — 다음 요건 충족 시
  1. 동기가 상대방에게 **표시**되었을 것 (명시적·묵시적 모두 가능)
  2. 중요부분의 착오에 해당할 것
  (대판 2000다12259)
- **상대방으로부터 유발된** 동기의 착오: 표시 없이도 취소 가능 (판례)

## 효과

### 취소권 행사 (제109조 제1항)
- 법률행위를 **취소**할 수 있음 (무효가 아님)
- 취소는 **소급효** → 처음부터 무효 (제141조)

### 제3자 보호 (제109조 제2항)
- 취소로 인한 무효는 **선의의 제3자에게 대항 불가**
- 제3자의 과실 유무는 문제되지 않음 (선의만 요구)

### 착오와 손해배상
- 착오 취소 시 상대방에 대한 **손해배상 의무 없음** (민법에 명문규정 없음)
- 다만 계약체결상의 과실 문제는 별도

## 착오와 다른 제도와의 관계

### 착오와 사기
- 사기에 의해 착오에 빠진 경우 → **사기 취소**(제110조)와 **착오 취소**(제109조) 경합 가능
- 어느 것을 주장해도 무방

### 착오와 하자담보책임
- 하자담보책임의 요건에 해당하면서 동시에 착오에도 해당 → 경합 가능

## 핵심 암기 포인트

- **취소요건**: ① 중요부분의 착오 + ② 중대한 과실 없을 것
- **중대한 과실 입증책임**: **상대방** (유효 주장자)
- **동기의 착오**: 원칙 취소 불가, 예외: **표시** + 중요부분이면 취소 가능
- **상대방 유발 동기의 착오**: 표시 없이도 취소 가능
- 중대한 과실 있어도 상대방이 **착오를 알고 이용** → 취소 허용
- **제3자 보호**: 선의의 제3자에게 대항 불가 (선의만, 무과실 불요)

## OX 퀴즈

**Q1.** 동기의 착오는 상대방에게 표시되지 않더라도 항상 취소할 수 있다. → **X** (원칙 취소 불가, 표시되고 중요부분이면 취소 가능)

**Q2.** 착오에 의한 의사표시에서 중대한 과실의 입증책임은 표의자에게 있다. → **X** (상대방, 즉 유효를 주장하는 자에게 있다)

**Q3.** 표의자에게 중대한 과실이 있으면 어떤 경우에도 취소할 수 없다. → **X** (상대방이 착오를 알고 이용한 경우 취소 허용)

**Q4.** 상대방으로부터 유발된 동기의 착오는 표시가 없어도 취소할 수 있다. → **O**

**Q5.** 착오에 의한 의사표시는 무효이다. → **X** (취소할 수 있는 것이지 무효가 아니다)

**Q6.** 착오 취소 시 선의·무과실의 제3자만 보호된다. → **X** (선의만 요구, 무과실 불요)

## 함정 노트

⚠️ "중대한 과실 입증책임 = 표의자" → **상대방**(유효 주장자)에게 입증책임

⚠️ "중대한 과실 = 절대 취소 불가" → 상대방이 **착오를 알고 이용**한 경우 취소 허용

⚠️ "동기의 착오 = 항상 취소 불가" → 상대방에게 **표시**되고 중요부분이면 취소 가능

⚠️ "착오 = 무효" → **취소** (무효가 아님)

⚠️ "동기의 착오 표시 = 명시적이어야" → 명시적·**묵시적** 모두 가능
