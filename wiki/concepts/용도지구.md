# 용도지구

[[용도지역]]의 제한을 **강화하거나 완화**하여 적용함으로써 용도지역의 기능을 증진시키고 경관·안전 등을 도모하기 위해 **도시·군관리계획으로 결정**하는 지역.

## 특징 ([[용도지역]]과의 차이)
- **국지적**으로 지정 (용도지역은 전국토 대상)
- 용도지역 위에 **부가적으로** 지정 (용도지역의 지정을 전제)
- **상호 중복지정 가능** (용도지역은 중복 불가)

## 용도지구의 종류

### 법률에 의한 용도지구 (9종)

| 용도지구 | 목적 | 세분 |
|---------|------|------|
| **경관지구** | 경관 보전·관리·형성 | 자연경관, 시가지경관, 특화경관 |
| **고도지구** | 건축물 높이의 **최고한도** 규제 | - |
| **방화지구** | 화재 위험 예방 | - |
| **방재지구** | 풍수해·산사태 등 재해 예방 | 시가지방재, 자연방재 |
| **보호지구** | 문화재·시설물·생태계 보호 | 역사문화환경보호, 중요시설물보호, 생태계보호 |
| **취락지구** | 취락 정비 | 자연취락, 보호취락, 집단취락 |
| **개발진흥지구** | 개발·정비 촉진 | 주거·산업유통·관광휴양·복합·특정 개발진흥 |
| **특정용도제한지구** | 오염물질·청소년 유해시설 등 입지 제한 | - |
| **복합용도지구** | 복합적 토지이용 촉진 | - |

### 방재지구 의무지정
- **연안침식관리구역**으로 지정된 지역
- 최근 **10년 이내 2회 이상** 동일 재해 발생으로 인명피해 입은 지역

### 복합용도지구 지정 가능 지역
- **일반주거지역**, **일반공업지역**, **계획관리지역**에 한함
- 해당 용도지역 전체 면적의 **1/3 이하** 범위

## 건축제한

| 용도지구 | 건축제한 방식 |
|---------|-------------|
| 경관·방화·방재·보호·특정용도제한·복합용도 | **조례**로 정함 |
| **고도지구** | **도시·군관리계획**으로 정하는 높이 초과 불가 |
| 자연취락·보호취락 | **시행령** 별표로 정함 |
| 집단취락 | 개발제한구역법령에 따름 |
| 개발진흥지구 | 지구단위계획 또는 개발계획에 따름 |

### 자연취락지구 건축 가능 건축물 (**4층 이하**)
- 단독주택, 제1종·제2종 근린생활시설(안마시술소 제외), 운동시설, 농림축수산업용 창고, 동식물관련시설, 교정·국방·군사시설, 방송통신시설, 발전시설

## 시험 핵심 포인트
- 고도지구: 건축물 **최고한도** 규제 (최저한도는 지구단위계획에서 정함)
- 방화지구에 걸치는 건축물: **전부에** 방화지구 규정 적용 (방화벽 구획 시 예외)
- 고도지구에 걸치는 건축물: **전부에** 고도지구 규정 적용

## 핵심 암기 포인트

- **용도지구 9종**: 경관·고도·방화·방재·보호·취락·개발진흥·특정용도제한·복합용도
- **상호 중복지정 가능** (용도지역은 불가)
- **고도지구**: **최고한도** 규제 (최저한도는 지구단위계획)
- **방재지구 의무지정**: 연안침식관리구역, 10년 이내 **2회 이상** 동일 재해로 인명피해
- **복합용도지구**: 일반주거·일반공업·계획관리에 한함, 전체 면적의 **1/3 이하**
- **자연취락지구**: **4층 이하** 건축 가능
- **방화지구 걸침**: 건축물 **전부에** 방화지구 규정 적용 (방화벽 구획 시 예외)
- **고도지구 걸침**: 건축물 **전부에** 고도지구 규정 적용

## OX 퀴즈

**Q1.** 용도지구는 상호 중복하여 지정할 수 없다. → **X** (중복지정 가능, 중복 불가는 용도지역)

**Q2.** 고도지구는 건축물 높이의 최저한도를 규제하기 위해 지정한다. → **X** (최고한도, 최저한도는 지구단위계획)

**Q3.** 방화지구에 걸치는 건축물에는 그 전부에 대하여 방화지구의 규정을 적용한다. → **O** (방화벽 구획 시 예외)

**Q4.** 복합용도지구는 준주거지역에 지정할 수 있다. → **X** (일반주거·일반공업·계획관리에 한함)

**Q5.** 자연취락지구에서는 5층 이하의 건축물을 건축할 수 있다. → **X** (4층 이하)

**Q6.** 방재지구는 최근 10년 이내 3회 이상 동일 재해가 발생한 지역에 의무적으로 지정한다. → **X** (2회 이상)

**Q7.** 복합용도지구는 해당 용도지역 전체 면적의 1/2 이하 범위에서 지정할 수 있다. → **X** (1/3 이하)

## 함정 노트

- ⚠️ **최고한도 vs 최저한도**: 고도지구는 **최고한도**, 지구단위계획에서 **최저한도** 정함. 바꿔 출제 빈출
- ⚠️ **중복지정**: 용도지역은 중복 **불가**, 용도지구는 중복 **가능**. 용도구역과도 구별
- ⚠️ **복합용도지구 지정 지역**: 일반주거·일반공업·계획관리 **3개만**. 준주거·준공업 등은 불가
- ⚠️ **1/3 vs 1/2**: 복합용도지구 면적은 전체의 **1/3** 이하
- ⚠️ **방화지구 vs 고도지구 걸침**: 둘 다 건축물 **전부**에 적용. 일부에만 적용한다는 선지는 오답
- ⚠️ **4층 이하**: 자연취락지구 건축 가능 층수. "3층"이나 "5층"은 오답
