# 감정평가 - 가격제원칙

## 1. 가격제원칙의 의의

**가격제원칙**이란 부동산의 가격이 어떻게 형성되고 유지되는가에 관한 **법칙성**을 추출하여 부동산평가활동의 **지침**으로 삼으려는 하나의 행위기준을 말한다.

## 2. 가격제원칙의 체계

가격제원칙은 **최유효이용의 원칙**을 핵심으로 하여 다음과 같이 체계화된다.

```
┌─────────── 토대(기본)가 되는 원칙 ───────────┐
│  변동의 원칙  /  예측의 원칙                    │
├─────────── 내부측면의 원칙 ──────────────────┤
│  균형의 원칙  /  기여의 원칙  /  수익배분의 원칙  │
│  수익체증·체감의 원칙                           │
├─────────── 외부측면의 원칙 ──────────────────┤
│  적합의 원칙  /  경쟁의 원칙                    │
├──── 간접적으로 관련되는 원칙 ────────────────┤
│  대체의 원칙  /  수요·공급의 원칙  /  기회비용의 원칙 │
└──────────────────────────────────────────┘
          ↑ 모든 원칙의 중심 ↑
        ▶ 최유효이용의 원칙 ◀
```

| 분류 | 원칙 | 판정기준 |
|------|------|---------|
| 토대가 되는 원칙 | 변동, 예측 | - |
| 내부측면 | 균형, 기여, 수익배분, 수익체증·체감 | **내부** 판정기준 |
| 외부측면 | 적합, 경쟁 | **외부** 판정기준 |
| 간접 관련 | 대체, 수요·공급, 기회비용 | - |

## 3. 토대(기본)가 되는 원칙

### (1) 변동의 원칙 (Principle of Change)

| 항목 | 내용 |
|------|------|
| **정의** | 부동산가격은 가치형성요인이 변함에 따라 **변동과정에서 형성**된다는 원칙 |
| **이론적 근거** | **시점수정** 및 **기준시점 확정**의 근거 |
| **관련 방법** | 기준시점과 관련 |

### (2) 예측의 원칙 (Principle of Anticipation)

| 항목 | 내용 |
|------|------|
| **정의** | 감정평가활동에서 가치형성요인의 추이나 동향에 대한 **예측을 해야 한다**는 원칙 |
| **핵심** | 부동산의 가치 = 장래 기대되는 편익을 **현재가치로 환원**한 값 |
| **관련 방법** | **수익환원법**의 이론적 근거 |

## 4. 내부측면의 원칙

### (3) 균형의 원칙 (Principle of Balance) — 비례의 원칙

| 항목 | 내용 |
|------|------|
| **정의** | 부동산이 최고의 가치를 구현하기 위해서는 **생산요소(내부구성요소)의 결합비율**이 적절한 균형을 이루어야 한다는 원칙 |
| **이론적 근거** | **기능적 감가**의 근거 |
| **관련 분석** | **개별분석**과 관련이 깊다 |

> 균형의 원칙은 **내부적** 관계의 원칙이다.

### (4) 기여의 원칙 (Principle of Contribution)

| 항목 | 내용 |
|------|------|
| **정의** | 부동산의 가격은 각 구성요소가 가치에 **기여하는 기여도(공헌도, 생산성)**의 영향을 받아 결정된다는 원칙 |
| **핵심 주의** | 가격은 구성요소의 **기여도**의 합이지 **생산비**의 합이 아니다 |
| **관련 방법** | **배분법**의 근거, **추가투자의 적정성** 판정에 유용 |

### (5) 수익배분의 원칙 (Principle of Return Distribution) — 잉여생산성의 원칙

| 항목 | 내용 |
|------|------|
| **정의** | 토지의 수익은 자본·노동·경영 등 각 생산요소에 배분되고 **남는 잔여수익**이 토지에 배분된다는 원칙 |
| **관련 방법** | **토지잔여법**, **수익분석법**의 이론적 근거 |

### (6) 수익체증·체감의 원칙

| 항목 | 내용 |
|------|------|
| **정의** | 기여의 원칙을 **입체적 개념**(한계효용, 층수의 결정 등)으로 적용한 것 |
| **핵심** | 추가투자에 따른 수익이 체증하다가 일정 시점 이후 **체감**한다는 원칙 |

## 5. 외부측면의 원칙

### (7) 적합의 원칙 (Principle of Conformity)

| 항목 | 내용 |
|------|------|
| **정의** | 부동산의 효용(유용성)이 최고로 발휘되기 위해서는 그 이용방법이 **그 부동산이 속한 지역의 환경에 적합**해야 한다는 원칙 |
| **이론적 근거** | **경제적 감가**의 근거 |
| **관련 분석** | **지역분석**과 관련이 깊다 |

> 적합의 원칙은 **외부적** 관계의 원칙이다.

### (8) 경쟁의 원칙 (Principle of Competition)

| 항목 | 내용 |
|------|------|
| **정의** | 부동산가격은 **초과이윤**을 얻기 위한 경쟁관계에 의해 형성된다는 원칙 |
| **핵심** | 초과이윤은 경쟁을 야기하며, 경쟁은 결국 초과이윤을 **감소 또는 소멸**시킨다 |
| **시사점** | 대상부동산의 가격을 실제 이상으로 **과대평가해서는 안 된다** |

## 6. 간접적으로 관련되는 원칙

### (9) 대체의 원칙 (Principle of Substitution)

| 항목 | 내용 |
|------|------|
| **정의** | 대상부동산의 가격은 **대체성을 지닌 유사한 부동산** 또는 재화 등의 가치와 상호작용과정에서 형성된다는 원칙 |
| **핵심** | 효용이 동일하다면 **가격이 싼 것**을 선택 |
| **관련 방법** | **거래사례비교법**의 이론적 근거 |

### (10) 수요·공급의 원칙 (Principle of Demand & Supply)

| 항목 | 내용 |
|------|------|
| **정의** | 부동산의 가격도 기본적으로 **수요와 공급과의 상호관계**에 의하여 결정된다는 원칙 |
| **관련** | 피드백(feed-back) 원리에 의한 **부동산가격의 이중성**의 이론적 근거 |

## 7. 최유효이용의 원칙 — 가격제원칙의 핵심

**최유효이용**이란 합법적이고, 물리적으로 가능하고, 재정적으로 타당하고, 최대의 생산성을 올릴 수 있는 이용방법을 말한다.

| 항목 | 내용 |
|------|------|
| **위치** | 모든 가격제원칙의 **중심·귀결점** |
| **내부 판정기준** | 균형의 원칙, 기여의 원칙 |
| **외부 판정기준** | 적합의 원칙, 경쟁의 원칙 |
| **지역분석과의 관계** | 지역분석은 대상지역 내 토지의 최유효이용을 판정 → **적합의 원칙**과 관련 |
| **개별분석과의 관계** | 개별분석은 대상부동산의 최유효이용을 판정 → **균형의 원칙**과 관련 |

## 8. 원칙별 관련 감정평가방법·감가요인 정리

| 원칙 | 관련 감정평가방법 | 관련 감가요인 | 관련 분석 |
|------|----------------|-------------|---------|
| **변동** | 시점수정, 기준시점 확정 | - | - |
| **예측** | 수익환원법 | - | - |
| **균형** | - | **기능적** 감가 | **개별**분석 |
| **기여** | 배분법, 추가투자 판정 | - | - |
| **수익배분** | 토지잔여법, 수익분석법 | - | - |
| **적합** | - | **경제적** 감가 | **지역**분석 |
| **경쟁** | - | - | - |
| **대체** | 거래사례비교법 | - | - |
| **수요·공급** | - | - | - |

## 핵심 암기 포인트

- 가격제원칙의 핵심: **최유효이용의 원칙**
- 균형의 원칙 → **기능적 감가** → **개별분석** (내부)
- 적합의 원칙 → **경제적 감가** → **지역분석** (외부)
- 예측의 원칙 → **수익환원법**
- 대체의 원칙 → **거래사례비교법**
- 수익배분의 원칙 → **토지잔여법**
- 기여의 원칙: 가격 = 생산비의 합이 아니라 **기여도의 합**
- 경쟁의 원칙: 초과이윤 → 경쟁 야기 → 초과이윤 **소멸**

## OX 퀴즈

**Q1.** 가격제원칙 중 가장 핵심이 되는 원칙은 대체의 원칙이다. → **X** (최유효이용의 원칙)

**Q2.** 변동의 원칙은 감정평가시 기준시점과 관련이 있다. → **O**

**Q3.** 예측의 원칙은 부동산의 가치를 장래 기대되는 편익을 현재가치로 환원한 값이라고 정의하는 것과 관련이 깊다. → **O**

**Q4.** 균형의 원칙은 경제적 감가의 이론적 근거가 된다. → **X** (기능적 감가의 근거, 경제적 감가는 적합의 원칙)

**Q5.** 적합의 원칙은 개별분석과 관련이 깊다. → **X** (지역분석과 관련, 개별분석은 균형의 원칙)

**Q6.** 기여의 원칙은 부동산의 가치는 각 구성부분의 생산비를 전부 합한 것이라는 원칙이다. → **X** (기여도의 합이지 생산비의 합이 아니다)

**Q7.** 수익배분의 원칙은 토지잔여법의 성립근거가 된다. → **O**

**Q8.** 경쟁의 원칙은 초과이윤은 경쟁을 야기하며 경쟁은 결국 초과이윤을 감소 또는 소멸시킨다는 원칙이다. → **O**

**Q9.** 대체의 원칙은 원가법의 이론적 근거가 된다. → **X** (거래사례비교법의 이론적 근거)

**Q10.** 수익체증·체감의 원칙은 부동산의 가치는 수요와 공급의 상호작용에 의해 결정된다는 원칙이다. → **X** (수요·공급의 원칙에 대한 설명이며, 수익체증·체감의 원칙은 기여의 원칙을 입체적으로 적용한 것)

## 함정 노트

- ⚠️ **균형 vs 적합**: 균형 = **내부**구성요소 간의 균형 = **기능적** 감가 = **개별**분석 / 적합 = **외부**환경과의 관계 = **경제적** 감가 = **지역**분석. 내외부·감가·분석이 **교차 출제**됨
- ⚠️ **기여의 원칙 ≠ 생산비**: "가격은 각 구성요소의 **기여도**의 합이지 **생산비**의 합이 아니다" — 시험 단골 함정
- ⚠️ **수익배분 vs 수요·공급**: "부동산의 가치는 수요와 공급의 상호작용에 의해 결정된다"는 수요·공급의 원칙이지 수익배분의 원칙이 아니다
- ⚠️ **수익체증·체감 vs 경쟁의 원칙**: "초과이윤 → 경쟁 → 소멸"은 **경쟁**의 원칙이지 수익체증·체감의 원칙이 아니다
- ⚠️ **추가투자 적정성 판정**: **기여**의 원칙이지 예측의 원칙이 아니다 (시험 오답 유도)
