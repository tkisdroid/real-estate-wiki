# 부동산 개발·관리론

## 1. 부동산관리의 기본개념

- 부동산관리란 소유자의 목적 달성을 위해 부동산의 성질을 변경시키지 않는 범위 내에서 **최유효이용**을 할 수 있도록 취득·보존·이용·개량 등 운용과 관련되는 일체의 행위
- **처분의 개념은 포함되지 않는다** (시험 출제 포인트)

### 복합개념의 관리 분류

| 분류 | 다른 명칭 | 내용 |
|------|----------|------|
| **법률적 관리** | 보존관리 | 권리분석·조정, 보존등기, 지목변경, 계약관리 |
| **경제적 관리** | 경영관리 | 인력관리, 회계(수지)관리, 손익분기점 파악 |
| **기술적 관리** | 유지관리(**협의의 관리**) | 위생관리, 설비관리, 보안관리, 보전관리 |

> **주의**: 법률관리는 "포트폴리오·재무효율성"이 아니다 (자산관리와 혼동 금지). 건물부지 부적응 개선 = **기술적 관리** (경제적 관리 아님).

### 관리의 유형 (계층별)

| 유형 | 영문 | 내용 | 수준 |
|------|------|------|------|
| **자산관리(AM)** | Asset Management | 포트폴리오 관리, 투자위험 관리, 매입·매각, 자금조달, PF | **최상위**(적극적) |
| **재산관리(PM)** | Property Management | 수익목표 수립, 예산·비용통제, 임대차 활동, 마케팅 | 중간 |
| **시설관리(FM)** | Facility Management | 설비운전·보수, 에너지관리, 방범·방재, 청소 | 하위(**소극적**) |

### 관리방식

| 방식 | 적합 대상 | 장점 | 단점 |
|------|----------|------|------|
| **자가관리** | 소규모(단독주택 등) | 기밀유지, 의사결정 신속, 관리비 절약, 종합 운영 | 전문성 결여, 비효율적 |
| **위탁관리** | 대형건물 | 전문적·합리적 관리, 타성화 방지, 소유자 본업 전념, **소유와 경영 분리** | 관리비 부담, 기밀유지 불안 |
| **혼합관리** | 과도기 | 필요부분만 선별 위탁, 전문성 활용 | **책임소재 불분명** |

### 임대차 유형

| 유형 | 내용 | 적용 부동산 |
|------|------|------------|
| **총임대차**(gross lease) | 임차인이 총임대료 지불, 임대인이 운영비용(세금·보험) 부담 | **주거용** |
| **순임대차**(net lease) | 순수임대료 + 영업경비 별도 부담 | **공업용**(3차 순임대차) |
| **비율임대차**(percentage lease) | 임차인 총수입의 일정 비율을 임대료로 지불 | **상업용** |

**순임대차 세부 구분**:
- **1차 순임대차**: 순임대료 + 편익시설비용 + 부동산세금 + 특별부과금
- **2차 순임대차**: 1차 + **보험료**
- **3차 순임대차**: 1차 + 보험료 + **유지수선비**

**임차인 선정기준**: 주거용(**유대성**), 상업용(**가능매상고**), 사무용·공업용(**적합성**)

### 유지활동

| 유형 | 내용 | 비용 |
|------|------|------|
| **일상적 유지활동** | 청소·소독 등 정기적 활동 | - |
| **대응적 유지활동** | 문제 발생 후 **사후적** 처리 | 예방적보다 **비용 큼** |
| **예방적 유지활동** | 사전 유지계획에 따른 **사전적** 점검·진단 (**가장 중요**) | - |

### 보험

| 보험 | 내용 |
|------|------|
| **손해보험** | 화재·홍수·재해 등으로 **대상부동산**이 입은 손해 보상 |
| **책임보험** | 임차자·고객·방문자 등이 입은 손해에 대해 관리자·소유자가 책임지는 부분 보상 |
| **임대료손실보험**(업무장애보험) | 사고 시 공사 중 임대료 수입 상실분 보전 |

## 2. 부동산마케팅

### 마케팅 믹스 (4P)

| 요소 | 내용 |
|------|------|
| **Product(제품)** | 부동산 상품의 특성·차별화 |
| **Price(가격)** | 가격전략 |
| **Place(유통)** | 입지, 유통채널 |
| **Promotion(촉진)** | 광고, 판매촉진, 홍보 |

### STP 전략 (시장점유 마케팅 전략)

1. **S**(Segmentation): 시장세분화 - 이질적 전체시장을 **동질적** 세분시장으로 분할
2. **T**(Targeting): 표적시장 선정 - 세분화된 집단에서 경쟁상황·능력 고려하여 선택
3. **P**(Positioning): 제품차별화 - 경쟁자들 사이에서 자신의 상품 위치 결정

> **주의**: STP에서 P = **Positioning(차별화)**이지 Promotion(촉진)이 아니다

### 마케팅 전략 3분류

| 전략 | 관점 | 핵심 |
|------|------|------|
| **시장점유 마케팅** | **공급자** 중심 | STP전략 + 4P Mix전략 |
| **고객점유 마케팅** | **소비자** 중심 | **AIDA원리**: 주의(Attention)→관심(Interest)→**욕망(Desire)**→행동(Action) |
| **관계 마케팅** | 공급자·소비자 **상호작용** | **장기적·지속적** 관계 유지, 브랜드 전략 |

> **시험 함정**: AIDA에서 D = **Desire(욕망)**이지 Decision(결정)이 아니다

### 가격정책 (4P 중 Price)

| 정책 | 특징 |
|------|------|
| **고가정책**(스키밍) | 초기 고가→점차 하락, 빠른 자금회수, 우수고객 파악 |
| **저가정책**(침투가격) | 시장침투, 경쟁자 진입 차단, 장기 시장점유율 확대 |
| **시가정책** | 경쟁자와 동일한 가격수준 또는 추종 |
| **차별가격정책** | 용도·위치·방위·층·시간·지역에 따라 가격차별화 |

### 촉진전략 (4P 중 Promotion)

| 수단 | 특징 |
|------|------|
| **광고**(advertising) | **유료**, 대중매체 이용, **비인적** 커뮤니케이션 |
| **홍보**(publicity) | **무료**(대금 미지불), 뉴스·기사 형태 보도 |
| **인적판매** | 예상고객과 **직접 대면** 대화를 통한 판매 |
| **판매촉진** | **비상례적·단기적** 유인수단 (경품, 할인, 전시회 등) |

## 3. 부동산이용

### 집약적 토지이용

- 토지의 **부증성**으로 인해 물리적 공급이 어려우므로 집약적 이용 요구
- 건물의 고층화, 지하공간 활용 등

### 직주분리와 직주접근

| 구분 | 내용 |
|------|------|
| **직주분리** | 직장과 주거지가 분리되는 현상 |
| **직주접근** | 직장과 주거지를 가까이 하려는 정책 |

### 도시스프롤 현상

- 도시가 외곽으로 **무질서하게 확산**되는 현상
- **지가고**, 교통발달이 원인
- 인프라 비용 증가, 자연환경 훼손

### 침입적 토지이용

기존 토지이용과 다른 용도가 진입하는 현상.

## 4. 부동산개발

### 기본개념

- **부동산개발**: 토지의 용도를 변경하거나 건물을 신축·개축하여 가치를 창출하는 행위
- 개발주체: 공공개발, 민간개발, 민관합동개발

### 개발의 위험

| 위험 | 내용 |
|------|------|
| **시장위험** | 경기변동, 수요·공급 변화 |
| **비용위험** | 건설비 초과, 금리 변동 |
| **시간위험** | 공사 지연, 인허가 지연 |
| **법률적 위험** | 규제 변화, 인허가 문제 |

### 타당성 분석

| 분석 유형 | 내용 |
|-----------|------|
| **시장분석** | 수요·공급 분석, 흡수율 추정 |
| **재무적 타당성 분석** | 투자수익률, NPV, IRR |
| **법률적 타당성** | 관련 법규, 인허가 가능성 |
| **물리적(기술적) 타당성** | 지반, 환경, 기반시설 |

### 개발 유형

| 유형 | 내용 |
|------|------|
| **신개발** | 미개발지에 새로운 개발 |
| **재개발** | 기존 시가지의 정비·개선 |
| **리모델링** | 기존 건축물의 증축·개축·대수선 |

## 핵심 암기 포인트

- 관리 계층: **AM(자산관리, 최상위)** > **PM(재산관리, 중간)** > **FM(시설관리, 하위)**
- AM: 포트폴리오·투자의사결정 / PM: 임대차·수입지출·마케팅 / FM: 물리적 유지보수
- 마케팅 믹스 4P: **Product, Price, Place, Promotion**
- STP 전략: **S**(시장세분화) → **T**(표적시장) → **P**(포지셔닝)
- 도시스프롤: 도시 외곽으로 **무질서한 확산**, 원인은 지가고·교통발달
- 타당성 분석 4종: **시장분석, 재무적, 법률적, 물리적(기술적)** 타당성
- 개발 유형: **신개발**(미개발지) / **재개발**(기존 시가지) / **리모델링**(증축·개축·대수선)
- 집약적 토지이용 근거: 토지의 **부증성**

## OX 퀴즈

**Q1.** 자산관리(AM)는 건물의 물리적 유지·보수를 담당하는 하위 관리이다. → **X** (AM은 **최상위** 관리(포트폴리오·투자의사결정), 물리적 유지보수는 FM)

**Q2.** 마케팅 믹스 4P에는 Product, Price, Place, People이 포함된다. → **X** (4P는 Product, Price, Place, **Promotion**)

**Q3.** 도시스프롤 현상은 도시가 외곽으로 무질서하게 확산되는 것이다. → **O**

**Q4.** 재산관리(PM)는 자산관리(AM)보다 상위 개념이다. → **X** (AM이 **최상위**, PM은 중간)

**Q5.** 리모델링은 미개발지에 새로운 개발을 하는 것이다. → **X** (리모델링은 기존 건축물의 **증축·개축·대수선**, 미개발지 개발은 신개발)

**Q6.** STP 전략에서 T는 Targeting(표적시장 선정)을 의미한다. → **O**

**Q7.** 직주분리는 직장과 주거지를 가까이 하려는 정책이다. → **X** (직주**분리**는 직장과 주거지가 분리되는 현상, 가까이 하려는 것은 직주**접근**)

## 함정 노트

- ⚠️ "AM = 시설관리" → AM은 **자산관리(최상위)**, 시설관리는 **FM(하위)**
- ⚠️ "4P의 P = People" → 4번째 P는 **Promotion**(촉진), People은 서비스 마케팅 7P에 해당
- ⚠️ "PM > AM" → AM(자산관리)이 **최상위**, PM(재산관리)은 중간, FM(시설관리)은 하위
- ⚠️ "직주분리 = 직주접근" → 직주분리(현상)와 직주접근(정책)은 **반대** 개념
- ⚠️ "재개발 = 신개발" → 재개발은 **기존 시가지** 정비, 신개발은 **미개발지** 개발
