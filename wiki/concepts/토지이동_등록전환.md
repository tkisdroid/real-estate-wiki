# 토지이동 - 등록전환

## 1. 의의

등록전환이란 **임야대장 및 임야도**에 등록된 토지를 **토지대장 및 지적도**에 옮겨 등록하는 것을 말한다.

> 방향: **임야대장 → 토지대장** (토지대장 → 임야대장의 역방향도 개념적으로 포함되나 실무상 임야→토지가 대부분)

## 2. 대상

### 원칙적 대상
산지관리법, 건축법 등 관계법령에 따른 **토지의 형질변경** 또는 **건축물의 사용승인** 등으로 인하여 **지목을 변경하여야 할 토지**

### 지목변경 없이 등록전환 가능한 경우 (3가지)

| 번호 | 사유 |
|------|------|
| 1 | 대부분의 토지가 등록전환되어 **나머지를 임야도에 존치하는 것이 불합리**한 경우 |
| 2 | 임야도에 등록된 토지가 사실상 형질변경되었으나 **지목변경을 할 수 없는** 경우 |
| 3 | **도시·군관리계획선**에 따라 토지를 분할하는 경우 |

### 등록전환 제외
- 도시개발사업 등으로 **지적확정측량**을 하려는 지역 안에 임야도를 갖춘 토지 → 등록전환 **하지 않을 수 있음**

## 3. 신청

- **신청권자**: 토지소유자
- **신청기한**: 사유 발생일부터 **60일 이내** 지적소관청에 신청

## 4. 면적 결정 (핵심)

| 항목 | 내용 |
|------|------|
| **면적 결정 원칙** | 토지대장에 등록하는 면적은 **등록전환측량**의 결과에 의함 |
| **임야대장 면적** | 그대로 정리할 수 **없음** (반드시 측량 필요) |
| **허용범위 이내** | 등록전환될 면적을 **등록전환면적**으로 결정 |
| **허용범위 초과** | 임야대장의 면적 또는 임야도의 경계를 지적소관청이 **직권으로 정정** |

### 면적 오차 허용범위 산식
```
A = 0.026√(M²F)
A: 오차허용면적, M: 임야도 축척분모, F: 등록전환될 면적
```

## 5. 지번부여

- 신규등록과 동일: 인접토지의 **본번에 부번**을 붙여서 부여
- 해당 지번부여지역에서 인접토지의 본번이 없으면 **최종 본번 다음 순번**

## 6. 처리절차

| 절차 | 내용 |
|------|------|
| 측량 | 등록전환측량 실시 (필수) |
| 대장 정리 | 토지대장에 새로 등록, 임야대장에서 말소 |
| **등기촉탁** | 등기할 필요가 있는 경우 관할 등기소에 촉탁 |

## 핵심 암기 포인트

- 방향: **임야대장·임야도 → 토지대장·지적도**
- 임야대장 면적 그대로 사용 **불가** → 반드시 **등록전환측량** 필요
- 지목변경 없이 등록전환 가능한 3가지 사유 암기
- 면적 오차 허용범위 초과 시 → 지적소관청 **직권 정정**
- 신청기한: **60일** 이내

## OX 퀴즈

**Q1.** 등록전환 시 임야대장의 면적을 그대로 토지대장에 등록할 수 있다. → **X** (등록전환측량 결과에 의함)

**Q2.** 등록전환은 토지대장에서 임야대장으로 옮겨 등록하는 것이다. → **X** (임야대장에서 토지대장으로)

**Q3.** 도시·군관리계획선에 따라 분할하는 경우 지목변경 없이 등록전환이 가능하다. → **O**

**Q4.** 등록전환 시 등기촉탁이 가능하다. → **O** (신규등록과 달리 등기부가 이미 있으므로)

**Q5.** 등록전환의 신청기한은 90일 이내이다. → **X** (60일 이내)

## 함정 노트

- **신규등록 vs 등록전환**: 신규등록은 등기촉탁 **불가**, 등록전환은 등기촉탁 **가능**. 등기부 존재 여부에 따른 차이
- **임야대장 면적 사용 불가**: "임야대장 면적=토지대장 면적"으로 그대로 이기하면 안 됨. 반드시 측량 필요
- **허용범위 초과**: 지적소관청이 **직권**으로 정정 (토지소유자 신청이 아님)
- **지목변경 없는 등록전환**: 3가지 사유를 구별하여 암기. 특히 "나머지 토지 존치가 불합리한 경우"가 자주 출제
