# 양도소득세 (허브 페이지)

양도소득세는 토지·건물 등 자산의 **양도**로 발생하는 소득에 대하여 부과하는 **국세**이다. 매년 **4~6문항** 출제되며, 계산문제가 포함되는 최고 난이도 과목이다.

## 1. 특징

- **국세**, 보통세, 직접세
- **신고납부** (예정신고 + 확정신고)
- **개인**에게만 과세 (법인은 법인세로 과세)
- 실현된 소득에 대하여 과세 (**양도시점**)
- **초과누진세율** 적용 (8단계)
- 양도차손은 다른 소득과 **통산 불가**

## 2. 과세대상

| 분류 | 과세대상 |
|------|----------|
| **토지·건물** | 모든 토지와 건물 |
| **부동산에 관한 권리** | 지상권, 전세권, 등기된 부동산임차권 |
| **부동산을 취득할 수 있는 권리** | 조합원입주권, 분양권 등 |
| **기타 자산** | 사업용 고정자산과 함께 양도하는 영업권, 특정시설물 이용권 등 |
| **주식 등** | 상장·비상장 주식 |

## 3. 양도의 개념

**양도**란 자산에 대한 등기·등록에 관계없이 매도, 교환, 법인에 대한 현물출자 등으로 그 자산을 **유상으로 사전 이전**하는 것을 말한다 (소득세법 제88조).

### 양도에 해당하는 경우

- 매매, 교환, 현물출자
- 대물변제, 수용, 공매, 경매
- 부담부증여 시 채무인수액에 상당하는 부분

### 양도에 해당하지 않는 경우

- 신탁재산의 위탁자→수탁자 이전
- 환지처분으로 지목·면적 변경 없이 교체
- 도시개발법에 따른 환지
- 농지의 교환·분합 (비과세 요건 충족 시)

## 4. 양도 및 취득시기

### 원칙

| 구분 | 양도(취득)시기 |
|------|---------------|
| 대금청산일 | **잔금지급일** |
| 대금청산일 불분명 | **등기접수일** |
| 대금청산 전 소유권이전등기 | **등기접수일** |

### 특례

| 구분 | 시기 |
|------|------|
| 장기할부 | 소유권이전등기접수일, 인도일, 사용수익일 중 **빠른 날** |
| 자기건설 | **사용검사필증 교부일** (임시사용승인일) |
| 상속 | **상속개시일** |
| 증여 | **증여등기 접수일** |
| 수용 | **대금을 청산한 날** (수용개시일) |

## 5. 비과세

### 1세대1주택 비과세 (소득세법 제89조)

**요건**: 1세대가 1주택을 **2년 이상 보유** (조정대상지역은 2년 이상 **거주** 요건 추가)하고 양도하는 경우

> **고가주택** (양도가액 **12억원 초과**)은 12억원 초과분에 대해서만 과세

**1세대의 범위**: 거주자 + 배우자 + 같은 주소에서 생계를 같이 하는 가족

**일시적 2주택 비과세**: 종전 주택 취득 후 1년 이상 경과하여 신규주택 취득 → 신규주택 취득일부터 **3년 이내** 종전주택 양도 시 비과세 (조정대상지역 내 취득은 **2년** 이내)

### 기타 비과세

- **농지의 교환·분합**: 경작 농지의 교환·분합으로 발생하는 소득
- **8년 이상 자경농지**: 양도소득세 감면 (조세특례제한법)

## 6. 양도소득세 계산구조

```
양도가액
- 취득가액
- 필요경비 (자본적지출, 양도비용)
─────────────────────────
= 양도차익
- 장기보유특별공제
─────────────────────────
= 양도소득금액
- 양도소득기본공제 (연 250만원)
─────────────────────────
= 양도소득과세표준
× 세율
─────────────────────────
= 양도소득 산출세액
```

## 7. 양도차익의 산정

### 양도가액·취득가액

**실지거래가액** 원칙 (2007년 이후 전면 실거래가 과세)

### 필요경비

| 구분 | 내용 |
|------|------|
| **자본적지출액** | 용도변경·개량·이용편의 위한 지출 (증축, 방수공사 등) |
| **양도비** | 중개수수료, 공증비, 인지대, 양도세 신고서 작성비용 등 |

### 기준시가에 의한 양도차익 (예외)

실지거래가액을 확인할 수 없는 경우: **기준시가(환산취득가액)** 적용

**개산공제율** (실거래가 확인 불가 시):
- 토지·건물: 취득당시 기준시가의 **3%**
- 지상권·전세권·등기된 임차권: **7%**
- 미등기양도자산: 토지·건물 **0.3%**, 지상권·전세권 **1%**

## 8. 장기보유특별공제

### 토지·건물 (3년 이상 보유)

| 보유기간 | 공제율 |
|----------|--------|
| 3년 이상 ~ 4년 미만 | **6%** |
| 4년 이상 ~ 5년 미만 | **8%** |
| 5년 이상 ~ 6년 미만 | **10%** |
| 6년 이상 ~ 7년 미만 | **12%** |
| ... (매년 2%씩 증가) | |
| 15년 이상 | **30%** (한도) |

### 1세대1주택 (고가주택, 2년 이상 보유·거주)

**보유기간**: 매년 **4%** (최대 40%, 10년 이상)
**거주기간**: 매년 **4%** (최대 40%, 10년 이상)
→ **합계 최대 80%**

### 장기보유특별공제 적용 제외

- **미등기양도자산**
- 1년 미만 보유 자산
- 조정대상지역 내 다주택자 중과세 적용 주택 (한시 배제 기간 제외)

## 9. 양도소득기본공제

- 연간 **250만원** (거주자 1인당)
- 미등기양도자산에는 **적용 불가**
- 부동산 등과 주식 등 **소득별로 각각** 250만원 공제

## 10. 세율

### 기본세율 (2년 이상 보유, 8단계 초과누진)

| 과세표준 | 세율 |
|----------|------|
| 1,400만원 이하 | **6%** |
| 1,400만원 ~ 5,000만원 | 84만원 + 초과액의 **15%** |
| 5,000만원 ~ 8,800만원 | 624만원 + 초과액의 **24%** |
| 8,800만원 ~ 1억5천만원 | 1,536만원 + 초과액의 **35%** |
| 1억5천만원 ~ 3억원 | 3,706만원 + 초과액의 **38%** |
| 3억원 ~ 5억원 | 9,406만원 + 초과액의 **40%** |
| 5억원 ~ 10억원 | 1억7,406만원 + 초과액의 **42%** |
| 10억원 초과 | 3억8,406만원 + 초과액의 **45%** |

### 단기보유·미등기 세율

| 자산 구분 | 보유기간 | 세율 |
|-----------|----------|------|
| **주택·조합원입주권** | 1년 미만 | **70%** |
| | 1~2년 미만 | **60%** |
| | 2년 이상 | 기본세율 (6~45%) |
| **분양권** | 1년 미만 | **70%** |
| | 1년 이상 | **60%** |
| **미등기양도자산** | 기간 무관 | **70%** |
| **비사업용토지** (2년 이상) | | 기본세율 **+ 10%** |

### 조정대상지역 내 다주택 중과세

| 구분 | 세율 |
|------|------|
| 1세대 **2주택** | 기본세율 **+ 20%** |
| 1세대 **3주택 이상** | 기본세율 **+ 30%** |

> **한시적 중과배제**: 2022.5.10 ~ 2024.5.9 양도분은 기본세율 적용 + 장기보유특별공제 허용

## 11. 미등기양도자산

취득에 관한 등기를 하지 않고 양도하는 토지·건물·부동산 관련 권리

**세제상 불이익:**
1. 비과세·감면 **배제**
2. 장기보유특별공제 **배제**
3. 양도소득기본공제 **배제**
4. 세율 **70%** 중과
5. 필요경비 개산공제 **0.3%** (일반 3%)

**미등기 제외 자산** (미등기로 보지 않는 경우):
- 장기할부조건으로 등기 불가능한 자산
- 법률·법원결정에 의해 등기 불가능한 자산
- 1세대1주택 무허가건물 등

## 12. 양도소득금액 계산의 특례

### 배우자·직계존비속 증여재산 이월과세

- **10년 이내** 증여받은 자산을 양도하는 경우
- **증여자의 취득가액**으로 양도차익 계산
- 납세의무자: **수증자** (양도소득세에 대한 연대납세의무 없음)
- 증여세 상당액은 **필요경비에 산입**
- 보유기간: **증여자의 취득일**부터 기산

### 부당행위계산부인 (특수관계인 거래)

- 시가와 거래가액의 차액이 **3억원 이상** 또는 시가의 **5%** 이상인 경우 적용
- 시가에 의하여 양도·취득가액 재계산

### 증여 후 양도의 부당행위계산부인 (우회양도 부인)

- 특수관계인(배우자·직계존비속 제외)에게 증여 후 **10년 이내** 양도
- 수증자의 (증여세 + 양도세) < 증여자가 직접 양도 시 양도세 → 증여자가 직접 양도한 것으로 봄
- 납세의무자: **증여자** (양도소득세 연대납세의무 있음)
- 증여세 **비과세**

## 13. 신고·납부

### 예정신고

| 구분 | 신고기한 |
|------|----------|
| 일반 양도 | 양도일이 속하는 달의 말일부터 **2개월 이내** |
| 부담부증여 (채무인수액) | 양도일이 속하는 달의 말일부터 **3개월 이내** |

### 확정신고

- 다음 연도 **5월 1일 ~ 5월 31일**
- 예정신고를 한 경우에도 확정신고 의무 (2건 이상 양도 등)

### 부가세: 지방소득세

- 양도소득세 세율의 **10분의 1**
- 예정신고: 양도소득세 예정신고기한에 **2개월** 더한 날까지
- 확정신고: 양도소득세 확정신고기한에 **2개월** 더한 날까지

## 핵심 암기 포인트

- **과세대상**: 토지·건물, 부동산에 관한 권리(지상권·전세권·등기된 임차권), 부동산취득권리(입주권·분양권), 주식 등
- **양도시기**: 원칙 = 잔금지급일, 불분명 = 등기접수일, 청산 전 등기 = 등기접수일
- **1세대1주택 비과세**: 2년 이상 보유(조정지역 거주 추가), 고가주택 12억 초과분만 과세
- **일시적 2주택**: 종전주택 1년 이상 보유 후 신규취득 → 3년 이내(조정 2년) 종전주택 양도
- **기본세율 8단계**: 6%→15%→24%→35%→38%→40%→42%→45%
- **단기보유 세율**: 주택 1년 미만 70%, 1~2년 60% / 분양권 1년 미만 70%, 1년 이상 60%
- **미등기양도**: 70%, 장기보유특별공제·기본공제 배제, 비과세·감면 배제, 개산공제 0.3%
- **조정지역 다주택 중과**: 2주택 +20%, 3주택+ +30%
- **비사업용토지**: 기본세율 +10%
- **장기보유특별공제**: 일반 매년 2%(15년 30% 한도), 1세대1주택 보유 매년 4%(10년 40%)+거주 매년 4%(10년 40%)=최대 80%
- **양도소득기본공제**: 연 250만원 (미등기 불가)
- **예정신고**: 양도일 속하는 달 말일부터 2개월(부담부증여 3개월)
- **확정신고**: 다음 해 5/1~5/31
- **이월과세**: 배우자·직계존비속 10년 이내 증여, 증여자 취득가액·취득일 적용, 납세의무자 = 수증자
- **부당행위계산부인**: 특수관계인(배우자·직계존비속 제외) 증여 후 10년 내 양도, 납세의무자 = 증여자, 증여세 비과세

## 함정 노트

- ⚠️ **세율 숫자 혼동**: 미등기 70% vs 주택 1년 미만 70% vs 1~2년 60% vs 분양권 1년 이상 60% — 비슷한 세율 구분 필수
- ⚠️ **예정신고 기한**: 일반 2개월 vs 부담부증여 3개월 — '3개월'로 통일 출제하는 함정
- ⚠️ **이월과세 vs 부당행위계산부인**: 납세의무자가 다름(이월=수증자, 부당=증여자), 연대납세의무 유무(이월=없음, 부당=있음), 증여세 처리(이월=필요경비산입, 부당=비과세)
- ⚠️ **비과세 배제 요건**: 미등기양도자산은 1세대1주택 비과세도 적용 불가 (무허가건물 등 등기불능 자산은 예외)
- ⚠️ **장기보유특별공제 한도**: 일반 30% vs 1세대1주택 80% — 혼동 주의
