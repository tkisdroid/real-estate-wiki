# 감정평가이론

## 1. 감정평가의 기본개념

- **감정평가**: 토지 등의 경제적 가치를 판정하여 그 결과를 가액으로 표시하는 것
- 시장가치 기준 평가가 원칙

### 가격(price)과 가치(value)

| 구분 | 가격 | 가치 |
|------|------|------|
| 성격 | **과거** 사실 | **미래** 예측 |
| 결정 | **시장**에서 결정 (객관적) | **평가주체**가 판정 (주관적 요소) |
| 특징 | 실제 거래된 금액 | 적정한 시장가치 추정 |

### 가격과 가치의 관계 (심화)

| 구분 | 가격(price) | 가치(value) |
|------|------------|------------|
| 의미 | 실제 지불된 금액 | 장래 기대편익의 현재가치 |
| 시점 | **과거**의 값 | **현재**의 값 |
| 수량 | 기준시점에서 **하나만** 존재 | 기준시점에서 **무수히 많음** |
| 성격 | **객관적·구체적** | **주관적·추상적** |

- 가격의 기초에는 가치가 있고, 가격은 가치의 **화폐적 표현**
- 화폐가치 **상승** → 가격 **하락** / 가치 증가 → 가격 증가
- 단기: 가격과 가치 **불일치** 가능 / 장기: **일치** (가치 = 가격의 장기 균형점)
- **가격 = 가치 / 화폐가치**

### 시장가치 (원칙)

- 통상적 시장에서 **충분한 기간** 공개된 후, 내용에 **정통한** 당사자 사이에 **신중하고 자발적인** 거래가 있을 경우 성립될 가능성이 **가장 높다고 인정되는** 대상물건의 **가액**
- 감정평가액은 시장가치를 기준으로 결정 (원칙)

### 시장가치 외의 가치 (예외)

| 가치 유형 | 내용 | 성격 |
|----------|------|------|
| **사용가치** | 특정 용도로 사용 시의 가치 | 주관적 |
| **교환가치** | 거래될 가능성이 가장 높은 구매력 크기 | 시장가치와 유사 |
| **투자가치** | 장래 기대수익의 현재가치 환원 | 주관적 |
| **장부가치** | 취득가격 - 감가상각분 (장부상 잔존가치) | - |
| **공익가치** | 최유효이용이 공공목적 비경제적 이용일 때 | - |

- 시장가치 외 가치 기준 평가 가능한 경우: 법령 규정, 의뢰인 요청, 사회통념상 필요

### 감정평가의 절차 (감정평가에 관한 규칙 제8조)

1. **기본적 사항의 확정** (의뢰인, 대상물건, 목적, 기준시점, 조건, 기준가치 등)
2. **처리계획의 수립**
3. **대상물건의 확인** (실지조사 원칙, 예외적 생략 가능)
4. **자료수집 및 정리** (확인자료, 요인자료, 사례자료)
5. **자료의 검토 및 가치형성요인 분석**
6. **감정평가방법의 선정 및 적용**
7. **감정평가액의 결정 및 표시**

> **기준시점** = 가격조사를 **완료**한 날짜 (개시 아님, 감정평가서 작성 완료일 아님). 미리 정한 경우 가격조사 **가능한 경우에만** 적용

### 감정평가의 분류

| 분류 | 내용 |
|------|------|
| **현황기준 평가**(원칙) | 기준시점의 이용상황(불법·일시적 이용 제외) 및 공법상 제한 기준 |
| **조건부 평가**(예외) | 가치형성요인을 실제와 다르게 가정 (법령, 의뢰인 요청, 사회통념상 필요) |
| **소급평가** | **과거** 시점 기준 | 
| **기한부평가** | **미래** 시점 기준 |
| **개별평가**(원칙) | 대상물건마다 개별 평가 |
| **일괄평가** | 일체 거래 또는 용도상 불가분 관계 |
| **구분평가** | 하나의 물건에서 가치를 달리하는 부분을 구분 |
| **부분평가** | 일체 이용 물건의 **일부분**만 평가 (특수 목적·합리적 이유 필요) |

> **주의**: 구분평가 ≠ 부분평가. 구분 = 가치가 다른 부분 / 부분 = 일체 이용 물건의 일부

## 2. 부동산 가격의 3면성

| 측면 | 내용 | 관련 방식 |
|------|------|----------|
| **비용** 측면 | 생산비, 재조달원가 | **원가방식** |
| **시장** 측면 | 시장에서의 교환가치 | **비교방식** |
| **수익** 측면 | 장래 기대수익의 현재가치 | **수익방식** |

## 3. 가격제원칙

### 최유효이용의 원칙 (가장 중요)

- 객관적으로 보아 **양식·통상적 이용방법**에 의한 가장 효율적인 이용
- **합리적**, **합법적**, **물리적으로 가능**, **재정적으로 실현가능**, **최대 수익**
- 토지의 **용도의 다양성**이 근거

### 기타 가격원칙

| 원칙 | 내용 |
|------|------|
| **기여의 원칙** | 부분이 전체에 기여하는 정도로 가치 판단 (공헌의 원칙) |
| **수익체증·체감의 원칙** | 투입을 증가시키면 수익도 체증하나 어느 시점부터 체감 |
| **수익배분의 원칙** | 총수익을 토지·노동·자본·경영 각 요소에 배분 |
| **균형의 원칙** | 구성요소 간 균형 시 최고가치 실현 |
| **적합의 원칙** | 주위환경에 적합할 때 최고가치 (외부균형) |
| **대체의 원칙** | 대체 가능한 재화 중 최저가격 기준 (비교방식의 근거) |
| **경쟁의 원칙** | 초과이윤 → 경쟁 → 초과이윤 소멸 |
| **예측의 원칙** | 장래 편익에 대한 예측이 가치에 반영 |
| **변동의 원칙** | 가격형성요인은 끊임없이 변동, 시점 수정 필요 |

## 4. 지역분석 및 개별분석

### 지역분석

| 지역 유형 | 내용 |
|-----------|------|
| **인근지역** | 대상부동산이 속한 유사한 이용·가격 형성 지역 |
| **유사지역** | 인근지역과 유사한 특성을 가진 다른 지역 |
| **동일수급권** | 대상부동산과 대체·경쟁 관계에 있는 지역 범위 |

### 개별분석

대상부동산의 **개별적 특성**을 분석하여 최유효이용을 판정하는 것.

- 대지의 형상, 면적, 접면도로, 방위, 고저
- 건물의 구조, 용도, 경과연수, 관리상태

## 5. 가치발생요인과 가치형성요인

### 가치발생요인

| 요인 | 내용 | 유형 |
|------|------|------|
| **효용(유용성)** | 욕구 충족 능력. 주거지=쾌적성, 상업지=수익성, 공업지=생산성 | 경제적 |
| **상대적 희소성** | 수요에 비해 공급이 부족한 상태 | 경제적 |
| **유효수요** | 구매의사(욕구) + **구매능력**을 갖춘 수요 | 경제적 |
| **이전성** | 소유권·권리의 이전 가능성 | **법률적** |

- 가치발생요인은 **상호결합**에 의해 가치 발생 (하나만으로 불충분)

### 가치형성요인

- 가치발생요인에 **영향을 주는** 요인
- **일반요인**(전국적·광역적): 사회적·경제적·행정적 요인
- **지역요인**: 지역 특성을 형성하는 요인 (인근지역의 가격에 전반적 영향)
- **개별요인**: 개별 부동산의 특수한 상태·조건

## 핵심 암기 포인트

- 가격 = **과거** 사실(시장 결정) / 가치 = **미래** 예측(평가주체 판정)
- 가격의 3면성: **비용**(원가방식), **시장**(비교방식), **수익**(수익방식)
- 최유효이용 조건: **합리적, 합법적, 물리적 가능, 재정적 실현가능, 최대 수익**
- 최유효이용의 근거 = 토지의 **용도의 다양성**
- 기여의 원칙 = 공헌의 원칙: **부분**이 **전체**에 기여하는 정도
- 대체의 원칙: **비교방식**의 이론적 근거
- 예측의 원칙: 장래 편익 예측이 가치에 반영
- 지역분석: **인근지역, 유사지역, 동일수급권**
- 가격 다원설: **효용 + 상대적 희소성 + 유효수요** 3요소 결합

## OX 퀴즈

**Q1.** 가격(price)은 미래 예측이고, 가치(value)는 과거 사실이다. → **X** (가격 = **과거** 사실, 가치 = **미래** 예측)

**Q2.** 최유효이용의 근거는 토지의 부증성이다. → **X** (최유효이용의 근거는 **용도의 다양성**)

**Q3.** 대체의 원칙은 수익방식의 이론적 근거이다. → **X** (대체의 원칙은 **비교방식**(및 원가방식)의 근거, 수익방식은 예측·수익배분의 원칙)

**Q4.** 경쟁의 원칙에 의하면 초과이윤은 경쟁을 유발하고 결국 소멸한다. → **O**

**Q5.** 인근지역과 유사지역은 동일한 개념이다. → **X** (인근지역은 대상부동산이 속한 지역, 유사지역은 인근지역과 유사한 **다른** 지역)

**Q6.** 적합의 원칙은 부동산이 주위환경에 적합할 때 최고가치가 실현된다는 것이다. → **O**

**Q7.** 가격 다원설에 의하면 부동산 가격은 효용과 희소성만으로 결정된다. → **X** (효용 + 상대적 희소성 + **유효수요** 3요소)

## 함정 노트

- ⚠️ "가격 = 미래, 가치 = 과거" → 정반대. 가격 = **과거** 사실, 가치 = **미래** 예측
- ⚠️ "최유효이용 근거 = 부증성" → 부증성도 관련되나 직접 근거는 **용도의 다양성**
- ⚠️ "대체의 원칙 = 수익방식 근거" → 대체의 원칙은 **비교·원가**방식 근거, 수익방식은 **예측·수익배분** 원칙
- ⚠️ "균형의 원칙 = 외부균형" → 균형의 원칙은 구성요소 간 **내부균형**, **외부균형**은 적합의 원칙
- ⚠️ "인근지역 = 유사지역" → 인근지역(대상부동산 소속)과 유사지역(유사한 다른 지역)은 **별개** 개념
