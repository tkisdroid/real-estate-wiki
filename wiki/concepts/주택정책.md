# 주택정책

## 1. 주택문제의 측정지표

| 지표 | 산식 | 의미 |
|------|------|------|
| **PIR** | 주택가격 / 연간가계소득 | 높을수록 주택구입능력 **낮음** |
| **RIR** | 주택임대료 / 연간가계소득 | 높을수록 주거비부담 **큼** |
| **슈바베지수** | 주거비 / 생계비(총소비지출) | 저소득층일수록 **높음** |

- **슈바베 법칙**: 소득↑ → 주거비/생계비 비율 **감소**

## 2. 임대료 규제정책 (최고가격제·상한제)

- 시장 균형임대료보다 **낮은 수준**으로 최고가격 설정
- **공급 완전비탄력적**일 때 소득재분배 효과 **최대**
- 규제임대료가 균형임대료보다 **높으면** 시장에 **영향 없음**

### 장기적 효과 (규제임대료 < 시장임대료)

| 효과 | 내용 |
|------|------|
| **초과수요** | 수요↑, 공급↓ → 주택부족 |
| **관리소홀** | 수익성 하락 → 임대주택 질적 수준 저하 |
| **공급감소** | 투자 기피, 다른 용도로 전용 |
| **주거이동 저하** | 이주할 주택 부족 |
| **암시장** 형성 | 법정임대료와 암시장임대료의 **이중가격** |
| **조세수입 감소** | |
| **사회적 비용 증가** | 후생손실 |

## 3. 임대료보조금 정책 (간접적 개입)

### 소비자보조 (수요 측면)

| 유형 | 내용 | 장점 |
|------|------|------|
| **집세보조** | 주택 상대가격 인하 | **주택소비 증가** 효과 우월 |
| **소득보조(현금보조)** | 소득 자체를 증가 | **효용** 면에서 우월 (선택 폭 넓음) |

- 단기: 수요↑ → 임대료 상승 → 기존 임대인 초과이익
- 장기: 공급↑ → 임대료 원래 수준으로 하락 → 임차인 실질임대료 하락

### 주택바우처 제도

- 교환권으로 임대료 보조 → 임차인이 **거주지 직접 선택** 가능
- 소득보조의 장점(선택권) + 집세보조의 장점(누수 방지)

### 생산자보조 (공급 측면)

- 건설자금 저리 지원 → 생산비 하락 → 공급 확대
- **단기에는 효과 없음**, 장기적으로 임대료 하락

## 4. 공공임대주택

| 유형 | 임대기간 | 대상 |
|------|----------|------|
| **영구임대** | **50년 이상** 또는 영구 | 최저소득 계층 |
| **국민임대** | **30년 이상** | 저소득 서민 |
| **행복주택** | - | **대학생·사회초년생·신혼부부** 등 젊은 층 |
| **통합공공임대** | - | 최저소득·저소득·젊은 층·장애인·국가유공자 등 |
| **장기전세** | - | 전세계약 방식 |
| **분양전환공공임대** | 일정기간 후 | 분양전환 목적 |

- 공공임대주택 공급 → 민간임대주택 수요의 가격탄력성 **증가**
- **소득재분배 효과** 기대 가능

## 5. 분양가 상한제

- 신규 주택가격을 균형가격보다 **낮게** 규제 (최고가격제)
- 분양가격 = **택지비 + 건축비** (토지임대부는 건축비만)
- 도시형 생활주택은 **제외**

### 효과

- 건설업체 수익성 악화 → 주택공급 **위축**
- 주택 과소비, 자원 낭비, 공급 질 저하
- 장기적으로 **중고주택 가격 상승** → 투기적 수요 발생
- 초과수요 발생

## 6. 선분양 vs 후분양

| 구분 | 선분양제도 | 후분양제도 |
|------|-----------|-----------|
| 시장 성격 | **공급자** 주도 | **소비자** 주도 |
| 장점 | 자금조달 용이, 분할납부 | 품질비교 가능, 시공사 부도위험 해소 |
| 단점 | 시공사 부도 시 피해, 투기 조장 | 목돈 부담, 대형사 독과점 우려 |

## 핵심 암기 포인트

- PIR = **주택가격 / 연간가계소득** (높을수록 구입능력 낮음)
- RIR = **주택임대료 / 연간가계소득** (높을수록 주거비부담 큼)
- 슈바베지수 = **주거비 / 생계비(총소비지출)** (저소득층일수록 높음)
- 임대료 규제: 공급 **완전비탄력적**일 때 소득재분배 효과 **최대**
- 집세보조: **주택소비 증가** 효과 우월 / 소득보조(현금보조): **효용** 면에서 우월
- 영구임대: **50년 이상** / 국민임대: **30년 이상**
- 행복주택 대상: **대학생·사회초년생·신혼부부** 등 젊은 층
- 분양가격 = **택지비 + 건축비** (토지임대부는 건축비만)
- 선분양: **공급자** 주도 / 후분양: **소비자** 주도
- 공공임대주택 공급 → 민간임대주택 수요의 가격탄력성 **증가**

## OX 퀴즈

**Q1.** PIR이 높을수록 주택구입능력이 높다. → **X** (PIR 높을수록 구입능력 **낮음**)

**Q2.** 슈바베 법칙에 의하면 소득이 증가하면 주거비/생계비 비율이 감소한다. → **O**

**Q3.** 임대료 규제 시 공급이 완전탄력적일 때 소득재분배 효과가 최대이다. → **X** (공급 **완전비탄력적**일 때 소득재분배 효과 최대)

**Q4.** 소득보조(현금보조)는 집세보조보다 주택소비 증가 효과가 크다. → **X** (주택소비 증가 효과는 **집세보조**가 우월, 효용 면에서는 소득보조가 우월)

**Q5.** 영구임대주택의 임대기간은 30년 이상이다. → **X** (**50년 이상** 또는 영구, 30년 이상은 국민임대)

**Q6.** 분양가 상한제 시행 시 장기적으로 중고주택 가격이 상승할 수 있다. → **O**

**Q7.** 행복주택은 최저소득 계층을 대상으로 한다. → **X** (행복주택은 **대학생·사회초년생·신혼부부** 등 젊은 층 대상, 최저소득층은 영구임대)

**Q8.** 후분양제도는 공급자 주도의 시장 성격을 가진다. → **X** (후분양은 **소비자** 주도, 선분양이 공급자 주도)

## 함정 노트

- ⚠️ "PIR 높음 = 구입능력 높음" → PIR은 **역관계**, 높을수록 구입능력 **낮음**
- ⚠️ "소득보조 = 주택소비 증가 우월" → 주택소비 증가는 **집세보조** 우월, **효용**은 소득보조 우월
- ⚠️ "영구임대 30년, 국민임대 50년" → 영구임대 **50년 이상**, 국민임대 **30년 이상**
- ⚠️ "행복주택 = 저소득층" → 행복주택은 **젊은 층** 대상, 저소득층은 영구·국민임대
- ⚠️ "분양가 상한제 → 공급 증가" → 분양가 상한제는 건설업체 수익성 악화로 공급 **위축**
