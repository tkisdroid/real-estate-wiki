# 유치권 (제320조~제328조)

## 의의

타인의 물건 또는 유가증권을 점유한 자가 그 **물건에 관하여 생긴 채권**이 변제기에 있는 경우 변제받을 때까지 **유치할 수 있는** 권리 (제320조)

## 법적 성질

### 법정담보물권
- 일정 요건 충족 시 **법률에 의해 당연히 성립** → 등기 불요
- 당사자 약정에 의해 유치권을 **성립시킬 수 없음**
- 다만 유치권 배제(포기) 특약은 **유효** (제320조는 임의규정, 대판 2010마1544)

### 담보물권적 성질
- **부종성·수반성**: 피담보채권 없으면 유치권 불인정, 피담보채권과 점유 이전 시 유치권도 이전
- **불가분성**: 채권 전부 변제 시까지 유치물 전부에 대하여 권리 행사 (대판 2005다16942)

### 인정되지 않는 성질
- **우선변제권 없음**
- **물상대위성 없음**
- **추급력 없음** (점유 상실 시 소멸)

## 성립요건

### 1. 타인의 물건 (또는 유가증권)
- **채무자뿐 아니라 제3자 물건**도 가능 (채무자 소유 불요)
- 부동산 일부에도 성립 가능
- 수급인 재료·노력으로 건축된 독립 건물 → 수급인 소유이므로 유치권 **불가** (대판 91다14116)
- 토지에 부합한 정착물(독립 건물 아닌 것) → 토지의 부합물이므로 유치권 **불가** (대판 2007마98)

### 2. 적법한 점유
- **점유는 성립요건이자 존속요건** → 점유 상실 시 유치권 소멸
- 직접점유·간접점유 모두 가능, 단 **채무자를 직접점유자로 하는 간접점유는 불가**
- **불법행위로 인한 점유**는 유치권 불성립
- 임대차 해지·기간만료 후 동시이행항변권에 기하지 않고 점유하면서 지출한 필요비 → 유치권 **불인정**
- 점유가 적법한 것으로 **추정**됨 → 부적법 주장하는 자에게 입증책임 (대판 2009다5162)

### 3. 물건에 관하여 생긴 채권 (견련관계)

#### 견련관계 인정 (유치권 성립)

| 채권 유형 | 예시 |
|-----------|------|
| **필요비·유익비 상환청구권** | 건물 수선비 등 |
| **수급인의 공사대금채권** | 건물 신축·증축 공사대금 |
| **목적물로 인한 손해배상청구권** | 물건 하자로 인한 손해 |

#### 견련관계 부정 (유치권 불성립)

| 채권 유형 | 이유 |
|-----------|------|
| **임차보증금반환청구권** | 건물에 관하여 생긴 채권이 아님 |
| **권리금반환청구권** | 건물 자체와 견련관계 없음 |
| **매매대금채권** | 건물에 관하여 생긴 채권이 아님 |
| **건축자재대금채권** | 건물에 관하여 생긴 채권이 아님 |
| **계약명의신탁의 매수자금** | 견련관계 불인정 (대판 2008다34828) |

- 목적물의 점유 중에 피담보채권이 발생할 것을 **요하지 않음** → 점유 이전에 채권 발생 후 점유 취득해도 유치권 성립

### 4. 채권의 변제기 도래
- 변제기는 **성립요건** → 변제기 미도래 시 유치권 불성립
- 유익비상환청구권에 대해 법원이 상당한 기간을 허여한 경우 → 유치권 **불성립**

### 5. 유치권 배제특약의 부존재
- 유치권 포기 특약은 **유효** (대판 2010마1544)
- 배제특약의 효력은 특약 상대방뿐 아니라 **그 밖의 사람도 주장 가능**
- 원상복구 약정 → 유익비·필요비 포기 취지로 해석 → 유치권 배척 (대판 73다2010)

## 유치권의 효력

### 유치적 효력 (유치할 권리)
- **모든 사람**(대세)에 대해 유치권 주장 가능
- 경매절차 매수인에 대하여도 피담보채권 변제 시까지 인도 거절 가능
- 다만 매수인에게 피담보채권의 **변제를 청구할 권리는 없음** (대판 95다8713)

### 경매개시결정과의 관계
- 경매개시결정 기입등기 후(압류 효력 발생 후) 점유 이전받아 유치권 취득 → 매수인에게 **대항 불가** (대판 2010마1544)
- 압류 효력 발생 **전**에 취득한 유치권 → 매수인에게 **대항 가능**

### 환가를 위한 경매권 (제322조 제1항)
- 유치권자는 채권 변제를 위해 유치물을 **경매** 가능
- **우선변제를 위한 경매가 아님** → 형식적 경매(환가 경매)

### 간이변제충당 (제322조 제2항)
- 정당한 사유 시 감정인 평가에 의해 유치물로 **직접 변제 충당** 가능 (법원 허가)
- 허가 결정 시 유치권자가 유치물 소유권 취득 → **등기 없이** 물권 변동 (제187조)

### 과실수취권 (제323조)
- 유치물의 과실을 수취하여 다른 채권보다 **먼저** 채권 변제에 충당 가능
- 과실이 금전 아닌 때 → 경매하여야 함
- 충당 순서: 이자 먼저 → 잉여 시 원본

## 유치권자의 의무

### 선관주의의무 (제324조 제1항)
- 선량한 관리자의 주의로 유치물 점유
- **자기 재산과 동일한 주의**보다 무거운 의무

### 무단 사용·대여·담보제공 금지 (제324조 제2항)
- **채무자의 승낙 없이** 사용·대여·담보제공 금지
- 예외: **보존에 필요한 사용**은 채무자 승낙 없이 가능
- 공사대금채권 유치권자가 주택에 거주 → **보존에 필요한 사용** 해당 (대판 2011다107009)
- 보존에 필요한 사용이라도 차임 상당 이득 → 소유자에게 **반환 의무** 있음
- 소유자 동의 없이 유치권자로부터 임차한 자의 점유 → 경락인에게 대항 **불가** (대판 2002마3516)

### 의무위반의 효과 (제324조 제3항)
- 의무 위반 시 채무자가 유치권 **소멸 청구** 가능 (형성권)
- 위반만으로 자동 소멸하는 것이 **아님** → 소멸청구가 있어야 소멸
- 여러 필지 중 일부에서 위반 → 위반 필지에 대해서만 소멸청구 가능 (대판 2018다301350)

## 유치권의 소멸

### 점유의 상실 (제328조)
- 점유 상실 → 유치권 **소멸**
- 점유침탈 후 점유회수의 소로 승소하여 점유 회복 → 유치권 **부활**
- 단지 점유회수의 소를 제기할 수 있다는 사정만으로는 부활 **불가**

### 다른 담보의 제공 (제327조)
- 채무자(또는 소유자)가 **상당한 담보 제공** → 유치권 소멸 청구 가능
- 담보 제공에 대해 유치권자의 승낙 또는 이에 갈음하는 판결 필요

### 기타
- 피담보채권의 소멸, 혼동, 공용수용, 포기, 몰수
- 유치권 자체는 **소멸시효에 걸리지 않음** (목적물에 대한 타인의 시효취득으로도 소멸하지 않음)
- 단, 유치권 행사는 피담보채권의 **소멸시효 진행에 영향 없음** (제326조)

## 부동산경매와 유치권의 관계

| 상황 | 유치권 대항 여부 |
|------|------------------|
| 압류 효력 발생 **전** 유치권 취득 | 매수인에게 **대항 가능** |
| 압류 효력 발생 **후** 유치권 취득 | 매수인에게 **대항 불가** |
| 매수인에 대한 변제 청구 | **불가** (인도 거절만 가능) |

## 핵심 암기 포인트

- **법정담보물권**: 요건 충족 시 법률에 의해 당연 성립, 등기 불요
- **성립요건 5가지**: 타인의 물건 + 적법한 점유 + 견련관계 + 변제기 도래 + 배제특약 부존재
- **우선변제권 없음**, 물상대위 없음, 추급력 없음
- **점유 = 성립요건 + 존속요건**: 상실 시 소멸
- **경매개시결정 후 취득 유치권**: 매수인에게 대항 불가
- **견련관계 부정**: 임차보증금, 권리금, 매매대금, 건축자재대금
- **선관주의의무**: 채무자 승낙 없이 사용·대여·담보제공 금지 (보존 필요 사용은 허용)
- **의무위반 → 소멸청구 필요** (자동 소멸 아님)

## OX 퀴즈

**Q1.** 유치권에는 우선변제권이 있다. → **X** (유치권에는 우선변제권이 없다)

**Q2.** 유치권은 채무자 소유의 물건에 대해서만 성립한다. → **X** (제3자 소유 물건에도 성립 가능)

**Q3.** 유치권자가 의무에 위반하면 유치권은 즉시 소멸한다. → **X** (채무자의 소멸청구가 있어야 소멸)

**Q4.** 임차보증금반환청구권은 건물에 대한 유치권의 피담보채권이 될 수 있다. → **X** (견련관계 부정)

**Q5.** 경매개시결정 기입등기 후에 취득한 유치권도 매수인에게 대항할 수 있다. → **X** (압류 효력 발생 후 취득한 유치권은 매수인에게 대항 불가)

**Q6.** 유치권자는 채무자의 승낙 없이도 유치물의 보존에 필요한 사용은 할 수 있다. → **O** (제324조 제2항 단서)

**Q7.** 유치권의 행사는 피담보채권의 소멸시효 진행을 중단시킨다. → **X** (유치권 행사는 소멸시효 진행에 영향을 미치지 않는다, 제326조)

**Q8.** 유치권 배제(포기) 특약은 무효이다. → **X** (유치권 포기 특약은 유효하다, 제320조는 임의규정)

## 함정 노트

⚠️ "유치권 = 우선변제권 있음" → **없음** (유치적 효력만 있고 우선변제권은 없다)

⚠️ "의무위반 = 유치권 자동 소멸" → 채무자의 **소멸청구**가 있어야 비로소 소멸

⚠️ "유치권 행사 = 시효 중단" → 유치권 행사는 피담보채권의 소멸시효 **진행에 영향 없음**

⚠️ "유치권 배제특약 = 무효" → **유효** (임의규정)

⚠️ "경매에서 매수인에게 변제 청구 가능" → 인도 거절만 가능, **변제 청구권은 없음**
