# 표현대리

## 의의

대리권이 없거나 대리권의 범위를 넘었음에도 불구하고, 외관상 대리권이 있는 것처럼 보이는 경우에 **선의·무과실의 상대방을 보호**하기 위하여 유효한 대리행위로 인정하는 제도

## 3유형 비교

| 유형 | 조문 | 요건 | 적용범위 |
|------|------|------|----------|
| **대리권수여 표시** | 제125조 | 본인이 제3자에 대리권 수여를 표시 + 상대방 **선의·무과실** | 임의대리만 |
| **권한을 넘은 표현대리** | 제126조 | **기본대리권** 존재 + 권한 외 행위 + 상대방의 **정당한 이유** | 임의대리 + **법정대리** |
| **대리권 소멸 후** | 제129조 | 대리권 소멸 + 범위 내 행위 + 상대방 **선의·무과실** | 임의대리 + **법정대리** |

## 제125조: 대리권수여 표시에 의한 표현대리

### 요건
1. 본인이 제3자(상대방)에게 **대리권 수여를 표시**한 것 (실제 대리권 수여 불요)
2. 표시된 대리권의 범위 내에서 대리행위
3. 상대방이 **선의·무과실**

### 적용범위
- **임의대리에만** 적용
- 법정대리에는 적용 안됨

### 효과
- 본인에게 법률행위의 효과 귀속 (유효한 대리로 취급)

## 제126조: 권한을 넘은 표현대리

### 요건
1. **기본대리권**이 존재할 것
2. 대리인이 기본대리권의 **범위를 넘어** 행위할 것
3. 상대방에게 권한 있다고 믿을 만한 **정당한 이유**가 있을 것

### 기본대리권이 될 수 있는 것

| 인정 (기본대리권 가능) | 부정 (기본대리권 불가) |
|----------------------|---------------------|
| **등기신청행위** (공법상 행위, 대판 78다282) | **사실행위** (대판 91다32190) |
| 부부간 **일상가사대리권** (대판 98다18988) | |
| **표현대리권** (대판 79다234) | |
| **법정대리권** | |

### 적용범위
- **임의대리 + 법정대리** 모두 적용
- 가장 넓은 적용범위

### 정당한 이유의 판단
- 대리인의 직업, 거래의 종류, 대리행위의 내용, 대리인과 본인의 관계 등을 종합적으로 고려
- 상대방의 **선의·무과실**과 유사하나, 보다 넓은 개념

## 제129조: 대리권 소멸 후의 표현대리

### 요건
1. 대리권이 **소멸**한 후의 행위일 것
2. **과거 대리권의 범위 내**에서 행위할 것
3. 상대방이 대리권 소멸을 **알지 못했고, 알지 못한 것에 과실이 없을 것** (선의·무과실)

### 적용범위
- **임의대리 + 법정대리** 모두 적용

### 효과
- 본인에게 법률행위의 효과 귀속

## 표현대리의 효과

### 공통
- 표현대리가 성립하면 **유효한 대리**로서 법률효과가 **본인에게 귀속**
- 본인은 상대방에 대해 법률행위의 효력을 부인할 수 없음

### 표현대리와 무권대리인의 책임 (제135조)
- 표현대리 성립 시 → 유효한 대리 → 무권대리인 책임 문제 안됨
- 표현대리 불성립 시 → 무권대리 → 무권대리인의 책임 가능

### 표현대리의 주장
- 표현대리는 **상대방이 주장**해야 함 (무권대리인이 주장할 수 없음)

## 표현대리와 다른 제도와의 관계

### 표현대리와 무권대리
- 표현대리가 성립하지 않으면 → 무권대리 법리 적용 (추인, 최고, 철회, 제135조 책임)
- 상대방은 표현대리 주장과 무권대리 법리를 **선택적으로** 주장 가능

### 제125조와 제126조의 중첩
- 대리권 수여 표시 + 표시된 권한 초과 → 제125조와 제126조 **중첩 적용** 가능

## 핵심 암기 포인트

- **3유형**: 대리권수여표시(125조), 권한초과(126조), 소멸후(129조)
- **적용범위**: 제125조 = 임의대리만 / 제126조·제129조 = 임의대리+법정대리
- **제126조 기본대리권**: 등기신청(공법상 행위) O, 일상가사대리권 O, 표현대리권 O, **사실행위 X**
- **정당한 이유 vs 선의·무과실**: 제126조는 "정당한 이유", 제125·129조는 "선의·무과실"
- 표현대리 성립 → 유효한 대리 (본인 귀속)
- 표현대리 불성립 → 무권대리 (추인·최고·철회·제135조)

## OX 퀴즈

**Q1.** 사실행위에 대한 권한도 제126조의 기본대리권이 될 수 있다. → **X** (사실행위는 기본대리권 불가, 대판 91다32190)

**Q2.** 제125조(대리권수여표시)는 법정대리에도 적용된다. → **X** (임의대리에만 적용)

**Q3.** 등기신청행위(공법상 행위)도 제126조의 기본대리권이 될 수 있다. → **O** (대판 78다282)

**Q4.** 부부간 일상가사대리권은 제126조의 기본대리권이 될 수 없다. → **X** (될 수 있다, 대판 98다18988)

**Q5.** 표현대리가 성립하면 무권대리인이 제135조의 책임을 진다. → **X** (표현대리 성립 시 유효한 대리이므로 무권대리인 책임 문제 안됨)

**Q6.** 제126조의 '정당한 이유'와 제125조의 '선의·무과실'은 완전히 동일한 개념이다. → **X** (정당한 이유가 더 넓은 개념)

**Q7.** 표현대리권도 제126조의 기본대리권이 될 수 있다. → **O** (대판 79다234)

## 함정 노트

⚠️ "사실행위 = 기본대리권 가능" → **불가** (대판 91다32190)

⚠️ "제125조 = 법정대리에도 적용" → **임의대리에만** 적용 (제126·129조는 양쪽 적용)

⚠️ "기본대리권 = 사법상 행위만" → 등기신청(공법상 행위)도 **가능** (대판 78다282)

⚠️ "표현대리 주장 = 무권대리인이 가능" → **상대방만** 주장 가능
