# 용익물권

타인의 부동산을 일정한 범위에서 사용·수익하는 물권. **지상권, 지역권, 전세권**이 있다.

## 용익물권 비교표

| 구분 | [[용익물권_지상권\|지상권]] | [[용익물권_지역권\|지역권]] | [[용익물권_전세권\|전세권]] |
|------|--------|--------|--------|
| **조문** | 제279조~제290조 | 제291조~제302조 | 제303조~제319조 |
| **목적** | 건물·공작물·수목 소유 위한 토지 사용 | 요역지 편익 위한 승역지 이용 | 부동산 사용·수익 + 우선변제 |
| **대상** | 토지 | 토지 | 부동산 전부 (토지+건물) |
| **대가** | 지료 (요소 아님, 무상 가능) | 무상 원칙 | 전세금 (성립요소) |
| **점유** | 토지 점유 | 반드시 점유 수반 않음 | 부동산 점유 |
| **존속기간** | 최단: 30/15/5년, 최장 제한 없음 | 제한 없음 | 최장 10년 |
| **양도성** | 자유 (동의 불요, 강행규정) | 요역지와 분리 처분 불가 | 자유 (동의 불요) |
| **수선의무** | 지상권자 (필요비 부담) | - | 전세권자 (제309조) |
| **담보성** | 없음 | 없음 | 있음 (우선변제권·경매청구권) |
| **법정지상권** | 관습법상·민법 제366조 | - | 제305조 |
| **소멸시효** | 걸림 | 걸림 | 걸림 |

## 핵심 비교 포인트

| 비교 항목 | 지상권 | 전세권 | 비고 |
|-----------|--------|--------|------|
| 대가의 성격 | 지료 = 요소 아님 | 전세금 = 성립요소 | 지상권은 무상 가능, 전세권은 불가 |
| 양도 시 동의 | 불요 (강행규정) | 불요 (설정행위로 금지 가능) | 임차권은 동의 필요 |
| 수선의무 | 지상권자 | 전세권자 | 임대차는 임대인 |
| 담보적 기능 | 없음 | 우선변제권·경매청구권 | 전세권의 이중성 |
