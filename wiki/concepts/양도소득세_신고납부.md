# 양도소득세 - 신고납부

## 1. 양도소득세 계산구조 (요약)

양도소득세는 다음의 단계별 계산구조를 거쳐 산출된다.

| 단계 | 내용 |
|------|------|
| **양도가액** | 실지거래가액 (원칙) |
| **(-)** 필요경비 | 실지취득가액 + 자본적 지출액 + 양도비용 |
| **= 양도차익** | 양도차손 발생 시 같은 소득별로 합산 |
| **(-)** 장기보유특별공제 | 등기된 토지·건물, 3년 이상 보유 시 |
| **= 양도소득금액** | |
| **(-)** 양도소득기본공제 | 소득별 각각 연 **250만원** (미등기자산 제외) |
| **= 양도소득 과세표준** | |
| **(x)** 세율 | 초과누진세율(6~45%) 또는 차등비례세율 |
| **= 산출세액** | |
| **(-)** 감면세액·세액공제 | |
| **= 결정세액** | |

## 2. 예정신고납부

### 예정신고기한

| 구분 | 신고기한 |
|------|---------|
| **토지·건물·부동산에 관한 권리·기타자산** | 양도일이 속하는 달의 말일부터 **2개월** 이내 |
| **부담부증여의 채무액 부분** (양도로 보는 경우) | 양도일이 속하는 달의 말일부터 **3개월** 이내 |
| **토지거래허가구역** 내 토지 (허가 전 대금청산) | 허가일이 속하는 달의 말일부터 **2개월** 이내 |

> 양도차익이 없거나 양도차손이 발생한 때에도 **예정신고를 하여야 한다**.

### 예정신고 산출세액

- 예정신고 산출세액 = (양도차익 - 장기보유특별공제 - 양도소득기본공제) x 세율
- 해당 과세기간에 누진세율 적용대상 자산에 대한 예정신고를 **2회 이상** 하는 경우: 이미 신고한 양도소득금액과 **합산하여 신고**

### 예정신고 납부

- 산출세액에서 감면세액을 뺀 세액을 **납세지 관할세무서·한국은행·체신관서**에 납부
- 수시부과세액이 있으면 이를 공제하여 납부
- 법정기한 내 예정신고 납부 시 별도의 세액공제는 **없다**

## 3. 확정신고납부

### 확정신고기한

해당 과세기간의 양도소득금액이 있는 거주자는 **다음 연도 5월 1일부터 5월 31일까지** 납세지 관할세무서장에게 신고

| 특례 | 기한 |
|------|------|
| **거주자 사망** | 상속개시일이 속하는 달의 말일부터 **6개월** (상속인 출국 시 출국일 전날) |
| **국외이주 목적 출국** | 출국일 **전날**까지 |

### 확정신고 면제 (예정신고한 자)

예정신고를 한 자는 원칙적으로 확정신고를 하지 않을 수 있다. **다만 다음의 경우 확정신고 필수:**

1. 누진세율 적용대상 자산 예정신고를 **2회 이상** 하고 합산신고하지 않은 경우
2. 토지·건물 등을 **2회 이상** 양도하고 감면소득 규정 적용 시 산출세액이 달라지는 경우
3. 주식 등을 **2회 이상** 양도하고 감면소득 규정 적용 시 산출세액이 달라지는 경우

### 확정신고 납부

- 산출세액 - 감면세액 - 세액공제액 = 확정신고 납부세액
- 예정신고 산출세액, 결정·경정세액, 수시부과세액이 있으면 공제하여 납부

## 4. 분할납부(분납)

### 요건

납부할 세액이 각각 **1,000만원을 초과**하는 경우, 납부기한 경과 후 **2개월 이내** 분할납부 가능

### 분할납부할 수 있는 세액

| 납부할 세액 | 분할납부 세액 |
|------------|-------------|
| **2,000만원 이하** | 1,000만원을 **초과**하는 금액 |
| **2,000만원 초과** | 세액의 **50% 이하** |

### 신청

- 예정신고 시: 양도소득과세표준 예정신고 및 납부계산서에 기재하여 예정신고기한까지 신청
- 확정신고 시: 양도소득과세표준 확정신고 및 납부계산서에 기재하여 확정신고기한까지 신청

## 5. 가산세

### 신고불성실 가산세

| 유형 | 가산세율 |
|------|---------|
| **일반 무신고** | 무신고 납부세액의 **20%** |
| **부정(부당) 무신고** | 무신고 납부세액의 **40%** (국제거래 부정행위: 60%) |
| **일반 과소신고** | 과소신고분 납부세액의 **10%** |
| **부정 과소신고** | 과소신고분 납부세액의 **40%** (국제거래 부정행위: 60%) |

### 납부지연 가산세

| 구분 | 내용 |
|------|------|
| **미납·과소납부** | 미납세액 x 납부기한 다음날~자진납부일(또는 납세고지일) x **1일 0.022%** |
| **납부고지 후 미납** | 미납세액 x **3%** + 미납세액 x 납세고지 납부기한 다음날~납부일 x **1일 0.022%** (5년 한도) |

### 가산세 이중부과 금지

**예정신고**와 관련하여 무신고가산세 또는 과소신고가산세가 부과된 부분에 대해서는 **확정신고**와 관련하여 가산세를 **적용하지 않는다**.

### 가산세 감면

| 수정신고 시기 | 감면율 |
|-------------|-------|
| 법정신고기한 후 **1개월 이내** | 90% |
| 1개월 초과 **3개월 이내** | 75% |
| 3개월 초과 **6개월 이내** | 50% |
| 6개월 초과 **1년 이내** | 30% |
| 1년 초과 **1년 6개월 이내** | 20% |
| 1년 6개월 초과 **2년 이내** | 10% |

- 기한 후 신고 감면: 1개월 이내 50%, 1~3개월 30%, 3~6개월 20%
- 예정신고 미이행 후 확정신고기한까지 신고 시: 무신고 가산세액의 **50%** 감면

## 6. 부가세

- 양도소득세 **감면세액의 20%**: 농어촌특별세 (부가세)
- 양도소득세 납부세액: 지방소득세 (독립세, 별도 과세)

## 핵심 암기 포인트

- 예정신고: 양도일이 속하는 달의 **말일부터 2개월** 이내 (부담부증여는 3개월)
- 확정신고: 다음 연도 **5월 1일~5월 31일**
- 분납 요건: 납부세액 **1,000만원 초과** 시, 납부기한 후 **2개월 이내**
- 분납 한도: 2,000만원 이하는 1,000만원 초과분, 2,000만원 초과는 **50% 이하**
- 무신고가산세: 일반 **20%**, 부정 **40%**
- 과소신고가산세: 일반 **10%**, 부정 **40%**
- 납부지연가산세: 1일 **0.022%** (10만분의 22)
- 가산세 이중부과 금지: 예정신고분에 부과되면 확정신고분에 미부과

## OX 퀴즈

**Q1.** 양도소득세 예정신고기한은 양도일이 속하는 달의 말일부터 3개월 이내이다. → **X** (2개월 이내, 3개월은 부담부증여의 경우)

**Q2.** 양도차손이 발생한 경우에는 예정신고를 하지 않아도 된다. → **X** (양도차손 발생 시에도 예정신고 하여야 한다)

**Q3.** 확정신고기한은 다음 연도 5월 1일부터 5월 31일까지이다. → **O**

**Q4.** 예정신고를 한 자는 모든 경우 확정신고를 하지 않을 수 있다. → **X** (누진세율 적용대상 2회 이상 예정신고 등의 경우 확정신고 필수)

**Q5.** 납부할 세액이 800만원인 경우 분할납부가 가능하다. → **X** (1,000만원 초과 시에만 가능)

**Q6.** 부정한 방법으로 무신고한 경우 가산세는 무신고 납부세액의 40%이다. → **O**

**Q7.** 납부지연가산세의 일일 이자율은 0.022%이다. → **O** (10만분의 22)

**Q8.** 예정신고 관련 무신고가산세가 부과된 경우, 확정신고 관련으로도 무신고가산세를 부과할 수 있다. → **X** (이중부과 금지)

**Q9.** 양도소득세 분할납부는 납부기한 경과 후 3개월 이내에 하여야 한다. → **X** (2개월 이내)

**Q10.** 법정신고기한 후 1개월 이내에 수정신고하면 과소신고가산세의 90%를 감면받는다. → **O**

## 함정 노트

- ⚠️ **예정신고 2개월 vs 부담부증여 3개월**: 일반 양도는 양도일이 속하는 달의 말일부터 **2개월**, 부담부증여의 채무부분은 **3개월**
- ⚠️ **양도차손도 예정신고 필수**: 양도차익이 없거나 양도차손이 발생해도 예정신고 의무 있음
- ⚠️ **분납 기준 1,000만원 "초과"**: 1,000만원 이하이면 분납 불가, 1,000만원을 "초과"하여야 함
- ⚠️ **가산세 이중부과 금지**: 예정신고에서 가산세 부과 → 확정신고에서 같은 부분 가산세 미부과. 그러나 예정신고에서 가산세가 부과되지 않은 부분은 확정신고에서 부과 가능
- ⚠️ **법정기한 내 예정신고 납부해도 세액공제 없음**: 과거에는 신고세액공제가 있었으나 현행법에서는 폐지됨
