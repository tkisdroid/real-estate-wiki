# 원가법 (원가방식)

> 📖 상위 문서: [[감정평가3방식]]

## 1. 기본 공식

$$적산가액 = 재조달원가 - 감가수정액$$

- 원가법은 대상물건의 **비용성**에 착안한 감정평가 방식
- 기본 원칙: **대체의 원칙**
- 주로 **건물, 특수부동산** 등에 적합

## 2. 재조달원가

대상물건을 **기준시점에 다시 조달**하는 데 드는 비용

### 재조달원가의 유형

| 유형 | 내용 |
|------|------|
| **복제원가(복성식)** | **동일한** 자재·공법으로 **재생산**하는 비용. 기존 건물과 동일한 복제 |
| **대체원가(대치식)** | **동등한 효용**의 대체물을 만드는 비용. 현대적 자재·공법 사용 가능 |

- 복제원가: 구식 자재·공법까지 반영 → 보수적
- 대체원가: 현대적 기준 → 실용적

### 재조달원가 산정방법

| 방법 | 내용 |
|------|------|
| **직접법** | 대상 건물의 공사비를 직접 산정 |
| **간접법** | 유사 건물의 공사비를 이용하여 간접적으로 산정 |

## 3. 감가수정

### 감가수정의 유형

| 유형 | 원인 | 치유 가능성 |
|------|------|------------|
| **물리적 감가** | 마모, 파손, 노후화, 자연적 풍화 | 치유가능 / 치유불가능 |
| **기능적 감가(기능적 퇴화)** | 설계불량, 설비진부화, 기능적 부적합 | 치유가능 / 치유불가능 |
| **경제적 감가(외부적 퇴화)** | 외부환경 변화, 시장 침체, 입법 변화 | **치유불가능만** |

> **중요**: 토지는 영속성으로 인해 **물리적 감가상각 적용 안 됨** (경제적 감가는 고려 가능)

### 감가수정의 방법

| 방법 | 내용 |
|------|------|
| **내용연수법** | 정액법(매기 균등 감가), 정률법(매기 일정 비율 감가) |
| **관찰감가법** | 실제 상태를 관찰하여 감가액 판단 |
| **분해법** | 물리적·기능적·경제적으로 분해하여 각각 산정 후 합산 |
| **시장추출법** | 시장 거래자료에서 감가율 추출 |

### 경제적 감가의 특수성

- 대상 부동산 **외부**의 요인에 의한 감가
- **치유불가능**만 존재 (물리적·기능적과 차이)
- 부동산의 **영속성**은 감정평가 시 경제적 감가를 고려하는 근거

## 4. 적산법 (임대료 평가)

$$적산임대료 = 기초가액 \times 기대이율 + 필요경비$$

- 원가방식의 임대료 평가 방법
- 기초가액: 대상 부동산의 가격
- 기대이율: 투자자가 기대하는 수익률

---

## 핵심 암기 포인트

- 적산가액 = **재조달원가 - 감가수정액**
- 재조달원가: **복제원가(복성식)** = 동일 재생산 / **대체원가(대치식)** = 동등 효용
- 감가 3유형: **물리적·기능적·경제적**
- **경제적 감가만 치유불가능**만 존재 (외부적 퇴화)
- 토지 = 물리적 감가상각 **적용 안 됨** (영속성)
- 감가수정 4방법: 내용연수법, 관찰감가법, 분해법, 시장추출법
- 적산임대료 = **기초가액 x 기대이율 + 필요경비**

## OX 퀴즈

**Q1.** 재조달원가에서 감가수정액을 더하면 적산가액이 된다. → **X** (**차감**하면 적산가액)

**Q2.** 복제원가란 동등한 효용의 대체물을 만드는 비용이다. → **X** (동등한 효용 = **대체원가(대치식)**. 복제원가 = **동일한** 자재·공법으로 재생산)

**Q3.** 경제적 감가는 치유 가능한 경우도 있다. → **X** (경제적 감가는 **치유불가능만** 존재)

**Q4.** 토지에는 물리적 감가상각이 적용된다. → **X** (토지는 영속성으로 물리적 감가상각 **적용 안 됨**)

**Q5.** 기능적 감가는 설계불량이나 설비진부화로 인해 발생한다. → **O**

**Q6.** 분해법은 물리적·기능적·경제적 감가를 분해하여 산정하는 방법이다. → **O**

**Q7.** 원가법은 수익성에 착안한 감정평가 방식이다. → **X** (**비용성**에 착안. 수익성은 **수익환원법**)

## 함정 노트

- ⚠️ "재조달원가 + 감가수정액 = 적산가액" → 재조달원가 **-** 감가수정액 = 적산가액
- ⚠️ "복제원가 = 동등한 효용" → 복제원가 = **동일한** 재생산. 동등한 효용 = **대체원가**
- ⚠️ "경제적 감가: 치유 가능" → 경제적 감가는 **치유불가능만** 존재
- ⚠️ "토지도 물리적 감가" → 토지는 물리적 감가 **미적용** (경제적 감가만 고려)
- ⚠️ "영속성 = 원가방식의 근거" → 영속성은 **경제적 감가** 고려의 근거 (원가방식 전체의 근거는 아님)
