# 튀넨(J.H. von Thunen)의 위치지대설

> 📖 상위 문서: [[지대이론]]

## 1. 의의

- **1826년** '**고립국(The Isolate State)**' 이론에서 제시
- 리카도가 불완전하게 다룬 **위치의 문제**를 전면적으로 다룸
- **비옥도는 동일하다**고 가정 (리카도와 핵심 차이)
- 도시로부터의 **거리에 따라** 농작물의 재배형태가 달라지는 것은 **수송비의 차이**에서 비롯
- 수송비의 차이가 **지대의 차이**를 가져온다

## 2. 지대 산정 공식

$$지대(초과이윤, 잉여) = 매상고 - 생산비 - 수송비(단위당\ 교통비 \times 거리)$$

- 리카도 공식에 **수송비** 개념 추가
- 지대 발생 요인 = **거리(수송비)**: 수송비 절약이 지대화

## 3. 핵심 내용

### 거리와 지대의 관계

- 중심지에서 **거리가 멀수록** → 수송비 **증가** → 지대 **하락**
- 중심지에서 **거리가 가까울수록** → 수송비 **감소** → 지대 **상승**
- **한계지에서는 지대가 발생하지 않으며**, 곡물가격이 지대를 결정 (리카도와 동일)

### 동심원적 토지이용체계

- 시장(도시)을 중심으로 **가장 많은 지대를 지불할 능력이 있는 작물**이 선정
- 시장에 **가까운 지역**: **집약적** 농업 (고부가가치, 부패성 작물)
- 시장에서 **먼 지역**: **조방적** 농업 (곡물, 목축 등)
- 이러한 지대지불능력에 따른 토지이용을 **동심원적 형태**로 설명

### 한계지대곡선의 특성

- **우하향**하면서 **원점을 향하여 볼록**한 형태
- **집약적** 생산방식의 한계지대곡선 → 기울기 **가파름**
- **조방적** 생산방식의 한계지대곡선 → 기울기 **완만**
- 토지의 가치와 토지이용의 집약도는 **중심도시에 가까울수록 증가**

## 4. 튀넨 이론의 영향

- 동심원적 토지이용체계 → **버제스의 동심원이론**에 영향
- 한계지대곡선 → **알론소의 입찰지대곡선**의 토대 제공
- 농촌 토지이용구조를 도시에 적용한 최초의 시도를 가능하게 함

## 5. 리카도와의 비교

| 구분 | 리카도 | 튀넨 |
|------|--------|------|
| 지대 공식 | 매상고 - 생산비 | 매상고 - 생산비 - **수송비** |
| 비옥도 | 차이 있음 (핵심 변수) | **동일** 가정 |
| 지대 결정 요인 | 비옥도 차이 | **거리(수송비)** |
| 한계지 지대 | 미발생 | 미발생 |
| 지대 성격 | 잉여 | 잉여 |
| 토지이용 형태 | - | **동심원적** 체계 |

## 핵심 암기 포인트

- 튀넨 = **위치지대설(입지교차지대설)**, 1826년, **고립국**
- 비옥도 **동일** 가정 → **거리(수송비)**가 지대 결정
- 지대 = **매상고 - 생산비 - 수송비(단위당 교통비 x 거리)**
- 한계지 지대 **미발생**, 곡물가격이 지대 결정 (리카도와 동일)
- **동심원적** 토지이용: 가까우면 집약적, 멀면 조방적
- 한계지대곡선: **우하향, 원점 볼록**, 집약적 → 기울기 가파름
- **버제스 동심원이론**과 **알론소 입찰지대이론**에 영향

## 함정 노트

- ⚠️ "튀넨: 비옥도 차이로 지대 설명" → 비옥도 **동일** 가정, **거리(수송비)**가 핵심
- ⚠️ "튀넨의 지대공식에 수송비 없음" → 수송비 **포함** (매상고 - 생산비 - 수송비). 수송비 없는 것은 **리카도**
- ⚠️ "한계지대곡선: 우상향" → **우하향**, 원점 **볼록**
- ⚠️ "시장 가까이 = 조방적 농업" → 시장 가까이 = **집약적**, 시장에서 멀면 = 조방적
- ⚠️ "튀넨: 한계지에서 지대 발생" → 한계지 지대 **미발생** (발생은 **마르크스**)
