# 확인·설명사항 (영 제21조 제1항)

## 1. 확인·설명의 취지

- 개업공인중개사는 부동산거래 전문가로서 중개물건의 각종 사항을 조사·확인하여 의뢰인에게 설명할 책임
- **비전문가인 의뢰인의 기대에 부응**하여 필요한 사항을 설명하는 것이 핵심
- 확인·설명사항은 개업공인중개사의 **최소한의 의무**를 규정한 것 (영 제21조 제1항)

## 2. 설명 상대방

- **권리를 취득하고자 하는 의뢰인**
  - 매매: **매수인**
  - 교환: **쌍방** (양측 모두 권리 취득)
  - 임대차: **임차인**

## 3. 설명 방법

- **성실·정확하게 설명**하고 **근거자료를 제시** (법 제25조 제1항)
- 근거자료: 등기사항증명서, 신탁원부, 건축물대장, 토지대장, 임야대장 등 각종 공부
- 위반 시: **500만원 이하 과태료**

## 4. 설명 시기

- 중개의뢰를 받은 후 ~ **중개가 완성되기 전** (계약 성립 전)
- 거래계약 체결 후에는 **확인설명서를 작성**하여 양 당사자에게 교부

## 5. 확인·설명사항 10항목 상세

### ① 기본사항
- 중개대상물의 종류·소재지·지번·지목·면적·용도·구조·건축연도 등

### ② 권리관계
- 소유권·전세권·저당권·지상권·임차권 등
- **매도인의 진정한 권리자 확인의무** 포함 (대판 1993.5.11)
  - 부동산등기부와 주민등록증 등에 의하여 조사·확인
  - 필요 시 **등기권리증** 소지 여부·내용 확인

### ③ 거래예정금액·중개보수·실비
- 거래예정금액, **중개보수 및 실비의 금액과 산출내역**

### ④ 공법상 거래규제·이용제한
- 토지이용계획, 공법상 거래규제 및 이용제한에 관한 사항

### ⑤ 시설물 상태
- 수도·전기·가스·소방·열공급·승강기·배수 등 시설물의 상태

### ⑥ 벽면·바닥면·도배 상태
- 건축물 내부 상태에 관한 사항

### ⑦ 환경조건
- 일조·소음·진동 등 환경조건
- 비선호시설은 자료요구 항목에서 제외

### ⑧ 입지조건
- 도로·대중교통 연계성, 시장·학교 근접성, 지형 등

### ⑨ 조세
- 권리 취득 시 부담할 **조세의 종류 및 세율**

### ⑩ 주택임대차 중개 시 추가 설명사항
| 항목 | 내용 |
|---|---|
| ㉠ 임대인 정보제시의무 | 확정일자 부여현황, 보증금 중 일정액 보호 |
| ㉡ 전입세대확인서 | 열람 또는 교부에 관한 사항 |
| ㉢ 임대보증금 보증 | 민간임대주택특별법에 의한 민간임대주택에 한함 |
| ㉣ 관리비 | 금액과 산출내역 |

## 6. 물건상태 자료요구 (법 제25조 제2항)

### 요구 대상
- **매도의뢰인·임대의뢰인** 등 권리를 이전하고자 하는 의뢰인

### 자료요구의 성격
- **권리가 아닌 의무가 아님** (요구하지 않아도 법 위반 아님)
- 요구 범위: 중개대상물의 **상태**에 관한 자료만

### 의뢰인 불응 시
- 불응 사실을 매수·임차 의뢰인에게 **설명**
- 확인·설명서에 **기재** (영 제21조 제2항)

### 자료 제출 후에도
- 개업공인중개사는 다시 물건상태 등에 대해 **직접 확인하고 설명**해야 손해배상 책임을 면할 수 있음

## 7. 신분증명서 제시 요구
- 중개업무 수행 시 필요한 경우 의뢰인에게 **주민등록증**(모바일 포함) 등 신분 확인 증표 제시 요구 가능

## 8. 확인·설명 주체

| 주체 | 의무 여부 |
|---|---|
| **개업공인중개사** | 의무 |
| **소속공인중개사** | 할 수 있음 (의무 아님) |
| **중개보조원** | 불가 (대여행위로 처벌) |

## 9. 위반 시 제재

| 위반 내용 | 개업공인중개사 | 소속공인중개사 |
|---|---|---|
| 확인·설명 불이행/부실 | **500만원 이하 과태료** | 자격정지 |
| 근거자료 미제시 | **500만원 이하 과태료** | - |

## 10. 주요 판례

| 판례 | 핵심 내용 |
|---|---|
| 대판 1993.5.11 | 매도인의 진정한 권리자 확인: 등기부·주민등록증 조사, 필요시 등기권리증 확인 |
| 대판 1992.2.11 | 선량한 관리자 주의와 신의성실로 진정한 권리자 동일인 여부 확인 |
| 대판 2002.2.5 | 무료 중개라도 확인·설명의무 감경 안 됨 |
| 수원지법 1997.8.26 | 잔금 지급 전 등기사항증명서 재확인 의무 |
| 서울남부지원 2000.2.11 | 선순위 임차인 미설명 → 손해배상 |

## 핵심 암기 포인트

1. 설명 상대방: **권리를 취득하고자 하는 의뢰인** (매수인, 교환 시 쌍방, 임차인)
2. 설명 시기: 중개의뢰 후 ~ **중개완성 전**
3. 설명 방법: 성실·정확 + **근거자료 제시**
4. 설명사항: **10항목** (기본-권리-보수-공법-시설-벽면-환경-입지-조세-임대차추가)
5. 물건상태 자료요구: 의무 아님, **상태에 관한 자료**만
6. 의뢰인 불응 시: 매수·임차 의뢰인에게 **설명 + 기재** 의무 발생
7. 자료 제출 후에도 **직접 확인** 필요
8. 위반 시: **500만원 이하 과태료**

## OX 퀴즈

**Q1.** 확인·설명의 상대방은 매도인과 매수인 쌍방이다.
> **X** — 확인·설명 상대방은 **권리를 취득하고자 하는 의뢰인**(매수인, 임차인 등)이다.

**Q2.** 교환의 경우 확인·설명은 한쪽 당사자에게만 하면 된다.
> **X** — 교환의 경우 **쌍방** 모두 권리를 취득하므로 양쪽 모두에게 설명해야 한다.

**Q3.** 개업공인중개사는 매도의뢰인에게 물건상태에 관한 자료를 반드시 요구하여야 한다.
> **X** — 자료요구는 **의무가 아니며**, 요구하지 않아도 법 위반이 아니다.

**Q4.** 매도의뢰인이 자료요구에 불응하면 개업공인중개사는 중개를 거절해야 한다.
> **X** — 불응 사실을 매수·임차 의뢰인에게 **설명하고 확인설명서에 기재**하면 된다.

**Q5.** 무료로 중개한 경우 확인·설명의무가 감경된다.
> **X** — 무료 중개라도 확인·설명의무가 **감경되지 않는다** (대판 2002.2.5).

**Q6.** 소속공인중개사도 확인·설명의 의무가 있다.
> **X** — 소속공인중개사는 확인·설명을 할 수 **있으나**, 의무는 아니다.

**Q7.** 확인·설명의무 위반 시 개업공인중개사에게는 500만원 이하의 과태료가 부과된다.
> **O** — 정확하다.

**Q8.** 의뢰인이 자료를 제출하였으면 개업공인중개사는 별도로 직접 확인할 필요가 없다.
> **X** — 자료 제출 후에도 **직접 확인하고 설명**해야 손해배상 책임을 면할 수 있다.

## 함정 노트

| 함정 포인트 | 정확한 내용 |
|---|---|
| 설명 상대방 = 쌍방 | 상대방은 **권리취득 의뢰인**만 (매도인 제외, 교환은 쌍방) |
| 자료요구 = 의무 | 자료요구는 **의무 아님** (요구 안 해도 법 위반 아님) |
| 자료요구 범위 = 전부 | **상태**에 관한 자료만 요구 가능 |
| 자료 제출 → 확인 면제 | 자료 제출 후에도 **직접 확인** 필요 |
| 무료 중개 → 의무 감경 | 무료 중개라도 **감경 안 됨** |
| 중개보조원도 설명 가능 | 중개보조원의 설명은 **대여행위로 처벌** |
| 설명 시기 = 계약 후 | 설명은 **중개완성 전**까지 (계약서 작성은 계약 후) |
