# 토지이동 - 합병

## 1. 의의

합병이란 지적공부에 등록된 **2필지 이상**을 **1필지**로 합하여 등록하는 것을 말한다.

## 2. 합병 신청

### 임의적 합병
- 토지소유자가 지적소관청에 합병 신청

### 의무적 합병 (60일 이내 신청)
다음의 토지로서 합병하여야 할 토지가 있으면 사유 발생일부터 **60일 이내** 신청:
- 공동주택 부지, 도로, 제방, 하천, 구거, 유지, 공장용지
- 학교용지, 철도용지, 수도용지, 공원, 체육용지 등

## 3. 합병 제한 사유 (지적법상 — 10가지)

합병할 수 **없는** 경우:

| 번호 | 합병 제한 사유 | 키워드 |
|:----:|---------------|--------|
| 1 | 합병 대상 토지의 **소유자가 서로 다른** 경우 | 소유자 다름 |
| 2 | 합병 대상 토지가 **등기된 토지와 등기되지 않은** 토지인 경우 | 등기·미등기 혼합 |
| 3 | 합병 대상 토지의 **지적도·임야도의 축척이 서로 다른** 경우 | 축척 다름 |
| 4 | 합병 대상 토지의 **지번부여지역이 서로 다른** 경우 | 지번부여지역 다름 |
| 5 | 합병 대상 토지의 **지목이 서로 다른** 경우 | 지목 다름 |
| 6 | 합병 대상 각 필지가 **서로 연접하지 않는** 경우 | 비연접 |
| 7 | 합병 대상 토지에 다음의 등기 **외의** 등기가 있는 경우 (아래 상세) | 등기 제한 |
| 8 | 각 필지의 지목은 같으나 **일부 토지 용도가 다르게** 되어 분할대상인 경우 | 용도 다름 |
| 9 | 소유자별 **공유지분이 다르거나** 소유자의 **주소가 서로 다른** 경우 | 지분·주소 다름 |
| 10 | 구획정리·경지정리·축척변경 시행 지역의 토지와 **그 지역 밖** 토지인 경우 | 사업지역 경계 |

### 제7호 상세: 합병 가능한 등기 (이것만 있으면 합병 가능)

| 합병 가능한 등기 |
|-----------------|
| **소유권** 등기 |
| **지상권** 등기 |
| **전세권** 등기 |
| **임차권** 등기 |
| **승역지에 대한 지역권** 등기 |

> 위 등기 **외의** 등기(저당권·가등기·가처분 등)가 있으면 합병 **불가**

### 제7호 예외: 저당권·신탁등기가 있어도 합병 가능한 경우

| 예외 | 조건 |
|------|------|
| **저당권** | 합병 대상 토지 **전부**에 대한 등기원인·연월일·접수번호가 **동일**한 저당권 |
| **신탁등기** | 합병 대상 토지 **전부**에 대한 등기사항이 **동일**한 신탁등기 |

### 제8호 예외
- 합병신청과 **동시에 분할신청**을 하는 경우에는 합병 가능

## 4. 합병에 따른 지번부여

### 원칙
- **선순위 지번**: 본번이 있으면 본번 중 선순위 지번

### 예외
- 토지소유자가 **건축물이 위치한 지번**을 합병 후 지번으로 **신청**하면 그 지번 부여

## 5. 처리절차

| 항목 | 내용 |
|------|------|
| **측량** | 불요 (경계 말소 + 면적 합산) |
| **조사** | 합병요건 충족 여부 확인을 위한 **토지이동조사** 실시 |
| **대장** | 합병 후 1개 대장 유지, 나머지 대장 **폐쇄** |
| **면적** | 각 필지 면적의 **합산** |

## 핵심 암기 포인트

- **2필지 이상 → 1필지**
- 합병 제한 10가지: 소유자·등기·축척·지번부여지역·지목·연접·등기·용도·지분주소·사업지역
- 합병 가능 등기: **소유권·지상권·전세권·임차권·승역지 지역권**만 (저당권 등은 원칙 불가)
- 저당권 예외: 전 필지에 **동일 등기원인·접수번호**의 저당권은 합병 가능
- 측량 **불요** (면적 합산만)
- 지번: **선순위** 지번 (건축물 지번 신청 가능)

## OX 퀴즈

**Q1.** 소유자가 서로 다른 토지는 합병할 수 있다. → **X** (합병 제한)

**Q2.** 지목이 서로 다른 토지는 합병할 수 없다. → **O**

**Q3.** 합병 시 지적측량을 실시하여야 한다. → **X** (측량 불요, 토지이동조사만 실시)

**Q4.** 저당권이 설정된 토지는 절대로 합병할 수 없다. → **X** (전 필지에 동일 저당권이면 가능)

**Q5.** 합병 대상 토지가 서로 연접하지 않아도 합병할 수 있다. → **X** (연접해야 함)

**Q6.** 소유권·지상권·전세권 외의 등기가 있는 토지는 합병할 수 없다. → **X** (임차권·승역지 지역권도 합병 가능)

**Q7.** 각 필지의 지목은 같지만 일부 용도가 다른 경우에도 합병 불가이다. → **X** (합병과 동시에 분할 신청하면 가능)

**Q8.** 합병 후 지번은 건축물이 위치한 지번으로 부여해야 한다. → **X** (원칙은 선순위 지번, 건축물 지번은 소유자 신청 시 예외)

## 함정 노트

- **합병 가능 등기 5종 암기**: "소지전임승" — 소유권, 지상권, 전세권, 임차권, 승역지 지역권. **요역지 지역권은 해당 안 됨** (승역지만)
- **저당권 합병 예외**: "전 필지에 동일" 조건 — 1필지라도 저당권이 없으면 합병 불가
- **공유지분 다름 = 합병 불가**: 같은 소유자라도 공유지분이 다르면 합병 제한
- **축척 다름**: 합병 불가이지만, 축척변경을 먼저 한 후 합병하면 가능
- **분할과의 관계**: 합병은 측량 불요, 분할은 측량 필요 (원칙). 합병 전 경계대로 분할 시 측량 불요
