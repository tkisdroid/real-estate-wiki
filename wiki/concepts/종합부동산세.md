# 종합부동산세

종합부동산세는 고액 부동산 보유자에 대하여 [[재산세]]에 추가로 부과하는 **국세**(보유세)이다. 2005년 신설되었으며, 매년 **1~2문항** 출제된다.

## 1. 개요 및 특징

- **국세**, 보통세
- 과세기준일: 매년 **6월 1일**
- **보유세 이원화**: 1차 [[재산세]] (시·군·구) → 2차 종합부동산세 (국세)
- 원칙: **정부부과과세** (선택에 의해 신고납부 가능)
- **초과누진세율** 적용 (인세 성격)
- **분할납부** 가능
- **세부담 상한** 제도 있음
- 당해 재산에 부과된 조세 → 저당권보다 **항상 우선** 징수

## 2. 과세대상 및 납세의무자

### 주택

과세기준일 현재 주택분 [[재산세]] 납세의무자로서, 국내 소재 주택의 **공시가격 합계액**이 다음 금액을 초과하는 자:

| 구분 | 공제금액 |
|------|----------|
| **1세대1주택자** (단독명의) | **12억원** |
| **일반** (개인) | **9억원** |
| **법인** | 공제 없음 (0원) |

### 토지

| 구분 | 공제금액 |
|------|----------|
| **종합합산과세대상** 토지 | **5억원** |
| **별도합산과세대상** 토지 | **80억원** |

> **분리과세대상 토지**는 종합부동산세 **과세 제외**

## 3. 주택에 대한 종합부동산세

### 과세표준

**(공시가격 합계액 - 공제금액) × 공정시장가액비율(60%)**

### 세율 (개인)

| 과세표준 | 세율 (일반) |
|----------|-------------|
| 3억원 이하 | **0.5%** |
| 3억원 ~ 6억원 | 150만원 + 초과액의 **0.7%** |
| 6억원 ~ 12억원 | 360만원 + 초과액의 **1.0%** |
| 12억원 ~ 25억원 | 960만원 + 초과액의 **1.3%** |
| 25억원 ~ 50억원 | 2,650만원 + 초과액의 **1.5%** |
| 50억원 ~ 94억원 | 6,400만원 + 초과액의 **2.0%** |
| 94억원 초과 | 1억5,200만원 + 초과액의 **2.7%** |

### 세율 (법인)

| 구분 | 세율 |
|------|------|
| **2주택 이하** | **2.7%** (비례세율) |
| **3주택 이상** | **5.0%** (비례세율) |

### 재산세액 공제 (이중과세 방지)

종합부동산세 과세표준에 해당하는 금액에 대하여 주택분 [[재산세]]로 부과된 세액을 공제한다.

**세액 계산 흐름 (1세대1주택자):**

```
① (공시가격 합계 - 12억원) × 60% = 과세표준
② 과세표준 × 세율 = 종합부동산세액
③ ② - 재산세 공제액 = 산출세액
④ ③ - 연령별 공제 - 장기보유공제 = 최종 납부세액
```

### 1세대1주택자 세액공제

**연령별 공제** (단독소유, 만 60세 이상):

| 연령 | 공제율 |
|------|--------|
| 만 60세 이상 ~ 65세 미만 | **20%** |
| 만 65세 이상 ~ 70세 미만 | **30%** |
| 만 70세 이상 | **40%** |

**장기보유기간별 공제** (단독소유, 5년 이상 보유):

| 보유기간 | 공제율 |
|----------|--------|
| 5년 이상 ~ 10년 미만 | **20%** |
| 10년 이상 ~ 15년 미만 | **40%** |
| 15년 이상 | **50%** |

> 연령별 공제 + 장기보유공제 **합계 한도: 80%**

### 1세대1주택으로 보는 경우

| 유형 | 조건 |
|------|------|
| **일시적 2주택** | 신규주택 취득 후 과세기준일 현재 **3년** 미경과 |
| **상속받은 주택** | 상속개시일부터 **5년** 미경과, 또는 지분율 40% 이하, 또는 지분율 상당 공시가격 6억원(수도권 밖 3억원) 이하 |
| **지방 저가 주택** | 수도권 밖, 공시가격 **3억원 이하** |

### 세부담 상한

해당 연도 총세액(재산세 + 종합부동산세)이 직전 연도 총세액의 **150%**를 초과하는 경우 → 초과세액은 없는 것으로 봄

| 구분 | 세부담 상한 |
|------|-------------|
| 주택 2이하 | **150%** |
| 주택 3이상 | **150%** |
| **법인** | 세부담 상한 **미적용** |

### 공동명의 주택자 특례

- 세대원 1인이 배우자와 **공동으로 1주택**을 소유하고, 다른 주택이 없는 경우
- 신청 시 **1세대1주택자**로 봐서 12억원 공제, 연령별·보유기간별 공제 적용
- 신청기간: **9월 16일 ~ 9월 30일**

## 4. 토지에 대한 종합부동산세

### 종합합산과세대상 토지

**과세표준**: (공시가격 합계 - **5억원**) × 공정시장가액비율(**100%**)

| 과세표준 | 세율 |
|----------|------|
| 15억원 이하 | **1%** |
| 15억원 ~ 45억원 | 1,500만원 + 초과액의 **2%** |
| 45억원 초과 | 7,500만원 + 초과액의 **3%** |

### 별도합산과세대상 토지

**과세표준**: (공시가격 합계 - **80억원**) × 공정시장가액비율(**100%**)

| 과세표준 | 세율 |
|----------|------|
| 200억원 이하 | **0.5%** |
| 200억원 ~ 400억원 | 1억원 + 초과액의 **0.6%** |
| 400억원 초과 | 2억2천만원 + 초과액의 **0.7%** |

### 토지 세부담 상한

직전 연도 총세액의 **150%** 초과 시 초과세액은 없는 것으로 봄

## 5. 부과·징수

| 항목 | 내용 |
|------|------|
| 과세기준일 | 매년 **6월 1일** |
| 납세지 | 개인: 주소지(거소지) / 법인: 본점 소재지 |
| 부과·징수기간 | **12월 1일 ~ 12월 15일** |
| 납세고지서 발부 | 납부기간 개시 **5일 전**까지 |

### 분할납부

| 납부세액 | 분납금액 |
|----------|----------|
| **500만원 초과 ~ 1,000만원 이하** | 500만원 초과분 |
| **1,000만원 초과** | 세액의 **50% 이하** |

분납기한: 납부기한 경과 후 **6개월 이내**

### 부가세: 농어촌특별세

종합부동산세 납부세액의 **20%**

## 6. 재산세와 종합부동산세 비교

| 항목 | [[재산세]] | 종합부동산세 |
|------|----------|--------------|
| 과세주체 | 시·군·구 (지방세) | 세무서장 (국세) |
| 과세기준일 | 6월 1일 | 6월 1일 |
| 과세대상 | 토지, 건축물, 주택, 선박, 항공기 | **주택, 토지** (종합·별도합산) |
| 과세방법 | 보통징수 | 정부부과과세 (선택 신고납부) |
| 세율 | 누진 + 비례 | **초과누진** |
| 물납 | **가능** (1,000만원 초과) | 불가 |
| 분납 | 250만원 초과 | 500만원 초과 |
| 공제 | - | 재산세액, 연령별, 장기보유 |

## 핵심 암기 포인트

- **과세기준일**: 매년 **6월 1일** (재산세와 동일)
- **부과·징수기간**: **12월 1일 ~ 12월 15일**
- **주택 공제금액**: 1세대1주택 12억원, 일반 9억원, 법인 0원
- **토지 공제금액**: 종합합산 5억원, 별도합산 80억원
- **분리과세대상 토지**: 종부세 과세 제외
- **공정시장가액비율**: 주택 60%, 토지 100%
- **주택 개인 세율**: 0.5%~2.7% (7단계)
- **주택 법인 세율**: 2주택 이하 2.7%, 3주택 이상 5.0%
- **토지 종합합산 세율**: 1%~3% (3단계, 공제 5억)
- **토지 별도합산 세율**: 0.5%~0.7% (3단계, 공제 80억)
- **1세대1주택자 세액공제**: 연령별(60세 20%, 65세 30%, 70세 40%) + 장기보유(5년 20%, 10년 40%, 15년 50%). 합계 한도 80%
- **세부담 상한**: 150% (법인 미적용)
- **분할납부**: 500만원 초과 ~ 1,000만원 이하 → 500만원 초과분, 1,000만원 초과 → 50% 이하. 납부기한 후 6개월 이내
- **부가세**: 농어촌특별세 = 납부세액의 20%
- **공동명의 1주택 특례**: 9/16~9/30 신청 → 1세대1주택자로 취급(12억원 공제)
- **물납 불가** (재산세와 다름)

## 함정 노트

- ⚠️ **공제금액 혼동**: 1세대1주택 12억 vs 일반 9억 vs 법인 0원 / 종합합산토지 5억 vs 별도합산토지 80억 — 숫자 혼동 최다 출제
- ⚠️ **분할납부 기한**: 종부세 6개월 vs 재산세 3개월 — 반드시 구분
- ⚠️ **물납**: 재산세 = 가능, 종부세 = 불가 — 보유세끼리 다르므로 주의
- ⚠️ **공정시장가액비율**: 주택 60% vs 토지 100% — 동일 세목 내에서도 다름
- ⚠️ **법인 특이사항**: 공제 0원, 비례세율(2.7%/5.0%), 세부담 상한 미적용 — 개인과 완전히 다른 체계
