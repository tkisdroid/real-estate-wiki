# 거래계약서 작성의무 (법 제26조)

## 1. 개요
- 부동산 거래의 고액화·권리관계 복잡성 → 전문가인 개업공인중개사가 작성
- **법정서식 없음**: 양식은 자유, 다만 필요적 기재사항 준수

## 2. 작성 주체

| 주체 | 작성 가능 여부 |
|---|---|
| **개업공인중개사** | O (의무) |
| 소속공인중개사 | O (의무 아님) |
| **중개보조원** | **X** (대여행위로 처벌) |
| 개업공인중개사의 구술에 의한 고용인 대서 | O (신체적 장애 등의 경우) |

## 3. 필요적 기재사항 (영 제22조 제1항)

| 번호 | 기재사항 |
|---|---|
| ① | **거래당사자의 인적사항** |
| ② | 물건의 표시 |
| ③ | **계약일** |
| ④ | 거래금액·계약금액 및 그 지급일자 등 지급에 관한 사항 |
| ⑤ | 물건의 **인도일시** |
| ⑥ | **권리이전의 내용** |
| ⑦ | 계약의 조건·기한 (있는 경우) |
| ⑧ | **확인·설명서 교부일자** |
| ⑨ | 그 밖의 약정내용 |

- 위반 시: **업무정지**

## 4. 서명 및 날인 (법 제26조 제2항)

### 서명·날인 의무자
| 대상 | 의무 |
|---|---|
| 개업공인중개사 | **서명 및 날인** (법인: 대표자, 분사무소: 책임자) |
| 중개행위를 한 소속공인중개사 | **함께 서명 및 날인** |
| 중개보조원 | 의무 없음 (서명·날인 해도 되고 안 해도 됨) |

- 위반 시: 개업공인중개사 → **업무정지**, 소속공인중개사 → **자격정지**

## 5. 교부 및 보관 (법 제26조 제1항, 영 제22조 제2항)

| 항목 | 내용 |
|---|---|
| 교부 | 거래당사자에게 교부 |
| **보관** | **5년간 보관** |
| 면제 | 공인전자문서센터 보관 시 보관의무 없음 |
| 위반 시 | **업무정지** |

## 6. 거짓기재 및 이중계약서 금지 (법 제26조 제3항)

- **거래금액 등 거짓 기재 금지**
- 서로 다른 **2이상의 거래계약서 작성 금지**
- 위반 시: **임의 등록취소** 또는 업무정지
- 소속공인중개사: **자격정지 6개월**
- 형법상: 경합범 시 **벌금형 분리 선고** (이 법 위반 300만원 이상 구분 위해)

## 7. 거래계약서 서식
- 현재 **법정서식 없음**
- 국토교통부장관이 표준 서식을 정해 **사용 권장** 가능 (영 제22조 제3항)
- 아직 표준서식 미제정
