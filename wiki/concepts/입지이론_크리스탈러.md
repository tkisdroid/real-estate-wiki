# 크리스탈러(W. Christaller)의 중심지이론

> 📖 상위 문서: [[입지이론]]

## 1. 의의

- **재화의 도달거리**와 **최소요구치**의 관계를 통해 **중심지의 형성과정**을 설명한 이론
- 중심지 규모의 크기에 따라 **상권의 규모가 달라진다**는 것을 실증

## 2. 핵심 용어

| 용어 | 정의 |
|------|------|
| **중심지(central place)** | 서로 다른 재화를 제공하는 상점들이 서로 밀집되어 있는 지역. 배후지에 재화·서비스 제공 |
| **배후지(상권)** | 고객이 존재하는 실질적 권역. 점포 매출이 발생하는 공간적 넓이 |
| **최소요구치(threshold)** | 기업 존립을 위한 **최소한의 수요 수준(판매수준)**. 판매자가 정상이윤을 얻는 만큼의 충분한 소비자를 포함하는 경계까지의 거리 |
| **재화의 도달거리(range)** | 중심지 기능에 대한 수요가 **'0'이 되는 지점**까지의 거리. 고객에게 재화를 제공할 수 있는 최대한의 거리 |

## 3. 재화의 도달거리와 최소요구치의 관계

| 관계 | 결과 |
|------|------|
| **재화의 도달거리 > 최소요구치** | 기업 **운영 가능** (존립) |
| **재화의 도달거리 < 최소요구치** | 기업 **도산** |

- 기업이 운영되기 위해서는 최소요구치가 항상 **재화의 도달범위 내**에 있어야 함
- 최소요구치가 재화의 도달범위보다 클 경우 → 충분한 수요 확보 불가 → **도산**

## 4. 중심지의 계층성

### 고차중심지 vs 저차중심지

| 구분 | 고차중심지 | 저차중심지 |
|------|-----------|-----------|
| 규모 | **큼** | 작음 |
| 중심지 간 간격 | **넓음** | 좁음 |
| 수 | **적음** | 많음 |
| 기능 | **다양**하고 전문화 | 제한적 |
| 이용빈도 | 적음 | 많음 |

- 중심지 분포 = **피라미드형** (저차중심지 많고, 고차중심지 적음)
- 고차중심지는 저차중심지들을 **지배**하여 시장지역을 포섭

### 상권의 형태

- **단일 중심지**: 원형의 상권
- **다수 중심지**: **정육각형** 형태의 상권

## 5. 중심지 배분의 3원리

| 원리 | 내용 |
|------|------|
| **시장원리(K=3)** | 소비자의 시장접근성 극대화 |
| **교통원리(K=4)** | 교통망의 효율적 배치 |
| **행정원리(K=7)** | 행정적 통제의 효율성 |

- 시장·교통·행정원리 등에 따라 중심지의 수와 모형이 영향을 받음

## 6. 배후지의 특성

- 배후지 범위는 **상품가격**과 소비자의 **교통비**에 의해 결정
- 배후지 인구밀도 높고, 지역면적 넓고, 소득수준 높으면 좋은 배후지 조건
- 배후지 범위는 **가변적**이므로 끊임없이 변함
- 미래의 배후지분석도 매우 중요

## 핵심 암기 포인트

- 크리스탈러 = **중심지이론**, 재화의 도달거리와 최소요구치의 관계
- **재화의 도달거리 > 최소요구치** → 기업 존립 가능
- **재화의 도달거리 < 최소요구치** → 기업 **도산**
- 최소요구치 = 기업 존립 **최소 수요 수준** / 재화의 도달거리 = 수요 **0이 되는 거리**
- 고차중심지: 규모 크고, 간격 넓고, 수 적고, 다양한 기능
- 중심지 분포 = **피라미드형** (계층성)
- 단일 중심지: 원형 / 다수 중심지: **정육각형**
- 3원리: **시장(K=3)·교통(K=4)·행정(K=7)**

## OX 퀴즈

**Q1.** 크리스탈러의 중심지이론에서 재화의 도달거리가 최소요구치보다 작으면 기업이 존립한다. → **X** (도달거리 > 최소요구치여야 존립 가능, 작으면 **도산**)

**Q2.** 최소요구치란 중심지로부터 어느 기능에 대한 수요가 0이 되는 곳까지의 거리이다. → **X** (수요가 0이 되는 거리는 **재화의 도달거리**. 최소요구치는 최소한의 **수요 수준**)

**Q3.** 고차중심지일수록 중심지간의 거리가 더 멀고 규모가 크다. → **O**

**Q4.** 다수의 중심지가 분포하는 경우 각 중심지의 상권은 정사각형으로 형성된다. → **X** (**정육각형**)

**Q5.** 저차중심지에서 고차중심지로 갈수록 중심지의 수는 증가한다. → **X** (고차중심지로 갈수록 수 **감소** → 피라미드형)

**Q6.** 배후지의 범위는 상품가격과 교통비에 의해 결정된다. → **O**

**Q7.** 재화의 도달거리란 중심지 기능에 대한 수요가 '1'이 되는 지점까지의 거리이다. → **X** (수요가 **'0'**이 되는 지점까지의 거리)

## 함정 노트

- ⚠️ "최소요구치 = 수요가 0이 되는 거리" → 그것은 **재화의 도달거리**. 최소요구치 = **최소 수요 수준**
- ⚠️ "도달거리 < 최소요구치 → 존립" → **도산**. 존립하려면 도달거리 > 최소요구치
- ⚠️ "재화의 도달거리 = 수요가 '1'" → **'0'**이 되는 거리 (제24회 출제)
- ⚠️ "크리스탈러 = 동심원이론" → **중심지이론** (동심원이론은 **버제스**) (제35회 출제)
- ⚠️ "최소요구범위와 최소요구치 혼동" → 최소요구범위 = 판매자가 정상이윤 얻는 소비자 경계까지의 **거리**, 최소요구치 = **수요 수준**
