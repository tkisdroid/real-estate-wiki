# 부동산거래신고 절차

## 1. 신고 대상 거래

### 신고 대상
- **부동산 매매계약** (교환·임대차는 대상 아님)
- 법률에 따른 **부동산 공급계약** (주택법, 도시정비법 등)
- **분양권·입주권의 매매계약**

### 신고 대상 아닌 것
- 교환, 임대차, 증여
- 각종 권리설정 (저당권, 전세권, 지상권, 지역권)
- 입목, 광업재단, 공장재단 거래

## 2. 신고 의무자 구분

| 거래 유형 | 신고의무자 | 비고 |
|---|---|---|
| **직접거래** | 거래당사자 **공동** | 일방 거부 → 단독신고 가능 |
| **중개거래** | **개업공인중개사** | 공동중개 → 공동 신고 |
| 일방이 **국가등** | **국가등** 단독 | |

## 3. 신고 절차

### 시기 및 장소
- **계약체결일부터 30일 이내**
- 부동산 소재지 관할 **시장·군수·구청장**(신고관청)에 제출

### 거래당사자 직접거래 시
1. 부동산거래계약신고서에 **공동 서명·날인**
2. 거래당사자 중 **일방이** 신고관청에 **제출**
3. **신분증명서** 제시 (전자문서: 전자인증)

### 개업공인중개사 중개거래 시
1. 부동산거래계약신고서에 개업공인중개사 **서명·날인**
2. 신고관청에 **제출** (전자문서 가능)
3. 공동중개 시 공동으로 서명·날인
4. **소속공인중개사**가 대행 가능 (전자문서 불가, 방문만 가능)

### 단독신고
- 사유서 + 거래계약서 사본 첨부
- **전자문서 불가** (방문신고만)
- 신고거부자: **500만원 이하 과태료**

### 신고 대행
- 거래당사자의 **위임**을 받은 자가 대행 가능
- 위임장(자필서명, 법인은 인감날인) + 위임인 신분증 사본 첨부

## 4. 신고필증 발급
- 신고관청은 신고 내용 검토 후 **신고필증 발급**
- 소유권이전등기 신청 시 **신고필증 첨부**

## 5. 해제신고 (법 제3조의2)
- 신고한 계약이 **해제·무효·취소**된 경우
- 확정된 날부터 **30일 이내** 공동 신고
- 일방 거부 → 단독신고 가능

## 6. 주택임대차계약 신고 (법 제6조의2)

### 신고 대상
- **보증금 6천만원 초과** 또는 **월차임 30만원 초과**
- 보증금·차임 증감 없이 **기간만 연장하는 갱신은 제외**

### 신고 의무
- 계약 체결일부터 **30일 이내** 소재지 신고관청에 **공동 신고**
- 일방 거부 시 → 계약서 사본 등 첨부하여 **단독신고** 가능

### 의제
- **전입신고** 시 임대차계약 신고한 것으로 봄 (계약서 제출 필요)
- 신고 접수 완료 시 **확정일자 부여** 의제

### 변경·해제 신고 (법 제6조의3)
- 보증금·차임 변경 또는 계약 해제 시
- 확정된 날부터 **30일 이내** 공동 신고

## 7. 위반 시 제재

| 위반 | 제재 |
|---|---|
| 30일 이내 미신고 | **500만원 이하 과태료** |
| **거짓 신고** | **취득가액의 10/100 이하 과태료** |
| 공동신고 거부 | 500만원 이하 과태료 |
| 해제신고 미이행 | 500만원 이하 과태료 |
| 임대차 신고 미이행 | 100만원 이하 과태료 |

## 8. 부동산거래신고 세부 규정

### 신고서 기재사항 (영 별표1)
| 항목 | 내용 |
|---|---|
| 거래당사자 인적사항 | 성명, 주소, 주민등록번호 등 |
| 물건의 표시 | 소재지, 지번, 지목, 면적, 건축물 용도 등 |
| **실제 거래가격** | 계약금, 중도금, 잔금 포함 총 거래가격 |
| 계약일 | 계약체결일 |
| 중개업자 정보 | 중개거래 시 개업공인중개사 인적사항 |
| 자금조달계획 | 투기과열지구 등 해당 시 |

### 허위신고 과태료 세부
- 거래가격을 거짓 신고: 취득가액의 **100분의 10 이하** 과태료
- 개업공인중개사가 거래당사자의 거짓 신고를 **방조**한 경우에도 과태료 부과 가능

### 신고 후 검증 (법 제5조)
- 신고관청은 신고내용의 **적정성을 검증** 가능
- 검증 결과 거짓 의심 시 → 거래당사자에게 **소명자료 제출** 요구
- 정당한 사유 없이 소명 불응 시 → 과태료 부과

### 부동산 정보체계 관리 (법 제25조의2)
- 국토교통부장관은 부동산거래 동향 파악을 위해 **부동산거래관리시스템** 운영
- 신고관청은 신고 접수 시 시스템에 등록

### 중개거래 시 소속공인중개사의 신고 대행
- 소속공인중개사가 개업공인중개사를 대행하여 신고서 **방문 제출** 가능
- 단, **전자문서에 의한 신고는 불가** (방문신고만 가능)
