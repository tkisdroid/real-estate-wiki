# 프로젝트파이낸싱(PF)과 주택연금

> 📖 상위 문서: [[부동산금융론]]

## 1. 프로젝트 파이낸싱 (PF)

### 의의

- **사업성(현금흐름)**을 기반으로 자금 조달
- 사업주의 신용이 아닌 **프로젝트 자체의 수익성**에 근거
- **비소구금융(non-recourse)** 또는 **제한적 소구금융(limited recourse)**
  - 대출 상환 재원이 **프로젝트 수익에 한정**
  - 사업 실패 시 사업주 개인 재산에 대한 청구권 제한

### PF의 특징

| 특징 | 내용 |
|------|------|
| **적합 대상** | 대규모 개발사업 |
| **부외금융** | 사업주의 **재무제표에 영향 최소화** |
| **위험분산** | 참여자 간 위험 분산 효과 |
| **높은 금융비용** | 일반 대출보다 금리 높음 |
| **상환 재원** | 프로젝트에서 발생하는 **현금흐름** |

### 비소구 vs 제한적 소구

| 구분 | 비소구금융 | 제한적 소구금융 |
|------|----------|---------------|
| 사업주 책임 | 프로젝트 수익 범위 내 | 일부 사업주 책임 존재 |
| 대출자 위험 | **높음** | 중간 |
| 실무 적용 | 드문 편 | **대부분** 이 형태 |

## 2. 주택연금 (역모기지/역저당)

### 의의

- **역저당(역모기지)** 방식: 주택을 담보로 **매월 연금** 수령
- 일반 모기지와 **반대** 방향: 일반 = 대출받고 갚아나감, 역 = 연금 받고 사후 정산
- 한국주택금융공사가 **보증서** 발급 → 은행이 연금 지급

### 가입요건

| 항목 | 요건 |
|------|------|
| **연령** | 주택소유자 또는 배우자가 만 **55세 이상** |
| **국적** | 부부 중 1명 이상 대한민국 국민 |
| **주택가격** | 부부기준 공시가격 등 **12억원 이하** |
| **다주택** | 합산가격 12억원 이하이면 가능 |
| **초과 시** | 12억원 초과 2주택자는 **3년 이내** 1주택 처분 시 가입 가능 |

### 우대방식

- 1인 이상이 기초연금 수급권자일 경우
- 일반주택연금보다 월지급금을 **최대 20% 우대**

### 정산 방식

- 가입자 **사망** 시 주택 **처분**하여 대출금 정산
- 처분 후 잔액 있으면 **상속인에게 반환**
- 부족하면 한국주택금융공사가 **부담** (가입자 상속인에게 청구하지 않음)

## 핵심 암기 포인트

- PF = **사업성(현금흐름)** 기반 자금 조달, **비소구/제한적 소구** 금융
- PF = **부외금융** (재무제표 영향 최소화), 대규모 개발사업 적합
- 주택연금 = **역모기지(역저당)**, 만 **55세** 이상, 공시가격 **12억원** 이하
- 한국주택금융공사 **보증** → 은행 연금 지급
- 사망 시 주택 **처분**하여 정산

## OX 퀴즈

**Q1.** PF는 사업주의 신용을 기반으로 자금을 조달한다. → **X** (**프로젝트 자체의 수익성**에 근거)

**Q2.** PF에서 비소구금융이란 사업주가 모든 책임을 지는 것을 의미한다. → **X** (비소구 = 상환 재원이 **프로젝트 수익에 한정**, 사업주 개인 책임 **제한**)

**Q3.** PF는 사업주의 재무제표에 큰 영향을 미친다. → **X** (**부외금융**으로 재무제표 영향 **최소화**)

**Q4.** 주택연금은 만 60세 이상 주택소유자만 가입할 수 있다. → **X** (만 **55세** 이상)

**Q5.** 주택연금 가입 시 공시가격 12억원 초과 단독주택은 가입이 불가능하다. → **O** (12억원 이하만 가능)

**Q6.** 주택연금 가입자 사망 시 상속인이 잔여 대출금을 부담해야 한다. → **X** (부족분은 **한국주택금융공사가 부담**)

## 함정 노트

- ⚠️ "PF = 사업주 신용 기반" → **프로젝트 수익성** 기반 (사업주 신용은 일반 대출)
- ⚠️ "비소구 = 사업주 전액 책임" → 비소구는 사업주 책임 **제한** (프로젝트 범위 한정)
- ⚠️ "주택연금: 만 60세 이상" → 만 **55세** 이상 (2014년 하향 조정)
- ⚠️ "주택연금: 9억원 이하" → **12억원** 이하 (2023년 상향 조정)
- ⚠️ "다주택자 가입 불가" → 합산 12억원 이하이면 **가능**
