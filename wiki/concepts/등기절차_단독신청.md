# 등기절차 - 단독신청 (공동신청의 예외)

## 1. 단독신청의 의의

공동신청주의의 예외로서, 등기권리자 또는 등기의무자 **일방이 단독으로** 등기를 신청할 수 있는 경우

## 2. 원래 단독신청인 경우 (7가지)

| 번호 | 유형 | 근거 | 신청인 |
|------|------|------|--------|
| 1 | **소유권보존등기** (및 보존등기의 말소) | 법 제23조 제2항 | 소유자 |
| 2 | **상속·합병·포괄승계**에 따른 등기 | 법 제23조 제3항 | 상속인·포괄승계인 |
| 3 | **판결에 의한 등기** | 법 제23조 제4항 | 승소한 등기권리자 또는 등기의무자 |
| 4 | **부동산표시변경·경정** 등기 | 법 제23조 제5항 | 소유자 |
| 5 | **등기명의인표시변경·경정** 등기 | 법 제23조 제6항 | 등기명의인 |
| 6 | **신탁등기** | 법 제23조 제7항 | **수탁자** 단독 |
| 7 | **멸실등기** | - | 소유자 |

### 단독신청 이유별 분류

| 분류 | 해당 유형 | 이유 |
|------|----------|------|
| **등기의무자 부존재** | 보존등기, 상속, 표시변경·경정, 명의인표시변경·경정, 멸실 | 대응하는 등기의무자가 없음 |
| **상대방 협력 불요** | 판결에 의한 등기 | 판결로 진정성 담보 |
| **법률의 특별규정** | 신탁등기 | 신탁의 특수성 |

## 3. 판결에 의한 등기 (법 제23조 제4항)

### 판결의 요건

| 요건 | 내용 |
|------|------|
| **이행판결** | "○○○등기절차를 이행하라"는 판결 |
| **확정판결** | 확정되어야 함 (항소·상고 중이면 불가) |
| **내용 명시** | 등기권리자·등기의무자·등기 내용이 명시 |
| **시효** | 확정 후 **10년 경과**해도 신청 가능 (시효 무관) |

### 판결에 준하는 집행권원 (단독신청 가능)

| 인정되는 것 | 인정되지 않는 것 |
|-------------|-----------------|
| **조정조서** | **공증인 작성 공정증서** |
| **화해조서** | |
| **인낙조서** | |
| 조정에 갈음하는 결정 | |
| 화해권고결정 | |

> **공정증서 함정**: 금전채권에 대한 집행력은 있으나, 등기 **단독신청의 근거가 될 수 없음**

### 첨부서면

| 첨부서면 | 필요 여부 |
|----------|----------|
| **판결정본** | 필요 |
| **확정증명서** | 필요 (조정·화해·인낙조서는 **불요**) |
| **송달증명서** | **불요** |
| **등기의무자의 등기필정보** | **불요** (단, 승소한 등기의무자가 단독신청 시는 필요) |

## 4. 예외적 단독신청이 가능한 경우

### 말소등기의 단독신청

| 유형 | 내용 |
|------|------|
| 판결에 의한 말소 | 승소 판결로 단독 말소 |
| **권리소멸약정**에 의한 말소 | 약정 소멸사유 발생 시 등기권리자 단독 |
| **행방불명**에 의한 말소 | 등기의무자 행방불명 시 등기권리자 단독 |
| **혼동**에 의한 말소 | 권리의 혼동으로 인한 말소 |

### 가등기의 단독신청

| 방법 | 내용 |
|------|------|
| **승낙서 첨부** | 가등기의무자의 승낙서를 첨부하여 가등기권리자 단독 |
| **가등기가처분명령** | 법원의 가등기가처분명령정본을 첨부하여 단독 |

## 5. 단독신청 사유 암기법

> **"보상판표명신멸"** — 보존, 상속(포괄승계), 판결, 표시변경·경정, 명의인표시변경·경정, 신탁, 멸실

---

## 핵심 암기 포인트

- 단독신청 7가지: 보존, 상속·포괄승계, 판결, 표시변경·경정, 명의인표시변경·경정, 신탁, 멸실
- **판결 요건**: 이행판결 + 확정판결, 확정 후 10년 경과해도 가능
- **공정증서 불가**: 등기 단독신청 근거가 될 수 없음
- **신탁등기**: **수탁자** 단독 (위탁자와 공동이 아님)
- 조정·화해·인낙조서: **확정증명서 불요**
- 판결정본: 확정증명서 **필요**, 송달증명서 **불요**

## OX 퀴즈

**Q1.** 소유권보존등기는 공동신청으로 한다. → **X** (등기의무자 없으므로 단독신청)

**Q2.** 상속에 의한 등기는 공동신청이 원칙이다. → **X** (단독신청 — 법 제23조 제3항)

**Q3.** 공정증서에 의해 단독으로 등기를 신청할 수 있다. → **X** (공정증서는 단독신청 불가)

**Q4.** 판결에 의한 등기신청은 확정 후 10년 이내에만 가능하다. → **X** (10년 경과해도 가능)

**Q5.** 신탁등기는 위탁자와 수탁자가 공동으로 신청한다. → **X** (수탁자 단독)

**Q6.** 판결에 의한 등기에서 조정조서는 확정증명서가 필요하다. → **X** (불요)

**Q7.** 가등기의무자의 승낙서를 첨부하면 가등기를 단독으로 신청할 수 있다. → **O**

**Q8.** 확인판결에 의한 등기 단독신청이 가능하다. → **X** (이행판결이어야 함. 단, 보존등기 신청인 자격 확인판결은 별개)

**Q9.** 등기명의인의 주소변경등기는 공동신청으로 한다. → **X** (등기명의인표시변경으로 단독신청)

**Q10.** 건물멸실등기는 공동신청이 원칙이다. → **X** (단독신청)

## 함정 노트

- **이행판결 vs 확인판결·형성판결**: 등기 단독신청은 반드시 **이행판결**이어야 함. 확인판결·형성판결만으로는 불가 (보존등기 신청인 자격의 확인판결은 별개의 문제)
- **공정증서**: 금전채권 집행력은 인정되지만 등기 단독신청은 **불가**. "집행권원=단독신청 가능"이 아님
- **승소한 등기의무자 단독신청**: 등기의무자가 승소하여 단독신청 시에는 **등기필정보 필요** (등기권리자 승소와 다름)
- **확정증명서 불요**: 조정조서·화해조서·인낙조서는 확정증명서 불요. 판결정본만 확정증명서 필요
- **소유권보존등기의 말소**: 보존등기 자체가 단독신청이므로 말소도 **단독**
