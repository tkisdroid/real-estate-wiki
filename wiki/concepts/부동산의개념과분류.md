# 부동산의 개념과 분류

## 0. 부동산학의 의의

### 부동산학의 성격

| 성격 | 내용 |
|------|------|
| **종합과학** | 다른 학문의 지적 축적을 원용하여 새로운 체계 구성 |
| **응용과학** | 다른 학문 성과를 현실에 활용 (**순수과학이 아니다**) |
| **규범과학** | 바람직한 방향 제시 (이론적 성격, 과학성) |
| **경험과학(실천과학)** | 현실 문제 해결을 위한 실무적 학문 (기술성) |
| **사회과학** | 사회적 수요에 의해 탄생 (**자연과학이 아니다**) |

### 부동산학의 접근방법

| 방법 | 내용 |
|------|------|
| **종합식** | 복합개념의 사고로 부동산현상 분석. 우리나라 체계화에 기여 |
| **의사결정** | 인간은 합리적 존재, **자기이윤의 극대화** 전제. 민간부문 중심 |
| **제도적** | 법률·정책·조직 중심, **공적 개입** 전제 |
| **분산식** | 각 학문분야별 개별적·부분적으로 부동산을 다루는 방법 |
| **중점식** | 특정 측면(중개, 감정평가, 건축 등)에만 중점 |
| **행태과학적** | 인간적 요인(부동산행태)에 착안 |
| **현상학적** | 토지는 인간 노동의 산물이 아님, 인간적 활동 강조 |

### 부동산학의 이념

| 이념 | 내용 | 주체 |
|------|------|------|
| **효율성** | 최소 비용으로 최대 산출 (최유효이용), 수익률 크기 | **사적 주체** |
| **형평성** | 분배의 문제, 정의의 개념, 애타적 협동심 | **공적 주체** |

- 민간부문 한정 시 효율성 중시, 사회 전체적으로는 **효율성과 형평성의 균형점** 추구
- 부동산학의 최우선 가치는 **효율성**

### 부동산활동의 속성

| 속성 | 내용 |
|------|------|
| **과학성·기술성** | → 전문성 파생 |
| **사회성·공공성** | → **윤리성** 파생 |
| **정보활동성** | 부동성 → 통제 불가능 요인 많아 정보 중요 |
| **대인·대물활동** | 사람과 부동산 동시 대상 |
| **임장활동성** | 근거: 부동성(부증성 아님) + 대물활동(대인 아님) |
| **배려의 장기성** | 영속성 + 용도의 다양성 → 장래 배려 |
| **공간활동성** | 3차원 공간(지표·지하·공중) |
| **복합개념성** | 법률적·경제적·기술적 측면 |

### 부동산활동의 주체

| 주체 | 중시 가치 | 예 |
|------|----------|---|
| **공적 주체**(제1섹터) | 효율성 + **형평성** 균형 | 정부, 지자체, 공사 |
| **사적 주체**(제2섹터) | **효율성** | 민간기업(이윤극대화), 개인(효용극대화), 조합 |
| **제3섹터** | 공익 + 이익 결합 | 공공+민간 결합 형태 |

### 표준산업분류상 부동산업

- **부동산 임대 및 공급업**: 주거용/비주거용 건물 임대업, 주거용/비주거용 건물 개발 및 공급업
- **부동산 관련 서비스업**: 부동산 관리업(주거용/비주거용), 부동산 중개 및 대리업, 부동산 투자자문업, 부동산 감정평가업, 부동산 분양 대행업

> **주의**: 부동산 개발 및 공급업은 '부동산 관련 서비스업'이 아닌 '부동산 임대 및 공급업'에 해당

## 1. 복합개념의 부동산

부동산은 **법률적·경제적·기술적 측면**의 복합된 개념으로 이해한다.

| 측면 | 구분 | 내용 |
|------|------|------|
| **무형적** | 법률적 개념 | 협의: 토지 및 그 정착물 (민법 제99조 1항) / 광의: 협의 + [[준부동산]](의제부동산) |
| **무형적** | 경제적 개념 | 자산, 자본, 상품, 생산요소, 소비재 |
| **유형적** | 기술적 개념 | 자연, 공간, 위치, 환경 |

> **시험 포인트**: 생산요소·자산은 **경제적** 개념이다. 기술적 개념과 혼동하지 말 것.

## 2. 토지정착물

### 독립정착물 (토지와 독립된 부동산)

| 유형 | 설명 |
|------|------|
| **건물** | 토지와 별도 등기, 별개의 부동산 |
| **입목** | 입목에 관한 법률에 따라 등기된 수목의 집단. 소유권·저당권 목적 가능 |
| **수목** | 명인방법 공시 시 독립부동산. 저당권 목적물 불가 |
| **농작물** | 토지에 부합하지 않고 별개의 부동산. 명인방법 불필요 |

### 종속정착물 (토지의 일부)

- 교량, 터널, 돌담, 우물, 구거, 저수지, 도로포장
- 등기·명인방법 미비 입목, 다년생식물

## 3. 준부동산 (의제부동산)

민법상 부동산은 아니나, 등기·등록으로 공시하여 부동산에 준하여 취급되는 특정 동산 또는 재단.

| 공시방법 | 대상 |
|----------|------|
| **등기** | 선박(20톤 이상), 광업재단, 공장재단 |
| **등록** | 자동차, 건설기계, 항공기, 광업권, 어업권 |

## 4. 복합부동산

- 토지와 건물 등 독립된 거래 객체가 **결합된 상태**에서 부동산활동의 대상이 되는 경우
- 감정평가 시 **일괄평가**가 원칙, **거래사례비교법** 적용
- 복합개념(법률·경제·기술 측면)이나 복합건물(주상복합)과 구별

## 5. 토지의 분류

### 지목 (공간정보의 구축 및 관리에 관한 법률)

전국 토지를 **28개 지목**으로 구분하여 토지대장에 기입·관리.

> 전, 답, 과수원, 목장용지, 임야, 광천지, 주유소, 염전, 대, 공장용지, 학교용지, 도로, 철도용지, 주차장, 하천, 제방, 구거, 유지, 수도용지, 공원, 체육용지, 유원지, 종교용지, 사적지, 묘지, 양어장, 창고용지, 잡종지

### 토지이용활동상 분류

| 구분 | 내용 |
|------|------|
| **택지** | 건축용지(주거·상업·공업용). 나대지+건부지 포함 |
| **부지** | 일정한 용도목적에 사용되는 바닥토지 (택지보다 포괄적) |
| **대지** | 건축법상 용어. 지목 '대(垈)'와 반드시 일치하지 않음 |
| **소지(원지)** | 대지 등으로 개발되기 이전의 자연상태 토지. 주로 1차 산업용 |
| **한계지** | 택지이용의 최원방권 |

### 나지·건부지·공지

| 구분 | 내용 |
|------|------|
| **나지** | 건물 등 정착물 없고, 지상권 등 사용·수익 제한 사법상 권리 **미설정** 토지 |
| **나대지** | 지상에 건축물 없고 지목이 '대'인 토지 (나지와 구분) |
| **건부지** | 건축물로 사용 중인 바닥토지. 가격은 나지가격을 **상한**으로 결정 |
| **공지** | 건폐율 제한 등으로 건축하면서 남겨놓은 일정면적 부분 |

- **건부감가**: 최유효이용 미달 시 지상건물로 인한 감가. 건물이 **견고할수록·면적 클수록** 크며, 공법상 규제 **완화** 시 흔히 발생
- **건부증가**: 공법상 규제 **강화** 시 (개발제한구역 등) 건물 소재로 토지가격 증가

### 맹지·대지(袋地)

| 구분 | 내용 |
|------|------|
| **맹지(盲地)** | 타인 토지에 둘러싸여 도로에 **직접 연결 안 되는** 토지. 건축 불가하나 주위토지 통행 가능 |
| **대지(袋地)** | 자루형 토지. 좁은 통로에 의해 접속면 있음 (맹지처럼 완전히 막히지 않음) |

### 후보지·이행지

| 구분 | 전환 범위 | 지목변경 |
|------|----------|----------|
| **후보지** | 용도적 지역 **상호간** 전환 (택지↔농지↔임지) | 전환 후 지목 **변경** |
| **이행지** | 같은 용도지역 **내** 용도 상호간 전환 | 전환 후 지목 **불변경** 가능 |

- 감정평가: 전환 후 용도 기준이 **원칙**, 전환이 완만(성숙도 낮음)하면 전환 전 기준

### 필지·획지

| 구분 | 기준 | 단위 |
|------|------|------|
| **필지** | **소유** 중심 (공간정보법상 용어), 하나의 지번 | 토지의 **등록**단위 |
| **획지** | **이용** 중심, 가격수준 비슷한 일단의 토지 | 토지의 **이용**단위 |

- 1필지 = 여러 획지 가능 (**구분평가**) / 여러 필지 = 1획지 가능 (**일괄평가**)

### 기타 토지 분류

| 구분 | 내용 |
|------|------|
| **법지(法地)** | 법면. 경사토지로 측량면적 포함되나 **실제 사용 불가** |
| **빈지(濱地)** | 해안선~지적공부 등록지역 사이. 활용실익 있으나 **사적 소유권 불인정** |
| **포락지(捕落地)** | 지적공부 등록 토지가 물에 침식되어 **수면 밑으로 잠긴** 토지 |
| **휴한지** | 농지 비옥도 회복을 위해 **정상적으로 휴경**하는 토지 |
| **공한지** | 도시토지 중 **투기목적**으로 장기간 방치한 토지 |
| **선하지** | 고압선 아래 토지. **선하지 감가** 적용 |
| **공유지** | 1필지를 **2인 이상** 공동소유 |
| **일단지** | 용도상 불가분 관계에 있는 **2필지 이상** 토지 |

## 6. 주택의 분류 (건축법)

### 단독주택

| 유형 | 주요 요건 |
|------|----------|
| **단독주택** | 일반 단독주택 형태 |
| **다중주택** | 학생·직장인 장기거주, **독립된 주거형태 미갖춤**(욕실 O, 취사시설 X), 660㎡ 이하, **3층 이하** |
| **다가구주택** | **19세대 이하**, 660㎡ 이하, **3층 이하** (공동주택 아님) |
| **공관** | - |

### 공동주택

| 유형 | 주요 요건 |
|------|----------|
| **아파트** | **5층 이상** |
| **연립주택** | 1개동 바닥면적 합계 **660㎡ 초과**, **4층 이하** |
| **다세대주택** | 1개동 바닥면적 합계 **660㎡ 이하**, **4층 이하** |
| **기숙사** | 공동취사시설 이용 세대 수가 전체의 **50% 이상** |

> **필로티**: 아파트·연립주택은 **1층 전부**가 주차장이면 층수 제외. 다세대·다중·다가구는 1층의 **전부 또는 일부**가 주차장이고 나머지가 **주택 외 용도**면 제외.

## 7. 공간으로서 부동산

- **3차원 공간**: 수평공간(지표권), 공중공간(공중권), 지하공간(지하권)
- 민법 제212조: "토지의 소유권은 **정당한 이익이 있는 범위 내**에서 토지의 상하에 미친다"
- [[개발권양도제도]](TDR)는 공중권 논리에서 탄생
- 공적 공중권 = 항공권, 사적 공중권 = 일조권, 천공조망권, TDR

## 핵심 암기 포인트

- 민법 제99조 1항: 부동산 = **토지 및 그 정착물** (협의)
- 지목: **28개** (공간정보의 구축 및 관리에 관한 법률)
- 아파트: **5층 이상** / 연립주택: 바닥면적 합계 **660㎡ 초과** / 다세대주택: **660㎡ 이하**
- 준부동산 등기 대상: 선박(**20톤 이상**), 광업재단, 공장재단
- 준부동산 등록 대상: 자동차, 건설기계, 항공기, 광업권, 어업권
- 입목: 등기 → 소유권·**저당권** 목적 가능 / 수목: 명인방법 → **저당권 목적물 불가**
- 민법 제212조: 토지소유권은 **정당한 이익이 있는 범위 내**에서 상하에 미침
- 복합부동산 감정평가: **일괄평가** 원칙, **거래사례비교법** 적용

## OX 퀴즈

**Q1.** 부동산의 경제적 개념에는 자산, 자본, 자연, 상품이 포함된다. → **X** (자연은 **기술적** 개념이다)

**Q2.** 입목에 관한 법률에 의해 등기된 수목의 집단은 저당권의 목적이 될 수 있다. → **O** (입목은 소유권·저당권 목적 가능)

**Q3.** 명인방법으로 공시된 수목은 저당권의 목적물이 될 수 있다. → **X** (수목은 저당권 목적물 **불가**)

**Q4.** 농작물은 토지에 부합하며, 명인방법이 필요하다. → **X** (농작물은 토지에 부합하지 않고, 명인방법 불필요)

**Q5.** 20톤 미만의 선박도 준부동산에 해당한다. → **X** (20톤 **이상**의 선박만 등기 대상 준부동산)

## 함정 노트

- ⚠️ "자연·공간은 경제적 개념이다" → 자연·공간·위치·환경은 **기술적(유형적)** 개념이다
- ⚠️ "연립주택은 660㎡ 이하" → 660㎡ **초과**가 연립주택, **이하**가 다세대주택
- ⚠️ "수목도 저당권 설정 가능" → 입목(등기)만 저당권 가능, 수목(명인방법)은 **불가**
- ⚠️ "복합부동산 = 복합개념" → 복합부동산(토지+건물 결합)과 복합개념(법률·경제·기술 측면)은 전혀 다른 개념
- ⚠️ "공장재단은 등록 대상" → 공장재단·광업재단은 **등기** 대상, 자동차·건설기계·항공기는 **등록** 대상
