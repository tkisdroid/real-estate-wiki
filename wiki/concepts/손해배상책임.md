# 손해배상책임과 업무보증

공인중개사법은 개업공인중개사의 귀책사유로 발생하는 거래당사자의 피해를 최소화하기 위해 **손해배상책임제도**와 **업무보증설정의무**를 두고 있다. 업무보증은 개업공인중개사의 배상능력 부재 시 보증기관이 대신 배상하는 제도이다.

## 비교 정리

### 보증설정 vs 배상책임

| 구분 | 보증설정 | 배상책임 |
|---|---|---|
| **성격** | 사전적 예방·담보 | 사후적 손해전보 |
| **시기** | 업무 개시 **전** 설정 | 손해 발생 **후** 배상 |
| **방법** | 보증보험/공제/공탁 택1 | 보증기관 지급 또는 직접 배상 |
| **대상** | 재산상 손해 | 재산상 손해 (정신적 손해 제외) |
| **보증금액(개인)** | 2억원 이상 | 실제 손해액 |
| **보증금액(법인)** | 4억원 이상 + 분사무소당 2억원 | 실제 손해액 |

## 세부 내용
- **[[손해배상책임_보증설정|업무보증 설정]]**: 보증보험/공제/공탁 비교, 보증금액(개인 2억/법인 4억), 공탁금 회수제한(3년), 재설정(15일)
- **[[손해배상책임_배상책임범위|배상책임 범위]]**: 책임 성립요건 4가지, 경과실+중과실 포함, 정신적 손해 제외, 중개보조원 무과실책임

## 손해배상책임 성립요건 (법 제30조)

### 4가지 요건
1. **고의 또는 과실**이 있어야 함 (경과실 + 중과실 모두 포함)
2. **인과관계**가 있어야 함 (중개활동과 손해 사이)
3. **재산상 손해**가 발생하여야 함 (정신적 손해는 업무보증 배상에서 제외, 민사책임은 별도)
4. **중개행위**에 기인하여야 함

### 자기 사무소 제공 책임 (법 제30조 제2항)
- 자기 사무소를 **타인의 중개행위 장소로 제공**하여 손해 발생 → 배상 책임

## 업무보증 설정 세부사항

### 보증 금액 (영 제25조)
| 구분 | 금액 |
|---|---|
| 법인 | **4억원** 이상 |
| 법인 분사무소 | 분사무소마다 **2억원** 추가 |
| 개인 | **2억원** 이상 |
| 특수법인 | **2천만원** 이상 |

### 공탁금 회수 제한 (법 제30조 제4항)
- 폐업 또는 사망한 날부터 **3년 이내** 회수 불가
- **등록취소, 법인 해산** 시에도 동일 적용
- 법인 대표자 사망 시에는 적용 안 됨 (대표자 교체 가능)

### 보증의 변경 (영 제25조 제1항)
- 기존 보증 **효력 유지 중** 다른 보증 설정 → 신고 → 기존 보증 해제 (순서 중요)

### 보증 재설정 (영 제26조 제2항)
- 보증보험금·공제금·공탁금으로 손해배상 한 경우 → **15일 이내** 재가입 또는 부족액 보전

### 보증설정 신고 예외
- 보증기관(보증보험회사, 공제사업자, 공탁기관)이 **등록관청에 직접 통보**한 경우 → 신고 생략 가능

### 손해배상금 지급 청구 첨부서류 (영 제27조)
1. 개업공인중개사와의 **손해배상합의서**
2. **화해조서**
3. 확정된 **법원 판결 사본**
4. 그 밖에 이에 준하는 효력이 있는 서류

### 보증사항 설명·교부 (법 제30조 제5항)
- 중개 완성 시 거래당사자에게 **보장금액, 보증기관, 소재지, 보장기간** 등 설명
- **관계증서 사본 교부** 또는 전자문서 제공
- 위반 시: **100만원 이하 과태료**

### 휴업·업무정지 중 보증 만료
- 휴업·업무정지 기간 중 보증 만료 → 만료일까지 재설정 **불요** → **재개업 시** 재설정
