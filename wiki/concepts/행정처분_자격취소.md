# 행정처분 - 공인중개사 자격취소 (법 제35조)

## 1. 개요

| 항목 | 내용 |
|------|------|
| **처분 대상** | 공인중개사 |
| **처분청** | **시·도지사** (자격증을 교부한 시·도지사) |
| **처분 성격** | **절대적 (취소하여야 한다)** — 재량 없음 |
| **청문** | 대상 **O** |

## 2. 자격취소 사유 (4가지 — 절대적)

시·도지사는 공인중개사가 다음에 해당하는 경우 자격을 **취소하여야** 한다.

| 번호 | 사유 | 비고 |
|------|------|------|
| 1 | **부정한 방법**으로 자격 취득 | 시험 부정 등 |
| 2 | 자격증 **양도·대여** | 다른 사람에게 성명을 사용하여 중개업무를 하게 한 경우 포함 |
| 3 | **자격정지 기간 중** 중개업무 수행 | 이중소속(다른 개업공인중개사의 소속공인중개사·중개보조원·법인 사원·임원이 된 경우) 포함 |
| 4 | 이 법 또는 **형법상 범죄**로 **금고 이상의 형** 선고 | **집행유예 포함**, 아래 형법 조항 한정 |

### 형법상 해당 범죄 (4호 관련)

| 형법 조문 | 죄명 |
|-----------|------|
| 제114조 | 범죄단체조직 |
| 제231조 | 사문서 위조·변조 |
| 제234조 | 위조문서 등의 행사 |
| 제347조 | **사기** |
| 제355조 | **횡령·배임** |
| 제356조 | **업무상 횡령·배임** |

> **금고 이상의 형**: 실형 + 집행유예 모두 포함. **벌금형은 해당 없음**

## 3. 처분권자 특례

| 상황 | 처분 방법 |
|------|----------|
| 자격증 교부 시·도지사 = 사무소 소재지 시·도지사 | 자격증 교부 시·도지사가 처분 |
| 자격증 교부 시·도지사 ≠ 사무소 소재지 시·도지사 | 사무소 소재지 시·도지사가 **필요한 절차를 모두 이행** 후 자격증 교부 시·도지사에게 **통보** |

## 4. 자격증 반납 및 후속조치

| 항목 | 내용 |
|------|------|
| **반납 기한** | 취소일부터 **7일 이내** |
| **반납처** | 자격증을 교부한 **시·도지사** |
| **반납 불가 시** | 사유서를 시·도지사에게 제출 |
| **미반납 시 과태료** | **100만원 이하** 과태료 |
| **통보** | 시·도지사는 자격취소 후 **5일 이내** 국토교통부장관과 다른 시·도지사에게 통보 |

## 5. 결격사유 기간

자격이 취소된 자는 취소된 날부터 **3년 이내** 공인중개사 자격을 다시 취득하지 못한다 (법 제6조).

> **3년**: 자격취소 = 3년, 등록취소 = 3년으로 동일

## 6. 청문

- 공인중개사 자격취소 시 **청문** 실시 필요
- 자격정지·업무정지 시에는 청문 대상이 아님 (의견제출 기회만 부여)

## 핵심 암기 포인트

- 자격취소는 **절대적 처분** ("취소하여야 한다")
- 처분청: **시·도지사** (등록관청이 아님)
- 사유 4가지: 부정취득, 양도·대여, 자격정지 중 업무, 금고 이상 형(집행유예 포함)
- 자격증 반납: **7일 이내**, 미반납 시 **100만원** 이하 과태료
- 결격사유: 취소 후 **3년** 재취득 불가
- **청문 대상**임 (자격정지는 청문 대상 아님)

## OX 퀴즈

**Q1.** 공인중개사 자격취소는 등록관청이 한다. → **X** (시·도지사)

**Q2.** 자격정지 기간 중 중개업무를 수행하면 자격취소 사유에 해당한다. → **O**

**Q3.** 이 법에 위반하여 벌금형을 선고받은 경우 자격취소 사유이다. → **X** (금고 이상의 형이어야 하며, 벌금형은 등록 결격사유)

**Q4.** 자격취소 후 자격증을 10일 이내에 반납해야 한다. → **X** (7일 이내)

**Q5.** 자격취소 후 5년간 자격 재취득이 불가하다. → **X** (3년)

**Q6.** 사기죄로 집행유예를 선고받은 공인중개사의 자격은 취소하여야 한다. → **O** (형법 347조, 금고 이상 형에 집행유예 포함)

**Q7.** 공인중개사 자격취소 시 청문을 실시하여야 한다. → **O**

## 함정 노트

- **시·도지사 vs 등록관청**: 자격취소·자격정지는 **시·도지사**, 등록취소·업무정지는 **등록관청**. 처분청 혼동 주의
- **집행유예 포함**: 금고 이상의 형에 집행유예 선고도 포함. "실형만 해당"이라는 함정 출제
- **벌금형 ≠ 자격취소**: 벌금형은 **등록 결격사유**(개설등록 불가)이지 자격취소 사유가 아님
- **형법 조문 한정**: 모든 형법 위반이 아니라 114조·231조·234조·347조·355조·356조로 **한정**
- **1심 판결 확정 전**: 1심에서 징역형 선고 후 항소 중이면 아직 확정되지 않았으므로 자격취소 불가 (행정심판 판례)
