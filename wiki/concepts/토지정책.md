# 토지정책

## 1. 용도지역·지구제

- **부(-)의 외부효과 제거·완화**를 통한 토지이용규제 수단
- 용도지역 간 **형평성 문제** 발생 가능

### 용도지역 체계 (국토계획법)

| 대분류 | 중분류 | 소분류 |
|--------|--------|--------|
| **도시지역** | 주거지역 | 전용(1·2종), 일반(1·2·3종), 준주거 |
| | 상업지역 | 중심, 일반, 근린, 유통 |
| | 공업지역 | 전용, 일반, 준공업 |
| | 녹지지역 | 보전, 생산, 자연 |
| **관리지역** | | 보전관리, 생산관리, 계획관리 |
| **농림지역** | | |
| **자연환경보전지역** | | |

> **시험 주의**: 자연환경보전지역은 **도시지역이 아님**. 용도지역은 **중복지정 불가**, 용도지구는 **중복지정 가능**.

### 주요 계획 용어

- **도시·군기본계획**: 장기발전방향 제시, 도시·군관리계획의 **지침**
- **도시·군관리계획**: 개발·정비·보전 위한 계획
- **지구단위계획**: 도시·군계획 수립 대상 일부에 대한 도시·군**관리**계획

## 2. 개발권양도제도 (TDR)

- 현재 우리나라에서 **시행하지 않음**
- 규제지역의 개발권을 독립 재산권으로 인정, 시장에서 거래 허용
- 용도지역·지구제의 **형평성 문제 보완**
- **공중공간**의 효율적 활용 촉진

| 장점 | 단점 |
|------|------|
| 정부 재정부담 없이 형평성 해소 | 개발가능지역 과밀·혼잡 |
| 공공재 안정적 공급 | 남용 시 용도지역·지구제 와해 우려 |

## 3. 개발이익환수제도

- **개발이익** = 정상지가상승분을 초과하여 귀속되는 토지가액 증가분
- **개발부담금**으로 환수

### 환수 필요성

- 외부효과·불로소득의 환수
- 계층간 위화감 방지, 공평배분
- 토지투기·지가급등 방지
- 공공투자 확대 재원 확보

## 4. 토지은행제도

- **공공토지의 비축에 관한 법률**
- 장래 필요 토지를 미리 확보 → 공익목적으로 적기·적소 활용
- **한국토지주택공사**에 설치, 고유계정과 구분 회계처리
- 토지비축계획: **10년** 단위 종합계획 + 연도별 시행계획

| 구분 | 공공개발용 | 수급조절용 |
|------|-----------|-----------|
| 목적 | 공공개발사업 원활 수행 | 토지시장 안정 |
| 대상 | SOC, 산업, 주택용지 | 일반토지, 개발가능지 |
| 취득수단 | 협의매수, **수용** | 협의매수, **선매** |

### 장단점

| 장점 | 단점 |
|------|------|
| 토지수급 조절 → 시장 안정 | 막대한 매입비 필요 |
| 공공용지 저렴 공급 | 정부 매입 → 지가상승 조장 우려 |
| 보상비 절감 | |
| 무질서한 개발 방지 | |

## 5. 현재 시행하지 않는 정책수단

- **택지소유상한제**, **토지초과이득세**, **개발권양도제(TDR)**
- **종합토지세**, **공한지세**

## 핵심 암기 포인트

- 용도지역 4대분류: **도시지역, 관리지역, 농림지역, 자연환경보전지역**
- 도시지역 세분: 주거(전용1·2, 일반1·2·3, 준주거), 상업(중심·일반·근린·유통), 공업(전용·일반·준), 녹지(보전·생산·자연)
- 용도지역: **중복지정 불가** / 용도지구: **중복지정 가능**
- TDR(개발권양도제도): 현재 우리나라 **미시행**, 형평성 문제 보완 목적
- 토지은행 비축계획: **10년** 단위 종합계획
- 토지은행: **한국토지주택공사**에 설치
- 공공개발용: 협의매수 + **수용** / 수급조절용: 협의매수 + **선매**
- 현재 미시행 정책: 택지소유상한제, 토지초과이득세, TDR, 종합토지세, 공한지세
- 개발이익 = **정상지가상승분을 초과**하여 귀속되는 토지가액 증가분

## OX 퀴즈

**Q1.** 용도지역은 중복지정이 가능하다. → **X** (용도지역은 중복지정 **불가**, 용도지구는 중복지정 가능)

**Q2.** 자연환경보전지역은 도시지역에 포함된다. → **X** (도시지역, 관리지역, 농림지역, 자연환경보전지역은 각각 **별개**)

**Q3.** 개발권양도제도(TDR)는 현재 우리나라에서 시행 중이다. → **X** (현재 **미시행**)

**Q4.** 토지은행의 토지비축계획은 5년 단위로 수립한다. → **X** (**10년** 단위 종합계획)

**Q5.** 토지은행에서 수급조절용 토지는 수용으로 취득한다. → **X** (수급조절용은 협의매수 + **선매**, 수용은 공공개발용)

**Q6.** 지구단위계획은 도시·군관리계획의 일종이다. → **O**

**Q7.** 택지소유상한제는 현재 시행 중인 토지정책이다. → **X** (현재 **미시행**)

**Q8.** 용도지역·지구제는 부(-)의 외부효과를 제거·완화하기 위한 수단이다. → **O**

## 함정 노트

- ⚠️ "자연환경보전지역 = 도시지역" → 자연환경보전지역은 도시지역과 **별개** 분류
- ⚠️ "용도지역 중복지정 가능" → 용도**지역**은 불가, 용도**지구**만 중복지정 가능
- ⚠️ "토지은행 = 5년 계획" → **10년** 단위 종합계획 + 연도별 시행계획
- ⚠️ "수급조절용 = 수용 가능" → 수용은 **공공개발용**만, 수급조절용은 **선매**
- ⚠️ "TDR 시행 중" → TDR, 택지소유상한제, 토지초과이득세, 종합토지세, 공한지세 모두 **미시행**
