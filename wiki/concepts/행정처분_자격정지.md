# 행정처분 - 공인중개사 자격정지 (법 제36조)

## 1. 개요

| 항목 | 내용 |
|------|------|
| **처분 대상** | **소속공인중개사** (업무를 수행하는 기간 중) |
| **처분청** | **시·도지사** (자격증을 교부한 시·도지사) |
| **처분 성격** | **임의적 (정지할 수 있다)** — 재량 있음 |
| **기간 범위** | **6개월** 범위 내 |
| **청문** | 대상 **X** (의견제출 기회만 부여) |

## 2. 자격정지 사유 및 기간 (7가지)

| 번호 | 사유 | 정지기간 |
|------|------|----------|
| 1 | **2 이상** 중개사무소에 소속 (**이중소속**) | **6개월** |
| 2 | **인장등록** 미이행 또는 **미등록 인장** 사용 | 3개월 |
| 3 | 확인·설명을 **불성실·부정확**하게 하거나 **근거자료** 미제시 | 3개월 |
| 4 | 확인·설명서에 **서명·날인** 미이행 | 3개월 |
| 5 | **거래계약서**에 서명·날인 미이행 | 3개월 |
| 6 | 거래계약서에 거래내용 **거짓기재** 또는 **이중계약서** 작성 | **6개월** |
| 7 | 법 제33조 제1항의 **금지행위** 위반 | **6개월** |

### 6개월 사유 (가장 중한 3가지)
- 이중소속
- 거짓기재·이중계약서
- 금지행위

## 3. 가중·경감

- **1/2 범위** 내에서 가중 또는 경감 가능
- 가중하더라도 자격정지기간은 **6개월 초과 불가**
- 위반행위의 **동기·결과·횟수** 등을 참작

## 4. 처분권자 및 통보

| 항목 | 내용 |
|------|------|
| **처분권자** | 자격증 교부 시·도지사 |
| **통보 의무** | 등록관청은 자격정지 사유를 알게 된 때 지체 없이 **시·도지사에게 통보** |
| **교부·소재지 다른 경우** | 사무소 소재지 시·도지사가 절차 이행 후 자격증 교부 시·도지사에게 통보 |

## 5. 자격정지 중 업무수행 시

자격정지 기간 중 중개업무를 수행하면 → **자격취소** 사유 (법 제35조 제1항 제3호)

> 자격정지 위반 → 자격취소로 격상되는 구조

## 6. 자격취소와 자격정지 비교

| 비교 항목 | 자격취소 | 자격정지 |
|-----------|---------|---------|
| **성격** | 절대적 (하여야 한다) | 임의적 (할 수 있다) |
| **처분청** | 시·도지사 | 시·도지사 |
| **대상** | 공인중개사 | **소속공인중개사** |
| **기간** | 영구 (3년 후 재취득 가능) | 6개월 범위 |
| **청문** | O | X |
| **사유 수** | 4가지 | 7가지 |

## 핵심 암기 포인트

- **소속공인중개사에게만** 적용 (개업공인중개사에게는 등록취소·업무정지)
- **임의적 처분**: "정지할 수 **있다**"
- 기간: **6개월 범위** 내
- 가중·경감: **1/2** 범위 (가중해도 6개월 초과 불가)
- 6개월 사유: 이중소속, 거짓기재·이중계약서, 금지행위
- 자격정지 중 업무 → **자격취소**로 격상
- 등록관청이 사유를 알면 **시·도지사에게 통보** (처분은 시·도지사)

## OX 퀴즈

**Q1.** 자격정지는 등록관청이 처분한다. → **X** (시·도지사)

**Q2.** 자격정지는 절대적 처분이다. → **X** (임의적, "할 수 있다")

**Q3.** 자격정지기간은 최대 1년이다. → **X** (6개월)

**Q4.** 가중처분 시 자격정지기간이 6개월을 초과할 수 있다. → **X** (6개월 초과 불가)

**Q5.** 인장등록 미이행 시 자격정지 6개월이다. → **X** (3개월)

**Q6.** 금지행위를 한 소속공인중개사는 자격정지 6개월에 해당한다. → **O**

**Q7.** 자격정지 기간 중 업무를 수행하면 자격취소 사유가 된다. → **O**

**Q8.** 자격정지 시 청문을 실시하여야 한다. → **X** (의견제출 기회만 부여)

## 함정 노트

- **소속공인중개사 한정**: 개업공인중개사에게는 자격정지가 아닌 **등록취소·업무정지**가 적용됨
- **이중소속 → 자격정지 + 자격취소**: 이중소속은 자격정지(6개월) 사유이면서 동시에 **절대적 등록취소** 사유. 소속공인중개사는 자격정지, 개업공인중개사는 등록취소
- **인장 사용 위반**: 소속공인중개사는 **자격정지**(3개월), 개업공인중개사는 **업무정지** 사유
- **거짓기재·금지행위**: 자격정지(6개월)와 동시에 임의적 **등록취소** 사유이기도 함
