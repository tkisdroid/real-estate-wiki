# 등기절차

등기절차의 흐름: **신청 → 접수 → 조사(심사) → 기입 → 교합 → 등기필정보 작성·통지**

## 비교 정리

### 공동신청 vs 단독신청

| 비교 항목 | 공동신청 | 단독신청 |
|-----------|---------|---------|
| **원칙/예외** | 원칙 | 예외 |
| **신청인** | 등기권리자 + 등기의무자 | 일방만 |
| **대표 유형** | 매매 이전등기, 저당권 설정 | 보존, 상속, 판결, 신탁 등 |

### 등기신청의 강제 비교

| 구분 | 근거법 | 기한 |
|------|--------|------|
| 건물표제부변경·멸실등기 | 부동산등기법 | **1개월** 이내 |
| 소유권보존·이전등기 | 부동산등기특별조치법 | **60일** 이내 |
| 소유권이전등기 | 실명법 | **3년** 이내 |

### 신청주의의 원칙 (법 제22조)

등기는 **당사자의 신청** 또는 **관공서의 촉탁**에 따라 한다. 예외적으로 등기관의 **직권** 또는 법원의 **명령**에 의한다.

### 등기관의 직권등기 유형

| 유형 | 내용 |
|------|------|
| 보존 | 미등기부동산에 대한 처분제한등기 촉탁 시 소유권보존등기 |
| 변경 | 행정구역 변경에 의한 부동산표시변경·등기명의인 주소변경 |
| 경정 | 등기관의 과오로 인한 착오·유루 발생 시 |
| 말소 | 법 제29조 제1호·제2호 간과 등기, 가등기에 기한 본등기 시 중간처분등기 등 |

## 촉탁등기 사유

관공서가 촉탁하여 등기를 개시하는 경우:

| 촉탁 유형 | 사유 |
|-----------|------|
| **지적소관청의 촉탁** | 토지표시변경등기 (토지이동 후 등기부 일치를 위해) |
| **법원의 촉탁** | 경매개시결정, 가압류, 가처분, 파산선고 등기 |
| **수용·협의취득 촉탁** | 공익사업을 위한 수용등기 |
| **세무서장의 촉탁** | 체납처분에 의한 압류등기 |
| **지방자치단체의 촉탁** | 체납처분에 의한 압류등기 |
| **기타 관공서** | 법률에 근거한 처분제한 등기 |

> 촉탁에 따른 등기절차는 법률에 다른 규정이 없으면 **신청에 따른 등기에 관한 규정을 준용**

## 등기신청서 첨부정보

### 필수 첨부정보

| 첨부정보 | 내용 |
|----------|------|
| **등기원인증명정보** | 매매계약서, 설정계약서, 상속재산분할협의서, 판결정본 등 |
| **등기의무자의 등기필정보** | 공동신청 또는 승소한 등기의무자 단독신청 시 제공 |
| **제3자의 허가·동의·승낙서** | 등기원인에 대하여 제3자의 허가·동의·승낙이 필요한 경우 |
| **인감증명** | 등기의무자의 신원 확인 (소유권 등기명의인) |

### 등기원인증명정보 유형

| 등기 유형 | 등기원인증명정보 |
|-----------|-----------------|
| 소유권이전 (매매) | 매매계약서 |
| 소유권이전 (상속) | 가족관계증명서, 상속재산분할협의서 |
| 소유권이전 (증여) | 증여계약서 |
| 가등기 | 매매예약서, 변경계약서 |
| 권리설정 | 설정계약서 |
| 말소등기 | 해지증서 |
| 수용 | 협의성립확인서, 재결서 |
| 판결 | 판결정본 |
| 경매 | 매각허가결정정본 |

### 등기필정보가 없는 경우

- **원칙**: 등기의무자가 등기소에 **출석**하여 등기관으로부터 본인확인을 받아야 함
- **예외1**: 자격자대리인(변호사·법무사)이 위임확인 → **확인정보** 제공
- **예외2**: 신청서(위임장)에 등기의무자의 작성부분에 관하여 **공증**을 받은 경우

### 검인 제도

| 검인을 받는 경우 | 검인을 받지 않는 경우 |
|-----------------|---------------------|
| 계약서·판결서·조서 | 상속·수용·공매·경매·시효취득·권리포기 |
| 미등기건물·분양계약서 | 가등기, 국가·지자체 |
| 무허가건물 | 허가증·거래신고필증을 받은 경우 |
| | 진정명의회복등기, 소유권이전 말소등기 |
