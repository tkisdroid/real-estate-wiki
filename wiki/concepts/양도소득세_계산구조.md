# 양도소득세 계산구조

[[양도소득세]] 계산문제는 매년 출제되며, 전체 계산흐름을 정확히 이해해야 한다.

## 1. 전체 계산흐름

```
① 양도가액 (실지거래가액)
② - 취득가액 (실지거래가액)
③ - 기타 필요경비 (자본적지출 + 양도비)
───────────────────────────────
④ = 양도차익
⑤ - 장기보유특별공제
───────────────────────────────
⑥ = 양도소득금액
⑦ - 양도소득기본공제 (250만원)
───────────────────────────────
⑧ = 양도소득과세표준
⑨ × 세율 (6% ~ 45%)
───────────────────────────────
⑩ = 산출세액
⑪ - 감면세액
⑫ + 가산세
───────────────────────────────
⑬ = 결정세액 (납부세액)
```

## 2. 양도가액

**실지거래가액** 원칙: 양도자와 양수자 간에 실제로 거래한 가액

## 3. 취득가액 및 필요경비

### 실지거래가액을 아는 경우

| 항목 | 내용 |
|------|------|
| **취득가액** | 실제 매입가 + 취득세 + 중개수수료 등 부대비용 |
| **자본적지출** | 증축, 방수공사, 용도변경 비용 등 (수익적지출 제외) |
| **양도비** | 중개수수료, 공증비, 인지대, 소송비용 등 |

### 환산취득가액 (실거래가 모를 경우)

```
환산취득가액 = 양도 당시 실거래가액 × (취득 당시 기준시가 / 양도 당시 기준시가)
```

**개산공제율**: 취득당시 기준시가 × **3%** (토지·건물)

## 4. 장기보유특별공제

### 일반 토지·건물 (3년 이상 보유)

| 보유기간 | 공제율 (매년 2%씩 증가) |
|----------|-------------------------|
| 3년 이상 ~ 4년 미만 | **6%** |
| 4년 ~ 5년 | **8%** |
| 5년 ~ 6년 | **10%** |
| 6년 ~ 7년 | **12%** |
| 7년 ~ 8년 | **14%** |
| 8년 ~ 9년 | **16%** |
| 9년 ~ 10년 | **18%** |

| 13년 ~ 14년 | **26%** |
| 14년 ~ 15년 | **28%** |
| **15년 이상** | **30%** (한도) |

### 1세대1주택 고가주택 (보유 + 거주)

| 기간 | 보유기간 공제율 | 거주기간 공제율 |
|------|-----------------|-----------------|
| 2년 이상 ~ 3년 미만 | - | 8% |
| 3년 이상 ~ 4년 미만 | 12% | 12% |
| 4년 ~ 5년 | 16% | 16% |
| 5년 ~ 6년 | 20% | 20% |
| 6년 ~ 7년 | 24% | 24% |
| 7년 ~ 8년 | 28% | 28% |
| 8년 ~ 9년 | 32% | 32% |
| 9년 ~ 10년 | 36% | 36% |
| **10년 이상** | **40%** (한도) | **40%** (한도) |

> 보유 + 거주 합계 **최대 80%**

### 적용 제외

- **미등기양도자산**
- **조정대상지역 다주택 중과대상** (한시배제 제외)
- 보유기간 1년 미만
- 분양권

## 5. 양도소득기본공제

연간 **250만원** (거주자 1인당)

- 토지·건물 등과 주식 등 **각각** 250만원 공제 (소득별 구분)
- **미등기양도자산**에는 적용 불가
- 해당 과세기간에 양도소득 **2건 이상**: 양도소득기본공제는 연 1회만 적용

## 6. 세율

### 기본세율 (8단계 초과누진)

| 과세표준 | 세율 | 누진공제 |
|----------|------|----------|
| 1,400만원 이하 | **6%** | - |
| 5,000만원 이하 | **15%** | 126만원 |
| 8,800만원 이하 | **24%** | 576만원 |
| 1.5억원 이하 | **35%** | 1,544만원 |
| 3억원 이하 | **38%** | 1,994만원 |
| 5억원 이하 | **40%** | 2,594만원 |
| 10억원 이하 | **42%** | 3,594만원 |
| 10억원 초과 | **45%** | 6,594만원 |

### 특수 세율 정리

| 구분 | 세율 |
|------|------|
| 미등기양도 | **70%** |
| 주택 1년 미만 보유 | **70%** |
| 주택 1~2년 미만 | **60%** |
| 분양권 1년 미만 | **70%** |
| 분양권 1년 이상 | **60%** |
| 비사업용토지 | 기본세율 **+ 10%** |
| 조정대상지역 2주택 | 기본세율 **+ 20%** |
| 조정대상지역 3주택 이상 | 기본세율 **+ 30%** |

## 7. 계산 예시

**사례**: 일반 토지(비사업용 아님)를 8년 보유 후 양도

| 항목 | 금액 |
|------|------|
| 양도가액 | 5억원 |
| 취득가액 | 2억원 |
| 필요경비 | 1,000만원 |
| **양도차익** | **2억9,000만원** |
| 장기보유특별공제 (8년: 16%) | △4,640만원 |
| **양도소득금액** | **2억4,360만원** |
| 양도소득기본공제 | △250만원 |
| **과세표준** | **2억4,110만원** |
| 산출세액 (38% - 1,994만원) | **7,167.8만원** |

## 8. 양도차손의 처리

- 양도차손은 **같은 소득** 내에서만 통산 (소득별 구분)
- 같은 세율 적용 자산의 양도소득금액에서 먼저 공제
- 다른 세율 적용 자산의 양도소득금액에서 공제
- **다른 종합소득과 통산 불가**

## 핵심 암기 포인트

- **계산 흐름**: 양도가액 - 취득가액 - 필요경비 = 양도차익 - 장특공제 = 양도소득금액 - 기본공제(250만원) = 과세표준 × 세율 = 산출세액
- **기본세율 8단계**: 1,400만(6%), 5,000만(15%), 8,800만(24%), 1.5억(35%), 3억(38%), 5억(40%), 10억(42%), 10억초과(45%)
- **누진공제**: 0, 126만, 576만, 1,544만, 1,994만, 2,594만, 3,594만, 6,594만
- **장기보유특별공제(일반)**: 3년부터 매년 2%, 15년 이상 30% 한도
- **장기보유특별공제(1세대1주택 고가)**: 보유 매년 4%(10년 40%) + 거주 매년 4%(10년 40%) = 최대 80%
- **공제 적용 제외**: 미등기양도, 조정지역 다주택 중과, 1년 미만 보유, 분양권
- **양도소득기본공제**: 연 250만원, 미등기 불가, 소득별 각각
- **환산취득가액**: 양도실거래가 × (취득기준시가/양도기준시가)
- **개산공제율**: 토지·건물 3%, 지상권·전세권 7%, 미등기 0.3%/1%
- **특수세율**: 미등기 70%, 주택1년미만 70%, 주택1~2년 60%, 분양권1년미만 70%, 분양권1년이상 60%, 비사업용토지 +10%, 조정2주택 +20%, 조정3주택+ +30%

## 함정 노트

- ⚠️ **기본세율 8단계 구간**: 1,400만/5,000만/8,800만/1.5억/3억/5억/10억 — 구간 금액 정확히 암기
- ⚠️ **장기보유특별공제 한도**: 일반 30% vs 1세대1주택 80% — 혼동하면 계산 전체가 틀림
- ⚠️ **개산공제율**: 일반 3% vs 미등기 0.3% — 10배 차이. 지상권·전세권은 7%/1%
- ⚠️ **비사업용토지 +10% vs 조정2주택 +20% vs 조정3주택 +30%**: 가산세율 구분 필수
- ⚠️ **양도소득기본공제 250만원**: 미등기에는 적용 불가. 토지·건물과 주식 각각 250만원 별도 공제
