# 도시개발법 총론

도시개발법은 도시개발에 필요한 사항을 규정하여 계획적이고 체계적인 도시개발을 도모하고 쾌적한 도시환경의 조성과 공공복리의 증진에 이바지하기 위한 법률이다.

## 도시개발사업의 절차 (개요)

**수용방식**: 도시개발구역 지정·고시 → 개발계획 수립 → 시행자 지정 → 실시계획 작성·인가·고시

**환지방식**: 도시개발구역 지정·고시 → 개발계획 수립 → 시행자 지정 → 환지계획 작성·인가 → 환지예정지 지정 → 공사 → 환지처분 → 환지등기

## 도시개발구역의 지정

### 지정권자

| 지정권자 | 대상 |
|---------|------|
| **시·도지사** (대도시 시장 포함) | 원칙적 지정권자 |
| **국토교통부장관** | ①국가 실시 필요 ②관계 중앙행정기관장 요청 ③공공기관·정부출연기관 30만㎡↑ 국가계획 관련 제안 ④시·도지사 등 협의 불성립 ⑤긴급한 도시개발 필요 |

- **대도시 시장**: 서울·광역시 제외 인구 **50만 이상** 대도시
- **협의 지정**: 둘 이상 시·도에 걸치는 경우 관계 시·도지사 등이 협의하여 지정권자 결정

### 지정제안
- 시행자 지정 가능 자(국가·지자체·도시개발조합 **제외**)가 특별자치도지사·시장·군수·구청장에게 제안
- **민간 제안 시 동의요건**: 사업대상 토지면적의 **2/3 이상** 토지소유자 동의
- **수용 여부 통보**: **1개월 이내** (불가피 시 1개월 연장)

### 지정규모

| 지역 | 규모 |
|------|------|
| 주거·상업지역 | **1만㎡** 이상 |
| 공업지역 | **3만㎡** 이상 |
| 자연녹지 | **1만㎡** 이상 |
| 생산녹지 (지정면적의 30% 이하) | **1만㎡** 이상 |
| 도시지역 외 (원칙) | **30만㎡** 이상 |
| 도시지역 외 (예외: 아파트·연립 포함 등) | **10만㎡** 이상 |

### 지정요건 배제
- 취락지구·개발진흥지구·지구단위계획구역으로 지정된 지역
- 국가균형발전 목적 지역 (자연환경보전지역 제외)

### 지정절차
1. 기초조사 (임의)
2. **주민의견청취** (14일 이상 공람, 100만㎡ 이상 시 공청회)
3. 관계행정기관 협의 + **도시계획위원회 심의**
4. 지정·고시 (관보·공보)

## 도시개발구역의 분할·결합 지정

- **분할**: 분할 후 각 사업시행지구 면적이 각각 **1만㎡ 이상**
- **결합**: **1만㎡ 이상** 도시경관·문화재·군사시설 등 토지이용 제한 지역 포함 시
- 결합개발 제안 시: 지역별 대상 토지면적의 **2/3 이상** 토지소유자(지상권자 포함) 동의

## 도시개발구역의 지정·고시 효과

- **도시지역** 및 **지구단위계획구역** 결정·고시 의제
- **행위제한**: 건축물 건축·대수선·용도변경, 공작물 설치, 토지형질변경·토석채취·토지분할, 물건적치(1개월↑), 죽목 벌채·식재 → 허가 필요
- **기득권 보호**: 지정·고시 당시 이미 착수한 자는 **30일 이내** 신고 후 계속 시행 가능

## 도시개발구역의 지정해제

### 지정 전 개발계획 수립의 경우
- 지정·고시일부터 **3년** 이내 실시계획인가 미신청 → 3년 되는 날의 다음 날 해제

### 지정 후 개발계획 수립의 경우
- 지정·고시일부터 **2년** 이내 개발계획 미수립 → 해제 (330만㎡ 이상: **5년**)
- 개발계획 수립·고시일부터 **3년** 이내 실시계획인가 미신청 → 해제 (330만㎡ 이상: **5년**)

## 개발계획

### 수립시기
- **원칙**: 구역 지정 전에 수립
- **예외** (지정 후 수립 가능): 자연녹지, 생산녹지(30%↓), 도시지역 외, 국가균형발전 지역, 주거·상업·공업지역 합계 30%↓ 지역

### 환지방식 개발계획 수립 시 동의
- 토지면적의 **2/3 이상** 토지소유자 + 토지소유자 총수의 **1/2 이상** 동의
- **동의 생략**: 시행자가 국가·지방자치단체인 경우

## 도시개발사업의 시행방식

### 시행방식 선정

| 방식 | 요건 |
|------|------|
| **환지방식** | 토지의 교환·분합, 구획변경, 공공시설 정비 필요 또는 지가가 인근보다 현저히 높아 수용 곤란 |
| **수용방식** | 계획적·체계적 도시개발, 집단적 조성·공급 필요 |
| **혼용방식** | 분할혼용(지구별 분할) 또는 미분할혼용(분할 없이 혼용) |

### 시행방식의 변경
- 수용 → **전부 환지** (가능)
- 혼용 → **전부 환지** (가능)
- 수용 → **혼용** (가능, 조합 제외)
- 환지 → 수용·혼용 (불가) / 혼용 → 수용 (불가)
- 원칙: **원주민·투자자에게 유리한 방향**으로만 변경 허용

## 도시개발사업 시행자

### 시행자 지정
- 국가·지방자치단체, 한국토지주택공사, 지방공사, 토지소유자, 도시개발조합 등에서 **지정권자가 지정**

### 전부 환지방식의 경우
- **원칙**: 토지소유자 또는 조합을 시행자로 지정
- **예외**: ①1년 이내 미신청 ②공공시설 사업 병행 ③토지면적 1/2↑ + 소유자 총수 1/2↑ 동의 시 지자체 등 지정 가능

### 도시개발조합
- **설립요건**: 토지소유자 **7명 이상** + 정관 작성 → 지정권자 인가
- **조합원 자격**: 도시개발구역의 토지소유자
- **의결권**: 보유토지 면적과 **관계없는 평등한** 의결권
- **대의원회**: 의결권 조합원 **50명 이상** 시 설치 가능, 대의원 수 조합원 총수의 **10/100 이상**
- **법인격**: 법인, 민법상 사단법인 규정 준용

### 시행자 변경
- 실시계획 인가 후 **2년 이내** 사업 미착수
- 구역 지정 고시일부터 **1년 이내** (6개월 연장 가능) 실시계획 인가 미신청 등

## 수용 또는 사용방식

### 토지 수용
- 시행자가 토지 등을 수용·사용 가능
- 공익사업법 준용, 사업인정 및 고시 의제
- 재결신청: 개발계획 사업시행기간 **종료일**까지

### 토지상환채권
- 토지소유자가 원하면 매수대금 일부를 조성토지·건축물로 상환하는 채권 발행 가능
- 민간시행자: **금융기관 지급보증** 필수
- 발행규모: 분양토지·건축물 면적의 **1/2** 이내
- 지정권자 **사전 승인** 필요

### 조성토지 공급

| 방법 | 대상 |
|------|------|
| **경쟁입찰** (원칙) | 일반 |
| **추첨** | 국민주택규모 이하 주택건설용지, 공공택지, 330㎡ 이하 단독주택·공장용지 |
| **수의계약** | 공공용지, 2회↑ 유찰, 임대주택 등 |

- 감정가격 이하 공급 가능: 학교, 공공청사, 사회복지시설(유료 제외), 공장(이전 공장), 임대주택, 국민주택규모 이하, 200실↑ 호텔

## 환지방식

### 환지계획
- **포함사항**: 환지설계, 환지명세, 청산대상토지명세, **체비지**·보류지 명세
- **인가권자**: 행정청 아닌 시행자 → 특별자치도지사·시장·군수·구청장

### 환지계획 작성기준
- **동의에 따른 환지 제외**: 토지소유자 신청·동의 시 환지 미지정 가능 (임차권자 동의 필요)
- **체비지**: 사업경비 충당용 토지를 환지 미지정 → 보류지 중 일부를 체비지로 지정
- **입체환지**: 토지·건축물 소유자 신청 시 건축물 일부 + 토지 공유지분 부여

### 평균토지부담률
- 원칙 **50%** 이내
- 지정권자 인정 시 **60%**까지
- 토지소유자 총수 **2/3 이상** 동의 시 60% 초과 가능

### 환지예정지 지정
- 종전 토지소유자·임차권자 → 환지예정지에서 **종전과 같은 권리** 행사
- 종전 토지: **사용·수익 불가**
- 체비지 환지예정지: 시행자가 사용·수익·처분 가능

### 환지처분
- 공사완료 후 환지계획 확정을 위한 행정처분
- **효력발생**: 환지처분 **공고된 날의 다음 날**부터 종전 토지로 봄
- **권리소멸**: 환지 미지정 종전 토지의 권리 → 공고된 날이 **끝나는 때** 소멸
- **체비지 소유권**: 시행자가 공고 다음 날 취득 (이미 처분된 체비지: 매수자가 이전등기 시 취득)
- **청산금 확정**: 공고된 날의 **다음 날**
- **환지등기**: 공고 후 **14일 이내** 촉탁·신청

## 핵심 암기 포인트

- **지정권자**: 원칙 시·도지사(대도시 시장), 예외 국토교통부장관
- **대도시**: 서울·광역시 제외 인구 **50만↑**
- **민간 제안 동의**: 토지면적 **2/3↑** 소유자 동의
- **수용 여부 통보**: **1개월** 이내 (1개월 연장)
- **지정규모**: 주거·상업·자연녹지 **1만㎡**, 공업 **3만㎡**, 도시 외 **30만㎡** (예외 10만㎡)
- **환지 동의**: 토지면적 **2/3↑** + 소유자 총수 **1/2↑** (국가·지자체 시행 시 동의 불요)
- **지정해제**: 2년 미수립(330만㎡: 5년), 3년 미신청(330만㎡: 5년)
- **조합 설립**: 토지소유자 **7명↑**, 대의원회 **50명↑** 시 설치
- **시행자 변경**: 실시계획 인가 후 **2년** 미착수, 지정고시 후 **1년** 미신청
- **토지상환채권**: 분양면적 **1/2** 이내, 민간 **지급보증** 필수
- **평균토지부담률**: 원칙 **50%**, 지정권자 **60%**, 소유자 2/3 동의 시 60%↑
- **환지처분 효력**: 공고 **다음 날**, 권리소멸 공고 **끝나는 때**
- **행위제한 기득권**: **30일** 이내 신고

## OX 퀴즈

**Q1.** 도시개발구역의 지정권자는 원칙적으로 시장·군수·구청장이다. → **X** (시·도지사, 대도시 시장)

**Q2.** 민간이 도시개발구역 지정을 제안하려면 사업대상 토지면적의 3분의 2 이상의 토지소유자 동의를 받아야 한다. → **O**

**Q3.** 환지방식 개발계획 수립 시 시행자가 지방자치단체인 경우에도 토지소유자의 동의를 받아야 한다. → **X** (국가·지자체는 동의 불요)

**Q4.** 도시개발조합을 설립하려면 토지소유자 10명 이상이 정관을 작성하여야 한다. → **X** (7명 이상)

**Q5.** 도시개발사업의 시행방식을 환지방식에서 수용방식으로 변경할 수 있다. → **X** (원주민에게 유리한 방향만 가능, 환지→수용 불가)

**Q6.** 환지처분이 공고된 날의 다음 날에 체비지의 소유권은 시행자가 취득한다. → **O**

**Q7.** 토지상환채권의 발행규모는 분양토지·건축물 면적의 3분의 2를 초과할 수 없다. → **X** (2분의 1)

**Q8.** 도시개발구역 지정 후 개발계획을 수립하는 경우, 지정·고시일부터 3년 이내 개발계획을 수립하지 않으면 해제된다. → **X** (2년, 330만㎡↑은 5년)

## 함정 노트

- ⚠️ **지정권자 vs 허가권자**: 도시개발구역 지정은 **시·도지사**, 개발행위허가는 **시장·군수**
- ⚠️ **2/3 vs 1/2**: 민간 제안·환지 동의 모두 토지면적 **2/3**, 환지 소유자 총수 **1/2**
- ⚠️ **조합 7명 vs 정비조합**: 도시개발조합 **7명↑**, 정비조합은 추진위원회 **5명↑**
- ⚠️ **330만㎡ 기준**: 지정해제 기간이 2→5년, 3→5년으로 늘어나는 규모
- ⚠️ **시행방식 변경 방향**: 수용→환지(O), 혼용→환지(O), 수용→혼용(O, 조합제외), 환지→수용(X)
