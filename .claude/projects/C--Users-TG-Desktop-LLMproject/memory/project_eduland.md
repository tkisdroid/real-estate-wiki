이 LLM Wiki(공인중개사 시험)는 에듀랜드 홈페이지에 연결시킬 예정.

**Why:** 에듀랜드는 공인중개사 시험 교재 출판사이며, 이 위키의 교재 원본이 에듀랜드 기본서(2026)임. 웹 서비스로 연결하여 수험생에게 제공할 목적.

**How to apply:** 향후 웹 프론트엔드 구축, 콘텐츠 저작권/출처 표기, 에듀랜드 브랜딩 등을 고려해야 함. 위키 페이지의 sources 필드에 에듀랜드 교재 출처가 이미 기록되어 있으므로 이를 활용.
